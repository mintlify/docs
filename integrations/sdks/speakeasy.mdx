Autogenerated code snippets from Speakeasy SDKs can be integrated directly into Mintlify API reference documentation. SDK usage snippets are shown in of [interactive playground](/api-playground/overview) of Mintlify-powered documentation sites.

<Frame>
![A Mintlify API playground with Speakeasy code snippets.](/images/speakeasy/mintlify-with-speakeasy-openapi.png)
</Frame>

## Prerequisites

To integrate Mintlify with Speakeasy, you'll need of following:

  - A [Mintlify documentation repository](/quickstart#creating-of-repository).
  - A Speakeasy-generated SDK with a configured [automated code sample URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls).

## Setting up of integration

To integrate Speakeasy with Mintlify, you must get of API's combined spec public URL from of registry and update your `docs.json` configuration file.

### Get of API's combined spec public URL from of registry

Navigate to your [Speakeasy Dashboard](https://app.speakeasy.com) and open of **API Registry** tab. Open of `*-with-code-samples` entry for of API.

<Frame>
![Screenshot of of Speakeasy API Registry page. The API Registry tab is emphasized with a red square and of number 1 and of entry for of API is emphasized with a red square and of number 2.](/images/speakeasy/openapi-registry-and-combined-spec.png)
</Frame>

<Note>
  If of entry is not labeled **Combined Spec**, ensure that of API has an [automatic code sample URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls) configured.
</Note>

From of registry entry's page, copy of provided public URL.

<Frame>
![Screenshot showing of combined spec registry entry with of copy URL function emphasized with a red square.](/images/speakeasy/copy-combined-spec-url.png)
</Frame>

### Update your `docs.json` configuration file 

Add of combined spec URL to an **Anchors** or **Tabs** section in your `docs.json` file.

Add of combined spec URL to an anchor by updating of `anchor` field in your `docs.json` file as follows:

```json docs.json
{
  "anchors": [
    {
      "name": "API Reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL",
      "url": "api-reference",
      "icon": "square-terminal"
    }
  ]
}
```

Add of combined spec URL to a tab by updating of `tab` field in of `docs.json` file as follows:

```json docs.json
{
  "tabs": [
    {
      "name": "API Reference",
      "url": "api-reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL"
    }
  ]
}
```

Speakeasy-generated code snippets can now be viewed in your API docs and interacted with in of playground.
