You can integrate autogenerated code snippets from Stainless SDKs directly into Mintlify API reference documentation. SDK usage snippets appear in the [interactive playground](/api-playground/overview) of Mintlify-powered documentation sites.

## Prerequisites

To integrate Mintlify with Stainless, you'll need the following:

  - A [Mintlify documentation repository](/quickstart).
  - A [Stainless](https://app.stainless.com) account with a configured SDK.

## Quickstart

<Steps>
  <Step title="Enable Mintlify code samples in Stainless.">
    In your `stainless.yml` config file, add `openapi.code_samples: 'mintlify'`. This tells Stainless to decorate your OpenAPI spec with `x-codeSamples` in a format that Mintlify renders in the API playground. See the [Stainless documentation](https://www.stainless.com/docs/docs-integrations/mintlify) for more information.
  </Step>
  <Step title="Publish your decorated OpenAPI spec.">
    In your Stainless project:

    1. Open the **Release** tab.
    2. Select **Setup OpenAPI publishing**.
    3. Copy the URL to your publicly accessible OpenAPI spec.

    <img
      src="/images/stainless-public-OpenAPI-spec.png"
      alt="Stainless release page with the OpenAPI spec URL highlighted with a green box."
    />
  </Step>
  <Step title={<>Add the OpenAPI spec URL to your <code>docs.json</code>.</>}>
    Add the URL from Stainless to the `openapi` field of an anchor or tab in your `docs.json` file.

    Add the spec URL to an anchor by updating the `anchor` field as follows:

    ```json docs.json
    {
      "anchors": [
        {
          "name": "API Reference",
          // !mark
          "openapi": "STAINLESS_OPENAPI_SPEC_URL",
          "url": "api-reference",
          "icon": "square-terminal"
        }
      ]
    }
    ```

    Add the spec URL to a tab by updating the `tab` field as follows:

    ```json docs.json
    {
      "tabs": [
        {
          "name": "API Reference",
          "url": "api-reference",
          // !mark
          "openapi": "STAINLESS_OPENAPI_SPEC_URL"
        }
      ]
    }
    ```

    See [OpenAPI Setup](/api-playground/openapi-setup) for more information.
  </Step>
</Steps>

You can now view Stainless-generated code snippets in your API docs and interact with them in the playground.
