---
title: "Heap"
---

Add the following to your `docs.json` file to send insights to Heap.

<CodeGroup>

```json Insights options in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
