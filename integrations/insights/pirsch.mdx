---
title: "Pirsch"
---

Add the following to your `docs.json` file to send insights to Pirsch.

You can get your site ID from Settings \> Developer \> Identification Code.

<CodeGroup>

```json Insights options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
