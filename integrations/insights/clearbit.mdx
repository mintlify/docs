---
title: "Clearbit"
---

Add the following to your `docs.json` file to send insights to Clearbit.

<CodeGroup>

```json Insights options in docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
