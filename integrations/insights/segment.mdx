---
title: "Segment"
---

Add your Segment write key to your `docs.json` file to send insights to Segment.

<CodeGroup>

```json Insights options in docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Example
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
