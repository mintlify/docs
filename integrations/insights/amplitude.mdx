---
title: "Amplitude"
---

Add the following to your `docs.json` file to send insights to Amplitude.

<CodeGroup>

```json Insights options in docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Example
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
