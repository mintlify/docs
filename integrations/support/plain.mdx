</Info>

The [Plain](https://plain.com) integration enables an AI-powered support agent that can automatically respond to customer inquiries using your documentation as context.

## Prerequisites

- A Plain workspace with API access
- Support Agent entitlement enabled for your Mintlify organization

## Setup

<Steps>

### Get your Plain API key

1. Log into your [Plain workspace](https://app.plain.com)
2. Go to **Settings** → **API keys**
3. Create a new API key with permissions to send messages and manage threads
4. Copy the API key for the next step

### Get your HMAC secret

1. In Plain, go to **Settings** → **Webhooks**
2. Create a new webhook or select an existing one
3. Copy the **Request signing secret** (HMAC secret)

### Configure the webhook URL

Add Mintlify's webhook URL as a destination in your Plain webhook settings:

```
https://api.mintlify.com/plain-webhook/{your-subdomain}
```

Replace `{your-subdomain}` with your Mintlify documentation subdomain.

<Note>
Use Plain API version `2024-09-18` when configuring the webhook.
</Note>

### Enter credentials in Mintlify

1. Go to your [Mintlify dashboard](https://dashboard.mintlify.com)
2. Navigate to **Products** → **Support Agent**
3. Enter your Plain API key and HMAC secret
4. Click **Save**

### Test the connection

Click **Test connection** to verify that Mintlify can communicate with your Plain workspace. A successful test displays your workspace name.

</Steps>

## How it works

When a customer sends a message through Plain, the support agent:

1. Receives the message via webhook
2. Searches your documentation for relevant content
3. Generates a response using AI based on your docs
4. Sends the response back through Plain

The agent uses your documentation as its knowledge base, ensuring responses are accurate and consistent with your published content.
