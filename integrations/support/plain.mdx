<Info>
  The Plain support agent is available on [Enterprise plans](https://mintlify.com/pricing?ref=plain-support).
</Info>

The Plain support agent uses AI to automatically respond to customer support threads in Plain. When a customer creates a thread, the agent analyzes your documentation to provide relevant answers. If the agent cannot resolve the issue, it escalates the thread to your support team.

## Prerequisites

- A [Plain](https://plain.com) workspace
- Enterprise plan on Mintlify

## Configure the support agent

<Steps>
  <Step title="Navigate to the support agent settings">
    In your Mintlify dashboard, go to **Products** > **Support Agent**.
  </Step>
  <Step title="Copy the webhook URL">
    Copy the webhook URL displayed on the configuration page. You'll add this to Plain in the next step.
  </Step>
  <Step title="Add the webhook in Plain">
    1. In your Plain workspace, go to **Settings** > **Webhooks**.
    2. Create a new webhook with the URL you copied from Mintlify.
    3. Set the API version to `2024-09-18`.
    4. Copy the HMAC secret that Plain generates.
  </Step>
  <Step title="Create a Plain API key">
    1. In Plain, go to **Settings** > **API keys**.
    2. Create a new API key with permissions to read threads and send messages.
    3. Copy the API key.
  </Step>
  <Step title="Save your credentials in Mintlify">
    1. Back in the Mintlify dashboard, enter your Plain API key and HMAC secret.
    2. Click **Save**.
  </Step>
  <Step title="Test the connection">
    Click **Test connection** to verify that the integration is configured correctly.
  </Step>
</Steps>

## How it works

When a customer creates a support thread in Plain:

1. Plain sends a webhook event to Mintlify.
2. The support agent analyzes the customer's question against your documentation.
3. If the agent finds a relevant answer, it sends a response in the thread.
4. If the agent cannot answer the question, it escalates the thread to your team.

The agent uses the same documentation that powers your [assistant](/ai/assistant), ensuring consistent answers across all customer touchpoints.
