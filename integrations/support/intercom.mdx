---
title: "Intercom"
---

Add the following to your `docs.json` file to add an [Intercom](https://www.intercom.com) widget.

<CodeGroup>

```json Integration options in docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json Example
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

</CodeGroup>
