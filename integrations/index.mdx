Mintlify connects with the tools your team already uses. Add analytics providers to track how users engage with your docs, embed support widgets so visitors can get help without leaving the page, pull autogenerated SDK code samples into your API playground, and manage cookie consent and privacy compliance — all through your `docs.json` configuration.
