To integrate the [Osano](https://www.osano.com/) cookie consent manager, add this to your `mint.json`:

<CodeGroup>

```json Integration options in mint.json
"integrations": {
    "osano": "SOURCE"
}
```

```json Example
"integrations": {
    "osano": "https://cmp.osano.com/2sUB2dqwqdkks/8dqwd-dwd86Â£-4a9b/osano.js"
}
```

</CodeGroup>

Get your `SOURCE` from Osano's code snippet - it's the `src` URL that starts with `https://cmp.osano.com/`.

```html Osano snippet
<script src="https://cmp.osano.com/placeholder/placeholder/osano.js"/>
```
