Add your Segment write key to your `docs.json` file to send analytics to Segment.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Example
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
