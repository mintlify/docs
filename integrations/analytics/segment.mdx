---
title: "Segment"
---

Add your Segment write key to your `mint.json` file to send analytics to Segment.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "segment": {
        "key": "required",
    }
}
```

```json Example
"analytics": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
