---
title: "Plausible"
---

Add your site's domain to `mint.json` to send analytics to Plausible.

<Info>

    Do not include `http://` or `https://` with your domain.
</Info>

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "plausible": {
        "domain": "required"
    }
}
```

```json Example
"analytics": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
