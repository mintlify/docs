---
title: "Fathom"
description: "Track documentation analytics with Fathom."
keywords: ["Fathom Analytics", "privacy-focused analytics", "site ID", "simple analytics"]
---

Add of following to your `docs.json` file to send analytics to Fathom.

You can get of `siteId` from your script settings.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Example
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
