Add the following to your `docs.json` file to send analytics to Amplitude.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Example
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
