Add the following to your `docs.json` file to send analytics to Heap.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>

Replace `1234567890` with your Heap app ID. You can find this in your [Heap account settings](https://heapanalytics.com/app/settings).

## Tracked events

Mintlify automatically tracks the following user interactions and sends them to Heap:

- Page views
- Search queries
- Feedback submissions
- Context menu interactions
- Navigation clicks

See the [full list of tracked events](/integrations/analytics/overview#tracked-events) for details on all events sent to your analytics provider.
