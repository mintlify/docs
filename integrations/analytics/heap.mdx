---
title: "Heap"
---

Add the following to your `docs.json` file to send analytics to Heap.

<CodeGroup>

```json Analytics options in docs.json
integrations: {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
integrations: {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
