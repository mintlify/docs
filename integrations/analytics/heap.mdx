Add the following to your `docs.json` file to send analytics to Heap.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>

### Configuration options

<ParamField path="appId" type="string" required>
  Your Heap app ID. You can find this in your [Heap account settings](https://heapanalytics.com/app/settings).
</ParamField>

## Tracked events

Mintlify automatically sends the following events to Heap:

- **Page views**: A `pageview` event fires on every page navigation with the page path.
- **Custom events**: Any custom events configured through the Mintlify analytics system are forwarded to Heap using `heap.track()`.
