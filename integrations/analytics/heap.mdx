---
title: "Heap"
description: "Track user interactions with Heap analytics."
keywords: ["Heap Analytics", "automatic event tracking", "Heap app ID", "user interaction tracking"]
---

Add the following to your `docs.json` file to send analytics to Heap.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
