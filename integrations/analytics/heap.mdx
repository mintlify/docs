Add the following to your `docs.json` file to send analytics to Heap.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>

Replace `1234567890` with your Heap app ID. You can find this in your [Heap project settings](https://heapanalytics.com/app/settings).

### Configuration options

<ParamField path="appId" type="string" required>
  Your Heap application ID.
</ParamField>

## Tracked events

Mintlify automatically sends the following events to Heap:

- **Page views** — tracked on every page navigation
- **Custom events** — user interactions like search queries, feedback submissions, and navigation clicks
