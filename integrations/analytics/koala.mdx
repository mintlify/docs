Add the following to your `docs.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123def456"
    }
}
```

</CodeGroup>

Replace `pk_abc123def456` with your Koala public API key. You can find this in your [Koala account settings](https://app.getkoala.com/goto/settings/install).
