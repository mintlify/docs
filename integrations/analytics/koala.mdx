Add the following to your `docs.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123xyz"
    }
}
```

</CodeGroup>

## Get your public API key

1. Sign in to your [Koala dashboard](https://app.getkoala.com/).
2. Go to **Settings** and find your **Public API Key**.
3. Copy the key and add it to your `docs.json` file.
