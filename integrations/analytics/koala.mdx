Add the following to your `docs.json` file to send analytics to Koala.

You can find your public API key in your [Koala settings](https://app.getkoala.com/goto/settings/install).

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123def456"
    }
}
```

</CodeGroup>

### Configuration options

<ParamField path="publicApiKey" type="string" required>
  Your Koala public API key.
</ParamField>
