---
title: "Koala"
description: "Identify and track visitors on your documentation site with Koala for intent-based sales intelligence."
keywords: ["Koala analytics", "visitor identification", "intent signals", "public API key"]
---

Add the following to your `docs.json` file to enable Koala on your documentation site.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123def456"
    }
}
```

</CodeGroup>

Replace `pk_abc123def456` with your Koala public API key. You can find this in your [Koala workspace settings](https://app.getkoala.com/goto/settings/install).
