---
title: "Koala"
description: "Identify visitors and track engagement on your documentation with Koala."
keywords: ["Koala integration", "visitor identification", "public API key", "engagement tracking"]
---

Add the following to your `docs.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123def456"
    }
}
```

</CodeGroup>
