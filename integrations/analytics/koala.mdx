Add the following to your `docs.json` file to enable Koala on your documentation site.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_76a6caa274e800f3ceff0b2bc6b9b9d82ab8"
    }
}
```

</CodeGroup>

### Configuration options

<ParamField path="publicApiKey" type="string" required>
  Your Koala public API key. You can find this in your [Koala dashboard](https://app.getkoala.com) under **Settings**.
</ParamField>
