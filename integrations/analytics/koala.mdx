---
title: "Koala"
description: "Identify companies and users visiting your documentation with Koala."
keywords: ["Koala integration", "visitor identification", "public API key", "account-based analytics"]
---

Add the following to your `docs.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123xyz"
    }
}
```

</CodeGroup>
