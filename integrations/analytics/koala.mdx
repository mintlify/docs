---
title: "Koala"
description: "Identify companies and visitors on your documentation site with Koala."
keywords: ["Koala integration", "visitor identification", "account scoring", "public API key"]
---

Add the following to your `docs.json` file to enable Koala visitor identification.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123def456"
    }
}
```

</CodeGroup>
