---
title: "Koala"
description: "Integrate Koala with your Mintlify documentation to identify visiting companies, track intent signals, and prioritize high-value accounts."
keywords: ["Koala integration", "visitor identification", "intent signals", "public API key"]
---

Add the following to your `docs.json` file to send analytics to Koala.

You can find your public API key in your [Koala dashboard](https://app.getkoala.com/) under Settings.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_a1b2c3d4e5f6"
    }
}
```

</CodeGroup>
