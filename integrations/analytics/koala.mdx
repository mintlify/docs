Add the following to your `docs.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "your-koala-public-api-key"
    }
}
```

</CodeGroup>

## Get your public API key

1. Sign up or log in to [Koala](https://getkoala.com).
2. Navigate to your workspace settings.
3. Copy your public API key from the installation instructions.
