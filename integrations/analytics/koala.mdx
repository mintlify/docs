Add the following to your `docs.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_abc123xyz"
    }
}
```

</CodeGroup>

### Configuration options

<ParamField path="publicApiKey" type="string" required>
  Your Koala public API key. Find this in your Koala workspace settings under **Settings > Install**.
</ParamField>
