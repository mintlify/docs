title: "HotJar"
description: "Add Hotjar to your documentation for heatmaps, session recordings, and user feedback collection."
keywords: ["Hotjar integration", "user feedback", "session recordings", "behavior analytics"]
---

Add the following to your `docs.json` file to send analytics to HotJar.

```json Analytics options in docs.json
"integrations": {
    "hotjar": {
        "hjid": "required",
        "hjsv": "required"
    }
}
```
