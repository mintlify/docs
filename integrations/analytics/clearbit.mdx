---
title: "Clearbit"
description: "Enrich visitor data with Clearbit to identify companies and users visiting your documentation."
keywords: ["Clearbit integration", "user enrichment", "visitor intelligence", "public API key"]
---

Add the following to your `docs.json` file to send analytics to Clearbit.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
