---
title: "Adobe Analytics"
description: "Track documentation engagement with Adobe Analytics."
keywords: ["Adobe Analytics integration", "report suite ID", "documentation analytics", "engagement tracking"]
---

Add the following to your `docs.json` file to send analytics to Adobe Analytics.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "adobeAnalytics": {
        "reportSuiteId": "required"
    }
}
```

```json Example
"integrations": {
    "adobeAnalytics": {
        "reportSuiteId": "your-report-suite-id"
    }
}
```

</CodeGroup>

<ParamField path="reportSuiteId" type="string" required>
  Your Adobe Analytics report suite ID. Find this in Adobe Analytics under Admin > Report Suites.
</ParamField>
