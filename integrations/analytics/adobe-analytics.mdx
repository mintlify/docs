Add the following to your `docs.json` file to send analytics to Adobe Analytics.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "adobeAnalytics": {
        "reportSuiteId": "required",
        "trackingServer": "required"
    }
}
```

```json Example
"integrations": {
    "adobeAnalytics": {
        "reportSuiteId": "mycompany-docs-prod",
        "trackingServer": "mycompany.sc.omtrdc.net"
    }
}
```

</CodeGroup>

## Configuration options

| Property | Required | Description |
| :--- | :--- | :--- |
| `reportSuiteId` | Yes | Your Adobe Analytics Report Suite ID where data will be sent. |
| `trackingServer` | Yes | Your Adobe Analytics tracking server domain. |

## Finding your credentials

### Report Suite ID

1. Log in to Adobe Analytics
2. Go to **Admin** > **Report Suites**
3. Find your report suite and copy the ID

### Tracking server

Your tracking server is typically in the format `yourcompany.sc.omtrdc.net`. You can find it in:

1. Adobe Analytics **Admin** > **Data Collection** settings
2. Your Adobe Experience Platform Launch (Tags) configuration
3. Contact your Adobe administrator if you're unsure
