---
title: "Common Room"
description: "Identify and track companies visiting your documentation with Common Room's visitor intelligence signals."
keywords: ["Common Room integration", "visitor intelligence", "company identification", "site ID"]
---

Add the following to your `docs.json` file to send analytics to Common Room.

You can get the `siteId` from your Common Room Signals settings. It must be a valid UUID.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "commonRoom": {
        "siteId": "required"
    }
}
```

```json Example
"integrations": {
    "commonRoom": {
        "siteId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
    }
}
```

</CodeGroup>
