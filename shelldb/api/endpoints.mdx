# API endpoints

Complete reference for all ShellDB API endpoints.

## Database operations

### List databases
```http
GET /databases
```

Response:
```json
{
  "success": true,
  "data": [
    {
      "name": "myapp",
      "description": "My application database",
      "created_at": "2024-01-28T10:00:00Z",
      "size": "45.2MB",
      "tables": 8
    }
  ]
}
```

### Create database
```http
POST /databases
```

Request:
```json
{
  "name": "myapp",
  "description": "My application database"
}
```

### Delete database
```http
DELETE /databases/{name}
```

## Query execution

### Execute query
```http
POST /query
```

Request:
```json
{
  "database": "myapp",
  "query": "SELECT * FROM users WHERE active = $1",
  "parameters": [true]
}
```

### Execute transaction
```http
POST /transaction
```

Request:
```json
{
  "database": "myapp",
  "queries": [
    {
      "query": "INSERT INTO orders (user_id, total) VALUES ($1, $2)",
      "parameters": [123, 99.99]
    },
    {
      "query": "UPDATE users SET last_order = NOW() WHERE id = $1",
      "parameters": [123]
    }
  ]
}
```

## Schema operations

### List tables
```http
GET /databases/{name}/tables
```

### Get table schema
```http
GET /databases/{name}/tables/{table}
```

### Create table
```http
POST /databases/{name}/tables
```

## Backup and restore

### Create backup
```http
POST /databases/{name}/backup
```

### List backups
```http
GET /databases/{name}/backups
```

### Restore backup
```http
POST /databases/{name}/restore
```
