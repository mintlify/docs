# Installing ShellDB

ShellDB can be installed on macOS, Linux, and Windows. Choose your preferred installation method below.

## Prerequisites

- Operating system: macOS, Linux, or Windows
- Available disk space: 50MB minimum
- Network connection (for initial download)

## Installation methods

### Package managers

<Tabs>
  <Tab title="macOS (Homebrew)">
    ```bash
    brew install shelldb
    ```
  </Tab>
  <Tab title="Linux (apt)">
    ```bash
    sudo apt update
    sudo apt install shelldb
    ```
  </Tab>
  <Tab title="Linux (yum)">
    ```bash
    sudo yum install shelldb
    ```
  </Tab>
  <Tab title="Windows (Chocolatey)">
    ```powershell
    choco install shelldb
    ```
  </Tab>
</Tabs>

### Direct download

Download the latest release for your platform:

1. Visit the [ShellDB releases page](https://github.com/shelldb/releases)
2. Download the appropriate binary for your platform
3. Extract the archive to your preferred location
4. Add the binary to your system PATH

### Build from source

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/shelldb/shelldb.git
    cd shelldb
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    make install-deps
    ```
  </Step>
  <Step title="Build ShellDB">
    ```bash
    make build
    ```
  </Step>
  <Step title="Install binary">
    ```bash
    make install
    ```
  </Step>
</Steps>

## Verify installation

After installation, verify that ShellDB is working correctly:

```bash
shelldb --version
```

You should see output similar to:

```
ShellDB version 1.0.0
```

## Configuration

Create a configuration file at `~/.shelldb/config.yaml`:

```yaml
# Default database directory
data_dir: ~/.shelldb/data

# Default connection settings
connection:
  host: localhost
  port: 5432
  
# Logging configuration
logging:
  level: info
  file: ~/.shelldb/logs/shelldb.log
```
