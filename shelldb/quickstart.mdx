# ShellDB quickstart

Get up and running with ShellDB in just a few minutes. This guide will walk you through creating your first database and performing basic operations.

## Prerequisites

Before you begin, make sure you have:
- [ShellDB installed](/shelldb/installation) on your system
- Basic familiarity with command-line interfaces

## Step 1: Initialize a new database

Create a new database called `myapp`:

```bash
shelldb create myapp
```

This creates a new database directory and configuration files:

```
myapp/
├── config.yaml
├── data/
└── logs/
```

## Step 2: Connect to your database

Connect to the database you just created:

```bash
shelldb connect myapp
```

You'll see the ShellDB prompt:

```
ShellDB [myapp]>
```

## Step 3: Create your first table

Create a simple users table:

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    email TEXT UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Step 4: Insert data

Add some sample data to your table:

```sql
INSERT INTO users (name, email) VALUES 
    ('Alice Johnson', 'alice@example.com'),
    ('Bob Smith', 'bob@example.com'),
    ('Carol Davis', 'carol@example.com');
```

## Step 5: Query your data

Retrieve the data you just inserted:

```sql
SELECT * FROM users;
```

You should see output similar to:

```
+----+--------------+-------------------+---------------------+
| id | name         | email             | created_at          |
+----+--------------+-------------------+---------------------+
| 1  | Alice Johnson| alice@example.com | 2024-01-28 10:30:00 |
| 2  | Bob Smith    | bob@example.com   | 2024-01-28 10:30:01 |
| 3  | Carol Davis  | carol@example.com | 2024-01-28 10:30:02 |
+----+--------------+-------------------+---------------------+
```

## Step 6: Create an index

Improve query performance by adding an index on the email column:

```sql
CREATE INDEX idx_users_email ON users(email);
```

## Step 7: Try advanced queries

<Tabs>
  <Tab title="Filtering">
    ```sql
    SELECT name, email FROM users 
    WHERE name LIKE 'A%';
    ```
  </Tab>
  <Tab title="Ordering">
    ```sql
    SELECT * FROM users 
    ORDER BY created_at DESC;
    ```
  </Tab>
  <Tab title="Counting">
    ```sql
    SELECT COUNT(*) as total_users FROM users;
    ```
  </Tab>
</Tabs>

## Step 8: Exit and backup

Exit the database connection:

```sql
.exit
```

Create a backup of your database:

```bash
shelldb backup myapp myapp_backup.sql
```

## Common commands

Here are some frequently used ShellDB commands:

| Command | Description |
|---------|-------------|
| `shelldb create <name>` | Create a new database |
| `shelldb connect <name>` | Connect to a database |
| `shelldb list` | List all databases |
| `shelldb backup <db> <file>` | Create a backup |
| `shelldb restore <file> <db>` | Restore from backup |
| `.schema` | Show table schemas |
| `.tables` | List all tables |
| `.exit` | Exit database connection |
