## Connect your Linear workspace

1. Open the agent panel in your [dashboard](https://dashboard.mintlify.com/products/agent).
2. Click the **Settings** button.
3. In the Linear integration section, click **Connect**.
4. Follow the Linear prompts to authorize the Mintlify app in your workspace.
5. Test that the agent is working by mentioning it in a Linear issue.

## Use the agent in Linear

Once connected, you can mention the agent in Linear issues and comments to trigger documentation updates. The agent reads the issue context, including the description, comments, and linked content, and creates a pull request in your connected documentation repository.

You can:

- Mention the agent in an issue to generate or update documentation based on the issue content.
- Include links to relevant pull requests or resources in the issue for additional context.
- Continue the conversation in issue comments to iterate on changes.

## Update documentation

Use the agent to update your documentation from a Linear issue or comment.

- **New request**: Mention the agent in a Linear issue with instructions on what to document.
- **Existing thread**: Reply in an issue comment and mention the agent with instructions on what to update.

The agent reads the issue context, including the title, description, and comments, and creates a pull request with the proposed documentation changes.

## Best practices

- **Be specific**: Tell the agent exactly what you want documented and where it should go.
- **Add context**: If the issue doesn't contain all the necessary information, include additional details in your message to the agent.
- **Review carefully**: Always review pull requests that the agent creates before merging them.
- **Link related PRs**: Include links to relevant code changes so the agent can reference implementation details.
