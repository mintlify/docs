<Note>
  The agent is available on [Pro and Enterprise plans](https://mintlify.com/pricing?ref=agent).
</Note>

## Connect your Linear workspace

1. Open the agent panel in your dashboard.
2. Click the **Settings** button.
3. In the Linear integration section, click **Connect**.
4. Follow the Linear prompts to authorize the Mintlify app for your workspace.

Once connected, the Mintlify agent appears as an assignable and mentionable entity in your Linear workspace.

## Use the agent in Linear

Trigger the agent by assigning or delegating a Linear issue to it. When the agent receives an issue, it:

1. Reads the issue title, description, and any comments for context.
2. Analyzes your existing documentation to determine what changes are needed.
3. Creates a pull request with the proposed documentation updates.
4. Reports progress and results back to the Linear issue as activity updates.
5. Moves the issue to **In Review** after the pull request is created.

## Continue a conversation

Reply in the agent session thread on a Linear issue to provide additional instructions or ask for follow-up changes. The agent picks up where it left off with full context from the previous interaction.

## Stop the agent

You can stop a running agent session from Linear at any time. The agent halts processing and no further changes are made for that run.

## Disconnect Linear

1. Open the agent panel in your dashboard.
2. Click the **Settings** button.
3. In the Linear integration section, click **Disconnect**.
