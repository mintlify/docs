Agent integrations give the agent access to your team's tools. Once connected, the agent can search Slack threads, read Notion pages, look up Linear issues, and pull context from other apps to inform its responses and documentation updates.

## Connect an integration

Integrations are connected per organization. Any admin or editor can connect an integration from the dashboard.

1. Go to [Settings → Organization → Integrations](https://app.mintlify.com/settings/organization/integrations) in your dashboard.
2. Find the integration you want to connect and click **Connect**.
3. Follow the OAuth prompts to authorize Mintlify to access your account.

Some integrations — like Jira and Salesforce — open an additional configuration step to enter your workspace URL or subdomain before the OAuth flow begins.

Once connected, the integration is available to the agent for your entire organization.

## Personal connections

Some integrations support both a shared workspace connection and personal connections. A shared connection is authenticated once for the whole organization. A personal connection lets individual team members link their own account for access to content that requires their personal permissions.

To add a personal connection, connect the integration from your own account. Your personal connection takes priority over the shared connection when the agent is responding to you.

## How the agent uses integrations

The agent uses connected integrations as tools. When you ask the agent a question or give it a task, it can search and retrieve content from your connected apps to build context.

For example:
- "Summarize the Slack thread about the v2 API migration and write it up as a docs page."
- "Check the Linear ticket for this feature and document the behavior."
- "What did the team decide about rate limiting? Check Slack."

The agent only accesses integration data when it's relevant to the request. It does not proactively read from connected apps.

<Note>
  The Slack integration here gives the agent access to Slack channel content as a data source. This is separate from [adding the agent to Slack](/agent/slack), which lets your team interact with the agent directly inside Slack.
</Note>

## Disconnect an integration

1. Go to [Settings → Organization → Integrations](https://app.mintlify.com/settings/organization/integrations) in your dashboard.
2. Find the connected integration and click **Configure**.
3. Click **Disconnect**.

Disconnecting removes the integration for your organization. The agent loses access to that tool immediately.
