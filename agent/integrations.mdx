The **Integrations** settings page lets admins connect and disconnect the third-party services that the agent uses as context. Connections set up here apply to your entire organization and are available to every teammate by default.

The agent uses [Composio](https://docs.composio.dev/toolkits) to support a broad catalog of third-party services, including issue trackers, knowledge bases, CRMs, and developer tools.

## Prerequisites

- An admin role on your Mintlify organization.
- A Mintlify plan that includes the agent. See [Pricing](https://mintlify.com/pricing?ref=integrations).

<Note>
  You must configure GitHub and GitLab through Mintlify, not Composio. Use the [GitHub app](/deploy/github) or [GitLab integration](/deploy/gitlab) to control which repositories the agent can access.
</Note>

## Open the Integrations page

Navigate to [Settings → Workspace → Integrations](https://app.mintlify.com/settings/organization/integrations) in your dashboard.

The page lists every integration that is supported for the current organization, split into:

- **Enabled**: Integrations your organization has connected.
- **Available**: Integrations that you can connect but haven't set up yet.

Use the search field to filter the list. Integrations that require auth configuration your team can't supply through the dashboard are hidden automatically.

## Connect an integration

1. Find the integration on the **Integrations** page.
2. Select **Connect**.
3. In the new tab that opens, sign in to the third-party service and approve the requested permissions.
4. Return to the dashboard. The integration moves to the **Enabled** section and is immediately available to the agent.

If authorization fails or you close the tab before completing it, the integration stays in the **Available** section and you can retry.

## Disconnect an integration

1. Find the integration in the **Enabled** section.
2. Select **Disconnect**.
3. Confirm the action.

Disconnecting removes the connection for your entire organization. The agent stops using the integration as context immediately. You can reconnect at any time from the **Available** section.

## How integrations interact with Slack

Integrations connected from the dashboard apply to your entire organization. If a teammate also connects an integration from Slack, that personal connection takes precedence when the teammate uses the agent in Slack.

For example:

- A teammate without a personal Linear connection uses the organization's Linear connection from the dashboard.
- A teammate with a personal Linear connection in Slack uses their own credentials, not the organization's.

See [Connect additional integrations from Slack](/agent/slack#connect-additional-integrations-from-slack) for details on per-user connections.

## Expired or revoked connections

Mintlify keeps the status of your connected services in sync with Composio. A connection moves back to the **Available** section when:

- The third-party provider expires or revokes the credentials.
- The provider marks the connection as inactive.
- An admin disconnects it from the dashboard.

To restore an expired connection, reconnect the integration from the **Available** section.
