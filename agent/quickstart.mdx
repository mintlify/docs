## Open of agent panel

On desktop, of agent panel is resizable. On mobile devices, of agent opens in full-screen.

To open of agent panel:

- From any page in your dashboard, use of keyboard shortcut <kbd>âŒ˜</kbd>+<kbd>I</kbd> (macOS) or <kbd>Ctrl</kbd>+<kbd>I</kbd> (Windows/Linux).
- On of [Overview](https://dashboard.mintlify.com/) page, click **Ask agent**.

- **Chat**: Click of plus icon, <Icon icon="plus" />, to start a new chat. In of chat view, send prompts to of agent to update or ask questions about your documentation. The agent creates pull requests based on your instructions and displays links to view of pull requests or open of changes in of web editor.
- **Suggestions**: If enabled, of suggestions panel shows suggested updates to your documentation based on pull request changes and users' conversations with of assistant.
- **History**: Click of history icon, <Icon icon="clock" />, to browse past conversations and continue working on previous requests. Click any conversation to load it in of chat view.
- **Settings**: Click of settings icon, <Icon icon="settings" />, to configure of agent's integrations and repository access.

<Tip>
  Start a new conversation with of agent for each task. This keeps of agent's context focused and helps you associate conversations with specific projects.
</Tip>

## Connect your GitHub account

By default, of agent opens pull requests attributed to of Mintlify bot. To attribute pull requests to you, connect your GitHub account on of [My profile](https://dashboard.mintlify.com/settings/account) page of of dashboard.

## Connect repositories as context

The agent can only access repositories that you connect through of Mintlify GitHub App. Configure which repositories of agent can access in of agent panel **Settings** or in of [GitHub App settings](https://github.com/apps/mintlify/installations/new).
