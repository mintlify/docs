## Configuration essentials

<AccordionGroup>
  <Accordion title="Basic docs.json structure">
    ```json
    {
      "name": "Acme Docs",
      "navigation": [
        {
          "group": "Get started",
          "pages": ["index", "quickstart"]
        }
      ],
      "colors": {
        "primary": "#0D9373",
        "light": "#07C983",
        "dark": "#0D9373"
      }
    }
    ```
  </Accordion>
  
  <Accordion title="OpenAPI integration">
    ```json
    {
      "openapi": [
        "openapi.json",
        "v2-openapi.yaml"
      ],
      "navigation": [
        {
          "group": "API Reference",
          "openapi": "openapi.json",
          "pages": [
            "GET /users",
            "POST /users"
          ]
        }
      ]
    }
    ```
  </Accordion>
  
  <Accordion title="Authentication setup">
    ```json
    {
      "api": {
        "auth": {
          "method": "bearer"
        }
      }
    }
    ```
    
    Supported methods: `bearer`, `key`, `basic`
  </Accordion>
</AccordionGroup>

## Troubleshooting checklist

<Steps>
  <Step title="Local development not working">
    - Check Node.js version (16+ required)
    - Run `npm install` in project root
    - Verify `docs.json` syntax with JSON validator
  </Step>
  
  <Step title="Pages not displaying">
    - Confirm file paths match navigation entries
    - Check frontmatter YAML syntax
    - Ensure MDX files have `.mdx` extension
  </Step>
  
  <Step title="API playground issues">
    - Validate OpenAPI spec at [editor.swagger.io](https://editor.swagger.io)
    - Check CORS configuration on API server
    - Verify authentication method in `docs.json`
  </Step>
  
  <Step title="Deployment failures">
    - Check build logs in dashboard
    - Verify GitHub App permissions
    - Ensure no syntax errors in configuration
  </Step>
</Steps>

## Performance optimization

<CardGroup cols={3}>
  <Card title="Images" icon="image">
    - Use WebP format when possible
    - Optimize file sizes < 1MB
    - Add alt text for accessibility
  </Card>
  
  <Card title="Navigation" icon="list">
    - Limit top-level groups to 6-8
    - Use descriptive page titles
    - Group related content together
  </Card>
  
  <Card title="Content" icon="file-text">
    - Keep pages focused and scannable
    - Use headers to structure content
    - Link between related pages
  </Card>
</CardGroup>
