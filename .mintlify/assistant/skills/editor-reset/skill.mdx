# Resetting the Editor

If the Mintlify web editor is broken, showing stale content, or failing to load your file tree — especially after a Git sync — a force reset usually fixes it.

## How to reset the editor

1. Open the web editor at [app.mintlify.com](https://app.mintlify.com)
2. Click the **gear/settings icon** in the editor toolbar
3. Scroll down to the **Reset Editor** section
4. Click **Reset Editor**

The editor will reload and re-sync from your Git repository. This clears local editor state without affecting your published docs or Git history.

## When to use this

- The file tree won't load or appears empty
- Content in the editor doesn't match what's in your GitHub repo
- The editor appears frozen or unresponsive
- After merging changes in GitHub, the editor still shows old content
- After a failed sync between the editor and Git

If the issue persists after resetting, it may indicate a deeper sync conflict — escalate to the support team.
