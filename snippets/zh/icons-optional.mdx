<ResponseField name="icon" type="string">
  要显示的图标。

  可选值：

  - [Font Awesome](https://fontawesome.com/icons) 图标名称（例如 `github`）
  - [Lucide](https://lucide.dev/icons) 图标名称（例如 `rocket`）
  - 指向外部托管图标的 URL
  - 项目中图标文件的路径
  - 用花括号包裹的 SVG 代码

  对于自定义 SVG 图标：
  1. 使用 [SVGR 转换器](https://react-svgr.com/playground/) 转换你的 SVG。
  1. 将 SVG 代码粘贴到 SVG 输入框。
  2. 从 JSX 输出框中复制完整的 `<svg>...</svg>` 元素。
  3. 用花括号包裹可用于 JSX 的 SVG 代码：`icon={<svg ...> ... </svg>}`。
  4. 根据需要调整 `height` 和 `width`。
</ResponseField>

<ResponseField name="iconType" type="string">
  [Font Awesome](https://fontawesome.com/icons) 的图标样式。仅在使用 Font Awesome 图标时生效。 
  
  可选值：`regular`、`solid`、`light`、`thin`、`sharp-solid`、`duotone`、`brands`。
</ResponseField>
