## Configuración de GitLab

Para usar repositorios de GitLab en un workflow, conecta cada proyecto a través de la página de configuración de [OAuth de GitLab](https://app.mintlify.com/settings/organization/gitlab-oauth). Conecta cada repositorio que el workflow utilice: tu repositorio de documentación y cualquier repositorio de activación o contexto. Debes tener al menos el rol de Maintainer en cada proyecto.

<Note>
  Los workflows requieren un plan de pago de GitLab. El agente utiliza tokens de acceso de proyecto de corta duración para acceder a los repositorios, lo cual el plan gratuito de GitLab no admite.
</Note>
