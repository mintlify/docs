Use AI ads to display contextual, intelligent advertisements that adapt to your documentation content and user behavior. AI ads leverage machine learning to show relevant promotions, product recommendations, and sponsored content.

## Basic setup

Add AI ads to your documentation by configuring the `aiAds` property in your `docs.json`:

<CodeGroup>
  ```json Basic AI ads
  "aiAds": {
    "enabled": true,
    "provider": "mintlify-ai",
    "placement": ["sidebar", "content"]
  }
  ```

  ```json Advanced configuration
  "aiAds": {
    "enabled": true,
    "provider": "mintlify-ai",
    "placement": ["sidebar", "content", "footer"],
    "contextual": true,
    "frequency": "medium",
    "categories": ["developer-tools", "saas", "ai-services"]
  }
  ```

  ```json Custom provider
  "aiAds": {
    "enabled": true,
    "provider": "custom",
    "apiEndpoint": "https://your-ai-ads-api.com/v1/ads",
    "apiKey": "your-api-key",
    "placement": ["content"]
  }
  ```
</CodeGroup>

## Configuration options

<ResponseField name="enabled" type="boolean" required>
  Enable or disable AI ads across your documentation site.
</ResponseField>

<ResponseField name="provider" type="string" required>
  The AI ads provider. Options: `"mintlify-ai"`, `"custom"`.
</ResponseField>

<ResponseField name="placement" type="array" required>
  Where to display ads. Options: `"sidebar"`, `"content"`, `"footer"`, `"header"`.
</ResponseField>

<ResponseField name="contextual" type="boolean">
  Enable contextual targeting based on page content. Defaults to `true`.
</ResponseField>

<ResponseField name="frequency" type="string">
  Ad display frequency. Options: `"low"`, `"medium"`, `"high"`. Defaults to `"medium"`.
</ResponseField>

<ResponseField name="categories" type="array">
  Allowed ad categories to filter relevant content.
</ResponseField>

<ResponseField name="apiEndpoint" type="string">
  Custom API endpoint for external ad providers (required when provider is `"custom"`).
</ResponseField>

<ResponseField name="apiKey" type="string">
  API key for authentication with custom providers.
</ResponseField>

## Placement options

### Sidebar ads
Display ads in the documentation sidebar for persistent visibility:

```json
"aiAds": {
  "enabled": true,
  "provider": "mintlify-ai",
  "placement": ["sidebar"],
  "sidebarConfig": {
    "position": "top",
    "maxAds": 2
  }
}
```

### Content ads
Embed ads within documentation content:

```json
"aiAds": {
  "enabled": true,
  "provider": "mintlify-ai", 
  "placement": ["content"],
  "contentConfig": {
    "insertAfterParagraphs": 3,
    "maxAdsPerPage": 1
  }
}
```

### Footer ads
Show ads at the bottom of each page:

```json
"aiAds": {
  "enabled": true,
  "provider": "mintlify-ai",
  "placement": ["footer"],
  "footerConfig": {
    "style": "banner",
    "maxAds": 1
  }
}
```

## Analytics integration

AI ads automatically integrate with your existing analytics setup to track:

- Ad impressions
- Click-through rates
- User engagement
- Revenue attribution

Events are sent to your configured analytics providers with the `docs.ai_ads.` prefix:

| Event name | Description |
| :--------- | :---------- |
| `docs.ai_ads.impression` | When an ad is displayed to a user |
| `docs.ai_ads.click` | When a user clicks on an ad |
| `docs.ai_ads.dismiss` | When a user dismisses a dismissible ad |
| `docs.ai_ads.load_error` | When an ad fails to load |

## Best practices

<Tip>
  **Start with low frequency**: Begin with `"frequency": "low"` to test user response before increasing ad density.
</Tip>

<Warning>
  **Respect user experience**: Avoid placing too many ads or using intrusive placements that disrupt documentation flow.
</Warning>

- Use contextual targeting to show relevant ads
- Monitor analytics to optimize placement and frequency
- Test different ad categories for your audience
- Consider user feedback when adjusting settings

## Custom ad providers

For custom ad providers, implement the following API contract:

```json API Request
POST /v1/ads
{
  "pageUrl": "https://docs.example.com/api/users",
  "pageContent": "API documentation content...",
  "userContext": {
    "sessionId": "abc123",
    "previousPages": ["/api", "/quickstart"]
  },
  "placement": "sidebar",
  "maxAds": 2
}
```

```json API Response
{
  "ads": [
    {
      "id": "ad-123",
      "title": "Developer Tool",
      "description": "Boost your API development",
      "imageUrl": "https://example.com/ad-image.png",
      "clickUrl": "https://example.com/product",
      "category": "developer-tools"
    }
  ]
}
```
