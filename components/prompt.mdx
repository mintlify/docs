Use the Prompt component to display pre-built AI prompts that users can copy to their clipboard or open directly in [Cursor](https://www.cursor.com).

## Examples

### Copyable prompt

<Prompt description="Use this pre-built prompt to get started faster.">
# Tailwind CSS Expert

You are a **Tailwind CSS expert**. Always use utility classes over custom CSS.  
**Rule #1:** Prefer `@apply` only in component layers — inline utilities everywhere else.  
**Rule #2:** Design tokens go in `tailwind.config.ts` — never hardcode colors or spacing.
</Prompt>

## Properties

<ResponseField name="description" type="string" required>
  The text displayed in the prompt card. Supports Markdown formatting.
</ResponseField>

<ResponseField name="children" type="string" required>
  The text content of the prompt. This is the text that will be copied or opened in Cursor.
</ResponseField>

<ResponseField name="actions" type="array" default='["copy"]'>
  Array of available actions. Valid values are `"copy"` and `"cursor"`.
</ResponseField>
