Use the Prompt component to display pre-built AI prompts that users can copy to their clipboard or open directly in Cursor.

## Basic usage

<Prompt description="Use this pre-built prompt to get started faster.">
# Tailwind CSS Expert

You are a **Tailwind CSS expert**. Always use utility classes over custom CSS.  
**Rule #1:** Prefer `@apply` only in component layers — inline utilities everywhere else.  
**Rule #2:** Design tokens go in `tailwind.config.ts` — never hardcode colors or spacing.
</Prompt>

```mdx

```

## Props

<ResponseField name="description" type="string" required>
  A brief description of the prompt displayed above the prompt content.
</ResponseField>

<ResponseField name="children" type="string" required>
  The prompt text content. This is the text that will be copied or opened in Cursor.
</ResponseField>

<ResponseField name="actions" type="array" default='["copy"]'>
  Array of available actions. Valid values are `"copy"` and `"cursor"`.
</ResponseField>

<ResponseField name="icon" type="string | ReactNode">
  Icon to display next to the description. Can be a Lucide icon name or a custom React component.
</ResponseField>

<ResponseField name="iconType" type="IconType">
  Type of icon when using a Lucide icon name. Options include `"regular"`, `"solid"`, etc.
</ResponseField>
