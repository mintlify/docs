Use popups to display supplemental content in a floating overlay when a user clicks a trigger element. Popups are useful for showing additional context, definitions, or interactive content without navigating away from the page.

<Popup>
  <PopupTrigger>
    <button style={{ padding: '6px 12px', borderRadius: '6px', border: '1px solid #e2e8f0', cursor: 'pointer', fontSize: '14px' }}>Click me</button>
  </PopupTrigger>
  <PopupContent>
    This is additional context displayed in a popup overlay.
  </PopupContent>
</Popup>

```mdx Popup example
<Popup>
  <PopupTrigger>
    <button>Click me</button>
  </PopupTrigger>
  <PopupContent>
    This is additional context displayed in a popup overlay.
  </PopupContent>
</Popup>
```

## Structure

A popup requires three components used together:

- `<Popup>` — the wrapper that manages open/close state
- `<PopupTrigger>` — the element that opens the popup when clicked
- `<PopupContent>` — the floating overlay content

```mdx Basic structure
<Popup>
  <PopupTrigger>
    {/* Your trigger element */}
  </PopupTrigger>
  <PopupContent>
    {/* Your popup content */}
  </PopupContent>
</Popup>
```

## Examples

### Term definition

Use popups to define terms without cluttering the page.

<Popup>
  <PopupTrigger>
    <span style={{ textDecoration: 'underline', textDecorationStyle: 'dotted', cursor: 'pointer' }}>What is SSR?</span>
  </PopupTrigger>
  <PopupContent>
    **Server-Side Rendering** — the process of rendering a web page on the server before sending it to the client.
  </PopupContent>
</Popup>

```mdx Term definition
<Popup>
  <PopupTrigger>
    <span style={{ textDecoration: 'underline', textDecorationStyle: 'dotted', cursor: 'pointer' }}>What is SSR?</span>
  </PopupTrigger>
  <PopupContent>
    **Server-Side Rendering** — the process of rendering a web page on
    the server before sending it to the client.
  </PopupContent>
</Popup>
```

### Rich content

Popups can contain any MDX content, including formatted text and code.

<Popup>
  <PopupTrigger>
    <button style={{ padding: '6px 12px', borderRadius: '6px', border: '1px solid #e2e8f0', cursor: 'pointer', fontSize: '14px' }}>View configuration</button>
  </PopupTrigger>
  <PopupContent>
    Set the `timeout` value in your configuration file:

    ```json
    {
      "timeout": 30
    }
    ```
  </PopupContent>
</Popup>

````mdx Rich content
<Popup>
````
