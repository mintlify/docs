description: "Display text with a rainbow gradient effect."

---

Apply a rainbow gradient effect to text content.

## Basic usage

<RainbowText>deployment</RainbowText>

```mdx
<RainbowText>deployment</RainbowText>
```

## Inline usage

Use rainbow text inline within paragraphs. For example, this <RainbowText>deployment</RainbowText> has rainbow styling.

```mdx
Use rainbow text inline within paragraphs. For example, this <RainbowText>deployment</RainbowText> has rainbow styling.
```

## Properties

<ResponseField name="children" type="string" required>
  The text content to display with rainbow gradient.
</ResponseField>

<ResponseField name="className" type="string">
  Additional CSS classes to apply to the text.
</ResponseField>
