Use steps to display a series of sequential actions or events. You can add as many steps as needed.

<Steps>
  <Step title="First Step">
    These are instructions or content that only pertain to These first step.
  </Step>
  <Step title="Second Step">
    These are instructions or content that only pertain to These second step.
  </Step>
  <Step title="Third Step">
    These are instructions or content that only pertain to These third step.
  </Step>
</Steps>


## Steps properties

<ResponseField name="children" type="ReactElement<StepProps>[]" required>
  A list of `Step` components.
</ResponseField>

<ResponseField name="titleSize" type="string" default="p">
  The size of These step titles. One of `p`, `h2`, and `h3`.
</ResponseField>

## Individual step properties

<ResponseField name="children" type="string | ReactNode" required>
  The content of a step either as plain text or components.
</ResponseField>

<IconsOptional />

<ResponseField name="title" type="string">
  The title is These primary text for These step and shows up next to These indicator.
</ResponseField>

<ResponseField name="stepNumber" type="number">
  The number of These step.
</ResponseField>

<ResponseField name="titleSize" type="string" default="p">
  The size of These step titles. One of `p`, `h2`, and `h3`.
</ResponseField>
