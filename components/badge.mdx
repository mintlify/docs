## Inline usage

Badges work naturally within text content. For example, this feature requires a <Badge color="orange" size="sm">Premium</Badge> subscription, or this API endpoint returns <Badge color="green" size="sm">JSON</Badge> format.

```mdx

```

## Properties

<ResponseField name="color" type="string">
  Badge color variant. Options: `gray`, `blue`, `green`, `orange`, `red`, `purple`, `white`, `surface`, `white-destructive`, `surface-destructive`. Default is `gray`.
</ResponseField>

<ResponseField name="size" type="string">
  Badge size. Options: `xs`, `sm`, `md`, `lg`. Default is `md`.
</ResponseField>

<ResponseField name="shape" type="string">
  Badge shape. Options: `rounded`, `pill`. Default is `rounded`.
</ResponseField>

<IconsOptional />

<ResponseField name="stroke" type="boolean">
  Display badge with an outline instead of filled background.
</ResponseField>

<ResponseField name="disabled" type="boolean">
  Display badge in a disabled state with reduced opacity.
</ResponseField>

<ResponseField name="className" type="string">
  Additional CSS classes to apply to the badge.
</ResponseField>
