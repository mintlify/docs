Use the `Popup` component to display rich content in an overlay when a user clicks a trigger element. Unlike [tooltips](/components/tooltips), popups support arbitrary content such as text, images, code blocks, and other components.

<Popup>
  <PopupTrigger>Click me to see the popup</PopupTrigger>
  <PopupContent>
    This is the popup content. You can include **rich text**, links, and other components here.
  </PopupContent>
</Popup>

## Example

```mdx Popup example
<Popup>
  <PopupTrigger>Click me to see the popup</PopupTrigger>
  <PopupContent>
    This is the popup content. You can include **rich text**, links,
    and other components here.
  </PopupContent>
</Popup>
```

## Components

The popup requires three components used together:

| Component | Description |
| :--- | :--- |
| `Popup` | Wrapper that manages the open/close state. |
| `PopupTrigger` | The clickable element that toggles the popup. |
| `PopupContent` | The content displayed inside the popup overlay. |
