---
title: "Popup"
description: "Display additional content in a floating popup triggered by clicking or hovering over an element."
keywords: ["popup", "popover", "tooltip", "floating content", "interactive"]
---

Use the Popup component to display supplementary content in a floating panel anchored to a trigger element. Popups are useful for definitions, previews, or contextual details without navigating away from the current page.

## Examples

<Popup>
  <PopupTrigger>Hover over this text</PopupTrigger>
  <PopupContent>A popup displays floating content anchored to a trigger element.</PopupContent>
</Popup>

```mdx Basic popup example
<Popup>
  <PopupTrigger>Hover over this text</PopupTrigger>
  <PopupContent>
    A popup displays floating content anchored to a trigger element.
  </PopupContent>
</Popup>
```

You can include rich content inside the popup:

<Popup>
  <PopupTrigger>API rate limits</PopupTrigger>
  <PopupContent>
    **Rate limits** protect your API from excessive requests.

    - Free plan: 100 requests/min
    - Pro plan: 1,000 requests/min
  </PopupContent>
</Popup>

```mdx Popup with rich content
<Popup>
  <PopupTrigger>API rate limits</PopupTrigger>
  <PopupContent>
    **Rate limits** protect your API from excessive requests.

    - Free plan: 100 requests/min
    - Pro plan: 1,000 requests/min
  </PopupContent>
</Popup>
```

## Sub-components

The Popup component uses three sub-components:

<ResponseField name="Popup" type="component" required>
  The root wrapper that manages the popup state. Contains `PopupTrigger` and `PopupContent` as children.
</ResponseField>

<ResponseField name="PopupTrigger" type="component" required>
  The element that activates the popup. Renders inline so you can use it within text.
</ResponseField>

<ResponseField name="PopupContent" type="component" required>
  The floating content displayed when the popup is active. Supports Markdown formatting including bold, italic, lists, links, and other MDX components.
</ResponseField>
