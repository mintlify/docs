Use the Timeline component to display a sequence of events in chronological order. Timelines are useful for changelogs, release histories, or onboarding steps.

## Basic timeline

<Timeline>
  <Timeline.Item title="v2.0 released" date="2025-03-01">
    Major release with breaking changes and new API endpoints.
  </Timeline.Item>
  <Timeline.Item title="v1.5 released" date="2025-01-15">
    Added support for webhooks and event-driven workflows.
  </Timeline.Item>
  <Timeline.Item title="v1.0 released" date="2024-11-01">
    Initial public release with core API functionality.
  </Timeline.Item>
</Timeline>

## With icons

Add icons to timeline items to indicate the type of event.

<Timeline>
  <Timeline.Item title="New feature" date="2025-03-01" icon="sparkles" color="blue">
    Added real-time collaboration support.
  </Timeline.Item>
  <Timeline.Item title="Bug fix" date="2025-02-14" icon="bug" color="red">
    Fixed an issue with authentication token refresh.
  </Timeline.Item>
  <Timeline.Item title="Improvement" date="2025-01-20" icon="arrow-up" color="green">
    Improved API response times by 40%.
  </Timeline.Item>
</Timeline>

## With nested content

Timeline items can contain any components including code blocks, callouts, and links.

<Timeline>
  <Timeline.Item title="Migrate to v2" date="2025-03-01">
    Update your API calls to use the new authentication header:

    ```bash
    curl -H "Authorization: Bearer YOUR_TOKEN" https://api.example.com/v2/resource
    ```

    <Note>See the [migration guide](/guides/migrating-from-mdx) for detailed instructions.</Note>
  </Timeline.Item>
  <Timeline.Item title="Deprecation notice" date="2025-02-01">
    The `/v1/users` endpoint is deprecated and will be removed on June 1, 2025.
  </Timeline.Item>
</Timeline>


## Properties

### Timeline

<ResponseField name="children" type="Timeline.Item" required>
  One or more timeline items to display.
</ResponseField>

### Timeline.Item

<ResponseField name="title" type="string" required>
  The title of the timeline event.
</ResponseField>

<ResponseField name="date" type="string">
  A date or label displayed alongside the title.
</ResponseField>

<ResponseField name="color" type="string" default="gray">
  Color of the timeline marker.

  Options: `gray`, `blue`, `green`, `yellow`, `orange`, `red`, `purple`.
</ResponseField>
