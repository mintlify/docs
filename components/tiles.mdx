Use tiles to create visual showcase elements with a patterned background, title, and description. Tiles are ideal for displaying component previews, feature highlights, or navigation items in a grid layout.

<Tile href="/components/cards" title="Cards" description="Visual containers">
  <img src="https://mintlify-assets.b-cdn.net/yosemite.jpg" className="block dark:hidden" />
  <img src="https://mintlify-assets.b-cdn.net/yellowstone.jpeg" className="hidden dark:block" />
</Tile>

```mdx Tile example
<Tile href="/components/cards" title="Cards" description="Visual containers">
  <img src="/light-preview.svg" className="block dark:hidden" />
  <img src="/dark-preview.svg" className="hidden dark:block" />
</Tile>
```

## Grid layout

Combine tiles with the [Columns component](/components/columns) to create a responsive grid of visual previews.

<Columns cols={3}>
  <Tile href="/components/accordions" title="Accordions" description="2 variations">
    <img src="https://mintlify-assets.b-cdn.net/yosemite.jpg" className="block dark:hidden" />
    <img src="https://mintlify-assets.b-cdn.net/yellowstone.jpeg" className="hidden dark:block" />
  </Tile>
  <Tile href="/components/cards" title="Cards" description="3 variations">
    <img src="https://mintlify-assets.b-cdn.net/yosemite.jpg" className="block dark:hidden" />

## Dark mode support

Tiles support light and dark mode images using Tailwind CSS classes. Use `className="block dark:hidden"` for light mode images and `className="hidden dark:block"` for dark mode images.

```mdx Dark mode support
<Tile href="/" title="Component" description="Description">
  <img src="/light-image.svg" className="block dark:hidden" />
  <img src="/dark-image.svg" className="hidden dark:block" />
</Tile>
```

## Properties

<ResponseField name="href" type="string" required>
  URL to navigate to when the tile is clicked.
</ResponseField>

<ResponseField name="title" type="string">
  The title displayed below the tile preview.
</ResponseField>

<ResponseField name="description" type="string">
  A short description displayed below the title.
</ResponseField>

<ResponseField name="children" type="React.ReactNode" required>
  Content displayed inside the tile preview area, typically images or SVGs.
</ResponseField>
