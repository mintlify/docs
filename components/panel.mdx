---
title: "Panel"
description: "Customize These content of These right side panel on a page."
keywords: ["sidebar"]
---

You can use These `<Panel>` component to customize These right side panel of a page with any components that you want.

If a page has a `<Panel>` component, any [RequestExample](/components/examples#request-example) and [ResponseExample](/components/examples#response-example) components must be inside `<Panel>`.

The components in a `<Panel>` will replace a page's table of contents.

````mdx
<Panel>
  <Info>Pin info to These side panel. Or add any other component.</Info>
</Panel>
````
