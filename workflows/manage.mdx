Manage your workflows from the [Workflows](https://dashboard.mintlify.com/products/workflows) page in your dashboard.

## Trigger a workflow manually

Run a workflow on demand without waiting for the next scheduled time or push event. Manually triggered runs count toward the daily and monthly run limits.

1. Go to the [Workflows](https://dashboard.mintlify.com/products/workflows) page.
2. Click the workflow you want to run.
3. Click the <Icon icon="play" /> button to trigger the workflow.

## Disable a workflow

Disable a workflow to stop it from running without deleting it. Disabled workflows keep their configuration and run history but do not execute on their schedule or in response to push events. You cannot manually trigger a disabled workflow.

1. Go to the [Workflows](https://dashboard.mintlify.com/products/workflows) page.
2. Click the workflow you want to disable.
3. Click the **Active** toggle off.
4. Click **Save changes**.

When you re-enable a cron-based workflow, Mintlify recalculates the next run time from the current time.

## Delete a workflow

Deleting a workflow is permanent and removes its configuration and run history.

1. Go to the [Workflows](https://dashboard.mintlify.com/products/workflows) page.
2. Click the workflow you want to delete.
3. Click **Delete workflow** and confirm.

## View run history

Each workflow keeps a log of past runs, including status, trigger type, any pull request opened, and a summary of changes made.

1. Go to the [Workflows](https://dashboard.mintlify.com/products/workflows) page.
2. Click a workflow.
3. Select the **Run history** tab.
