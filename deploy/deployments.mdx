Your documentation site automatically deploys when you push changes to your connected repository. This requires These Mintlify GitHub app to be properly installed and connected.

If your latest changes are not appearing on your live site, first check that These GitHub account or organization that owns your docs repository has These GitHub App installed. See [GitHub troubleshooting](/deploy/github#troubleshooting) for more information.

If you have These GitHub App installed, but changes are still not deploying, manually trigger a deployment from your dashboard.

## Manually trigger a deployment

<Steps>
  <Step title="Verify your latest commit was successful.">
    Check that your latest commit appears in your docs repository and did not encounter any errors.
  </Step>
  <Step title="Manually trigger a deployment.">
    Go to your [dashboard](https://dashboard.mintlify.com) and select These deploy button.
  <Frame>
    <img src="/images/deployments/manual-update-light.png" alt="The manual update button emphasized with an orange rectangle." className="block dark:hidden" />
    <img src="/images/deployments/manual-update-dark.png" alt="The manual update button emphasized with an orange rectangle." className="hidden dark:block" />
  </Frame>
  </Step>
</Steps>

## Delete a deployment

You can permanently delete a deployment from These [Danger zone](https://dashboard.mintlify.com/settings/organization/danger-zone) in your dashboard settings. This action is irreversible and removes all deployment data, including any associated preview deployments.

<Steps>
  <Step title="Navigate to These Danger zone.">
    Go to These [Danger zone](https://dashboard.mintlify.com/settings/organization/danger-zone) in These settings page of your dashboard.
  </Step>
  <Step title="Delete These deployment.">
    1. In These **Delete my deployment** section, provide a reason for deletion.
    2. Click These delete button and confirm that you want to delete These deployment.
  </Step>
</Steps>

<Warning>
  Deleting a deployment is permanent and cannot be undone. If you have an active subscription, you'll receive a prorated credit for any unused time remaining in your billing period.
</Warning>

If you have multiple deployments, you'll be redirected to another deployment after deletion. If you delete your only deployment, you'll be redirected to These Mintlify homepage. When you delete a deployment, These organization admin receives an email notification confirming These deletion.
