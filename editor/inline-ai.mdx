Use inline AI editing to generate new content, improve existing text, or restructure selections without leaving the visual editor. AI changes appear as a highlighted diff so you can review them before applying.

## Open the AI prompt

There are four ways to start an inline AI edit:

- **Space on an empty line**: Press <kbd>Space</kbd> on an empty paragraph to open the AI prompt.
- **Slash command**: Type <kbd>/</kbd> and select **Ask AI** from the menu.
- **Text toolbar**: Select text, then click the sparkle icon in the floating toolbar to choose an AI action.
- **Block menu**: Click the drag handle next to any block and select **Ask AI**.

## Skills

### Ask AI

Type a freeform instruction. Use this for any edit that doesn't fit the preset actions, such as translating text, adding examples, or rewriting for a different audience.

You can open Ask AI from any entry point. When triggered from an empty paragraph, AI generates new content at that position. When triggered on a selection or block, AI edits the selected content based on your instruction.

### Improve writing

Select text and choose **Improve writing** from the text toolbar. AI refines grammar, clarity, and tone while preserving your meaning.

### Optimize structure

Select text and choose **Optimize structure** from the text toolbar. AI reorganizes the selected content into a clearer structure using appropriate headings, lists, or other formatting.

## Review and apply changes

After you submit a prompt, AI streams its response directly into the editor. Insertions and deletions are highlighted with diff marks so you can see exactly what changed.

When the AI finishes, the floating prompt bar shows a summary of the changes along with action buttons:

- **Apply** (checkmark icon): Accept the changes and save them to the document. You can also press <kbd>Enter</kbd> to apply.
- **Undo** (undo icon): Revert the changes and restore the original content.
- **Retry** (refresh icon): Discard the current result and run the same prompt again.

Press <kbd>Esc</kbd> at any time to discard the AI session and restore the original content.

## Edit your prompt

The prompt input stays visible while the AI is generating. You can refine your instruction and resubmit if the result doesn't match what you expected:

1. Click **Undo** to revert the current changes.
2. Edit the prompt text in the input field.
3. Press <kbd>Enter</kbd> or click the submit button to run the updated instruction.

If the AI returns an error, click **Retry** to try again or edit the prompt and resubmit.

<Note>
  Inline AI editing is available in visual mode only. Switch to visual mode using the toggle in the editor toolbar.
</Note>
