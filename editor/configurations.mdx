Configure pages, navigation elements, and media using configuration sheets.

## What are configuration sheets

Configuration sheets provide a visual interface for editing page metadata, navigation properties, and media settings without directly editing frontmatter or configuration files.

Use configuration sheets to:

- Set page titles, descriptions, and SEO metadata
- Configure navigation element properties like icons and tags
- Update image alt text and file paths
- Adjust page display modes and sidebar settings

## Accessing configuration sheets

Open configuration sheets from different areas of the editor:

- **From the navigation tree**: Right-click a navigation item and select **Configure**
- **From the file tree**: Right-click a file and select **Configure**
- **From the media manager**: Right-click an image or asset and select **Configure**

The configuration sheet opens as a side panel with fields for the selected item.

## Page configuration

Configure page properties using the page configuration sheet.

### Basic properties

- **Title**: The page title that appears in navigation and browser tabs
- **Description**: A brief description of the page for SEO and navigation
- **File path**: The file location (editable to move or rename files)

### Display properties

- **Icon**: Select an icon from the [Lucide icon library](https://lucide.dev) to display next to the page title
- **Mode**: Choose how the page displays:
  - `default`: Standard page layout
  - `wide`: Full-width layout
  - `custom`: Custom layout with configurable width
  - `frame`: Embedded iframe layout
  - `center`: Centered content layout

### Navigation properties

- **Hide from navigation**: Toggle to hide the page from your site's navigation sidebar
- **Sidebar**: Configure sidebar behavior and position

### SEO properties

- **Meta tags**: Set custom meta tags for search engines
- **OpenGraph**: Configure social media preview images and descriptions

### Saving changes

Click **Save** or press <kbd>Enter</kbd> to save your configuration changes. Press <kbd>Escape</kbd> to close without saving.

## Navigation element configuration

Configure navigation structure elements like groups, links, and other navigation types.

### Group configuration

Configure navigation groups:

- **Group name**: The label that appears in the navigation sidebar
- **Icon**: Select an icon from the [Lucide icon library](https://lucide.dev)
- **Tag**: Add a tag label (like "NEW" or "BETA") that appears next to the group
- **Expanded**: Set whether the group is expanded or collapsed by default

### Link configuration

Configure external links in your navigation:

- **Link text**: The text that appears in the navigation
- **URL**: The destination URL
- **Icon**: Select an icon from the [Lucide icon library](https://lucide.dev)

### Other navigation elements

Configuration sheets also support tabs, anchors, and dropdowns. See [Navigation](/organize/navigation) for details on these navigation types and their configuration options.

## Media configuration

Configure images and other assets using the media configuration sheet.

### Image properties

- **Alt text**: Descriptive text for screen readers and SEO
- **File path**: The asset location (editable to move or rename)
- **Display settings**: Configure how the image displays in your documentation

## Keyboard shortcuts

Use keyboard shortcuts in configuration sheets:

- <kbd>Enter</kbd>: Save changes
- <kbd>Escape</kbd>: Close without saving
