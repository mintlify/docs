Configuration sheets provide a visual interface for editing page metadata, navigation properties, and media settings without directly editing frontmatter or configuration files.

## Access configuration sheets

Open configuration sheets by right-clicking an item and selecting **Configure**:

- **Pages** - Right-click a file in the file tree or a page in the navigation tree
- **Navigation elements** - Right-click a group, tab, or other navigation item in the navigation tree
- **Media** - Right-click an image or asset in the media manager

The configuration sheet opens as a side panel. Press <kbd>Enter</kbd> to save changes or <kbd>Escape</kbd> to close without saving.

## Page configuration

Configure how pages appear in your documentation and search results.

### Content and navigation

- **Title** - The main heading for your page. Appears in navigation, browser tabs, and search results.
- **Sidebar title** - Show different text in navigation than the page title. Use this when the full title is too long for the sidebar.
- **Description** - Brief summary of the page (1-2 sentences). Shown in search results and used for SEO.
- **Icon** - Add a visual marker next to the page in navigation. Choose from the [Lucide icon library](https://lucide.dev).

### Page layout

- **Mode** - Control how the page displays:
  - `default` - Standard page layout with table of contents
  - `wide` - Full-width layout for content that needs more horizontal space
  - `custom` - Set a specific maximum width for the content area
  - `center` - Centered content with constrained width for better readability
  - `frame` - Embed external content in an iframe

### Visibility

- **Hide from navigation** - Remove the page from your site's navigation sidebar. The page remains accessible via direct URL.

### SEO and social

- **Keywords** - Help users find this page in search. Add relevant terms.
- **OpenGraph image** - Set a custom preview image for social media shares.

See [Pages reference](/organize/pages) for complete frontmatter options.

## Navigation configuration

Configure navigation groups, links, and other navigation elements.

### Groups

- **Group name** - The label that appears in the navigation sidebar.
- **Icon** - Add a visual marker next to the group. Choose from the [Lucide icon library](https://lucide.dev).
- **Tag** - Display a label like "NEW" or "BETA" next to the group name.
- **Expanded by default** - Control whether the group starts open or collapsed when users visit your site.

### External links

- **Link text** - The text that appears in the navigation.
- **URL** - The destination URL when users click the link.
- **Icon** - Add a visual marker next to the link. Choose from the [Lucide icon library](https://lucide.dev).

### Other navigation types

Configuration sheets also support tabs, anchors, and dropdowns. See [Navigation](/organize/navigation) for details on these navigation types and their configuration options.

## Media configuration

Configure images and other assets.

- **Alt text** - Describe the image for screen readers and search engines. Required for accessibility.
- **File path** - The asset location. Edit this to move or rename the file.
