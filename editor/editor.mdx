Quick reference guide for the Mintlify web editor's features, modes, and keyboard shortcuts.

## Editor modes

| Feature | Visual Mode | Markdown Mode |
|:--------|:------------|:--------------|
| **Interface** | WYSIWYG editor | Raw MDX code |
| **Best for** | Quick edits, content creation | Precise control, component properties |
| **Component insertion** | `/` menu | Manual MDX syntax |
| **Real-time preview** | Yes | No |
| **Syntax highlighting** | No | Yes |
| **Advanced formatting** | Limited | Full control |

Switch between modes anytime using the toggle in the editor toolbar.

## Automated Git operations

The web editor handles these Git operations automatically:

| Action | Git Operation | Result |
|:-------|:--------------|:-------|
| **Open file** | `git fetch` + checkout | Latest version loaded |
| **Save changes** | `git commit` | Changes committed to branch |
| **Create branch** | `git checkout -b` | New branch created |
| **Publish (deployment branch)** | `git push` | Direct deployment |
| **Publish (feature branch)** | `git push` + PR creation | Pull request opened |

## Publishing behavior

| Branch Type | Publish Action | Result |
|:------------|:---------------|:-------|
| **Deployment branch** (main) | Direct publish | Immediate live deployment |
| **Feature branch** | Publish pull request | Review workflow, manual merge required |

## File operations

| Task | Method | Notes |
|:-----|:-------|:------|
| **Find files** | Sidebar explorer or <kbd>Cmd/Ctrl</kbd> + <kbd>P</kbd> | Search by filename |
| **Create file** | + icon in sidebar | Must end in `.mdx` |
| **Switch branches** | Branch dropdown in toolbar | Unsaved changes are lost |
| **Add to navigation** | Edit `docs.json` | Update `navigation.pages` array |

## Component insertion

In visual mode, press <kbd>/</kbd> to access:

- Text blocks (headings, paragraphs, lists)
- Callouts (Note, Tip, Warning, Info)  
- Code blocks with syntax highlighting
- Accordions and expandable content
- Frames and images
- Custom Mintlify components
