The web editor includes built-in AI editing tools that help you refine documentation without leaving the editor. Select any content on a page and use AI to improve clarity, reorganize structure, or apply custom instructions.

## Use AI editing

1. Open a page in Visual mode.
2. Highlight the text you want to modify.
3. Click the **AI** button in the floating toolbar that appears above your selection, or choose one of the available actions.

The editor streams AI-generated changes in real time. When the edit is complete, you can review the changes and accept or discard them.

## Available actions

### Improve writing

Revises selected text for clarity, flow, and readability. This action fixes spelling, grammar, and punctuation while improving sentence structure and word choice. It preserves the original meaning without adding new information.

Use this when you want to polish a draft or clean up rough notes into publication-ready documentation.

### Auto-structure

Reorganizes selected content to improve scannability and information architecture. The editor may reorder content, convert prose into lists or tables, and introduce components like callouts, tabs, or cards where they improve readability.

Use this when you have a block of text that would be easier to scan as structured content.

### Ask AI

Applies a custom instruction to the selected text. Type a prompt describing what you want, and the AI follows your instruction while preserving valid MDX syntax and technical accuracy.

Use this for tasks like:

- Simplifying technical language for a broader audience
- Adding context or examples to an explanation
- Converting content between formats, such as turning a paragraph into a step-by-step guide
- Adjusting tone or formality

<Note>
  AI editing works with both Markdown and MDX content. When editing `.mdx` files, the AI recognizes Mintlify components like callouts, cards, tabs, and accordions.
</Note>

## AI syntax fixer

When the editor detects a parsing error in your page, it offers to fix the syntax automatically. The AI corrects broken MDX or Markdown syntax with minimal changes, without altering your content or wording.

## AI merge conflict resolution

When you encounter merge conflicts between your branch and the upstream branch, the editor uses AI to resolve conflicts automatically. The resolution preserves changes from both versions, with your local changes taking priority when direct conflicts occur.

## Tips for effective use

- **Select specific content.** Highlight only the section you want to change rather than the entire page for more targeted edits.
- **Write clear prompts.** When using Ask AI, be specific about the desired outcome. For example, "Rewrite this for developers who are new to REST APIs" produces better results than "Make this simpler."
- **Review before accepting.** Always review AI-generated changes before accepting them to ensure accuracy and consistency with your documentation style.
