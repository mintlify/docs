## Editor actions and their Git equivalents

| Action in the editor | Git operation |
|---|---|
| Edit a page | Changes auto-save to Mintlify servers — no Git commit yet |
| Publish on your deployment branch | `git commit` + `git push` → triggers a deployment |
| Save in branch | `git commit` to the current feature branch |
| Create pull request | `git push` + opens a pull request against your deployment branch |
| Merge and publish | Merges the pull request → triggers a deployment |
| Create a branch | `git checkout -b <branch-name>` |
| Switch branches | `git checkout <branch-name>` |
| External push or CLI update | Incoming changes sync into the editor automatically (three-way merge) |

## How auto-sync works

When someone pushes changes to your repository outside the editor — from the CLI, a merged pull request, or any other Git client — the editor incorporates those changes automatically using a three-way merge.

The merge compares three versions:

- **Base**: The last version both the editor and Git agreed on.
- **Local**: Your current editor content, including unsaved edits.
- **Remote**: The newly pushed version from Git.

Non-overlapping changes apply automatically. If a remote change and your local edit affect the same part of a page, the editor highlights the conflict so you can resolve it.

## Key concepts

| Term | Definition |
|------|-----------|
| **Commit** | A saved snapshot of your changes at a point in time, with a message describing what changed. Publishing creates a commit. |
| **Branch** | A separate line of work. Changes on a branch don't affect your live site until merged. |
| **Deployment branch** | The branch that builds your live site, typically `main`. |
| **Pull request** | A proposal to merge changes from one branch into another, with a review step. |
| **Merge** | Combining changes from one branch into another. |
| **Conflict** | Incompatible changes to the same file on two branches. The editor highlights conflicts and prompts you to resolve them. |
| **Diff** | A view showing what changed between two versions of a file. |
