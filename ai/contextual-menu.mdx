The contextual menu provides quick access to AI-optimized content and direct integrations with AI tools. Users can copy content as context or open conversations in ChatGPT, Claude, Perplexity, or custom tools with documentation pre-loaded.

## Menu options

| Option | Identifier | Description |
|:--------|:------------|:-------------|
| **Copy page** | `copy` | Copies the current page as Markdown |
| **View as Markdown** | `view` | Opens the current page as Markdown |
| **Open in ChatGPT** | `chatgpt` | Creates a ChatGPT conversation with page context |
| **Open in Claude** | `claude` | Creates a Claude conversation with page context |
| **Open in Perplexity** | `perplexity` | Creates a Perplexity conversation with page context |
| **Copy MCP server URL** | `mcp` | Copies your MCP server URL |
| **Connect to Cursor** | `cursor` | Installs your MCP server in Cursor |
| **Connect to VS Code** | `vscode` | Installs your MCP server in VS Code |

## Enabling the contextual menu

Add the `contextual` field to your `docs.json`:

```json
{
 "contextual": {
   "options": ["copy", "view", "chatgpt", "claude", "perplexity", "mcp", "cursor", "vscode"]
 }
}
```

## Adding custom options

Create custom options by adding an object to the `options` array:

<ResponseField name="title" type="string" required>
  The title of the option.
</ResponseField>

<ResponseField name="description" type="string" required>
  The description displayed beneath the title.
</ResponseField>

<IconsRequired />

<ResponseField name="href" type="string | object" required>
  The href. Use a string for simple links or an object for dynamic links.
  
  <Expandable title="href object">
    <ResponseField name="base" type="string" required>
      The base URL.
    </ResponseField>

    <ResponseField name="query" type="object" required>
      Query parameters. Use placeholders: `$page` (page content), `$path` (page path), `$mcp` (MCP server URL).
    </ResponseField>
  </Expandable>
</ResponseField>

Example:

```json {9-14} wrap
{
    "contextual": {
        "options": [
            "copy",
            "chatgpt",
            {
                "title": "Request a feature",
                "description": "Join the discussion on GitHub",
                "icon": "plus",
                "href": "https://github.com/orgs/mintlify/discussions/categories/feature-requests"
            }
        ]
    }
}
```
