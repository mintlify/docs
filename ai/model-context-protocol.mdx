## About MCP servers

The Model Context Protocol (MCP) creates standardized connections between AI applications and external services. Mintlify generates an MCP server from your documentation that any MCP client (Claude, Cursor, VS Code, etc.) can connect to.

Your MCP server exposes a search tool for AI applications to query your documentation. The LLM can proactively search your docs during response generation based on conversation context.

## Accessing your MCP server

<Note>
  MCP servers can only be generated for public documentation.
</Note>

Your MCP server is hosted at your documentation URL with the `/mcp` path (e.g., `https://mintlify.com/docs/mcp`). Find your URL in your [dashboard](https://dashboard.mintlify.com/products/mcp).

## Using your MCP server

Help users connect to your MCP server:

<Tabs>
<Tab title="Contextual menu">
  Add options in the [contextual menu](/ai/contextual-menu):

  | Option | Identifier | Description |
  | :----- | :--------- | :---------- |
  | **Copy MCP server URL** | `mcp` | Copies URL to clipboard |
  | **Connect to Cursor** | `cursor` | Installs in Cursor |
  | **Connect to VS Code** | `vscode` | Installs in VS Code |
</Tab>
<Tab title="Claude">
1. Navigate to [Connectors](https://claude.ai/settings/connectors) in Claude settings.
2. Select **Add custom connector**.
3. Add your MCP server name and URL.
4. When using Claude, select the attachments button and choose your MCP server.

See the [MCP documentation](https://modelcontextprotocol.io/docs/tutorials/use-remote-mcp-server#connecting-to-a-remote-mcp-server) for details.
</Tab>
<Tab title="Claude Code">
```bash
claude mcp add --transport http <name> <url>
```

See the [Claude Code documentation](https://docs.anthropic.com/en/docs/claude-code/mcp#installing-mcp-servers) for details.
</Tab>
<Tab title="Cursor">
1. Open command palette (<kbd>Command</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd>).
2. Search for "Open MCP settings" and select **Add custom MCP**.
3. Configure in `mcp.json`:

```json
{
  "mcpServers": {
    "<your-mcp-server-name>": {
      "url": "<your-mcp-server-url>"
    }
  }
}
```

See the [Cursor documentation](https://docs.cursor.com/en/context/mcp#installing-mcp-servers) for details.
</Tab>
<Tab title="VS Code">
Create `.vscode/mcp.json`:

```json
{
  "servers": {
    "<your-mcp-server-name>": {
      "type": "http",
      "url": "<your-mcp-server-url>"
    }
  }
}
```

See the [VS Code documentation](https://code.visualstudio.com/docs/copilot/chat/mcp-servers) for details.
</Tab>
</Tabs>

### Example: Connecting to Mintlify MCP server

<Tabs>
<Tab title="Contextual menu">

</Tab>
<Tab title="Claude">
1. Navigate to [Connectors](https://claude.ai/settings/connectors).
2. Add: Name: `Mintlify`, URL: `https://mintlify.com/docs/mcp`
3. Select the attachments button and choose Mintlify.
