Autopilot monitors your selected Git repositories and suggests documentation updates. Autopilot flags potential user-facing changes in your application and prepares context for the agent to draft a pull request to update your documentation.

Use autopilot to identify documentation gaps and proactively keep your documentation up to date when new features ship.

## Configure autopilot

Configure autopilot settings from your [dashboard](https://dashboard.mintlify.com/products/autopilot/settings). Click the **Configuration** button to access the autopilot settings page.
