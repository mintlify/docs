The [agent](/ai/agent) monitors your Git repositories and suggests documentation updates when it identifies user-facing changes. Create pull requests with one click.

## Prerequisites

Install the [Mintlify GitHub App](/deploy/github) with access to your documentation repository and at least one other repository.

## Configure suggestions

### Select repositories to monitor

1. Click **Ask agent** to open the agent panel.
1. Click **Settings**.
    <Frame>
    <img src="/images/agent/dashboard-settings-light.png" alt="The settings button in light mode." className="block dark:hidden" />
    <img src="/images/agent/dashboard-settings-dark.png" alt="The settings button in dark mode." className="hidden dark:block" />
    </Frame>
1. In **Monitor Repositories**, click an organization and toggle repositories on.

The agent monitors the default branch. A **Mintlify Autopilot** check runs on PRs to analyze them for documentation updates.

<Note>
  When you first enable monitoring, the agent creates suggestions for PRs merged in the last seven days.
</Note>

### Stop monitoring

Toggle repositories off in the **Monitor Repositories** section. Existing suggestions remain until dismissed.

## Review suggestions

The **Ask agent** button displays the number of pending suggestions.

<Frame>
  <img src="/images/suggestions/count-light.png" alt="The Ask agent button showing suggestions in light mode." className="block dark:hidden" />
  <img src="/images/suggestions/count-dark.png" alt="The Ask agent button showing suggestions in dark mode." className="hidden dark:block" />
</Frame>

### Create pull requests

1. Click **Ask agent**.
1. Click **Add to chat** on a suggestion.
1. Submit the prompt to create a PR.

### Dismiss suggestions

Click **Dismiss** to remove suggestions you don't need. Dismissed suggestions cannot be retrieved.
