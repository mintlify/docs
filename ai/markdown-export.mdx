Markdown provides structured text that AI tools process more efficiently than HTML, resulting in better accuracy, faster processing, and lower token usage.

Mintlify automatically generates Markdown versions of pages optimized for AI tools.

## .md URL extension

Add `.md` to any page's URL to view a Markdown version.

<PreviewButton href="https://mintlify.com/docs/ai/markdown-export.md">Open this page as Markdown</PreviewButton>

## Keyboard shortcut

Press <kbd>Command</kbd> + <kbd>C</kbd> (<kbd>Ctrl</kbd> + <kbd>C</kbd> on Windows) to copy a page as Markdown.
