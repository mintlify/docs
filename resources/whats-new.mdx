## Latest releases

### January 2026

<AccordionGroup>
  <Accordion title="install.md auto-generation" icon="file-code">
    Documentation sites now automatically generate an `install.md` file that aggregates installation-related pages for AI agents. Custom files take priority over auto-generated versions.
  </Accordion>

  <Accordion title="Web editor improvements" icon="pen-line">
    - **Live preview**: Real-time changes without preview deployments
    - **Unused files section**: Drag-and-drop support for unused pages
    - **Branch search**: Easier navigation between deployment branches
  </Accordion>

  <Accordion title="Assistant message tiers" icon="bot">
    Select tiers of assistant message allotments per month from the dashboard.
  </Accordion>

  <Accordion title="Tab synchronization" icon="arrows-left-right">
    Tabs and code groups now synchronize selection across pages for consistent language or framework choices.
  </Accordion>
</AccordionGroup>

### December 2025

<AccordionGroup>
  <Accordion title="Agent suggestions" icon="wand-magic-sparkles">
    The agent monitors Git repositories and suggests documentation updates when user-facing code changes are detected.
  </Accordion>

  <Accordion title="Q&A bots" icon="message-circle">
    Add bots powered by the assistant to Slack workspaces and Discord servers for real-time answers.
  </Accordion>

  <Accordion title="Preview widget" icon="eye">
    Interactive widget displays all changed files in preview deployments with one-click navigation.
  </Accordion>

  <Accordion title="Last modified dates" icon="clock">
    Display automatic timestamps on pages to show when content was last updated.
  </Accordion>
</AccordionGroup>
