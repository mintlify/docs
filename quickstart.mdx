[Create an account](https://mintlify.com/start) to deploy your site to `https://<your-project-name>.mintlify.app`.

## Edit your docs

Use the **Editor** in your [dashboard](https://dashboard.mintlify.com/) or install the CLI for local development:

```bash
npm i -g mint && mint dev
```

Push changes to GitHub to deploy automatically.
