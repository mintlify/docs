Get your documentation site live in minutes. After completing this guide, you'll have a deployed site ready to customize.

<Info>
**Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Your site is live

After onboarding, your documentation automatically deploys to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

## Local development

Install the CLI to work locally:

<CodeGroup>
  ```bash npm
  npm i -g mint
  ```

  ```bash pnpm
  pnpm add -g mint
  ```
</CodeGroup>

Preview your changes:

```bash
mint dev
```

Your preview will be available at `localhost:3000`. Push changes to your repository to deploy automatically.

## Custom domain (optional)

Add a custom domain in your [dashboard settings](https://dashboard.mintlify.com/settings/deployment/custom-domain). Configure your DNS with a CNAME record pointing to `cname.mintlify.app`.
