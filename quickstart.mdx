Deploy your documentation site in minutes and start editing.

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account and create a repository. Your site deploys automatically at `https://<your-project-name>.mintlify.app`.

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open the editor">
        Navigate to the [web editor](https://dashboard.mintlify.com/editor) in your dashboard.
      </Step>
      <Step title="Edit a page">
        Open the **Introduction** page and update the description.
      </Step>
      <Step title="Publish">
        Click **Publish** in the top-right toolbar. Your changes deploy automatically.
      </Step>
    </Steps>
  </Tab>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        ```bash
        npm i -g mint
        ```
        Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher. See [Install the CLI](/installation) for details.
      </Step>
      <Step title="Edit and preview">
        Open `index.mdx` and update the description. Run `mint dev` to preview at `http://localhost:3000`.
      </Step>
      <Step title="Deploy">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```
        Your changes deploy automatically.
      </Step>
    </Steps>
  </Tab>
</Tabs>
