## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account, create a repository for your docs, and install the GitHub App.

After onboarding, your site is live at `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/).

<Tip>
  If your code is already in a public GitHub repository, [auto-generate your site](https://mintlify.com/explore) by replacing `github.com` with `mintlify.com` in the URL.
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+.

        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Edit and preview">
        Clone your repository, open `index.mdx`, and update the description. Then preview locally:

        ```bash
        mint dev
        ```
      </Step>
      <Step title="Push your changes">
        ```bash
        git add . && git commit -m "Update description" && git push
        ```

        Mintlify deploys automatically. Check status on the [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit a page">
        Open the [web editor](https://dashboard.mintlify.com/editor), select the **Introduction** page, and update the description.
      </Step>
      <Step title="Publish">
        Click **Publish** in the toolbar. Your changes deploy automatically.
      </Step>
    </Steps>
  </Tab>
</Tabs>
