## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) to deploy your site. Connect your GitHub account, select a repository, and install the GitHub App for automatic deployments.

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    Install the CLI and clone your repository:

    ```bash npm
    npm i -g mint
    ```

    Edit `index.mdx` and update the description. Preview with `mint dev`, then push:

    ```bash
    git add .
    git commit -m "Update description"
    git push
    ```
  </Tab>
  <Tab title="Web editor">
    Open the [web editor](https://dashboard.mintlify.com/editor), edit the **Introduction** page, and click **Publish**.
  </Tab>
</Tabs>
