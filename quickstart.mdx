Deploy your documentation site and make your first change.

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and complete onboarding. Connect your GitHub account and select a repository. Your site deploys to `<your-project-name>.mintlify.app`.

## View your site

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Clone your repository">
        Clone your repository locally if you haven't already.
      </Step>
      <Step title="Edit a page">
        Open `index.mdx` and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        ```bash
        mint dev
        ```
      </Step>
      <Step title="Push your changes">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open the web editor">
        Go to the [web editor](https://dashboard.mintlify.com/editor).
      </Step>
      <Step title="Edit a page">
        Open the **Introduction** page and update the description.
      </Step>
      <Step title="Publish">
        Click **Publish** in the top-right.
      </Step>
      <Step title="View live">
        Check deployment status on the [Overview](https://dashboard.mintlify.com/) page, then refresh your site.
      </Step>
    </Steps>
  </Tab>
</Tabs>
