Get your documentation site live in minutes.

<Info>
**Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Your site is ready

After onboarding, your site is automatically deployed at:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

## Web editor

1. Go to your [dashboard](https://dashboard.mintlify.com)
2. Click **Editor** in the sidebar
3. Select any file to start editing
4. Click **Publish** when ready

<Tip>
Type `/` in the editor to access formatting tools and components.
</Tip>

## Code editor

### Install the CLI

```bash
npm i -g mint
```

### Edit locally

1. Open your repository from onboarding
2. Edit any `.mdx` file
3. Run `mint dev` to preview at `localhost:3000`
4. Push changes to deploy automatically

## Add a custom domain

1. Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain)
2. Enter your domain (e.g., `docs.yourcompany.com`)
3. Add this DNS record:

| Type | Name | Value |
|------|------|-------|
| CNAME | docs | cname.mintlify.app |

<Info>
DNS changes can take up to 48 hours to propagate.
</Info>
