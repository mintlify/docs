## Getting started

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

### Install the GitHub App

1. Navigate to **Settings** in your Mintlify dashboard.
2. Select **GitHub App** from the sidebar.
3. Select **Install GitHub App**.
4. Select the organization or user account where you want to install the app.
5. Select the repositories to connect.

  <img
    src="/images/quickstart/github-app-installation-light.png"

### Authorize your GitHub account

1. Navigate to **Settings** in your Mintlify dashboard.
2. Select **My Profile** from the sidebar.
3. Select **Authorize GitHub account**.

## Code-based workflow

### Install the CLI

<Info>
  **Prerequisite**: [Node.js](https://nodejs.org/en) v20.17.0 or higher through v24.
</Info>

Install the [mint](https://www.npmjs.com/package/mint) CLI:

<CodeGroup>

```bash npm
npm i -g mint
```


```bash pnpm
pnpm add -g mint
```

</CodeGroup>

### Create a new project

Run `mint new` to create a new documentation project. See the [CLI installation guide](/installation#create-a-new-project) for details.

### Edit the documentation

Open `index.mdx` and update the title:

```mdx index.mdx {2}
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

### Preview the changes

To preview the changes locally, run the following command:

```bash
mint dev
```

Your preview is available at `localhost:3000`.

### Push the changes

Push to your repository to deploy. Mintlify automatically builds and deploys your changes. Monitor deployment status in your GitHub commit history or the [dashboard](https://dashboard.mintlify.com).

## Adding a custom domain

Navigate to the [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) page in your dashboard.

<Frame>
  <img
    src="/images/quickstart/custom-domain-light.png"
    alt="Custom Domain"

  />

  <img
    src="/images/quickstart/custom-domain-dark.png"
    alt="Custom Domain"
    className="hidden dark:block"
  />
</Frame>

Enter your domain (for example, `docs.yourcompany.com`) and configure DNS settings:

<Table>
  | Record Type | Name                | Value                | TTL  |
  | ----------- | ------------------- | -------------------- | ---- |
  | CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
</Table>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Local preview not working">
    Ensure Node.js v20.17.0 or higher is installed and run `mint dev` from the directory containing your `docs.json` file.
  </Accordion>
  <Accordion title="Changes not reflecting on live site">
    Deployment takes a few minutes. Check GitHub Actions or deployment logs in the Mintlify dashboard for build errors.
  </Accordion>
  <Accordion title="Custom domain not connecting">
    Verify DNS records are correct and allow up to 48 hours for propagation. Use [DNSChecker](https://dnschecker.org) to verify your CNAME record.
  </Accordion>
</AccordionGroup>
