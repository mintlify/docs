## Installation

<Steps>
  <Step title="Install the Mintlify CLI">
    ```bash
    npm i -g mintlify
    ```
  </Step>

  <Step title="Navigate to your docs directory">
    ```bash
    cd path/to/docs
    ```
  </Step>

  <Step title="Initialize your documentation">
    ```bash
    mintlify init
    ```
  </Step>

  <Step title="Start the development server">
    ```bash
    mintlify dev
    ```
    Your documentation will be available at `http://localhost:3000`.
  </Step>
</Steps>

## Create your first page

Create a new file called `getting-started.mdx`:

```mdx getting-started.mdx
---
title: Getting started
description: Learn the basics of our platform
---

# Welcome to our platform

Install our package using npm:

```bash
npm install our-package
```

## Basic usage

```javascript
import { initialize } from 'our-package';

const app = initialize({
  apiKey: 'your-api-key'
});
```
```

Add the page to `docs.json`:

```json docs.json
{
  "navigation": {
    "pages": ["quickstart", "getting-started"]
  }
}
```

## Deploy

1. Push your documentation to GitHub
2. Go to [dashboard.mintlify.com](https://dashboard.mintlify.com)
3. Sign in and select your repository
4. Click "Deploy"
