[Create an account](https://mintlify.com/start) and complete onboarding. Your site deploys to `https://<your-project-name>.mintlify.app`.

## Make your first edit

<Tabs>
  <Tab title="Web editor">
    1. Open **Editor** in your dashboard.
    2. Select `index.mdx` and update the title.
    3. Click **Publish** to deploy.
  </Tab>
  
  <Tab title="Code editor">
    Install the CLI:

    ```bash
    npm i -g mint
    ```

    Create a project:

    ```bash
    mint new
    ```

    Edit `index.mdx`, then push to GitHub to deploy.
  </Tab>
</Tabs>
