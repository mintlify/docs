Deploy your documentation site in minutes and make your first edit.

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding. You'll connect your GitHub account, select a repository, and install the GitHub App.

Your site deploys automatically at `https://<your-project-name>.mintlify.app`.

<Tip>
  Already have a public GitHub repository? [Auto-generate your docs](https://mintlify.com/explore) by replacing `github.com` with `mintlify.com` in your repo URL.
</Tip>

## Make your first edit

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor)
    2. Edit the **Introduction** page description
    3. Click **Publish**
    4. View your changes live at your site URL
  </Tab>
  <Tab title="CLI">
    1. Install the CLI: `npm i -g mint`
    2. Clone your repository locally
    3. Edit `index.mdx` and update the description
    4. Preview with `mint dev`
    5. Push your changes: `git push`

    See [CLI installation](/installation) for details.
  </Tab>
</Tabs>
