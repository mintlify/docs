## Deploy your site

1. Go to [mintlify.com/start](https://mintlify.com/start) and complete onboarding
2. Connect your GitHub account and select a repository
3. Your site deploys at `https://<your-project-name>.mintlify.app`

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

<Tip>
  You can skip GitHub connection during onboarding to start quickly. Mintlify creates a private repository for you that you can migrate later.
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    Install the CLI ([Node.js](https://nodejs.org/en) v20.17.0+ required):

    ```bash
    npm i -g mint
    ```

    Clone your repo, then edit `index.mdx` and preview locally:

    ```bash
    mint dev
    ```

    Push changes to trigger deployment:

    ```bash
    git add . && git commit -m "Update description" && git push
    ```

    See [Install the CLI](/installation) for details.
  </Tab>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor)
    2. Edit the **Introduction** page
    3. Click **Publish**

    View deployment status on the [Overview](https://dashboard.mintlify.com/) page.
  </Tab>
</Tabs>
