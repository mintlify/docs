## Make your first edit

<Tabs>
  <Tab title="Web editor">
    1. Open **Editor** in your dashboard.
    2. Select `index.mdx` and update the title to "Hello World".
    3. Click **Publish** to deploy.

    <Tip>
      Type <kbd>/</kbd> to open the command menu.
    </Tip>
  </Tab>
  
  <Tab title="Code editor">
    Install the CLI:

    <CodeGroup>
      ```bash npm
      npm i -g mint
      ```

      ```bash pnpm
      pnpm add -g mint
      ```
    </CodeGroup>

    Create a project:

    ```bash
    mint new
    ```

    Open `index.mdx` and update the title:

    ```mdx index.mdx {2}
    ---
    title: "Hello World"
    description: "This is the introduction to the documentation"
    ---
    ```

    Preview locally:

    ```bash
    mint dev
    ```

    Push to GitHub to deploy automatically.
  </Tab>
</Tabs>

## Custom domain

Add your domain in [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and configure DNS:

| Record Type | Name | Value | TTL |
|-------------|------|-------|-----|
| CNAME | docs | cname.vercel-dns.com | 3600 |
