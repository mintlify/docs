<Info>
  **Prerequisite**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

After onboarding, your site deploys automatically to `https://<your-project-name>.mintlify.app`. Find your URL in the [dashboard](https://dashboard.mintlify.com/).

## Connect GitHub

Install the GitHub App to enable automatic deployments:

1. Go to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your repositories.

## Make your first edit

Choose your preferred workflow:

<Tabs>
  <Tab title="Web editor">
    1. Select **Editor** in your dashboard sidebar.
    2. Open `index.mdx` and change the title.
    3. Select **Publish** to deploy.
  </Tab>
  <Tab title="Local development">
    Install the CLI (requires [Node.js](https://nodejs.org/en) v20.17.0+):

    ```bash
    npm i -g mint
    ```

    Clone your repo, edit files, and preview locally:

    ```bash
    mint dev
    ```

    Push changes to deploy automatically.
  </Tab>
</Tabs>

## Add a custom domain

Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard and add a CNAME record:

| Record Type | Name                | Value                |
| ----------- | ------------------- | -------------------- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com |
