## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) to deploy. Connect your GitHub account, select a repository, and install the GitHub App for automatic deployments.

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    Install the CLI with `npm i -g mint` (requires Node.js v20.17.0+). Clone your repository, edit `index.mdx`, and run `mint dev` to preview at `http://localhost:3000`. Push your changes to deploy automatically.
  </Tab>
  <Tab title="Web editor">
    Open the [web editor](https://dashboard.mintlify.com/editor), edit `index.mdx`, and click **Publish**. View deployment status on your [dashboard](https://dashboard.mintlify.com/).
  </Tab>
</Tabs>
