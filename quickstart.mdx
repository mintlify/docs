[Create an account](https://mintlify.com/start) and complete onboarding. Your site deploys to `https://<your-project-name>.mintlify.app`.

## Edit your docs

<Tabs>
  <Tab title="Web editor">
    Open **Editor** in your [dashboard](https://dashboard.mintlify.com), make changes, and click **Publish**.
  </Tab>
  <Tab title="CLI">
    ```bash
    npm i -g mint    # Install CLI (requires Node.js v20.17+)
    mint new         # Create project
    mint dev         # Preview at localhost:3000
    ```
    Push to your repository to deploy.
  </Tab>
</Tabs>
