Mintlify is docs-as-code: every page is a file in your <Tooltip tip="Your documentation's source code where all files and their history are stored.">repository</Tooltip>, edited locally or in the web editor and synced to Git.

<Tip>
  Using an AI coding tool? Add the Mintlify [skill](/ai/skillmd) and [MCP server](/ai/model-context-protocol) for better results.
</Tip>

<Prompt description="Install the Mintlify skill and MCP server" actions={["copy", "cursor"]}>
Install the Mintlify skill to get context on Mintlify project structure, components, and documentation best practices:

npx skills add https://mintlify.com/docs

Then add the Mintlify MCP server for access to documentation search. Follow the setup instructions at https://www.mintlify.com/docs/ai/model-context-protocol.md
</Prompt>

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and complete onboarding: connect GitHub, pick a repo, and install the GitHub App. Your site deploys to `https://<your-project>.mintlify.app` — find the exact URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

<AccordionGroup>
  <Accordion title="Skip Git during onboarding">
    Skip the Git connection to start in the web editor immediately — Mintlify provisions a private repo for you. Migrate to your own repo later from [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings). See [Clone to your own repository](/deploy/github#clone-to-your-own-repository).
  </Accordion>
</AccordionGroup>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0\+ (LTS recommended). See [Install the CLI](/cli/install) for troubleshooting.

        <CodeGroup>

        ```bash npm
        npm i -g mint
        ```

        ```bash pnpm
        pnpm add -g mint
        ```

        </CodeGroup>
      </Step>
      <Step title="Clone and edit">
        [Clone your repository](/cli/install#clone-your-repository), then update the description in `index.mdx`:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview and push">
        Preview at `http://localhost:3000`, then push to deploy:

        ```bash
        mint dev
        git add . && git commit -m "Update description" && git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit a page">
        Open the [web editor](https://dashboard.mintlify.com/editor) and update the **Introduction** description.

        <Frame>
          <img
            alt="Introduction page open in the web editor with the description edited to say Hello world!."
            className="block dark:hidden"
            src="/images/quickstart/hello-world-light.png"
          />
          <img
            alt="Introduction page open in the web editor with the description edited to say Hello world!."
            className="hidden dark:block"
            src="/images/quickstart/hello-world-dark.png"
          />
        </Frame>
      </Step>
      <Step title="Publish">
        Click **Publish** in the toolbar. Track deployment status on the [Overview](https://dashboard.mintlify.com/) page.
      </Step>
    </Steps>
  </Tab>
</Tabs>
