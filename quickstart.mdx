[Create an account](https://mintlify.com/start) and complete onboarding. Your site deploys automatically to `https://<your-project-name>.mintlify.app`.

## Connect GitHub

1. Go to **Settings** > **GitHub App** and install the app.
2. Go to **Settings** > **My Profile** and authorize your account.

## Choose your workflow

Pick the editing workflow that fits your team:

### Code-based workflow

<Info>
  **Prerequisite**: Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.
</Info>

Install the CLI:

<CodeGroup>

```bash npm
npm i -g mint
```

```bash pnpm
pnpm add -g mint
```

</CodeGroup>

Create a project:

```bash
mint new
```

Edit your files (example: update `index.mdx`):

```mdx index.mdx
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

Preview locally:

```bash
mint dev
```

Push to GitHub to deploy automatically.

### Web editor workflow

Access the editor:

1. Go to your [dashboard](https://dashboard.mintlify.com).
2. Select **Editor** in the sidebar.

Edit your files in the visual interface, then select **Publish** to deploy.

<Tip>
  Type <kbd>/</kbd> to open the command menu and access formatting tools.
</Tip>

## Add a custom domain

Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and enter your domain (e.g., `docs.yourcompany.com`).

Add this DNS record with your domain provider:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>
