Deploy a live documentation site in minutes and make your first edit.

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and connect your GitHub account. Mintlify will create a repository and deploy your site to `https://<your-project-name>.mintlify.app`.

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first edit

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor) in your dashboard
    2. Edit the **Introduction** page description
    3. Click **Publish** to deploy your changes
  </Tab>
  <Tab title="CLI">
    1. Install the CLI: `npm i -g mint`
    2. Clone your repository and open `index.mdx`
    3. Update the description in the frontmatter
    4. Run `mint dev` to preview at `http://localhost:3000`
    5. Push your changes: `git add . && git commit -m "Update description" && git push`
  </Tab>
</Tabs>
