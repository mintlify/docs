Get your documentation site live in minutes.

## Deploy your documentation site

1. Go to [mintlify.com/start](https://mintlify.com/start)
2. Connect your GitHub account
3. Create or select a repository for your documentation

Your site is deployed at `https://<your-project-name>.mintlify.app`. Find the exact URL on your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="Web editor">
    1. Go to the [web editor](https://dashboard.mintlify.com/editor) in your dashboard
    2. Open `index.mdx` and update the description:

    ```mdx
    ---
    title: "Introduction"
    description: "Your custom description here"
    ---
    ```

    3. Click **Publish** to deploy your changes
  </Tab>
  <Tab title="CLI">
    1. Install the CLI: `npm i -g mint` (requires [Node.js](https://nodejs.org/en) v20.17.0+)
    2. Clone your repository locally
    3. Edit `index.mdx` and update the description
    4. Preview: `mint dev`
    5. Deploy: `git add . && git commit -m "Update description" && git push`
  </Tab>
</Tabs>
