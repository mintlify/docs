## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding process. Connect your GitHub account and select a repository for your documentation.

Your site deploys automatically at `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor) in your dashboard
    2. Edit the **Introduction** page
    3. Click **Publish**
    4. View your changes live at your site URL
  </Tab>
  <Tab title="CLI">
    Install the CLI (requires [Node.js](https://nodejs.org/en) v20.17.0+):

    ```bash
    npm i -g mint
    ```

    Clone your repository, edit `index.mdx`, and preview locally:

    ```bash
    mint dev
    ```

    Push your changes to deploy:

    ```bash
    git add .
    git commit -m "Update description"
    git push
    ```
  </Tab>
</Tabs>
