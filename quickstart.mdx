## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and complete onboarding. You'll connect your GitHub account, select a repository, and install the GitHub App.

Your site deploys at `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/).

<Tip>
  Have a public GitHub repository? [Auto-generate your docs](https://mintlify.com/explore) by replacing `github.com` with `mintlify.com` in any repo URL.
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    Install the CLI (requires Node.js v20.17.0+):

    ```bash
    npm i -g mint
    ```

    Clone your repo, edit `index.mdx`, then preview and deploy:

    ```bash
    mint dev                    # Preview at localhost:3000
    git add . && git commit -m "Update" && git push
    ```
  </Tab>
  <Tab title="Web editor">
    Open the [web editor](https://dashboard.mintlify.com/editor), edit any page, and click **Publish**.
  </Tab>
</Tabs>
