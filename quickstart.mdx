## Your documentation site

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL in the [dashboard](https://dashboard.mintlify.com/).

## Make your first edit

Choose your preferred workflow:

<Tabs>
  <Tab title="Web editor">
    1. Open **Editor** in your [dashboard](https://dashboard.mintlify.com).
    2. Select `index.mdx` in the file explorer.
    3. Update the title to "Hello World".
    4. Select **Publish**.

    <Tip>
      Type <kbd>/</kbd> in the editor to access formatting tools and components.
    </Tip>
  </Tab>
  <Tab title="Code-based">
    Install the CLI:

    ```bash
    npm i -g mint
    ```

    Create a project and preview locally:

    ```bash
    mint new
    mint dev
    ```

    Edit `index.mdx`, then push to deploy:

    ```mdx index.mdx
    ---
    title: "Hello World"
    ---
    ```
  </Tab>
</Tabs>

## Add a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard and add a CNAME record:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
