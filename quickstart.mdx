Get your documentation site up and running quickly.

<Info>
**Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Your site is live

After onboarding, your documentation automatically deploys to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the [dashboard](https://dashboard.mintlify.com/) Overview page.

## Install the CLI

Install the Command Line Interface to work locally:

<CodeGroup>
  ```bash npm
  npm i -g mint
  ```

  ```bash pnpm
  pnpm add -g mint
  ```
</CodeGroup>

## Edit your documentation

1. Open your repository from onboarding
2. Edit `index.mdx` and update the title:

```mdx index.mdx
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

## Preview locally

Run this command to preview changes:

```bash
mint dev
```

Your preview will be available at `localhost:3000`.

## Deploy changes

Push your changes to your repository. Mintlify automatically builds and deploys your updates.

## Add a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard and add your domain (e.g., `docs.yourcompany.com`).

Configure DNS with your provider:

<Table>
| Record Type | Name | Value | TTL |
|-------------|------|-------|-----|
| CNAME | docs | cname.mintlify.app | 3600 |
</Table>
