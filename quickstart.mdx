After you complete this guide, you'll have a live documentation site ready to customize and update.

## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding. Connect your GitHub account, select a repository, and install the GitHub App to enable automatic deployments.

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Install with npm or pnpm (requires Node.js v20.17.0+):

        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Edit and preview">
        Open `index.mdx` and update the description. Run `mint dev` to preview at `http://localhost:3000`.
      </Step>
      <Step title="Deploy">
        Push your changes to trigger automatic deployment:

        ```bash
        git add . && git commit -m "Update description" && git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit">
        Open `index.mdx` in the [web editor](https://dashboard.mintlify.com/editor) and update the description.
      </Step>
      <Step title="Publish">
        Click **Publish** to deploy your changes.
      </Step>
    </Steps>
  </Tab>
</Tabs>
