## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding process. You'll connect your GitHub account, create or select a repository, and install the GitHub App for automatic deployments.

After onboarding, your site deploys at `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

<Tip>
  Have a public GitHub repository? [Auto-generate your docs](https://mintlify.com/explore) by replacing `github.com` with `mintlify.com` in the URL.
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+.

        ```bash npm
        npm i -g mint
        ```
      </Step>
      <Step title="Edit and preview">
        Clone your repo, edit `index.mdx`, then run `mint dev` to preview at `http://localhost:3000`.
      </Step>
      <Step title="Deploy">
        ```bash
        git add . && git commit -m "Update docs" && git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit">
        Open the [web editor](https://dashboard.mintlify.com/editor) and edit any page.
      </Step>
      <Step title="Publish">
        Click **Publish** in the toolbar.
      </Step>
    </Steps>
  </Tab>
</Tabs>
