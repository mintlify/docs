## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) to deploy your site. You'll connect GitHub, select a repository, and install the GitHub App.

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor)
    2. Edit the **Introduction** page description
    3. Click **Publish**
  </Tab>
  <Tab title="CLI">
    1. Install the CLI: `npm i -g mint`
    2. Clone your repository
    3. Edit `index.mdx`
    4. Preview with `mint dev`
    5. Push your changes: `git push`
  </Tab>
</Tabs>
