[Create an account](https://mintlify.com/start) to deploy your site.

## Edit your docs

**CLI**: `npm i -g mint` then `mint dev` to preview locally. Push to GitHub to deploy.

**Web**: Open [Editor](https://dashboard.mintlify.com), make changes, select **Publish**.
