[Create an account](https://mintlify.com/start) to deploy your site at `https://<project>.mintlify.app`.

## Edit your docs

<Tabs>
  <Tab title="CLI">
    ```bash
    npm i -g mint    # Install CLI (requires Node.js 20.17+)
    mint new         # Create project
    mint dev         # Preview at localhost:3000
    ```
    Push to GitHub to deploy.
  </Tab>
  <Tab title="Web editor">
    Open **Editor** in your [dashboard](https://dashboard.mintlify.com), make changes, and select **Publish**.
  </Tab>
</Tabs>

## Custom domain

Add a CNAME record pointing your subdomain to `cname.vercel-dns.com` in [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain).
