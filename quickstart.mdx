Deploy a live documentation site in minutes.

## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) and complete onboarding. Connect your GitHub account, select a repository, and install the GitHub App.

Your site deploys automatically at `https://<your-project-name>.mintlify.app`.

<Tip>
  Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.

        <CodeGroup>
        ```bash npm
        npm i -g mint
        ```

        ```bash pnpm
        pnpm add -g mint
        ```
        </CodeGroup>
      </Step>

      <Step title="Clone your repository">
        Follow [Clone your repository](/installation#clone-your-repository).
      </Step>

      <Step title="Edit a page">
        Open `index.mdx` and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>

      <Step title="Preview locally">
        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>

      <Step title="Push your changes">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        View deployment status on your [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>

  <Tab title="Web editor">
    <Steps>
      <Step title="Open the web editor">
        Navigate to the [web editor](https://dashboard.mintlify.com/editor).
      </Step>
      <Step title="Edit a page">
        Open `index.mdx` and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Publish">
        Click **Publish** in the top-right toolbar.
      </Step>
      <Step title="View live">
        Check deployment status on your [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
</Tabs>
