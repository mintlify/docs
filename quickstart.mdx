Deploy a live documentation site and make your first change.

## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding process. Connect your GitHub account, select a repository, and install the GitHub App to enable automatic deployments.

Your site deploys to `https://<your-project-name>.mintlify.app`.

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Install the CLI with npm or pnpm. Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.

        <CodeGroup>

        ```bash npm
        npm i -g mint
        ```

        
        ```bash pnpm
        pnpm add -g mint
        ```

        </CodeGroup>
      </Step>
      <Step title="Clone and edit">
        Clone your repository, then open `index.mdx` and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        Run `mint dev` from your documentation directory and view at `http://localhost:3000`.
      </Step>
      <Step title="Deploy">
        Push your changes to deploy:

        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit a page">
        Open the [web editor](https://dashboard.mintlify.com/editor), select `index.mdx`, and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Publish">
        Click **Publish** in the top-right toolbar. View deployment status on your [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
</Tabs>
