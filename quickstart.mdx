## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account and create your documentation repository. Your site deploys automatically at `https://<your-project-name>.mintlify.app`.

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    Install the CLI and preview changes locally:

    ```bash npm
    npm i -g mint
    ```

    Edit `index.mdx` and update the description:

    ```mdx
    ---
    title: "Introduction"
    description: "Your custom description here"
    ---
    ```

    Preview locally with `mint dev`, then push your changes:

    ```bash
    git add .
    git commit -m "Update description"
    git push
    ```
  </Tab>
  <Tab title="Web editor">
    Open the [web editor](https://dashboard.mintlify.com/editor), edit `index.mdx`, and click **Publish** to deploy your changes.
  </Tab>
</Tabs>
