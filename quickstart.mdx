## Getting started

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/) Overview page.

### Install the GitHub App

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your organization and repositories.

### Authorize your GitHub account

Navigate to **Settings** > **My Profile** and select **Authorize GitHub account**.

## Code-based workflow

### Install the CLI

<Info>
  **Prerequisite**: Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher through v24.
</Info>

<CodeGroup>

```bash npm
npm i -g mint
```

```bash pnpm
pnpm add -g mint
```

</CodeGroup>

### Create and edit

Run `mint new` to create a project. Edit files like `index.mdx`:

```mdx index.mdx {2}
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

### Preview and deploy

Run `mint dev` to preview at `localhost:3000`. Push changes to your repository to deploy automatically.

## Web editor workflow

### Access and edit

1. Log in to your [dashboard](https://dashboard.mintlify.com) and select **Editor**.
2. Find `index.mdx` in the sidebar and edit the title field.

<Tip>
  Type <kbd>/</kbd> to open the command menu and access formatting tools.
</Tip>

### Publish

Select **Publish** in the top-right corner to deploy changes immediately. See [web editor documentation](/editor) for details on branches and pull requests.

## Adding a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and enter your domain (e.g., `docs.yourcompany.com`). Configure DNS:

<Table>
  | Record Type | Name                | Value                | TTL  |
  | ----------- | ------------------- | -------------------- | ---- |
  | CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
</Table>

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>
