## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) to connect GitHub and deploy your docs. Your site will be live at `<your-project-name>.mintlify.app`.

<Tip>
  Using an AI coding tool? Add the [skill](/ai/skillmd) and [MCP server](/ai/model-context-protocol) for better results.
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+.

        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Edit and preview">
        Clone your repo, edit `index.mdx`, then preview:

        ```bash
        mint dev
        ```
      </Step>
      <Step title="Deploy">
        ```bash
        git add . && git commit -m "Update docs" && git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit">
        Open the [web editor](https://dashboard.mintlify.com/editor) and modify any page.
      </Step>
      <Step title="Publish">
        Click **Publish** in the toolbar.
      </Step>
    </Steps>
  </Tab>
</Tabs>
