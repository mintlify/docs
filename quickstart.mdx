<Info>
  **Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Your site URL

After onboarding, your site deploys to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the [dashboard](https://dashboard.mintlify.com/) Overview page.

## Install the GitHub App

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App**.
3. Choose your organization and repositories to connect.

## Edit your docs

Choose your preferred workflow:

<Tabs>
  <Tab title="Code-based">
    ### Install the CLI

    <Info>
      Requires [Node.js](https://nodejs.org/en) v20.17.0+ through v24.
    </Info>

    ```bash npm
    npm i -g mint
    ```

    ### Create and edit

    ```bash
    mint new    # Create a new project
    mint dev    # Preview at localhost:3000
    ```

    Edit your MDX files, then push to GitHub. Mintlify automatically deploys your changes.
  </Tab>
  <Tab title="Web editor">
    1. Open **Editor** in your [dashboard](https://dashboard.mintlify.com).
    2. Select a file and make your edits.
    3. Select **Publish** to deploy.

    <Tip>
      Type <kbd>/</kbd> in the editor to access formatting tools and components.
    </Tip>
  </Tab>
</Tabs>

## Add a custom domain

1. Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard.
2. Enter your domain (e.g., `docs.yourcompany.com`).
3. Add a CNAME record with your DNS provider:

| Record Type | Name | Value | TTL |
| ----------- | ---- | ----- | --- |
| CNAME | docs | cname.vercel-dns.com | 3600 |

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>
