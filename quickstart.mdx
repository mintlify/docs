## Getting started

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/).

### Install the GitHub App

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your organization and repositories.

### Authorize your GitHub account

Navigate to **Settings** > **My Profile** and select **Authorize GitHub account**.

## Code-based workflow

### Install the CLI

<Info>
  Requires [Node.js](https://nodejs.org/en) v20.17.0+ (LTS preferred).
</Info>

<CodeGroup>

```bash npm
npm i -g mint
```

```bash pnpm
pnpm add -g mint
```

</CodeGroup>

### Edit and preview

1. Run `mint new` to create a project, or clone your existing docs repo.
2. Edit your MDX files (e.g., change the title in `index.mdx`).
3. Run `mint dev` to preview at `localhost:3000`.
4. Push changes to deploy automatically.

## Web editor workflow

1. Open **Editor** in your [dashboard](https://dashboard.mintlify.com).
2. Select a file and make your edits.
3. Select **Publish** to deploy.

<Tip>
  Type <kbd>/</kbd> in the editor to access formatting tools. See the [web editor documentation](/editor) for more details.
</Tip>

## Custom domain

Add a custom domain in [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain). Configure a CNAME record pointing to `cname.vercel-dns.com`. See [custom domain setup](/customize/custom-domain) for details.
