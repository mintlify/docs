After onboarding, your site deploys automatically to `https://<your-project-name>.mintlify.app`. Find your URL in the [dashboard](https://dashboard.mintlify.com/).

## Web editor

1. Open **Editor** in your [dashboard](https://dashboard.mintlify.com).
2. Select a file and make your changes.
3. Click **Publish** to deploy.

<Tip>
  Type <kbd>/</kbd> in the editor to access formatting tools and components.
</Tip>

## Code-based

### Install the CLI

<Info>
  Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.
</Info>

<CodeGroup>

```bash npm
npm i -g mint
```

```bash pnpm
pnpm add -g mint
```

</CodeGroup>

### Create and preview

```bash
mint new      # Create a new project
mint dev      # Preview at localhost:3000
```

### Deploy

Push changes to your GitHub repository. Mintlify automatically builds and deploys your site.

## Add a custom domain

1. Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard.
2. Enter your domain and configure DNS:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>
