## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) to create your documentation site. Connect your GitHub account and select a repository.

Your site deploys automatically to `https://<your-project-name>.mintlify.app`.

## Make your first change

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor)
    2. Edit the **Introduction** page
    3. Click **Publish**
  </Tab>
  <Tab title="CLI">
    Install the CLI:

    ```bash
    npm i -g mint
    ```

    Edit `index.mdx` and preview locally:

    ```bash
    mint dev
    ```

    Push your changes:

    ```bash
    git add .
    git commit -m "Update description"
    git push
    ```
  </Tab>
</Tabs>
