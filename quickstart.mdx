Deploy your documentation site in minutes and make your first edit.

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account and create a repository. The GitHub App enables automatic deployments.

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.

        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Edit a page">
        Open `index.mdx` and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>
      <Step title="Push your changes">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        View deployment status on your [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open the web editor">
        Navigate to the [web editor](https://dashboard.mintlify.com/editor).
      </Step>
      <Step title="Edit and publish">
        Open the **Introduction** page, update the description, and click **Publish**.
      </Step>
      <Step title="View live">
        Check deployment status on your [dashboard](https://dashboard.mintlify.com/), then refresh your site.
      </Step>
    </Steps>
  </Tab>
</Tabs>
