## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and complete onboarding. Connect your GitHub account and select a repository for your documentation.

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor)
    2. Edit `index.mdx` and update the description
    3. Click **Publish**
  </Tab>
  <Tab title="CLI">
    1. Install the CLI: `npm i -g mint`
    2. Clone your repository locally
    3. Edit `index.mdx` and update the description
    4. Preview with `mint dev`
    5. Push your changes: `git add . && git commit -m "Update" && git push`
  </Tab>
</Tabs>
