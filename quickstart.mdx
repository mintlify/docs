## Getting started

After onboarding, your site deploys automatically to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

### Install the GitHub App

1. Navigate to **Settings** in your Mintlify dashboard.
2. Select **GitHub App** from the sidebar.
3. Select **Install GitHub App**.
4. Select the organization and repositories to connect.

## Code-based workflow

### Install the CLI

<Info>
  **Prerequisite**: Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher through v24.
</Info>

Install the CLI:

<CodeGroup>

```bash npm
npm i -g mint
```


```bash pnpm
pnpm add -g mint
```

</CodeGroup>

### Create a new project

Run `mint new` to create a new documentation project. See the [CLI installation guide](/installation#create-a-new-project) for details.

### Edit the documentation

Update the title of the introduction page:

1. Open `index.mdx`:

```mdx index.mdx
---
title: "Introduction"
description: "This is the introduction to the documentation"
---
```

2. Update the `title` field to `"Hello World"`:

```mdx index.mdx {2}
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

### Preview the changes

Run the following command:

```bash
mint dev
```

Your preview is available at `localhost:3000`.

Push your changes to your repository. Mintlify automatically builds and deploys the updates. Monitor deployment status in your GitHub commit history or the [dashboard](https://dashboard.mintlify.com).

## Adding a custom domain

Navigate to the [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) page in your dashboard.

<Frame>
  <img
    src="/images/quickstart/custom-domain-light.png"
    alt="Custom Domain"
    className="block dark:hidden"
  />

  <img
    src="/images/quickstart/custom-domain-dark.png"
    alt="Custom Domain"
    className="hidden dark:block"
  />
</Frame>

Enter your domain (for example, `docs.yourcompany.com`) and configure DNS settings:

<Table>
  | Record Type | Name                | Value                | TTL  |
  | ----------- | ------------------- | -------------------- | ---- |
  | CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
</Table>

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Local preview not working">
    Make sure you have Node.js v20.17.0 or higher installed and run `mint dev` from the directory containing your `docs.json` file.
  </Accordion>
  <Accordion title="Changes not reflecting on live site">
    Deployment can take a few minutes. Check your GitHub Actions or deployment logs in the Mintlify dashboard for build errors.
  </Accordion>
  <Accordion title="Custom domain not connecting">
    Verify your DNS records are correct and allow time for DNS propagation (up to 48 hours). Use [DNSChecker](https://dnschecker.org) to verify your CNAME record.
  </Accordion>
</AccordionGroup>
