## Getting started

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/) Overview page.

### Install the GitHub App

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your organization and repositories.

### Authorize your GitHub account

1. Navigate to **Settings** > **My Profile**.
2. Select **Authorize GitHub account**.

## Choose your workflow

### Code-based workflow

<Info>
  **Prerequisite**: Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.
</Info>

Install the CLI:

<CodeGroup>

```bash npm
npm i -g mint
```

```bash pnpm
pnpm add -g mint
```

</CodeGroup>

Create a project with `mint new`, then edit your documentation files. For example, update `index.mdx`:

```mdx index.mdx {2}
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

Preview locally with `mint dev` at `localhost:3000`, then push changes to deploy automatically.

### Web editor workflow

1. Log in to your [dashboard](https://dashboard.mintlify.com) and select **Editor**.
2. Navigate to `index.mdx` and update the title field.
3. Select **Publish** to deploy changes immediately.

<Tip>
  Type <kbd>/</kbd> in the editor to access formatting tools and components.
</Tip>

## Adding a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and enter your domain (e.g., `docs.yourcompany.com`). Configure DNS with your provider:

<Table>
  | Record Type | Name                | Value                | TTL  |
  | ----------- | ------------------- | -------------------- | ---- |
  | CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
</Table>

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>
