After you complete this guide, you'll have a live documentation site ready to customize and update.

## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding process. You'll connect your GitHub account, create or select a repository, and install the GitHub App for automatic deployments.

Your documentation site is deployed at `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

<Tip>
  Before sharing with users, you may want to add a [custom domain](/customize/custom-domain).
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Install the CLI with Node.js v20.17.0 or higher:

        ```bash npm
        npm i -g mint
        ```

        See [Install the CLI](/installation) for details.
      </Step>
      <Step title="Clone and edit">
        Clone your repository and open `index.mdx` in your editor. Update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>
      <Step title="Push your changes">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        View deployment status on your [Overview](https://dashboard.mintlify.com/) page.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open the web editor">
        Navigate to the [web editor](https://dashboard.mintlify.com/editor).
      </Step>
      <Step title="Edit and publish">
        Open the **Introduction** page, update the description, and click **Publish**.
      </Step>
      <Step title="View live">
        Check deployment status on your [Overview](https://dashboard.mintlify.com/) page and refresh your site.
      </Step>
    </Steps>
  </Tab>
</Tabs>
