Deploy a live documentation site in minutes.

## Choose your repository

<CardGroup cols={2}>
  <Card title="Mintlify-hosted" icon="cloud">
    Mintlify creates a private GitHub repository for you. Best for getting started quickly.
  </Card>

  <Card title="Your own repository" icon="github">
    Connect your Git repository. Best for local development and existing repos.
  </Card>
</CardGroup>

## Deploy your site

<Tabs>
  <Tab title="Mintlify-hosted">
    <Steps>
      <Step title="Create account">
        Go to [mintlify.com/start](https://mintlify.com/start).
      </Step>
      <Step title="Skip GitHub">
        Click **Skip** when prompted to connect GitHub.
      </Step>
      <Step title="Complete setup">
        Follow onboarding prompts. Your site deploys at `<your-project>.mintlify.app`.
      </Step>
    </Steps>
  </Tab>

  <Tab title="Your own repository">
    <Steps>
      <Step title="Create account">
        Go to [mintlify.com/start](https://mintlify.com/start).
      </Step>
      <Step title="Connect GitHub">
        Authenticate your GitHub account.
      </Step>
      <Step title="Install GitHub App">
        Install the [GitHub App](https://dashboard.mintlify.com/api/github/install-app) for automatic deployments.
      </Step>
      <Step title="Select repository">
        Choose your repository on [Git settings](https://dashboard.mintlify.com/settings/deployment/git-settings). It must contain a valid `docs.json` file.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Make your first change

<Steps>
  <Step title="Open editor">
    Go to the [web editor](https://dashboard.mintlify.com/editor).
  </Step>
  <Step title="Edit content">
    Open `index.mdx` and update the description:
    
    ```mdx
    ---
    title: "Introduction"
    description: "Your custom description here"
    ---
    ```
  </Step>
  <Step title="Publish">
    Click **Publish** in the toolbar.
  </Step>
</Steps>
