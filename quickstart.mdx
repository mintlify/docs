Deploy a live documentation site in minutes.

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) and connect your GitHub account. Select or create a repository for your documentation and install the GitHub App.

Your site deploys automatically to `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

<Tip>
  **Public GitHub repo?** Replace `github.com` with `mintlify.com` in your repository URL to [auto-generate your site](https://mintlify.com/explore).
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.

        <CodeGroup>

        ```bash npm
        npm i -g mint
        ```

        
        ```bash pnpm
        pnpm add -g mint
        ```

        </CodeGroup>
      </Step>
      <Step title="Clone and edit">
        Clone your repository and open `index.mdx`. Update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>
      <Step title="Deploy">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        Your changes deploy automatically.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open the editor">
        Go to the [web editor](https://dashboard.mintlify.com/editor) in your dashboard.
      </Step>
      <Step title="Edit and publish">
        Open the **Introduction** page, update the description, and click **Publish**.
      </Step>
    </Steps>
  </Tab>
</Tabs>
