Complete this guide to deploy a live documentation site ready to customize.

## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) and connect your GitHub account. Select or create a repository and install the GitHub App for automatic deployments.

Your site deploys to `<your-project-name>.mintlify.app`.

## View your deployed site

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+.

        <CodeGroup>

        ```bash npm
        npm i -g mint
        ```

        
        ```bash pnpm
        pnpm add -g mint
        ```

        </CodeGroup>
      </Step>
      <Step title="Clone your repository">
        Clone your repository locally. See [Clone your repository](/installation#clone-your-repository).
      </Step>
      <Step title="Edit a page">
        Open `index.mdx` and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>
      <Step title="Push your changes">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        View deployment status on your [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open the web editor">
        Go to the [web editor](https://dashboard.mintlify.com/editor).
      </Step>
      <Step title="Edit a page">
        Open **Introduction** and update the description.

        <Frame>
          <img
            src="/images/quickstart/hello-world-light.png"
            alt="Introduction page open in the web editor with the description edited to say Hello world!."
            className="block dark:hidden"
          />

          <img
            src="/images/quickstart/hello-world-dark.png"
            alt="Introduction page open in the web editor with the description edited to say Hello world!."
            className="hidden dark:block"
          />
        </Frame>
      </Step>
      <Step title="Publish">
        Click **Publish** in the top-right.
      </Step>
      <Step title="View live">
        Check deployment status on your [dashboard](https://dashboard.mintlify.com/). Refresh your site when complete.
      </Step>
    </Steps>
  </Tab>
</Tabs>
