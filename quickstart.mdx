[Create an account](https://mintlify.com/start) and complete onboarding. Your site deploys to `https://<your-project-name>.mintlify.app`.

## Edit your docs

**Web editor**: Open **Editor** in your [dashboard](https://dashboard.mintlify.com), make changes, and click **Publish**.

**Locally**: Install the CLI and push changes to GitHub:

```bash
npm i -g mint   # Requires Node.js v20.17+
mint dev        # Preview at localhost:3000
```
