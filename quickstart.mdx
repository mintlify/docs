Get your documentation site up and running in minutes.

<Info>
**Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Your documentation site

After onboarding, your documentation automatically deploys to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

## Code-based workflow

### Install the CLI

<CodeGroup>
  ```bash npm
  npm i -g mint
  ```

  ```bash pnpm
  pnpm add -g mint
  ```
</CodeGroup>

### Edit and preview

1. Open your repository and edit any `.mdx` file
2. Run `mint dev` to preview at `localhost:3000`
3. Push changes to deploy automatically

## Web editor workflow

1. Open the [Editor](https://dashboard.mintlify.com/editor) in your dashboard
2. Select any file to edit
3. Click **Publish** to deploy your changes

<Tip>
Type <kbd>/</kbd> in the editor to access formatting tools and components.
</Tip>

## Add a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and add your domain (e.g., `docs.yourcompany.com`).

Configure your DNS with a CNAME record:

| Record Type | Name | Value | TTL |
|-------------|------|-------|-----|
| CNAME | docs | cname.mintlify.app | 3600 |
