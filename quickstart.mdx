<Info>
  **Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/) Overview page.

## Install the GitHub App

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your organization and repositories.

## Code-based workflow

<Info>
  **Prerequisite**: [Node.js](https://nodejs.org/en) v20.17.0+ (LTS preferred).
</Info>

### Install the CLI

<CodeGroup>
```bash npm
npm i -g mint
```

```bash pnpm
pnpm add -g mint
```
</CodeGroup>

### Create and edit

```bash
mint new    # Create a new project
mint dev    # Preview at localhost:3000
```

Edit your MDX files, then push to GitHub. Mintlify automatically deploys your changes.

## Web editor workflow

1. Open **Editor** in your [dashboard](https://dashboard.mintlify.com).
2. Select a file and make your edits.
3. Select **Publish** to deploy.

<Tip>
  Type <kbd>/</kbd> in the editor to access formatting tools and components.
</Tip>

## Add a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and add your domain (e.g., `docs.yourcompany.com`).

Configure your DNS:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
