After [signing up](https://mintlify.com/start), your site deploys to `https://<your-project-name>.mintlify.app`.

## Edit your docs

Use the [web editor](https://dashboard.mintlify.com) or edit locally with the CLI:

```bash
npm i -g mint && mint dev
```

Push to your repository to deploy.
