## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account and create your documentation repository. After onboarding, your site is deployed at `https://<your-project-name>.mintlify.app`.

## Make your first change

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor)
    2. Edit `index.mdx` and update the description
    3. Click **Publish**
  </Tab>
  <Tab title="CLI">
    1. Install the CLI: `npm i -g mint`
    2. Clone your repository and edit `index.mdx`
    3. Preview with `mint dev`
    4. Push your changes to deploy
  </Tab>
</Tabs>
