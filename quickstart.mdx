After you complete this guide, you'll have a live documentation site ready to customize.

## Before you begin

Mintlify uses a docs-as-code approach — every page on your site has a corresponding file in your documentation <Tooltip tip="Your documentation's source code where all files and their history are stored.">repository</Tooltip>. Connect your repository to work locally or in the web editor, with changes synced automatically.

## Deploy your site

<Tip>
  If your code is already in a public GitHub repository, [auto-generate your docs](https://mintlify.com/explore) by replacing `github.com` with `mintlify.com` in the URL.
</Tip>

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account, select a repository, and install the GitHub App. Your site deploys automatically and is accessible at `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+. See [Install the CLI](/cli/install) for details.

        <CodeGroup>

        ```bash npm
        npm i -g mint
        ```

        ```bash pnpm
        pnpm add -g mint
        ```

        </CodeGroup>
      </Step>
      <Step title="Clone and edit">
        [Clone your repository](/cli/install#clone-your-repository), then open `index.mdx` and update the frontmatter:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview and push">
        Preview locally with `mint dev` at `http://localhost:3000`, then push:

        ```bash
        git add . && git commit -m "Update description" && git push
        ```

        Mintlify deploys automatically. Check status on your [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit a page">
        Open the [web editor](https://dashboard.mintlify.com/editor), select the **Introduction** page, and update the description.

        <Frame>
          <img
            src="/images/quickstart/hello-world-light.png"
            alt="Introduction page open in the web editor with the description edited."
            className="block dark:hidden"
          />

          <img
            src="/images/quickstart/hello-world-dark.png"
            alt="Introduction page open in the web editor with the description edited."
            className="hidden dark:block"
          />
        </Frame>
      </Step>
      <Step title="Publish">
        Click **Publish** in the toolbar. View deployment status on the [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
</Tabs>
