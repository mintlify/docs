</Info>

## Your documentation site

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/) Overview page.

## Connect GitHub

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your organization and repositories.

<Info>
  You may also need to authorize your GitHub account under **Settings** > **My Profile**.
</Info>

## Make your first edit

Choose your preferred workflow:

<Tabs>
  <Tab title="Code-based">
    ### Install the CLI

    <Info>
      Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.
    </Info>

    ```bash npm
    npm i -g mint
    ```

    ### Edit and preview

    1. Open `index.mdx` in your repository.
    2. Change the `title` field to `"Hello World"`.
    3. Run `mint dev` to preview at `localhost:3000`.
    4. Push your changes to deploy.
  </Tab>
  <Tab title="Web editor">
    1. Select **Editor** in your [dashboard](https://dashboard.mintlify.com).
    2. Open `index.mdx` from the file explorer.
    3. Update the title to "Hello World".
    4. Select **Publish** to deploy.

    <Tip>
      Type <kbd>/</kbd> in the editor to access formatting tools and components.
    </Tip>
  </Tab>
</Tabs>

## Add a custom domain

1. Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard.
2. Enter your domain (e.g., `docs.yourcompany.com`).
3. Add a CNAME record pointing to `cname.vercel-dns.com`.
