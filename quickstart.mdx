[Create an account](https://mintlify.com/start) and complete onboarding. Your site deploys automatically to `https://<your-project-name>.mintlify.app`.

## Connect GitHub

1. Go to **Settings** > **GitHub App** and install the app.
2. Go to **Settings** > **My Profile** and authorize your account.

## Edit your docs

**Code editor**: Install the CLI with `npm i -g mint`, run `mint dev` to preview, and push to GitHub to deploy.

**Web editor**: Go to **Editor** in your dashboard, make changes, and select **Publish**.

## Add a custom domain

Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and enter your domain (e.g., `docs.yourcompany.com`).

Add this DNS record with your domain provider:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>
