<Info>
  **Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Getting started

After onboarding, your site deploys automatically to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

### Install the GitHub App

1. Navigate to **Settings** in your dashboard.
2. Select **GitHub App** from the sidebar.
3. Select **Install GitHub App**.
4. Select the organization and repositories to connect.

## Code-based workflow

### Install the CLI

<Info>
  **Prerequisite**: Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher through v24.
</Info>

Install the CLI:

<CodeGroup>

```bash npm
npm i -g mint
```


```bash pnpm
pnpm add -g mint
```

</CodeGroup>

### Create a new project

Run `mint new` to create a new documentation project.

### Edit the documentation

Update the title in `index.mdx`:

```mdx index.mdx {2}
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

### Preview the changes

Run:

```bash
mint dev
```

Preview at `localhost:3000`.

Push to your repository. Mintlify automatically builds and deploys your updates. Monitor deployment in your GitHub commit history or [dashboard](https://dashboard.mintlify.com).

## Adding a custom domain

Navigate to the [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) page in your dashboard.

<Frame>
  <img
    src="/images/quickstart/custom-domain-light.png"
    alt="Custom Domain"
    className="block dark:hidden"
  />

  <img
    src="/images/quickstart/custom-domain-dark.png"
    alt="Custom Domain"
    className="hidden dark:block"
  />
</Frame>

Enter your domain (e.g., `docs.yourcompany.com`) and configure DNS settings:

<Table>
  | Record Type | Name                | Value                | TTL  |
  | ----------- | ------------------- | -------------------- | ---- |
  | CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
</Table>

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Local preview not working">
    Ensure Node.js v20.17.0+ is installed and run `mint dev` from the directory containing `docs.json`.
  </Accordion>
  <Accordion title="Changes not reflecting on live site">
    Deployment takes a few minutes. Check GitHub Actions or dashboard logs for build errors.
  </Accordion>
  <Accordion title="Custom domain not connecting">
    Verify DNS records and allow time for propagation. Use [DNSChecker](https://dnschecker.org) to verify your CNAME.
  </Accordion>
</AccordionGroup>
