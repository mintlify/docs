<Info>

</Info>

## Your documentation site

After onboarding, your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL in the [dashboard](https://dashboard.mintlify.com/).

## Connect GitHub

Install the GitHub App to automate deployments:

1. Go to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your repositories.

## Make your first edit

Choose your preferred workflow:

<Tabs>
  <Tab title="Web editor">
    1. Select **Editor** in your dashboard sidebar.
    2. Open `index.mdx` and update the title.
    3. Select **Publish** to deploy your changes.

    <Tip>
      Type <kbd>/</kbd> in the editor to access formatting tools and components.
    </Tip>
  </Tab>
  <Tab title="Code-based">
    Install the CLI ([Node.js](https://nodejs.org/en) v20.17.0+ required):

    ```bash
    npm i -g mint
    ```

    Preview locally:

    ```bash
    mint dev
    ```

    Push changes to your repository to trigger deployment.
  </Tab>
</Tabs>

## Add a custom domain

Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard and add a CNAME record:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
