[Create an account](https://mintlify.com/start) and complete onboarding to get your live documentation site at `https://<your-project-name>.mintlify.app`.

## Install the GitHub App

1. Go to **Settings** in your [dashboard](https://dashboard.mintlify.com/).
2. Select **GitHub App** from the sidebar.
3. Select **Install GitHub App** and choose your repositories.

## Make your first edit

Choose your preferred workflow:

### Web editor

1. Open the **Editor** in your dashboard.
2. Select `index.mdx` in the file explorer.
3. Update the title field to "Hello World".
4. Select **Publish** to deploy your changes.

### Local development

Install the CLI:

<CodeGroup>
  ```bash npm
  npm i -g mint
  ```

  ```bash pnpm
  pnpm add -g mint
  ```
</CodeGroup>

Edit your docs:

1. Run `mint new` to create a project.
2. Open `index.mdx` and update the title to "Hello World".
3. Run `mint dev` to preview at `localhost:3000`.
4. Push your changes to deploy automatically.

## Add a custom domain

Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) and add a CNAME record:

| Record Type | Name | Value | TTL |
|-------------|------|-------|-----|
| CNAME | docs | cname.vercel-dns.com | 3600 |
