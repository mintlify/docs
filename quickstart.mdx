<Info>
**Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Your documentation site

After onboarding, your site deploys automatically to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

## Install the GitHub App

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your organization.
3. Select the repositories to connect.

## Edit your documentation

Choose your preferred workflow:

<Tabs>
  <Tab title="Web editor">
    1. Select **Editor** in your [dashboard](https://dashboard.mintlify.com).
    2. Open `index.mdx` and make your changes.
    3. Select **Publish** to deploy.

    <Tip>Type <kbd>/</kbd> in the editor to access formatting tools and components.</Tip>
  </Tab>
  <Tab title="Code-based">
    Install the CLI (requires [Node.js](https://nodejs.org/en) v19+):

    ```bash
    npm i -g mint
    ```

    Create a new project:

    ```bash
    mint new
    ```

    Preview locally:

    ```bash
    mint dev
    ```

    Push changes to your repository to deploy automatically.
  </Tab>
</Tabs>

## Add a custom domain

1. Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard.
2. Enter your domain and configure DNS:

| Record Type | Name | Value | TTL |
|-------------|------|-------|-----|
| CNAME | docs (or subdomain) | cname.vercel-dns.com | 3600 |

<Info>DNS changes can take up to 48 hours to propagate.</Info>
