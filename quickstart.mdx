## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) to deploy. Connect your GitHub account and select a repository. Your site deploys to `https://<your-project-name>.mintlify.app`.

## Make your first change

<Tabs>
  <Tab title="Web editor">
    1. Open the [web editor](https://dashboard.mintlify.com/editor)
    2. Edit `index.mdx` and update the description
    3. Click **Publish**
  </Tab>
  <Tab title="CLI">
    1. Install: `npm i -g mint`
    2. Clone your repository
    3. Edit `index.mdx`
    4. Preview: `mint dev`
    5. Push changes: `git push`
  </Tab>
</Tabs>
