## Your site is live

After onboarding, your site deploys automatically to `https://<your-project-name>.mintlify.app`. Find your URL in the [dashboard](https://dashboard.mintlify.com/).

## Make your first edit

Choose your preferred workflow:

<Tabs>
  <Tab title="Web editor">
    1. Go to **Editor** in your [dashboard](https://dashboard.mintlify.com).
    2. Select a file and make your changes.
    3. Click **Publish**.

    <Tip>
      Type <kbd>/</kbd> in the editor to access formatting tools and components.
    </Tip>
  </Tab>
  <Tab title="Code-based">
    Install the CLI:

    ```bash
    npm i -g mint
    ```

    Preview locally:

    ```bash
    mint dev
    ```

    Push changes to your repository to deploy automatically.
  </Tab>
</Tabs>

## Add a custom domain

Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard and add a CNAME record:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |
