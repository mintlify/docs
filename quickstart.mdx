## Getting started

After onboarding, your docs deploy automatically to `https://<your-project-name>.mintlify.app`. Find your URL in the [dashboard](https://dashboard.mintlify.com/).

### Install the GitHub App

The GitHub App automates deployment when you push changes.

1. Go to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your repositories.

## Edit your docs

<Tabs>
  <Tab title="Web editor">
    1. Open **Editor** in your [dashboard](https://dashboard.mintlify.com).
    2. Select a file, make changes, and click **Publish**.

    <Tip>Type <kbd>/</kbd> to access formatting tools and components.</Tip>

    See the [web editor documentation](/editor) for more details.
  </Tab>
  <Tab title="Local development">
    Install the CLI ([Node.js](https://nodejs.org/en) v20.17.0+ required):

    ```bash
    npm i -g mint
    ```

    Preview locally:

    ```bash
    mint dev
    ```

    Push changes to your repository to deploy automatically.
  </Tab>
</Tabs>

## Add a custom domain

1. Go to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard.
2. Enter your domain and add a CNAME record pointing to `cname.vercel-dns.com`.

DNS changes can take up to 48 hours to propagate.
