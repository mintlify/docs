## Hark! A Prologue to Thy Documentation Journey

*When thou hast read this verse complete,*
*A living site shall be thy feat,*
*Ready to mold and customize,*
*Beneath the vast and starry skies.*

## Ere Thou Begin'st This Noble Quest

Fair Mintlify doth embrace the way
Of docs-as-code, both night and day.
Each page that graces thy fair site
Hath correspondence, stored in <Tooltip tip="Your documentation's source code where all files and their history are stored. The web editor connects to your documentation repository to access and modify content, or you can edit files locally in your preferred IDE.">repository's</Tooltip> light.

When thou dost bind thy repository
To Mintlify's deployment, verily,
Thou may'st work local or in editor's grace,
And sync thy changes to the remote place.

## The Summoning of Thy Documentation Realm

Venture forth to [mintlify.com/start](https://mintlify.com/start) with haste,
And let the onboarding flow be traced.
Connect thy GitHub with solemn vow,
Create or choose repository now,
Install the GitHub App with care,
That automatic deployments fill the air.

When onboarding's task is truly done,
Thy documentation site's begun!
Accessible at `.mintlify.app` 'twill be,
A domain of knowledge, wild and free.

## Craft Thy First Alteration Most Fair

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        *The CLI doth crave [Node.js](https://nodejs.org/en) most fine,*
        *Version twenty point seventeen, or higher in line.*
        *An LTS version for stability's sake.*

        <CodeGroup>

        ```bash npm
        npm i -g mint
        ```


        ```bash pnpm
        pnpm add -g mint
        ```

        </CodeGroup>

        *For details full and troubleshooting's art,*
        *See [Install the CLI](/installation), from the start.*
      </Step>
      <Step title="Clone your repository">
        *If thou hast not yet claimed thy repository near,*
        *Follow [Clone your repository](/installation#clone-your-repository) clear.*
      </Step>
      <Step title="Edit a page">
        *Open `index.mdx` in thy editor preferred,*
        *And change the description, as wisdom has stirred:*

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        *From thy documentation's sacred ground,*
        *Let this command make preview sound:*

        ```bash
        mint dev
        ```

        *Behold thy preview at `http://localhost:3000`, where wonders are found.*
      </Step>
      <Step title="Push your changes">
        *Commit and push thy alterations grand,*
        *That deployment may be at hand:*

        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        *Mintlify shall deploy with speed automatic,*
        *View thy status on [Overview](https://dashboard.mintlify.com/), most dramatic!*
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open the web editor">
        *Venture to the [web editor](https://dashboard.mintlify.com/editor) bright,*
        *Within thy dashboard's guiding light.*
      </Step>
      <Step title="Edit a page">
        *Open `index.mdx` and with thy quill so true,*
        *Update the frontmatter's descriptive view:*

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Publish">
        *Press **Publish** button, placed top-right with care,*
        *Upon the editor's toolbar there.*
      </Step>
      <Step title="View live">
        *On [Overview](https://dashboard.mintlify.com/) page of thy dashboard fair,*
        *Thy deployment's status thou may'st compare.*
        *When finished deploying, refresh with glee,*
        *Thy changes live upon thy site to see!*
      </Step>
    </Steps>
  </Tab>
</Tabs>
