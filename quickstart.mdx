After this guide, you'll have a live documentation site ready to customize.

## Deploy your site

<Tip>
  For public GitHub repositories, [auto-generate your docs](https://mintlify.com/explore) by replacing `github.com` with `mintlify.com` in the URL.
</Tip>

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account and create a documentation repository. Your site deploys automatically at `https://<project-name>.mintlify.app`.

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+.

        ```bash
        npm i -g mint
        ```

        See [Install the CLI](/installation) for details.
      </Step>
      <Step title="Edit and preview">
        Open `index.mdx`, update the description, then run:

        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>
      <Step title="Deploy">
        ```bash
        git add . && git commit -m "Update description" && git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit">
        Open the [web editor](https://dashboard.mintlify.com/editor) and update the **Introduction** page.
      </Step>
      <Step title="Publish">
        Click **Publish** in the toolbar.
      </Step>
    </Steps>
  </Tab>
</Tabs>
