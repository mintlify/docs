After completing this guide, you will have a live documentation site ready to customize.

<Info>
  **Prerequisites**: [Create an account](https://mintlify.com/start) and complete onboarding.
</Info>

## Your documentation site

After onboarding, your site automatically deploys to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

## Install the GitHub App

The GitHub App automates deployment when you push changes.

1. Navigate to **Settings** in your dashboard.
2. Select **GitHub App** from the sidebar.
3. Select **Install GitHub App** and follow the prompts.

## Code-based workflow

### Install the CLI

<Info>
  Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.
</Info>

<CodeGroup>

```bash npm
npm i -g mint
```

```bash pnpm
pnpm add -g mint
```

</CodeGroup>

### Edit and preview

1. Open `index.mdx` in your documentation repository.
2. Update the `title` field to `"Hello World"`.
3. Run `mint dev` to preview at `localhost:3000`.

### Deploy

Push your changes to GitHub. Mintlify automatically builds and deploys your site.

## Web editor workflow

1. Log in to your [dashboard](https://dashboard.mintlify.com).
2. Select **Editor** on the left sidebar.
3. Edit any file and select **Publish** to deploy.

<Tip>
  Type <kbd>/</kbd> in the editor to access formatting tools and components.
</Tip>

## Add a custom domain

1. Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard.
2. Enter your domain (e.g., `docs.yourcompany.com`).
3. Add a CNAME record pointing to `cname.vercel-dns.com`.
