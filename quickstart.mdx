Get your documentation site live in minutes using Mintlify's docs-as-code approach.

## Deploy your site

1. Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding
2. Connect your GitHub account and select a repository
3. Your site deploys automatically at `https://<your-project-name>.mintlify.app`

Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+.

        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Edit and preview">
        Clone your repo, edit `index.mdx`, then run:

        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>
      <Step title="Deploy">
        ```bash
        git add . && git commit -m "Update docs" && git push
        ```

        Changes deploy automatically.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open editor">
        Go to the [web editor](https://dashboard.mintlify.com/editor).
      </Step>
      <Step title="Edit and publish">
        Make your changes, then click **Publish** in the toolbar.
      </Step>
    </Steps>
  </Tab>
</Tabs>
