## Deploy your documentation site

Go to [mintlify.com/start](https://mintlify.com/start) and complete the onboarding process. Connect your GitHub account, create or select a repository, and install the GitHub App to enable automatic deployments.

Your site deploys at `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.

        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Edit a page">
        Open `index.mdx` and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview locally">
        ```bash
        mint dev
        ```

        View at `http://localhost:3000`.
      </Step>
      <Step title="Push your changes">
        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        View deployment status on your [dashboard](https://dashboard.mintlify.com/).
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit a page">
        Open `index.mdx` in the [web editor](https://dashboard.mintlify.com/editor) and update the description:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Publish">
        Click **Publish** in the top-right toolbar, then refresh your site to see changes live.
      </Step>
    </Steps>
  </Tab>
</Tabs>
