Deploy a live documentation site and make your first edit.

## Deploy your site

Go to [mintlify.com/start](https://mintlify.com/start) to connect your GitHub account and create or select a repository. The GitHub App enables automatic deployments.

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the **Overview** page of your [dashboard](https://dashboard.mintlify.com/).

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.

        <CodeGroup>

        ```bash npm
        npm i -g mint
        ```

        
        ```bash pnpm
        pnpm add -g mint
        ```

        </CodeGroup>
      </Step>
      <Step title="Edit a page">
        Clone your repository and open `index.mdx`:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview and deploy">
        Preview locally with `mint dev`, then push your changes:

        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Edit a page">
        Open the [web editor](https://dashboard.mintlify.com/editor), select the **Introduction** page, and update the description.
      </Step>
      <Step title="Publish">
        Click **Publish** in the toolbar. Check deployment status on the [Overview](https://dashboard.mintlify.com/) page.
      </Step>
    </Steps>
  </Tab>
</Tabs>
