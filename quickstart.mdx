## Your documentation site

After onboarding, your site deploys automatically to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the [dashboard](https://dashboard.mintlify.com/) Overview page.

## Connect GitHub

Install the GitHub App to automate deployments when you push changes:

1. Navigate to **Settings** > **GitHub App** in your dashboard.
2. Select **Install GitHub App** and choose your organization.
3. Select the repositories to connect.

<Info>
  You may also need to authorize your GitHub account under **Settings** > **My Profile**.
</Info>

## Edit your documentation

Choose your preferred workflow:

<Tabs>
  <Tab title="Code-based">
    ### Install the CLI

    <Info>
      Requires [Node.js](https://nodejs.org/en) v20.17.0 or higher.
    </Info>

    ```bash npm
    npm i -g mint
    ```

    ### Create and edit

    Run `mint new` to create a project, then edit your MDX files:

    ```mdx index.mdx
    ---
    title: "Hello World"
    description: "This is the introduction to the documentation"
    ---
    ```

    ### Preview locally

    ```bash
    mint dev
    ```

    Your preview is available at `localhost:3000`.

    ### Deploy

    Push to your repository. Mintlify automatically builds and deploys your changes.
  </Tab>
  <Tab title="Web editor">
    ### Access the editor

    Select **Editor** in your [dashboard](https://dashboard.mintlify.com) sidebar.

    ### Edit and publish

    1. Select a file from the sidebar.
    2. Make your changes in the visual editor.
    3. Select **Publish** to deploy.

    <Tip>
      Type <kbd>/</kbd> in the editor to access formatting tools and components.
    </Tip>
  </Tab>
</Tabs>

## Add a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard and add a CNAME record:

| Record Type | Name                | Value                | TTL  |
| ----------- | ------------------- | -------------------- | ---- |
| CNAME       | docs (or subdomain) | cname.vercel-dns.com | 3600 |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Local preview not working">
    Ensure Node.js v20.17.0+ is installed and run `mint dev` from the directory containing `docs.json`.
  </Accordion>
  <Accordion title="Changes not reflecting on live site">
    Check deployment logs in your dashboard for build errors. Deployment can take a few minutes.
  </Accordion>
  <Accordion title="Custom domain not connecting">
    Verify DNS records and allow up to 48 hours for propagation. Use [DNSChecker](https://dnschecker.org) to verify.
  </Accordion>
</AccordionGroup>
