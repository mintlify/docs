[Create an account](https://mintlify.com/start) to deploy your site to `https://<your-project-name>.mintlify.app`.

## Edit your docs

**Web editor**: Open **Editor** in your [dashboard](https://dashboard.mintlify.com), make changes, and click **Publish**.

**Locally**: Install the CLI and run `mint dev` to preview at `localhost:3000`. Push to deploy.

```bash
npm i -g mint
```

## Add a custom domain

In [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain), add a CNAME record pointing your subdomain to `cname.vercel-dns.com`.
