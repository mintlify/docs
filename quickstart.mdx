[Create an account](https://mintlify.com/start) and complete onboarding to get started.

## Your documentation site

After onboarding, your site deploys to:

```
https://<your-project-name>.mintlify.app
```

Find your URL on the Overview page of your [dashboard](https://dashboard.mintlify.com/).

## Install the GitHub App

1. Navigate to **Settings** in your Mintlify dashboard.
2. Select **GitHub App** from the sidebar.
3. Select **Install GitHub App**.
4. Select the organization and repositories to connect.

## Web editor

### Edit your documentation

1. Log in to your [dashboard](https://dashboard.mintlify.com).
2. Select **Editor** in the sidebar.
3. Select `index.mdx` in the file explorer.
4. Update the title field to "Hello World".

<Tip>
  Type <kbd>/</kbd> to open the command menu and access formatting tools.
</Tip>

### Publish changes

Select **Publish** in the top-right corner to deploy your changes.

## Code editor

### Install the CLI

Install the Mintlify CLI to preview changes locally:

<CodeGroup>
  ```bash npm
  npm i -g mint
  ```

  ```bash pnpm
  pnpm add -g mint
  ```
</CodeGroup>

<Info>
  Requires [Node.js](https://nodejs.org/en) v19 or higher.
</Info>

### Create a project

Run `mint new` to create a new documentation project.

### Edit your documentation

1. Open `index.mdx` in your code editor.
2. Update the title field to "Hello World".

```mdx index.mdx {2}
---
title: "Hello World"
description: "This is the introduction to the documentation"
---
```

### Preview changes

Run `mint dev` to preview at `localhost:3000`.

### Publish changes

Push your changes to GitHub. Mintlify automatically builds and deploys your site.

## Add a custom domain

Navigate to [Domain Setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) in your dashboard and add your domain (for example, `docs.yourcompany.com`).

Configure your DNS with a CNAME record:

| Record Type | Name | Value | TTL |
|-------------|------|-------|-----|
| CNAME | docs | cname.vercel-dns.com | 3600 |

<Info>
  DNS changes can take up to 48 hours to propagate.
</Info>
