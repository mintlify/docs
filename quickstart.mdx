Get a live documentation site ready to customize in minutes.

## Deploy your site

1. Go to [mintlify.com/start](https://mintlify.com/start)
2. Connect your GitHub account and select a repository
3. Install the GitHub App for automatic deployments

Your site deploys to `https://<your-project-name>.mintlify.app`. Find your URL on the [dashboard](https://dashboard.mintlify.com/).

<Tip>
  Using an AI coding tool? Add the Mintlify [skill](/ai/skillmd) and [MCP server](/ai/model-context-protocol) for better results.
</Tip>

## Make your first change

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Install the CLI">
        Requires [Node.js](https://nodejs.org/en) v20.17.0+.

        ```bash
        npm i -g mint
        ```
      </Step>
      <Step title="Clone and edit">
        Clone your repository and open `index.mdx`. Update the frontmatter:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>
      <Step title="Preview and push">
        ```bash
        mint dev          # Preview at localhost:3000
        git add . && git commit -m "Update description" && git push
        ```

        Mintlify automatically deploys your changes.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Web editor">
    <Steps>
      <Step title="Open editor">
        Go to the [web editor](https://dashboard.mintlify.com/editor).
      </Step>
      <Step title="Edit and publish">
        Update the **Introduction** page description and click **Publish**.
      </Step>
    </Steps>
  </Tab>
</Tabs>
