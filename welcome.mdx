Mintlify is an AI-native test platform that helps developers and teams ship beautiful, fast, and intelligent docs. This page expands on the [Introduction](/index) with more context on what Mintlify is, who it's built for, and how to get the most out of it.

## What is Mintlify?

Mintlify combines a modern documentation framework with AI-native tooling so that your docs are easy to write, easy to read, and easy for AI agents to consume. You author content in MDX, configure your site in `docs.json`, and Mintlify handles the rest — rendering, search, navigation, API playgrounds, analytics, and AI assistance.

A Mintlify site includes:

- **Beautiful defaults** — production-ready themes, typography, and components out of the box.
- **MDX content** — Markdown with JSX, so you can mix prose with interactive React components.
- **Interactive API playgrounds** — generated from your OpenAPI or AsyncAPI specs.
- **AI-native experiences** — an in-product assistant, an agent that can edit your docs, an MCP server, and `llms.txt` support for downstream AI tools.
- **Smart search** — fast, typo-tolerant search across your entire site.

## How a Mintlify site is structured

Every Mintlify site has the same core pieces:

1. **`docs.json`** — the single source of truth for navigation, theme, integrations, and site-wide settings. See the [docs.json schema reference](/organize/settings-reference).
2. **MDX pages** — one file per page, with YAML frontmatter for title, description, and metadata. Learn more in [Pages](/organize/pages).
3. **Navigation** — groups, tabs, anchors, and dropdowns defined in `docs.json` and reflected in your sidebar. See [Navigation](/organize/navigation).
4. **Assets** — images, videos, and other files served from your repo. See [Files](/create/files).

## Ways to author content

You can work with Mintlify in whichever workflow fits your team:

- **Git-based** — edit MDX files in your repo, preview locally with the [Mintlify CLI](/cli/index), and deploy on push.
- **Web editor** — make changes in the browser with a visual [editor](/editor/index) and open pull requests without leaving the page.
- **AI agent** — ask the [agent](/agent/index) to draft, refactor, or restructure pages for you.

## AI-native by default

Mintlify treats AI as a first-class reader and a first-class collaborator:

- The [assistant](/ai-native) answers reader questions grounded in your docs.
- The agent can edit pages, update navigation, and open PRs on your behalf.
- The [Admin MCP server](/ai/mintlify-mcp) gives tools like Claude and Cursor write access to your content and settings.
- `llms.txt` and `llms-full.txt` make your docs easy for other AI tools to ingest.
