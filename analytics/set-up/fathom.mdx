---
title: "Fathom"
---

Add the following to your `mint.json` file to send analytics to Fathom.

You can get the siteId from your script settings.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Example
"analytics": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
