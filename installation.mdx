Preview your documentation locally as you write and edit. The CLI also includes utilities for maintaining your documentation.

## Install the CLI

<Info>
  **Prerequisite**: [Node.js](https://nodejs.org/en) v20.17.0 or higher (LTS versions preferred).
</Info>

<CodeGroup>
  ```bash npm
  npm i -g mint
  ```

  ```bash pnpm
  pnpm add -g mint
  ```
</CodeGroup>

## Preview locally

Navigate to your documentation directory and run:

```bash
mint dev
```

Your documentation is available at `http://localhost:3000`.

To run without installing globally:

```bash
npx mint dev
```

### Custom ports

Use the `--port` flag to customize the port:

```bash
mint dev --port 3333
```

### Preview as a specific group

Preview as a specific authentication group:

```bash
mint dev --groups admin
```

## Create a new project

```bash
mint new [directory]
```

This clones the [starter kit](https://github.com/mintlify/starter) and prompts you for a project name and [theme](/customize/themes).

<Warning>
  Using `--force` or overwriting the current directory will delete existing files.
</Warning>

Available flags:
* `--theme`: Set the theme
* `--name`: Set the project name
* `--force`: Overwrite the current directory

Example:

```bash
mint new docs --name my-project --theme linden
```
## Update the CLI

```bash
mint update
```

If `mint update` is not available, reinstall:

<CodeGroup>
  ```bash npm
  npm i -g mint@latest
  ```

  ```bash pnpm
  pnpm add -g mint@latest
  ```
</CodeGroup>

## Additional commands

### Find broken links

```bash
mint broken-links
```

### Find accessibility issues

```bash
mint a11y
```

### Check OpenAPI spec

```bash
mint openapi-check <OpenAPI filename or URL>
```

### Rename files

```bash
mint rename <path/to/old-filename> <path/to/new-filename>
```

### Migrate MDX endpoint pages

```bash
mint migrate-mdx
```

See [Migrating from MDX](/guides/migrating-from-mdx) for details.

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module'>
    1. `npm uninstall -g mint`
    2. Upgrade Node.js
    3. `npm install -g mint`
  </Accordion>
  <Accordion title="Unknown error">
    Delete the `~/.mintlify` folder and run `mint dev` again.
  </Accordion>
  <Accordion title="Permission denied">
    Run `sudo npm i -g mint` and enter your password.
  </Accordion>
  <Accordion title="Local preview looks different from production">
    Run `mint update`.
  </Accordion>
  <Accordion title="mintlify vs. mint package">
    If you have both `mint` and `mintlify` installed, uninstall `mintlify`:
    
    ```bash
    npm uninstall -g mintlify
    npm cache clean --force
    npm i -g mint
    ```
  </Accordion>
</AccordionGroup>
