## Prerequisites

- A Mintlify account
- A GitHub repository connected to your project

## Create your first page

<Steps>
  <Step title="Create an MDX file">
    Add a new file called `hello-world.mdx` to your docs repository:

    ```mdx
    ---
    title: Hello World
    description: My first Mintlify page.
    ---

    Welcome to my documentation!
    ```
  </Step>
  <Step title="Add it to navigation">
    Open your `docs.json` file and add the page to a navigation group:

    ```json
    {
      "group": "Get started",
      "pages": ["hello-world"]
    }
    ```
  </Step>
  <Step title="Push and deploy">
    Commit your changes and push to your repository. Mintlify automatically deploys on push.

    ```bash
    git add hello-world.mdx docs.json
    git commit -m "Add hello world page"
    git push
    ```
  </Step>
  <Step title="Verify">
    Visit your documentation site and confirm the new page appears in the navigation.
  </Step>
</Steps>
