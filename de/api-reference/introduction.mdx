Die Mintlify REST API ermöglicht es Ihnen, programmgesteuert mit Ihrer Dokumentation zu interagieren, Updates anzustoßen und KI-gestützte Chat-Erlebnisse einzubetten.

<div id="endpoints">
  ## Endpunkte
</div>

- [Update auslösen](/de/api-reference/update/trigger): Lösen Sie bei Bedarf ein Update Ihrer Website aus.
- [Update-Status abrufen](/de/api-reference/update/status): Rufen Sie den Status eines Updates und weitere Details zu Ihrer Dokumentation ab.
- [Assistentennachricht erstellen](/de/api-reference/assistant/create-assistant-message): Betten Sie den auf Ihrer Dokumentation trainierten Assistenten in eine Anwendung Ihrer Wahl ein.
- [Dokumentation durchsuchen](/de/api-reference/assistant/search): Durchsuchen Sie Ihre Dokumentation.

<div id="authentication">
  ## Authentifizierung
</div>

Sie können einen API-Schlüssel über [das Dashboard](https://dashboard.mintlify.com/settings/organization/api-keys) erstellen. API-Schlüssel sind einer gesamten Organisation zugeordnet und können über mehrere Deployments hinweg verwendet werden.

<div id="admin-api-key">
  ### Admin-API-Schlüssel
</div>

Der Admin-API-Schlüssel wird für die Endpunkte [Update auslösen](/de/api-reference/update/trigger) und [Update-Status abrufen](/de/api-reference/update/status) verwendet.

Admin-API-Schlüssel beginnen mit dem Präfix `mint_`. Halten Sie Ihre Admin-API-Schlüssel geheim.

<div id="assistant-api-key">
  ### Assistant-API-Schlüssel
</div>

Der Assistant-API-Schlüssel wird für die Endpunkte [Assistant-Nachricht generieren](/de/api-reference/assistant/create-assistant-message) und [Dokumentation durchsuchen](/de/api-reference/assistant/search) verwendet.

Assistant-API-Schlüssel beginnen mit dem Präfix `mint_dsc_`.

Der Assistant-API-**Key** ist ein serverseitiges Token, das geheim gehalten werden sollte.

Das Assistant-API-**Token** ist ein öffentliches Token, das in Ihrem Frontend-Code verwendet werden kann.

<Note>
  Aufrufe mit dem Assistant-API-Token können Kosten verursachen – entweder durch Nutzung Ihrer AI-Assistent-Guthaben oder durch Überziehungen.
</Note>
