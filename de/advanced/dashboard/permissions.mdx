---
title: "Editor-Berechtigungen"
description: "Ermöglichen Sie mehr Teammitgliedern, Ihre Dokumentation zu aktualisieren"
---

Ein Editor hat Zugriff auf Ihr Dashboard und den Webeditor.

Jede Person kann zu Ihrer Dokumentation beitragen, indem sie lokal arbeitet und Änderungen in Ihr Repository pusht. Es gibt jedoch wesentliche Unterschiede darin, wie Änderungen bereitgestellt werden:

* **Editor-Änderungen**: Wenn ein Editor über den Webeditor veröffentlicht oder einen Pull Request in Ihr Docs-Repository merged, werden die Änderungen automatisch auf Ihrer Live-Website bereitgestellt.
* **Nicht-Editor-Änderungen**: Wenn eine Person ohne Editorrechte einen Pull Request in Ihr Repository merged, müssen Sie die Bereitstellung manuell über Ihr Dashboard auslösen, damit diese Änderungen auf Ihrer Live-Website erscheinen.

<div id="add-editors">
  ## Editor hinzufügen
</div>

Standardmäßig hat das Teammitglied, das Ihre Mintlify-Organisation erstellt hat, Editorzugriff. Fügen Sie weitere Editorinnen und Editoren auf der Seite [Members](https://dashboard.mintlify.com/settings/organization/members) in Ihrem Dashboard hinzu.

Editor-Sitze werden nutzungsbasiert abgerechnet, und Sie können so viele Editorinnen und Editoren haben, wie Sie benötigen. Weitere Details finden Sie auf unserer [Preisseite](https://mintlify.com/pricing).
