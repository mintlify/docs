Markdown bietet strukturierten Text, den KI-Tools effizienter als HTML verarbeiten können. Das führt zu höherer Antwortgenauigkeit, schnelleren Verarbeitungszeiten und geringerem Tokenverbrauch.

Mintlify generiert automatisch Markdown-Versionen von Seiten, die für KI-Tools und externe Integrationen optimiert sind.

<div id="md-url-extension">
  ## .md-URL-Erweiterung
</div>

Fügen Sie jeder Seiten-URL `.md` hinzu, um eine Markdown-Version anzuzeigen.

<PreviewButton href="https://mintlify.com/docs/ai/markdown-export.md">Diese Seite als Markdown öffnen</PreviewButton>

<div id="keyboard-shortcut">
  ## Tastenkürzel
</div>

Drücken Sie <kbd>Command</kbd> + <kbd>C</kbd> (<kbd>Strg</kbd> + <kbd>C</kbd> unter Windows), um eine Seite als Markdown in die Zwischenablage zu kopieren.
