Die [llms.txt-Datei](https://llmstxt.org) ist ein Industriestandard, der LLMs hilft, Inhalte effizienter zu indexieren – ähnlich wie eine Sitemap Suchmaschinen unterstützt. KI-Tools können diese Datei nutzen, um die Struktur Ihrer Dokumentation zu verstehen und Inhalte zu finden, die für Benutzeranfragen relevant sind.

Mintlify hostet automatisch eine `llms.txt`-Datei im Stammverzeichnis Ihres Projekts, die alle verfügbaren Seiten Ihrer Dokumentation auflistet. Diese Datei ist stets aktuell und erfordert keinerlei Wartung. Optional können Sie eine eigene `llms.txt`-Datei zum Stammverzeichnis Ihres Projekts hinzufügen.

Rufen Sie Ihre `llms.txt` auf, indem Sie `/llms.txt` an die URL Ihrer Dokumentations-Website anhängen.

<div id="llmstxt-structure">
  ## Struktur von llms.txt
</div>

Eine `llms.txt`-Datei ist eine einfache Markdown-Datei, die Folgendes enthält:

- **Seitentitel** als H1-Überschrift.
- **Strukturierte Inhaltsabschnitte** mit Links und einer Beschreibung jeder Seite in Ihrer Dokumentation.

```mdx Example llms.txt
# Example product docs

## Guides
- [Getting started](https://example.com/docs/start): Intro guide
- [Install](https://example.com/docs/install): Setup steps

## Reference
- [API](https://example.com/docs/api): Endpoint list and usage
```

Dieser strukturierte Ansatz ermöglicht es LLMs, Ihre Dokumentation auf hoher Ebene effizient zu verarbeiten und relevante Inhalte für Nutzeranfragen zu finden, wodurch Genauigkeit und Geschwindigkeit KI-gestützter Dokumentationssuchen verbessert werden.

<div id="llms-fulltxt">
  ## llms-full.txt
</div>

Die Datei `llms-full.txt` fasst Ihre gesamte Dokumentationssite in einer einzigen Datei zusammen, die als Kontext für KI-Tools dient, und wird von LLM-Traffic indexiert.

Mintlify hostet automatisch eine `llms-full.txt`-Datei im Stammverzeichnis Ihres Projekts. Sie können Ihre `llms-full.txt` anzeigen, indem Sie `/llms-full.txt` an die URL Ihrer Dokumentationssite anhängen.

<div id="custom-files">
  ## Benutzerdefinierte Dateien
</div>

Um eine benutzerdefinierte `llms.txt`- oder `llms-full.txt`-Datei hinzuzufügen, erstellen Sie eine `llms.txt`- oder `llms-full.txt`-Datei im Stammverzeichnis Ihres Projekts. Das Hinzufügen einer benutzerdefinierten Datei überschreibt die automatisch generierte Datei mit demselben Namen. Wenn Sie eine benutzerdefinierte Datei löschen, wird wieder die Standarddatei verwendet.

Ihre benutzerdefinierte `llms.txt`- oder `llms-full.txt`-Datei muss einen Website-Titel als H1-Überschrift enthalten. Weitere Inhalte sind optional. Siehe [Format](https://llmstxt.org/#format) in der `llms.txt`-Spezifikation für weitere Informationen zu optionalen Abschnitten und Best Practices.
