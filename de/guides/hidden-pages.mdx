---
title: "Versteckte Seiten"
description: "Seiten aus deiner Navigation ausschließen"

---

Versteckte Seiten werden aus der Navigation deiner Website entfernt, bleiben jedoch für alle öffentlich zugänglich, die ihre URL kennen.

Verwende versteckte Seiten für Inhalte, die auf deiner Website verfügbar sein oder als Kontext für KI-Tools dienen sollen, aber nicht über die Navigation auffindbar sein sollen.

Für Inhalte, die eine strikte Zugriffskontrolle erfordern, musst du die [Authentifizierung](/de/authentication-personalization/authentication-setup) konfigurieren.

Wenn du Seiten für bestimmte Nutzergruppen ausblenden möchtest, nutze die Personalisierung, um die [Seitensichtbarkeit](/de/authentication-personalization/overview#page-visibility) zu steuern.

<div id="hiding-a-page">
  ## Eine Seite ausblenden
</div>

Eine Seite ist ausgeblendet, wenn sie nicht in deiner `docs.json`-Navigation enthalten ist. Um eine Seite auszublenden, entferne sie aus deiner Navigationsstruktur.

Ausgeblendete Seiten verwenden dieselbe URL-Struktur wie reguläre Seiten auf Basis ihres Dateipfads. Zum Beispiel wäre `guides/hidden-page.mdx` unter `docs.yoursite.com/guides/hidden-page` erreichbar.

Sieh dir ein [Beispiel für eine ausgeblendete Seite](/de/guides/hidden-page-example) an.

<Note>
  Einige Navigationselemente wie Sidebars, Dropdowns und Tabs können auf ausgeblendeten Seiten leer erscheinen oder das Layout verschieben.
</Note>

<div id="hiding-a-group-of-pages">
  ## Eine Gruppe von Seiten ausblenden
</div>

Eine Gruppe von Seiten wird ausgeblendet, wenn die Eigenschaft `hidden` in deiner `docs.json` auf `true` gesetzt ist:

```json highlight={4}
"groups": [
  {
    "group": "Getting started",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  },
  {
    "group": "Guides",
    "pages": [
      "guides/hidden-page.mdx",
      "guides/hidden-groups.mdx"
    ]
  }
]
```

In diesem Beispiel ist die Gruppe `Getting started` ausgeblendet, die Gruppe `Guides` ist jedoch sichtbar.

<div id="hiding-a-tab">
  ### Einen Tab ausblenden
</div>

Du kannst auch einen Tab ausblenden, indem du die Eigenschaft `hidden` zu deiner `docs.json` hinzufügst:

```json highlight={4}
"tabs": [
  {
    "tab": "Home",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  }
]
```

<div id="search-seo-and-ai-indexing">
  ## Suche, SEO und KI-Indizierung
</div>

Standardmäßig sind versteckte Seiten von der Indizierung für Suchmaschinen, die interne Suche in deinen Docs und als Kontext für den KI-Assistenten ausgeschlossen. Um versteckte Seiten in Suchergebnissen und als Kontext für den Assistenten einzubeziehen, füge die Eigenschaft `seo` zu deiner `docs.json` hinzu:

```json
"seo": {
    "indexing": "all"
}
```

Um eine bestimmte Seite auszuschließen, füge `noindex: "true"` zu ihrem Frontmatter hinzu.
