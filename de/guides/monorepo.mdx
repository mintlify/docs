---
title: "Monorepo-Setup"
description: "Stellen Sie Ihre Doku aus einem Repository bereit, das mehrere Projekte enthält"
icon: "folder-git"
---

Konfigurieren Sie Mintlify so, dass die Dokumentation aus einem bestimmten Verzeichnis innerhalb eines Monorepos bereitgestellt wird. Dieses Setup ermöglicht es Ihnen, die Doku gemeinsam mit Ihrem Code in Repositories zu pflegen, die mehrere Projekte oder Services enthalten.

<div id="prerequisites">
  ## Voraussetzungen
</div>

* Adminzugriff auf Ihr Mintlify-Projekt.
* Dokumentationsdateien, die in einem eigenen Verzeichnis innerhalb Ihres Monorepos organisiert sind.
* Eine gültige `docs.json` in Ihrem Dokumentationsverzeichnis.

<div id="configure-monorepo-deployment">
  ## Monorepo-Bereitstellung konfigurieren
</div>

<Steps>
<Step title="Auf Git-Einstellungen zugreifen">
Navigiere in deinem Dashboard zu [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings).

<Frame>
<img
  className="block dark:hidden my-0 pointer-events-none"
  src="/images/monorepo-light.png"
  alt="Das Projekt-Einstellungsfenster im Menü Git Settings. Der Umschalter Set up as monorepo ist aktiviert und ein Pfad zum Verzeichnis /docs ist angegeben."
/>
<img
  className="hidden dark:block my-0 pointer-events-none"
  src="/images/monorepo-dark.png"
  alt="Das Projekt-Einstellungsfenster im Menü Git Settings. Der Umschalter Set up as monorepo ist aktiviert und ein Pfad zum Verzeichnis /docs ist angegeben."
/>
</Frame>

</Step>
<Step title="Pfad zur Dokumentation festlegen">
1. Aktiviere den Umschalter **Set up as monorepo**.
2. Gib den relativen Pfad zu deinem Docs-Verzeichnis ein. Wenn sich deine Docs beispielsweise im Verzeichnis `docs` befinden, gib `/docs` ein.
<Note>
  Füge am Ende keinen Slash in den Pfad ein.
</Note>
3. Wähle **Save changes**.
</Step>
</Steps>
