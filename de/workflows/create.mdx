<div id="prerequisites">
  ## Voraussetzungen
</div>

- Ein Mintlify-Projekt, das mit einem GitHub- oder GitLab-Repository verbunden ist
- Für GitHub: die [Mintlify GitHub-App](/de/deploy/github), die auf jedem Repository installiert ist, das Sie im Workflow verwenden möchten
- Für GitLab: ein verbundenes GitLab-Konto (siehe [GitLab-Einrichtung](#gitlab-setup) unten)

<div id="create-a-custom-workflow">
  ## Einen benutzerdefinierten Workflow erstellen
</div>

<Note>
  Der Agent läuft in einer isolierten Sandbox mit eingeschränktem Internetzugang. Der Agent kann zur Laufzeit keine zusätzlichen Pakete oder Tools installieren. Prompts, die auf nicht verfügbare Tools verweisen, können zu unerwarteten Ergebnissen führen oder fehlschlagen.
</Note>

1. Öffnen Sie die Seite [Workflows](https://dashboard.mintlify.com/products/workflows?tab=workflows) in Ihrem Dashboard.
2. Klicken Sie auf **Einen benutzerdefinierten Workflow erstellen**.
3. Konfigurieren Sie den Workflow-Namen, wann er ausgeführt wird, die Anweisungen und wie er Aktualisierungen vornimmt.
4. Klicken Sie auf **Workflow erstellen**.

<div id="best-practices">
  ## Bewährte Verfahren
</div>

Wirksame Workflow-Anweisungen konzentrieren sich auf eine einzelne Aufgabe und definieren ein klares, überprüfbares Ergebnis. Workflows weisen einige Variationen auf, da Agenten nicht-deterministisch sind, aber konsistente Anweisungen führen zu konsistenten Ergebnissen.

- **Beschreiben Sie das Ergebnis.** Sagen Sie dem Agenten, wie das Ergebnis aussehen soll. Beschreiben Sie, wie „fertig" aussieht.
- **Schreiben Sie eine Vorgehensweise.** Geben Sie dem Agenten eine Schritt-für-Schritt-Anleitung zum Befolgen.
- **Geben Sie den Kontext an.** Verweisen Sie auf die Dateien, Verzeichnisse oder Repositories, die der Agent verwenden soll.
- **Halten Sie den Umfang begrenzt.** Teilen Sie komplexe Aufgaben auf mehrere Workflows auf, anstatt sie in einem einzigen Prompt zu kombinieren.

<Tip>
  Überprüfen Sie die Ausgabe Ihrer benutzerdefinierten Workflows, um sicherzustellen, dass sie wie erwartet funktionieren. Aktualisieren Sie Ihre Anweisungen, um die Ergebnisse zu verbessern.
</Tip>
