<div id="adding-code-samples">
  ## Hinzufügen von Codebeispielen
</div>

Sie können Inline-Codeausschnitte oder Codeblöcke hinzufügen. Codeblöcke unterstützen Metaoptionen für Syntaxhervorhebung, Titel, Zeilenhervorhebung, Icons und mehr.

<div id="inline-code">
  ### Inline-Code
</div>

Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, schließen Sie es in Backticks (\`) ein.

```mdx
Um ein `Wort` oder eine `Phrase` als Code zu kennzeichnen, schließen Sie es in Backticks (`) ein.
```

<div id="code-blocks">
  ### Codeblöcke
</div>

Verwenden Sie [Fenced Code Blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), indem Sie Code in drei Backticks einschließen. Codeblöcke sind kopierbar, und wenn Sie den Assistenten aktiviert haben, können Nutzer die KI bitten, den Code zu erklären.

Geben Sie die Programmiersprache für die Syntaxhervorhebung an und um Metaoptionen zu aktivieren. Fügen Sie beliebige Metaoptionen, wie einen Titel oder ein Icon, hinter der Sprachangabe hinzu.

```java HelloWorld.java lines icon="java"
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

````mdx
```java HelloWorld.java lines icon="java"

        System.out.println("Hello, World!");
    }

```
````

<div id="code-block-options">
  ## Optionen für Codeblöcke
</div>

Sie können Meta-Optionen zu Ihren Codeblöcken hinzufügen, um deren Darstellung anzupassen.

<Note>
  Sie müssen eine Programmiersprache für einen Codeblock angeben, bevor Sie andere Meta-Optionen hinzufügen.
</Note>

<div id="option-syntax">
  ### Optionssyntax
</div>

- **String- und boolesche Optionen**: Mit `""`, `''` oder ohne Anführungszeichen umschließen.
- **Ausdrucksoptionen**: Mit `{}`, `""` oder `''` umschließen.

```java
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

<div id="expandable">
  ### Ausklappbar
</div>

Ermöglichen Sie Nutzern, lange Codeblöcke mit `expandable` ein- und auszuklappen.

```python Expandable Example expandable
from datetime import datetime, timedelta
from typing import Dict, List, Optional
from dataclasses import dataclass

@dataclass
class Book:
    title: str
    author: str
    isbn: str
    checked_out: bool = False
    due_date: Optional[datetime] = None

class Library:
    def __init__(self):
        self.books: Dict[str, Book] = {}
        self.checkouts: Dict[str, List[str]] = {}  # patron -> list of ISBNs

    def add_book(self, book: Book) -> None:
        if book.isbn in self.books:
            raise ValueError(f"Book with ISBN {book.isbn} already exists")
        self.books[book.isbn] = book

    def checkout_book(self, isbn: str, patron: str, days: int = 14) -> None:
        if patron not in self.checkouts:
            self.checkouts[patron] = []

        book = self.books.get(isbn)
        if not book:
            raise ValueError("Book not found")

        if book.checked_out:
            raise ValueError("Book is already checked out")

        if len(self.checkouts[patron]) >= 3:
            raise ValueError("Patron has reached checkout limit")

        book.checked_out = True
        book.due_date = datetime.now() + timedelta(days=days)
        self.checkouts[patron].append(isbn)

    def return_book(self, isbn: str) -> float:
        book = self.books.get(isbn)
        if not book or not book.checked_out:
            raise ValueError("Book not found or not checked out")

        late_fee = 0.0
        if datetime.now() > book.due_date:
            days_late = (datetime.now() - book.due_date).days
            late_fee = days_late * 0.50

        book.checked_out = False
        book.due_date = None

        # Remove from patron's checkouts
        for patron, books in self.checkouts.items():
            if isbn in books:
                books.remove(isbn)
                break

        return late_fee

    def search(self, query: str) -> List[Book]:
        query = query.lower()
        return [
            book for book in self.books.values()
            if query in book.title.lower() or query in book.author.lower()
        ]

def main():
    library = Library()

    # Add some books
    books = [
        Book("The Hobbit", "J.R.R. Tolkien", "978-0-261-10295-4"),
        Book("1984", "George Orwell", "978-0-452-28423-4"),
    ]

    for book in books:
        library.add_book(book)

    # Checkout and return example
    library.checkout_book("978-0-261-10295-4", "patron123")
    late_fee = library.return_book("978-0-261-10295-4")
    print(f"Late fee: ${late_fee:.2f}")

if __name__ == "__main__":
    main()
```

<div id="diff">
  ### Diff
</div>

Zeige einen visuellen Diff von hinzugefügten oder entfernten Zeilen in deinen Codeblöcken. Hinzugefügte Zeilen sind grün hervorgehoben, entfernte Zeilen rot.

Um Diffs zu erstellen, füge diese speziellen Kommentare am Ende der betreffenden Zeilen in deinem Codeblock hinzu:

- `// [!code ++]`: Markiert eine Zeile als hinzugefügt (grüne Hervorhebung).
- `// [!code --]`: Markiert eine Zeile als entfernt (rote Hervorhebung).

Für mehrere aufeinanderfolgende Zeilen gib die Anzahl nach einem Doppelpunkt an:

- `// [!code ++:3]`: Markiert die aktuelle Zeile plus die nächsten zwei Zeilen als hinzugefügt.
- `// [!code --:5]`: Markiert die aktuelle Zeile plus die nächsten vier Zeilen als entfernt.

Die Kommentar-Syntax muss zu deiner Programmiersprache passen (zum Beispiel `//` für JavaScript oder `#` für Python).

```js Diff Example icon="code" lines
const greeting = "Hello, World!"; // [!code ++]
function sayHello() {
  console.log("Hello, World!"); // [!code --]
  console.log(greeting); // [!code ++]
}
sayHello();
```
