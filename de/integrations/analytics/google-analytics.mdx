---
title: "Google Analytics 4"
---

Sie müssen eine neue <Tooltip tip="Google Analytics 4">GA4</Tooltip>-Property für die Verwendung mit Mintlify erstellen. Die erfassten Daten fließen in dasselbe Projekt wie Ihre anderen Google-Analytics-Daten.

Wenn Sie die ältere Version von Google Analytics, Universal Analytics, verwenden, können Sie trotzdem eine <Tooltip tip="Google Analytics 4">GA4</Tooltip>-Property erstellen. <Tooltip tip="Google Analytics 4">GA4</Tooltip>-Daten unterscheiden sich leicht von UA-Daten, werden aber weiterhin im selben Projekt erfasst.

<div id="how-to-connect-ga4-to-mintlify">
  ## So verbindest du GA4 mit Mintlify
</div>

<div id="create-a-web-stream">
  ### Webdatenstream erstellen
</div>

Du musst einen Webdatenstream erstellen, um die Measurement ID zu erhalten, die du in Mintlify einfügst.

Klicke auf das Zahnrad unten links im Google-Analytics-Bildschirm. Klicke dann auf Data Streams.

<Frame>![](/images/ga4-web-streams.png)</Frame>

Erstelle einen Webdatenstream und gib die URL deiner Mintlify-Dokumentationsseite als Stream-URL ein.

Deine Measurement ID sieht aus wie `G-XXXXXXX` und wird direkt nach dem Erstellen des Webdatenstreams unter Stream Details angezeigt.

<div id="put-measurement-id-in-docsjson">
  ### Measurement ID in docs.json eintragen
</div>

Füge deine Measurement ID in deine Datei `docs.json` wie folgt ein:

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<div id="wait">
  ### Warten
</div>

Google Analytics benötigt zwei bis drei Tage, um deine Daten anzuzeigen.

Du kannst die [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en)-Erweiterung verwenden, um zu prüfen, ob Analytics korrekt aktiviert ist. Die Erweiterung protokolliert in der Konsole deines Browsers jedes Mal, wenn GA4 eine Anfrage stellt.

<Note>

Vorschau-Links haben Analytics deaktiviert.

</Note>
