---
title: "Google Tag Manager"
---

Fügen Sie Ihre Tag-ID in die Datei `docs.json` ein, und wir binden das Google-Tag-Manager-Skript auf all Ihren Seiten ein.

Sie sind selbst dafür verantwortlich, bei Bedarf Cookie-Einwilligungsbanner über den Google Tag Manager einzurichten.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
