---
title: "Plausible"
---

Fügen Sie die Domain Ihrer Website zu `docs.json` hinzu, um Analysedaten an Plausible zu senden.

<Info>

    Geben Sie für Domain oder Server kein `https://` an.

</Info>

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Example
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
