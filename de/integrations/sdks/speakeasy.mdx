Automatisch generierte Code-Snippets aus Speakeasy-SDKs lassen sich direkt in die Mintlify-API-Referenzdokumentation integrieren. SDK-Beispiele werden in der [interaktiven API-Spielwiese](/de/api-playground/overview) von Mintlify-basierten Dokumentationsseiten angezeigt.

<Frame>
  ![Eine Mintlify-API-Spielwiese mit Speakeasy-Code-Snippets.](/images/speakeasy/mintlify-with-speakeasy-openapi.png)
</Frame>

<div id="prerequisites">
  ## Voraussetzungen
</div>

Um Mintlify mit Speakeasy zu integrieren, benötigen Sie Folgendes:

- Ein [Mintlify-Dokumentations-Repository](/de/quickstart#creating-the-repository).
- Ein von Speakeasy generiertes SDK mit einer konfigurierten [automatisierten Codebeispiel-URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls).

<div id="setting-up-the-integration">
  ## Einrichten der Integration
</div>

Um Speakeasy mit Mintlify zu integrieren, müssen Sie die öffentliche URL der kombinierten API-Spezifikation aus der Registry abrufen und Ihre `docs.json`-Konfigurationsdatei aktualisieren.

<div id="get-the-apis-combined-spec-public-url-from-the-registry">
  ### Öffentliche URL der kombinierten API-Spezifikation aus der Registry abrufen
</div>

Navigieren Sie zu Ihrem [Speakeasy-Dashboard](https://app.speakeasy.com) und öffnen Sie den Tab **API Registry**. Öffnen Sie den Eintrag `*-with-code-samples` für die API.

<Frame>
  ![Screenshot der Speakeasy-Seite „API Registry“. Der Tab „API Registry“ ist mit einem roten Quadrat und der Zahl 1 hervorgehoben, und der Eintrag für die API ist mit einem roten Quadrat und der Zahl 2 hervorgehoben.](/images/speakeasy/openapi-registry-and-combined-spec.png)
</Frame>

<Note>
  Wenn der Eintrag nicht mit **Combined Spec** gekennzeichnet ist, stellen Sie sicher, dass für die API eine [Automatic Code Sample URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls) konfiguriert ist.
</Note>

Kopieren Sie auf der Seite des Registry-Eintrags die angegebene öffentliche URL.

<Frame>
  ![Screenshot, der den Registry-Eintrag der kombinierten Spezifikation zeigt, wobei die Funktion „Copy URL“ mit einem roten Quadrat hervorgehoben ist.](/images/speakeasy/copy-combined-spec-url.png)
</Frame>

<div id="update-your-docsjson-configuration-file">
  ### Aktualisieren Sie Ihre `docs.json`-Konfigurationsdatei
</div>

Fügen Sie die URL der kombinierten Spezifikation in einen **Anchors**- oder **Tabs**-Abschnitt Ihrer `docs.json`-Datei ein.

Fügen Sie die URL der kombinierten Spezifikation zu einem Anchor hinzu, indem Sie das Feld `anchor` in Ihrer `docs.json`-Datei wie folgt aktualisieren:

```json docs.json
{
  "anchors": [
    {
      "name": "API Reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL",
      "url": "api-reference",
      "icon": "square-terminal"
    }
  ]
}
```

Fügen Sie die URL der kombinierten Spezifikation zu einem Tab hinzu, indem Sie das Feld `tab` in der Datei `docs.json` wie folgt aktualisieren:

```json docs.json
{
  "tabs": [
    {
      "name": "API Reference",
      "url": "api-reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL"
    }
  ]
}
```

Von Speakeasy generierte Codebeispiele können nun in Ihren API-Dokumenten angezeigt und in der API-Spielwiese verwendet werden.
