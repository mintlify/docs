<div id="enabling-data-privacy-integrations">
  ## Aktivieren von Datenschutzintegrationen
</div>

Sie können Datenschutzplattformen zu Ihren Docs hinzufügen. Fügen Sie das Feld `integrations` mit den entsprechenden Skripten in Ihre `docs.json`-Datei ein.

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

<div id="cookie-consent-and-disabling-telemetry">
  ## Cookie-Einwilligung und Deaktivierung der Telemetrie
</div>

Wenn Sie prüfen müssen, ob ein Nutzer bereits der Verwendung von Cookies im Sinne der DSGVO zugestimmt hat, können Sie unter `cookies` einen Local-Storage-Schlüssel und -wert angeben:

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

Wenn diese Werte gesetzt sind, wird der Local Storage geprüft, um festzustellen, ob der Nutzer der Verwendung von Cookies zugestimmt hat. Falls nicht, wird die Telemetrie deaktiviert.

Wenn Sie die Telemetrie für alle Nutzer deaktivieren möchten, fügen Sie Folgendes zu Ihrer `docs.json`-Datei hinzu:

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
