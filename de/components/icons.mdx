Verwenden Sie Icons aus Font Awesome, Lucide, SVGs, externen URLs oder Dateien in Ihrem Projekt, um Ihre Dokumentation aufzuwerten.

<Icon icon="flag" size={32} />

```mdx Icon example

```

<div id="inline-icons">
  ## Inline-Icons
</div>

Icons werden inline verwendet, wenn sie innerhalb eines Absatzes stehen.

<Icon icon="flag" iconType="solid" /> Die Dokumentation, die Sie brauchen – mühelos.

```markdown Inline-Icon-Beispiel
<Icon icon="flag" iconType="solid" /> Die Dokumentation, die Sie brauchen – mühelos.
```

<div id="properties">
  ## Eigenschaften
</div>

<IconsRequired />

<ResponseField name="color" type="string">
  Die Farbe des Icons als Hex-Code (zum Beispiel `#FF5733`).
</ResponseField>

<ResponseField name="size" type="number">
  Die Größe des Icons in Pixeln.
</ResponseField>
