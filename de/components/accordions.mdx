Akkordeons ermöglichen es Nutzerinnen und Nutzern, Inhaltsbereiche ein- und auszuklappen. Verwenden Sie Akkordeons für progressive Offenlegung und zur Strukturierung von Informationen.

<div id="single-accordion">
  ## Einzelnes Accordion
</div>

<Accordion title="Ich bin ein Accordion.">
  Du kannst hier beliebige Inhalte einfügen, einschließlich anderer Komponenten, zum Beispiel Code:
   ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
</Accordion>

<div id="accordion-groups">
  ## Accordion-Gruppen
</div>

Gruppieren Sie zusammengehörige Accordions mit `<AccordionGroup>`. So entsteht ein zusammenhängender Abschnitt aus Accordions, die jeweils einzeln ein- oder ausgeklappt werden können.

<AccordionGroup>
  <Accordion title="Erste Schritte">
    In Accordions können Sie auch andere Komponenten verwenden.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="Erweiterte Funktionen" icon="bot">
    Fügen Sie Symbole hinzu, um Accordions visuell unterscheidbarer und leichter erfassbar zu machen.
  </Accordion>
  
  <Accordion title="Fehlerbehebung">
    Halten Sie thematisch verwandte Inhalte in Gruppen organisiert.
  </Accordion>
</AccordionGroup>

<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="title" type="string" required>
  Titel in der Akkordeon-Vorschau.
</ResponseField>

<ResponseField name="description" type="string">
  Detail unter dem Titel in der Akkordeon-Vorschau.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Ob das Akkordeon standardmäßig geöffnet ist.
</ResponseField>
