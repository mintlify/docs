Verwenden Sie Cards, um visuelle Container für Inhalte zu erstellen. Cards sind flexible Container, die Text, Symbole, Bilder und Links enthalten können.

<div id="basic-card">
  ## Einfache Karte
</div>

<Card title="Kartentitel" icon="text" href="/de/components/columns">
  So verwendest du eine Karte mit Symbol und Link. Ein Klick auf diese Karte
  führt dich zur Seite „Columns“.
</Card>

```mdx Card example
<Card title="Card title" icon="text" href="/components/columns">
  This is how you use a card with an icon and a link. Clicking on this card
  brings you to the Columns page.
</Card>
```

<div id="card-variations">
  ## Kartenvarianten
</div>

Karten unterstützen mehrere Layout- und Stylingoptionen, um unterschiedliche Inhaltsanforderungen abzudecken.

<div id="horizontal-layout">
  ### Horizontales Layout
</div>

Fügen Sie die Eigenschaft `horizontal` hinzu, um Karten in einem kompakteren, horizontalen Layout darzustellen.

<Card title="Horizontale Karte" icon="text" horizontal>
  Dies ist ein Beispiel für eine horizontale Karte.
</Card>

```mdx Horizontal card example
<Card title="Horizontal card" icon="text" horizontal>
  This is an example of a horizontal card.
</Card>
```

<div id="image-cards">
  ### Bildkarten
</div>

Fügen Sie die Eigenschaft `img` hinzu, um oben auf der Karte ein Bild anzuzeigen.

<Card title="Bildkarte" img="https://mintlify-assets.b-cdn.net/yosemite.jpg">
  Dies ist ein Beispiel für eine Karte mit Bild.
</Card>

```mdx Image card example
<Card title="Image card" img="/images/card-with-image.png">
  This is an example of a card with an image.
</Card>
```

<div id="link-cards-with-custom-ctas">
  ### Linkkarten mit benutzerdefinierten CTAs
</div>

Sie können den Call-to-Action-Text anpassen und festlegen, ob ein Pfeil angezeigt wird. Standardmäßig werden Pfeile nur bei externen Links angezeigt.

<Card title="Linkkarte" icon="link" href="/de/components/columns" arrow="true" cta="Hier klicken">
  Dies ist ein Beispiel für eine Karte mit Symbol und Link. Ein Klick auf diese Karte führt Sie zur Seite Columns.
</Card>

```mdx Link card example
<Card
  title="Link card"
  icon="link"
  href="/components/columns"
  arrow="true"
  cta="Click here"
>
  This is an example of a card with an icon and a link. Clicking on this card brings you to the Columns page.
</Card>
```

<div id="grouping-cards">
  ## Karten gruppieren
</div>

Verwenden Sie die [Columns-Komponente](/de/components/columns), um mehrere Karten nebeneinander anzuordnen.

<Columns cols={2}>
  <Card title="Erste Karte" icon="panel-left-close">
    Dies ist die erste Karte.
  </Card>

  <Card title="Zweite Karte" icon="panel-right-close">
    Dies ist die zweite Karte.
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="First card" icon="panel-left-close">
    This is the first card.
  </Card>
  <Card title="Second card" icon="panel-right-close">
    This is the second card.
  </Card>
</Columns>
```

<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="title" type="string" required>
  Der auf der Karte angezeigte Titel
</ResponseField>

<IconsOptional />

<ResponseField name="color" type="string">
  Symbolfarbe als Hex-Code (z. B. `#FF6B6B`).
</ResponseField>

<ResponseField name="href" type="string">
  URL, zu der navigiert wird, wenn die Karte angeklickt wird.
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  Karte in einem kompakten, horizontalen Layout anzeigen.
</ResponseField>

<ResponseField name="img" type="string">
  URL oder lokaler Pfad zu einem Bild, das oben auf der Karte angezeigt wird.
</ResponseField>

<ResponseField name="cta" type="string">
  Benutzerdefinierter Text für die Schaltfläche.
</ResponseField>

<ResponseField name="arrow" type="boolean">
  Link-Pfeilsymbol ein- oder ausblenden.
</ResponseField>
