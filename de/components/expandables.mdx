---
title: "Aufklapper"
description: "Umschalten, um verschachtelte Eigenschaften anzuzeigen"
icon: "list-tree"
---

Verwenden Sie Aufklapper, um verschachtelte Inhalte innerhalb von Response-Feldern ein- und auszublenden. Aufklapper sind besonders hilfreich, um komplexe Objekteigenschaften in der API-Dokumentation darzustellen.

<ResponseField name="user" type="User Object">
  <Expandable title="properties">

<ResponseField name="full_name" type="string">
  Der vollständige Name des Benutzers
</ResponseField>

<ResponseField name="is_over_21" type="boolean">
  Ob der Benutzer über 21 Jahre alt ist
</ResponseField>

  </Expandable>
</ResponseField>

```mdx Beispiel für einen Aufklapper
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      Der vollständige Name des Benutzers
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Ob der Benutzer über 21 Jahre alt ist
    </ResponseField>
  </Expandable>
</ResponseField>
```

<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="title" type="string">
  Der Name des Objekts, das Sie anzeigen.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Auf `true` setzen, damit das ausklappbare Element beim Laden der Seite geöffnet ist
</ResponseField>
