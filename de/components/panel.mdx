---
title: "Panel"
description: "Legen Sie den Inhalt des rechten Seitenpanels fest"
icon: "panel-right"
---

Sie können die `<Panel>`-Komponente verwenden, um das rechte Seitenpanel einer Seite mit beliebigen Komponenten anzupassen.

Wenn eine Seite eine `<Panel>`-Komponente enthält, müssen alle [RequestExample](/de/components/examples#request-example)- und [ResponseExample](/de/components/examples#response-example)-Komponenten innerhalb von `<Panel>` platziert werden.

Die Komponenten in einem `<Panel>` ersetzen das Inhaltsverzeichnis der Seite.

```mdx
<Panel>
  <Info>Heften Sie Informationen an das Seitenpanel an. Oder fügen Sie eine andere Komponente hinzu.</Info>
</Panel>
```
