  Der neue Editor ist jetzt verfügbar mit Echtzeit-Zusammenarbeit, automatischer Speicherung über Geräte hinweg und verbesserter Synchronisierung. Siehe [Editor](/de/editor) für weitere Informationen.
</Warning>

Mit dem Web-Editor können Sie Dokumentation direkt in Ihrem Browser erstellen, bearbeiten und veröffentlichen.

<div id="editor-modes">
  ## Editor-Modi
</div>

Der Editor verfügt über zwei Modi, zwischen denen Sie über den Umschalter in der Symbolleiste wechseln können.

**Der visuelle Modus** zeigt eine Live-Vorschau Ihrer Inhalte beim Tippen an. Drücken Sie <kbd>/</kbd>, um Komponenten und Medien einzufügen oder Text zu formatieren.

**Der Markdown-Modus** gibt Ihnen direkten Zugriff auf den MDX-Quellcode Ihrer Seiten. Verwenden Sie ihn für die präzise Steuerung von Komponenten-Eigenschaften und Frontmatter.

<div id="publish-changes">
  ## Änderungen veröffentlichen
</div>

Der Branch, an dem Sie arbeiten, bestimmt, wie der Editor Ihre Änderungen veröffentlicht. Wenn Sie Branch-Schutzregeln in Ihrem Repository haben, setzt der Editor diese durch.

- **Deployment-Branch**: Beim Veröffentlichen wird Ihre Live-Website sofort aktualisiert.
- **Feature-Branches**: Beim Veröffentlichen wird ein Pull Request zur Überprüfung erstellt, bevor er auf Ihrer Live-Website bereitgestellt wird.
