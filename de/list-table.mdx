<div id="lists">
  ## Listen
</div>

Listen folgen der offiziellen [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Geordnete Liste
</div>

Um eine geordnete Liste zu erstellen, setzen Sie Zahlen gefolgt von einem Punkt vor die Listeneinträge.

1. Erstes Element
2. Zweites Element
3. Drittes Element
4. Viertes Element

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### Ungeordnete Liste
</div>

Um eine ungeordnete Liste zu erstellen, setzen Sie Bindestriche (`-`), Sternchen (`*`) oder Pluszeichen (`+`) vor die Listeneinträge.

- Erstes Element
- Zweites Element
- Drittes Element
- Viertes Element

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### Verschachtelte Liste
</div>

Rücken Sie Listeneinträge ein, um sie zu verschachteln.

- Erstes Element
- Zweites Element
  - Zusätzliches Element
  - Zusätzliches Element
- Drittes Element

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## Tabellen
</div>

Tabellen folgen der offiziellen [Markdown-Syntax](https://www.markdownguide.org/extended-syntax/#tables).

Um eine Tabelle hinzuzufügen, verwende drei oder mehr Bindestriche (`---`), um die Kopfzeile jeder Spalte zu erstellen, und nutze Pipes (`|`), um die Spalten zu trennen. Für die Kompatibilität solltest du außerdem am Anfang und am Ende jeder Zeile eine Pipe hinzufügen.

| Eigenschaft | Beschreibung                         |
| ----------- | ------------------------------------ |
| Name        | Vollständiger Name des Nutzers       |
| Alter       | Angegebenes Alter                    |
| Beitritt    | Ob der Nutzer der Community beigetreten ist |

```mdx
| Eigenschaft | Beschreibung                         |
| ----------- | ------------------------------------ |
| Name        | Vollständiger Name des Nutzers       |
| Alter       | Angegebenes Alter                    |
| Beitritt    | Ob der Nutzer der Community beigetreten ist |
```
