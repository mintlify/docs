Verwenden Sie CSS, um HTML-Elemente zu gestalten, oder fügen Sie benutzerdefiniertes CSS und JavaScript hinzu, um Erscheinungsbild und Verhalten Ihrer Dokumentation vollständig anzupassen.

<div id="styling-with-tailwind-css">
  ## Styling mit Tailwind CSS
</div>

Verwenden Sie Tailwind CSS v3, um HTML-Elemente zu gestalten. Sie können Layout, Abstände, Farben und andere visuelle Eigenschaften steuern. Einige gängige Klassen sind:

- `w-full` – volle Breite
- `aspect-video` – Seitenverhältnis 16:9
- `rounded-xl` – große abgerundete Ecken
- `block`, `hidden` – Anzeige
- `dark:hidden`, `dark:block` – Sichtbarkeit im Dark Mode

Arbitrary Values in Tailwind CSS werden nicht unterstützt. Für benutzerdefinierte Werte verwenden Sie stattdessen die `style`-Prop.

```html
<img style={{ width: '350px', margin: '12px auto' }} src="/path/image.jpg" />
```

<div id="custom-css">
  ## Benutzerdefiniertes CSS
</div>

Fügen Sie CSS-Dateien zu Ihrem Repository hinzu; deren definierte Klassennamen werden angewendet und stehen in all Ihren `MDX`-Dateien zur Verfügung.

<div id="adding-stylecss">
  ### Hinzufügen von `style.css`
</div>

Sie können beispielsweise die folgende Datei `style.css` hinzufügen, um die Gestaltung der Navbar und des Footers anzupassen.

```css
#navbar {
  background: "#fffff2";
  padding: 1rem;
}

footer {
  margin-top: 2rem;
}
```

<div id="custom-javascript">
  ## Benutzerdefiniertes JavaScript
</div>

Benutzerdefiniertes JS ermöglicht es Ihnen, eigenen ausführbaren Code global hinzuzufügen. Es entspricht dem Einfügen eines `<script>`-Tags mit JS-Code auf jeder Seite.

<div id="adding-custom-javascript">
  ### Benutzerdefiniertes JavaScript hinzufügen
</div>

Jede `.js`-Datei im Content-Verzeichnis Ihrer Dokumentation wird auf jeder Seite der Dokumentation eingebunden. Sie können zum Beispiel die folgende Datei `ga.js` hinzufügen, um [Google Analytics](https://marketingplatform.google.com/about/analytics) in der gesamten Dokumentation zu aktivieren.

```js
window.dataLayer = window.dataLayer || [];
function gtag() {
  dataLayer.push(arguments);
}
gtag('js', new Date());

gtag('config', 'TAG_ID');
```

<Warning>
  Bitte mit Vorsicht verwenden, um keine Sicherheitslücken zu verursachen.
</Warning>
