Füge deiner Dokumentation Bilder hinzu, bette Videos ein und binde interaktive Inhalte mit iframes ein.

Alle statischen Assets in deinem Docs-Repository werden automatisch unter dem entsprechenden Pfad auf deiner Domain bereitgestellt. Wenn du zum Beispiel `/images/my-logo.png` in deinem Repo hast, ist das Bild unter `https://docs.yoursite.com/images/my-logo.png` verfügbar.

<div id="images">
  ## Bilder
</div>

Fügen Sie Bilder hinzu, um Ihrer Dokumentation visuellen Kontext, Beispiele oder dekorative Elemente hinzuzufügen.

<div id="basic-image-syntax">
  ### Grundlegende Bildsyntax
</div>

Verwenden Sie die [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#images), um Ihrer Dokumentation Bilder hinzuzufügen:

```mdx
![Alt text describing the image](/path/to/image.png)
```

<Tip>
  Fügen Sie immer aussagekräftigen Alternativtext (Alt-Text) hinzu, um Barrierefreiheit und SEO zu verbessern. Der Alt-Text sollte klar beschreiben, was auf dem Bild zu sehen ist.
</Tip>

Bilddateien müssen kleiner als 20 MB sein. Größere Dateien sollten Sie über einen CDN-Dienst wie [Amazon S3](https://aws.amazon.com/s3) oder [Cloudinary](https://cloudinary.com) hosten.

<div id="html-image-embeds">
  ### HTML-Bildeinbettungen
</div>

Für mehr Kontrolle über die Bilddarstellung verwenden Sie HTML-`<img>`-Tags:

```html
<img 
  src="/images/dashboard.png" 
  alt="Main dashboard interface"
  height="300"
  className="rounded-lg"
/>
```

<div id="disable-zoom-functionality">
  #### Zoom-Funktion deaktivieren
</div>

Um den standardmäßigen Zoom beim Klicken zu deaktivieren, fügen Sie die Eigenschaft `noZoom` hinzu:

```html highlight="4"
<img 
  src="/images/screenshot.png" 
  alt="Descriptive alt text"
  noZoom 
  height="200"
/>
```

<div id="link-images">
  #### Bilder verlinken
</div>

Um ein Bild anklickbar zu machen, umschließen Sie es mit einem Anker-Tag und fügen Sie die Eigenschaft `noZoom` hinzu:

```html
<a href="https://mintlify.com" target="_blank">
  <img 
    src="/images/logo.png" 
    alt="Mintlify logo"
    noZoom 
    height="100"
  />
</a>
```

<Note>
  Bilder innerhalb von Anker-Tags zeigen automatisch einen Zeiger-Cursor an, der ihre Klickbarkeit signalisiert.
</Note>

<div id="light-and-dark-mode-images">
  #### Bilder für Light- und Dark-Mode
</div>

Um unterschiedliche Bilder für helle und dunkle Themes anzuzeigen, verwenden Sie Tailwind-CSS-Klassen:

```html
<!-- Light mode image -->
<img 
  className="block dark:hidden" 
  src="/images/light-mode.png" 
  alt="Light mode interface"
/>

<!-- Dark mode image -->
<img 
  className="hidden dark:block" 
  src="/images/dark-mode.png" 
  alt="Dark mode interface"
/>
```

<div id="videos">
  ## Videos
</div>

Mintlify unterstützt [HTML-Tags in Markdown](https://www.markdownguide.org/basic-syntax/#html) und bietet Ihnen die Flexibilität, reichhaltige Inhalte zu erstellen.

<Tip>
  Fügen Sie innerhalb von Videoelementen immer Fallback-Text für Browser ein, die keine Videowiedergabe unterstützen.
</Tip>

<div id="youtube-embeds">
  ### YouTube-Einbettungen
</div>

Binden Sie YouTube-Videos mit iframe-Elementen ein:

```html
<iframe
  className="w-full aspect-video rounded-xl"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>
```

<div id="self-hosted-videos">
  ### Selbst gehostete Videos
</div>

Verwenden Sie das HTML-Element `<video>` für selbst gehostete Videoinhalte:

```html
<video
  controls
  className="w-full aspect-video rounded-xl"
  src="link-to-your-video.com"
></video>
```

<div id="autoplay-videos">
  ### Automatische Wiedergabe von Videos
</div>

Um ein Video automatisch abzuspielen, verwenden Sie:

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video rounded-xl"
  src="/videos/demo.mp4"
></video>
```

<Note>
  Bei Verwendung der JSX-Syntax schreiben Sie mehrteilige Attribute in camelCase: `autoPlay`, `playsInline`, `allowFullScreen`.
</Note>

<div id="iframes">
  ## iframes
</div>

Externe Inhalte mit iframe-Elementen einbetten:

```html
<iframe 
  src="https://example.com/embed" 
  title="Embedded content"
  className="w-full h-96 rounded-xl"
></iframe>
```
