Mintlify can generate a complete documentation site for any public GitHub repository, whether there is already documentation in the repository or just code.

Replace `github.com` with `mintlify.com` in any public repository URL:

```text
https://mintlify.com/github-owner/repository-name
```

Browse already generated sites or create your own at [mintlify.com/explore](https://mintlify.com/explore).

## What Mintlify analyzes

To generate your site, Mintlify builds context by analyzing your entire repository, including:

- README files
- Programming languages used
- GitHub and repository metadata
- Homepage URL and branding assets such as logo, favicon, and colors

After building context and making a plan for your site, Mintlify generates your documentation site, reviews it, and deploys it.

## Claim your site

Click the **Use as starter template** button on the banner of your generated site to create a Mintlify account and claim the site.

After you claim your site, you can edit content, add a custom domain, and gain access to all other Mintlify features.
