<Info>
  Los despliegues de vista previa están disponibles en los [planes Pro y Enterprise](https://mintlify.com/pricing?ref=preview-deployments).
</Info>

Los despliegues de vista previa te permiten ver cómo se verán los cambios en tu documentación antes de integrarlos en producción. Cada vista previa crea una URL compartible que se actualiza automáticamente a medida que haces push de nuevos cambios.

<div id="creating-preview-deployments">
  ## Creación de despliegues de vista previa
</div>

Los despliegues de vista previa se crean automáticamente mediante pull requests o manualmente desde tu panel.

<div id="automatic-previews">
  ### Vistas previas automáticas
</div>

Cuando creas un pull request, el bot de Mintlify agrega automáticamente un enlace para ver el despliegue de vista previa en tu pull request. La vista previa se actualiza cada vez que haces push de nuevos commits a la rama.

<div id="manual-previews">
  ### Vistas previas manuales
</div>

Puedes crear manualmente una vista previa para cualquier rama.

1. Ve a tu [panel](https://dashboard.mintlify.com/).
2. Selecciona **Previews**.
3. Selecciona **Create preview**.
4. Ingresa el nombre de tu rama.
5. Selecciona **Create deployment**.

<div id="redeploying-a-preview">
  ## Volver a desplegar una vista previa
</div>

Vuelve a desplegar una vista previa para actualizar el contenido o reintentar tras un despliegue fallido.

1. Selecciona la vista previa desde tu [panel](https://dashboard.mintlify.com/).
2. Selecciona **Redeploy**.

<div id="sharing-preview-deployments">
  ## Compartir implementaciones de vista previa
</div>

Las implementaciones de vista previa pueden ser públicas o estar restringidas a usuarios autenticados.

**Acceso público**: Las URL de vista previa son públicas de forma predeterminada. Comparte un enlace de vista previa con cualquier persona que necesite revisar tus cambios.

**Acceso restringido**: Habilita la autenticación en las implementaciones de vista previa para limitar el acceso únicamente a usuarios autenticados de Mintlify. Consulta [Configuración de autenticación](/es/authentication-personalization/authentication-setup) para obtener más información.
