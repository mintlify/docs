Para configurar tu documentación en un dominio personalizado, debes establecer el dominio que quieras usar en la configuración de Mintlify y configurar el DNS con tu proveedor de dominios.

<Info>
  ¿Quieres configurar un subdirectorio personalizado como `mintlify.com/docs`? Consulta la documentación sobre [subdirectorio personalizado](/es/advanced/subpath/cloudflare).
</Info>

<div id="dashboard-settings">
  ## Configuración del dashboard
</div>

1. Ve a tu [dashboard](https://dashboard.mintlify.com)
2. Selecciona **Settings**
3. Selecciona **Domain Setup**
4. Ingresa el dominio personalizado que deseas. Por ejemplo, `docs.mintlify.com`

<div id="configuring-your-dns">
  ## Configurar tu DNS
</div>

1. Ve a la configuración de DNS de tu dominio en el sitio web de tu proveedor de dominio.
2. Crea una nueva entrada de DNS e introduce los siguientes valores:

```bash
CNAME | docs | cname.vercel-dns.com.
```

<div id="provider-specific-settings">
  ## Configuración específica del proveedor
</div>

<AccordionGroup>
  <Accordion title="Verificación de Vercel">
    Si Vercel es tu proveedor de domain, debes añadir un registro `TXT` de verificación. Esta información se mostrará en tu dashboard después de enviar tu domain personalizado y también la recibirás por correo electrónico.
  </Accordion>

  <Accordion title="Configuración de seguridad de Cloudflare">
    Si Cloudflare es tu proveedor de DNS, debes habilitar la opción de seguridad &quot;full strict&quot; en la configuración de HTTPS.
  </Accordion>
</AccordionGroup>
