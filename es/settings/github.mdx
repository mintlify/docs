Mintlify utiliza una app de GitHub para sincronizar automáticamente tu documentación con tu repositorio de GitHub.

## Instalación de la aplicación de GitHub

Instala la aplicación de GitHub de Mintlify en la cuenta u organización de GitHub que sea propietaria de tu repositorio de documentación. Debes ser propietario de la organización o tener permisos de administrador en el repositorio para instalar la aplicación. Si no cuentas con los permisos necesarios, el propietario del repositorio deberá aprobar la solicitud de instalación.

Instala la aplicación de GitHub de Mintlify desde tu [panel](https://dashboard.mintlify.com/settings/organization/github-app).

<Tip>
Recomendamos otorgar acceso solo al repositorio donde se aloja tu documentación.
</Tip>

## Permisos

Cuando instalas la aplicación de GitHub, se te pedirá que concedas los siguientes permisos:

Permisos de lectura:
* `metadata`: Información básica del repositorio

Permisos de lectura y escritura:
* `checks`: Crear comprobaciones de estado en pull requests
* `code`: Leer cambios de archivos cuando haces commit en tu rama de documentación
* `deployments`: Generar despliegues de vista previa para pull requests
* `pull requests`: Crear ramas y pull requests desde el editor web
* `workflows`: Administrar GitHub Actions para los despliegues de la documentación

<Info>
La aplicación solo accede a los repositorios a los que le otorgas acceso explícitamente. Si tienes habilitadas reglas de protección de ramas, la aplicación no puede hacer push directamente a ramas protegidas.
</Info>

## Gestión del acceso al repositorio

Al instalar nuestra aplicación de GitHub, puedes conceder acceso a todos tus repositorios o solo a algunos específicos. Recomendamos conceder acceso únicamente a los repositorios donde se encuentre tu documentación. Puedes modificar esta selección en cualquier momento en la [configuración de la aplicación de GitHub](https://github.com/apps/mintlify/installations/new).

## Configuración de la fuente de la documentación

Cambia la organización, el repositorio o la rama desde la que se compila tu documentación en la sección [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) de tu panel.

## Solución de problemas

Si tienes problemas con la aplicación de GitHub, restablecer la conexión suele resolver la mayoría.

<Steps>
  <Step title="Desinstala la aplicación de Mintlify desde GitHub.">
    1. En GitHub, ve a [installations](https://github.com/settings/installations) y selecciona **Configure** junto a la aplicación de Mintlify. Desplázate hacia abajo y selecciona **Uninstall**.
    2. Ve a [Authorized GitHub Apps](https://github.com/settings/apps/authorizations) y selecciona **Revoke** junto a la aplicación de Mintlify.
  </Step>
  <Step title="Vuelve a instalar la aplicación de Mintlify.">
    1. En tu panel de Mintlify, ve a [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) e instala la aplicación de GitHub.
    2. Autoriza tu cuenta en la sección [My Profile](https://dashboard.mintlify.com/settings/account) de tu panel.
  </Step>
</Steps>
