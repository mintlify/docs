<div id="enabling-data-privacy-integrations">
  ## Habilitar integraciones de privacidad de datos
</div>

Puedes añadir plataformas de privacidad de datos a tu sitio de documentación. Incluye el campo `integrations` en tu archivo `docs.json` con los scripts correspondientes.

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

<div id="cookie-consent-and-disabling-telemetry">
  ## Consentimiento de cookies y desactivación de la telemetría
</div>

Si necesitas verificar si un usuario ya ha dado su consentimiento para las cookies para cumplir con el RGPD, puedes especificar una key de almacenamiento local y un value en `cookies`:

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

Si se establecen estos valores, se verificará el almacenamiento local para comprobar si el usuario ha dado su consentimiento a las cookies. Si no, la telemetría se desactivará.

Si quieres desactivar la telemetría para todos los usuarios, puedes añadir lo siguiente a tu archivo `docs.json`:

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```

<Note>
  Si desactivas la telemetría, no podrás recopilar comentarios en tus páginas de documentación, incluso si habilitas los comentarios en tu dashboard.
</Note>
