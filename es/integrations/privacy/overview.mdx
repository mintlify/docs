<div id="enabling-data-privacy-integrations">
  ## Habilitar integraciones de privacidad de datos
</div>

Puedes agregar plataformas de privacidad de datos a tu documentación. Añade el campo `integrations` a tu archivo `docs.json` junto con los scripts correspondientes.

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

<div id="cookie-consent-and-disabling-telemetry">
  ## Consentimiento de cookies y desactivación de la telemetría
</div>

Si necesitas verificar si un usuario ya ha dado su consentimiento para las cookies para cumplir con el RGPD, puedes especificar una key y un value de almacenamiento local en `cookies`:

```json
  "integrations": {
    "cookies": {
      "key": "CLAVE DE ALMACENAMIENTO LOCAL",
      "value": "VALOR DE ALMACENAMIENTO LOCAL"
    }
  }
```

Si se establecen estos valores, se comprobará el almacenamiento local para ver si el usuario ha consentido el uso de cookies. Si no lo ha hecho, la telemetría se desactivará.

Si quieres desactivar la telemetría para todos los usuarios, puedes añadir lo siguiente a tu archivo `docs.json`:

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
