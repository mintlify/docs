Añade lo siguiente a tu archivo `docs.json` para incorporar un widget de [Front Chat](https://front.com).

<CodeGroup>

```json Opciones de integración en docs.json
"integrations": {
    "frontchat": "CHAT_ID"
}
```

```json Ejemplo
"integrations": {
    "frontchat": "1365d046d7c023e9b030ce90d02d093a"
}
```

</CodeGroup>
