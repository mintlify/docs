---
title: "Intercom"
description: "Conecta Intercom para la mensajería con clientes dentro de la aplicación."
---

Agrega lo siguiente a tu archivo `docs.json` para añadir un widget de [Intercom](https://www.intercom.com).

<CodeGroup>

```json Opciones de integración en docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json Ejemplo
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }

```
