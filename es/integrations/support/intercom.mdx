title: "Intercom"
---

Añade lo siguiente a tu archivo `docs.json` para incorporar un widget de [Intercom](https://www.intercom.com).

<CodeGroup>

```json Opciones de integración en docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json Ejemplo
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }

```
