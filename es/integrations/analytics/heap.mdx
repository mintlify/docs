Agrega lo siguiente a tu archivo `docs.json` para enviar Analytics a Heap.

<CodeGroup>
  ```json Opciones de Analytics en docs.json
  "integrations": {
      "heap": {
          "appId": "required"
      }
  }
  ```

  ```json Ejemplo
  "integrations": {
      "heap": {
          "appId": "1234567890"
      }
  }
  ```
</CodeGroup>

Reemplaza `1234567890` con tu ID de aplicación de Heap. Puedes encontrarlo en la [configuración de tu cuenta de Heap](https://heapanalytics.com/app/settings).
