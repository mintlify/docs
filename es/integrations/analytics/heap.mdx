---
title: "Heap"
---

Agrega lo siguiente a tu archivo `docs.json` para enviar analítica a Heap.

<CodeGroup>

```json Opciones de analítica en docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
