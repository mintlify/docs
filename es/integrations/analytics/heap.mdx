---
title: "Heap"
---

Añade lo siguiente a tu archivo `docs.json` para enviar analíticas a Heap.

<CodeGroup>

```json Opciones de analíticas en docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
