Agrega lo siguiente a tu archivo `docs.json` para enviar Analytics a Heap.

<CodeGroup>
  ```json Opciones de Analytics en docs.json
  "integrations": {
      "heap": {
          "appId": "required"
      }
  }
  ```

  ```json Ejemplo
  "integrations": {
      "heap": {
          "appId": "1234567890"
      }
  }
  ```
</CodeGroup>

Reemplaza `1234567890` con tu ID de aplicación de Heap. Puedes encontrarlo en la [configuración de tu cuenta de Heap](https://heapanalytics.com/app/settings).

## Eventos rastreados

Mintlify rastrea automáticamente las siguientes interacciones de usuario y las envía a Heap:

- Vistas de página
- Consultas de búsqueda
- Envíos de comentarios
- Interacciones del menú contextual
- Clics de navegación

Consulta la [lista completa de eventos rastreados](/es/integrations/analytics/overview#tracked-events) para obtener detalles sobre todos los eventos enviados a tu proveedor de Analytics.
