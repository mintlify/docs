Añade lo siguiente a tu archivo `docs.json` para enviar Analytics a Common Room.

Puedes obtener el `siteId` desde la configuración de Signals de Common Room. Debe ser un UUID válido.

<CodeGroup>

  ```json Opciones de Analytics en docs.json
  "integrations": {
      "commonRoom": {
          "siteId": "obligatorio"
      }
  }
  ```

  ```json Ejemplo
  "integrations": {
      "commonRoom": {
          "siteId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
      }
  }
  ```
</CodeGroup>
