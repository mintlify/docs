---
title: "Plausible"
---

Agrega el dominio de tu sitio en `docs.json` para enviar analíticas a Plausible.

<Info>

    No incluyas `https://` en el dominio ni en el servidor.

</Info>

<CodeGroup>

```json Opciones de analítica en docs.json
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Ejemplo
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
