---
title: "Plausible"
---

Añade el dominio de tu sitio a `docs.json` para enviar analíticas a Plausible.

<Info>

    No incluyas `https://` en el dominio ni en el servidor.

</Info>

<CodeGroup>

```json Opciones de analítica en docs.json
"integrations": {
    "plausible": {
        "domain": "obligatorio",
        "server": "opcional"
    }
}
```

```json Ejemplo
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
