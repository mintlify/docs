---
title: "Plausible"
---

Agrega el domain de tu sitio a `docs.json` para enviar Analytics a Plausible.

<Info>

    No incluyas `https://` en el domain ni en el servidor.

</Info>

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Ejemplo
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
