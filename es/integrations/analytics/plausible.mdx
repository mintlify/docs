Añade el domain de tu sitio a `docs.json` para enviar analytics a Plausible.

<Info>

    No incluyas `https://` en el domain ni en el servidor.

</Info>

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "plausible": {
        "domain": "required - el dominio de tu documentación",
        "server": "optional - la URL de tu servidor Plausible autoalojado"
    }
}
```

```json Ejemplo
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
