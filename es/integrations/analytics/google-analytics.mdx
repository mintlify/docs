title: "Google Analytics 4"
---

Necesitas generar una nueva propiedad de <Tooltip tip="Google Analytics 4">GA4</Tooltip> para usar con Mintlify. Los datos recopilados se incorporarán al mismo proyecto que tus otros datos de Google Analytics.

Si usas la versión anterior de Google Analytics, Universal Analytics, todavía podrás generar una propiedad de <Tooltip tip="Google Analytics 4">GA4</Tooltip>. Los datos de <Tooltip tip="Google Analytics 4">GA4</Tooltip> son ligeramente diferentes de los de UA, pero se siguen recopilando en el mismo proyecto.



<div id="how-to-connect-ga4-to-mintlify">
  ## Cómo conectar GA4 a Mintlify
</div>

<div id="create-a-web-stream">
  ### Crear un flujo web
</div>

Necesitas crear un flujo web para obtener el Measurement ID que debes introducir en Mintlify.

Haz clic en el ícono de engranaje en la esquina inferior izquierda de la pantalla de Google Analytics. Luego, haz clic en Data Streams.

<Frame>![](/images/ga4-web-streams.png)</Frame>

Crea un flujo web e introduce la URL de tu sitio de documentación de Mintlify como la URL del flujo.

Tu Measurement ID tiene el formato `G-XXXXXXX` y aparecerá en Stream Details inmediatamente después de crear el flujo web.

<div id="put-measurement-id-in-docsjson">
  ### Añadir el Measurement ID en docs.json
</div>

Añade tu Measurement ID a tu archivo `docs.json` de la siguiente manera:

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<div id="wait">
  ### Espera
</div>

Google Analytics tarda de dos a tres días en mostrar tus datos.

Puedes usar la extensión [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) para comprobar que Analytics esté habilitado correctamente. La extensión registrará en la consola de tu navegador cada vez que GA4 realice una solicitud.

<Note>
  Los enlaces de vista previa tienen Analytics desactivado.
</Note>
