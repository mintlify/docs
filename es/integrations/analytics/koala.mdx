---
title: "Koala"
description: "Identifica empresas y visitantes en tu sitio de documentación con Koala."
keywords: ["Integración con Koala", "identificación de visitantes", "scoring de cuentas", "public API key"]
---

Añade lo siguiente a tu archivo `docs.json` para habilitar la identificación de visitantes con Koala.

<CodeGroup>
  ```json Opciones de Analytics en docs.json
  "integrations": {
      "koala": {
          "publicApiKey": "required"
      }
  }
  ```

  ```json Ejemplo
  "integrations": {
      "koala": {
          "publicApiKey": "pk_abc123def456"
      }
  }
  ```
</CodeGroup>
