---
title: "Koala"
---

Agrega lo siguiente a tu archivo `docs.json` para enviar analíticas a Koala.

<CodeGroup>

```json Opciones de analíticas en docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
