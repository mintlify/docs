Añade lo siguiente a tu archivo `docs.json` para enviar Analytics a Clearbit.

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
