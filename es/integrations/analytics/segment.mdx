Agrega tu write key de Segment a tu archivo `docs.json` para enviar Analytics a Segment.

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Ejemplo
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
