---
title: "Segment"
---

Agrega tu clave de escritura de Segment a tu archivo `docs.json` para enviar analíticas a Segment.

<CodeGroup>

```json Opciones de analíticas en docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Ejemplo
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
