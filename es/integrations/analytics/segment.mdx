---
- title: Segment
---

Añade tu clave de escritura de Segment a tu archivo `docs.json` para enviar analíticas a Segment.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "segment": {
          "key": "required",
      }
  }
  ```

  ```json Example
  "integrations": {
      "segment": {
          "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
      }
  }
  ```
</CodeGroup>
