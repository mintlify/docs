---
title: "Fathom"
description: "Haz seguimiento de la analítica de tu documentación con Fathom."
keywords: ["Fathom Analytics", "analítica centrada en la privacidad", "site ID", "analítica simple"]
---

Agrega lo siguiente a tu archivo `docs.json` para enviar Analytics a Fathom.

Puedes obtener el `siteId` en la configuración de tu script.

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "fathom": {
        "siteId": "Obligatorio"
    }
}
```

```json Ejemplo
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
