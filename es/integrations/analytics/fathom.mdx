---
title: "Fathom"
---

Añade lo siguiente a tu archivo `docs.json` para enviar analíticas a Fathom.

Puedes obtener el `siteId` en la configuración de tu script.

<CodeGroup>

```json Opciones de analíticas en docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
