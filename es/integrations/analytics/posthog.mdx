---

Agrega lo siguiente a tu archivo `docs.json` para enviar analíticas a PostHog.

Solo necesitas incluir `apiHost` si alojas PostHog por tu cuenta. De forma predeterminada enviamos eventos a `https://app.posthog.com`.

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "posthog": {
        "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
        "apiHost": "optional"
    }
}
```

```json Ejemplo
"integrations": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<br />

<Warning>
    Al habilitar las analíticas de PostHog se deshabilitarán las analíticas en el dashboard de Mintlify.
</Warning>



<div id="session-recordings">
  ## Grabaciones de sesiones
</div>

Debes agregar la URL de tu sitio de documentación a “Authorized domains for recordings” de PostHog antes de poder recibir grabaciones de sesiones. La opción para agregar tu URL se encuentra en la configuración del proyecto de PostHog.
