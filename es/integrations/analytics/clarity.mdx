Añade lo siguiente a tu archivo `docs.json` para enviar datos de Analytics a Microsoft Clarity.

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "clarity": {
        "projectId": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "clarity": {
        "projectId": "abc123xyz"
    }
}
```

</CodeGroup>

<div id="get-your-project-id">
  ## Obtén tu ID de proyecto
</div>

1. Crea una [cuenta en Clarity](https://clarity.microsoft.com/projects).
2. Haz clic en **Obtener código de seguimiento.**
3. Copia tu ID de proyecto del código de seguimiento.
