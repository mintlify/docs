Añade lo siguiente a tu archivo `docs.json` para enviar Analytics a Amplitude.

<CodeGroup>

```json Opciones de Analytics en docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```
