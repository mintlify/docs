Mintlify se conecta con las herramientas que tu equipo ya utiliza. Agrega proveedores de análisis para rastrear cómo los usuarios interactúan con tu documentación, incorpora widgets de soporte para que los visitantes puedan obtener ayuda sin salir de la página, muestra fragmentos de código SDK autogenerados en tu playground de API y gestiona el consentimiento de cookies y el cumplimiento de privacidad, todo desde la configuración de tu `docs.json`.
