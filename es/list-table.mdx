## Listas

Las listas siguen la [sintaxis oficial de Markdown](https://www.markdownguide.org/basic-syntax/#lists-1).

### Lista ordenada

Para crear una lista ordenada, añade números seguidos de un punto antes de cada elemento.

1. Primer elemento
2. Segundo elemento
3. Tercer elemento
4. Cuarto elemento

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

### Lista desordenada

Para crear una lista desordenada, añade guiones (`-`), asteriscos (`*`) o signos más (`+`) antes de cada elemento.

- Primer elemento
- Segundo elemento
- Tercer elemento
- Cuarto elemento

```mdx
- First item
- Second item
- Third item
- Fourth item
```

### Lista anidada

Indenta los elementos de la lista para anidarlos.

- Primer elemento
- Segundo elemento
  - Elemento adicional
  - Elemento adicional
- Tercer elemento

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

## Tablas

Las tablas siguen la [sintaxis oficial de Markdown](https://www.markdownguide.org/extended-syntax/#tables).

Para agregar una tabla, usa tres o más guiones (`---`) para crear el encabezado de cada columna y usa barras verticales (`|`) para separar cada columna. Por compatibilidad, también debes agregar una barra vertical al inicio y al final de la fila.

| Propiedad | Descripción                          |
| --------- | ------------------------------------ |
| Nombre    | Nombre completo del usuario          |
| Edad      | Edad informada                       |
| Ingreso   | Si el usuario se unió a la comunidad |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
