---
title: "Modo rojo"
description: "Habilita un modo de visualización con tinte rojo para reducir la fatiga ocular en entornos con poca luz."
---

El modo rojo aplica un filtro de color rojo a tu documentación, reduciendo la emisión de luz azul. Esto es útil para:

- Leer documentación en entornos oscuros sin interrumpir la visión nocturna
- Reducir la fatiga ocular durante largas sesiones de programación nocturnas
- Usuarios que prefieren temperaturas de color más cálidas

## Habilitar el modo rojo

Agrega la opción `redMode` a tu configuración de `docs.json`:

```json docs.json
{
  "appearance": {
    "redMode": true
  }
}
```

## Opciones de configuración

<ResponseField name="redMode" type="boolean | object">
  Habilita el modo rojo para tu documentación. Establece en `true` para la configuración predeterminada, o pasa un objeto para configuración avanzada.

  <Expandable title="Opciones avanzadas">
    <ResponseField name="intensity" type="number" default="0.5">
      La intensidad del filtro rojo, de `0` (sutil) a `1` (rojo completo).
    </ResponseField>
    <ResponseField name="toggle" type="boolean" default="true">
      Mostrar un botón de alternancia en la barra de navegación para que los usuarios activen/desactiven el modo rojo.
    </ResponseField>
  </Expandable>
</ResponseField>

### Ejemplo con opciones avanzadas

```json docs.json
{
  "appearance": {
    "redMode": {
      "intensity": 0.7,
      "toggle": true
    }
  }
}
```

## Alternancia de usuario

Cuando `toggle` está habilitado, los usuarios pueden activar o desactivar el modo rojo usando el icono de luna en la barra de navegación superior. Su preferencia se guarda en el almacenamiento local.
