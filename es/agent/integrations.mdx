Las integraciones del agente dan al agente acceso al contexto de otras herramientas que utiliza tu equipo. Una vez conectadas, el agente puede buscar en hilos de Slack, leer páginas de Notion, consultar incidencias de Linear y obtener contexto desde otras aplicaciones para fundamentar sus respuestas y actualizaciones de contenido.

<div id="connect-an-integration">
  ## Conectar una integración
</div>

Los administradores y editores pueden conectar una integración para su organización desde el dashboard.

1. Ve a la página [Integraciones del agente](https://app.mintlify.com/settings/organization/integrations) en tu dashboard.
2. Busca la integración que quieres conectar.
3. Haz clic en **Connect**.
4. Sigue las indicaciones de OAuth para autorizar a Mintlify a acceder a tu cuenta.

Algunas integraciones, como Jira y Salesforce, abren un paso adicional de configuración para introducir la URL o el subdominio de tu espacio de trabajo antes de que comience el flujo de OAuth.

Una vez conectada, todos los miembros de tu organización pueden pedirle al agente que use la integración en Slack o en workflows que admitan integraciones.

  ## Integraciones compatibles
</div>

- Google Calendar
- Intercom
- Jira
- Linear
- Notion
- Salesforce
- Slack

<div id="how-the-agent-uses-integrations">
  ## Cómo usa el agente las integraciones
</div>

El agente usa las integraciones conectadas como herramientas. Cuando le haces una pregunta al agente o le asignas una tarea, puede buscar y recuperar contenido de tus aplicaciones conectadas para construir contexto.

Por ejemplo:
- "Resume el hilo de Slack sobre la migración a la API v2 y redáctalo para la base de conocimientos."
- "Revisa el ticket de Linear para esta funcionalidad y documenta su comportamiento."
- "¿Qué decidió el equipo sobre los límites de velocidad? Revisa Slack."

El agente solo accede a los datos de las integraciones cuando son relevantes para la solicitud. No lee de forma proactiva desde las aplicaciones conectadas.

<div id="disconnect-an-integration">
  ## Desconectar una integración
</div>

1. Ve a la página [Integraciones del agente](https://app.mintlify.com/settings/organization/integrations) en tu dashboard.
2. Busca la integración conectada que quieres desconectar.
3. Haz clic en **Configure**.
4. Haz clic en **Disconnect**.

Desconectar elimina la integración para tu organización. El agente pierde el acceso a esa herramienta de forma inmediata.
