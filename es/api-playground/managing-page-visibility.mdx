---
- title: Gestionar la visibilidad de la página
- description: Controla qué endpoints de API aparecen en la navegación de tu documentación.
---

Para endpoints de uso interno, operaciones obsoletas, funciones beta o endpoints que deberían ser accesibles mediante URL directa pero no detectables a través de la navegación del sitio, puedes controlar qué operaciones de OpenAPI se publican como páginas de documentación y su visibilidad en la navegación.

Si tus páginas se generan automáticamente a partir de un documento OpenAPI, gestiona la visibilidad de la página con las extensiones `x-hidden` y `x-excluded`.

## `x-hidden`

La extensión `x-hidden` crea una página para un endpoint, pero la oculta de la navegación. La página solo es accesible navegando directamente a su URL.

Casos de uso comunes para `x-hidden` are:

* Endpoints que quieres documentar, pero no promocionar.
* Páginas a las que enlazarás desde otro contenido.
* Endpoints para usuarios específicos.

## `x-excluded`

La extensión `x-excluded` excluye completamente un endpoint de tu documentación.

Casos de uso comunes para `x-excluded` are:

* Endpoints de uso interno.
* Endpoints obsoletos que no quieres documentar.
* Funciones beta que no están listas para documentación pública.

## Implementación

Añade la extensión `x-hidden` o `x-excluded` bajo el método HTTP en tu especificación OpenAPI.

Aquí hay ejemplos de cómo usar cada propiedad en un documento de esquema OpenAPI para un endpoint y una ruta de webhook.

```json Endpoint example {11, 19}
"paths": {
  "/plants": {
    "get": {
      "description": "Returns all plants from the store",
      "parameters": { /*...*/ },
      "responses": { /*...*/ }
    }
  },
  "/hidden_plants": {
    "get": {
      "x-hidden": true,
      "description": "Returns all somewhat secret plants from the store",
      "parameters": { /*...*/ },
      "responses": { /*...*/ }
    }
  },
  "/secret_plants": {
    "get": {
      "x-excluded": true,
      "description": "Returns all top secret plants from the store (do not publish this endpoint!)",
      "parameters": { /*...*/ },
      "responses": { /*...*/ }
    }
  }
},
```

```json Webhook example {9, 15}
"webhooks": {
  "/plants_hook": {
    "post": {
      "description": "Webhook for information about a new plant added to the store",
    }
  },
  "/hidden_plants_hook": {
    "post": {
      "x-hidden": true,
      "description": "Webhook for somewhat secret information about a new plant added to the store"
    }
  },
  "/secret_plants_hook": {
    "post": {
      "x-excluded": true,
      "description": "Webhook for top secret information about a new plant added to the store (do not publish this endpoint!)"
    }
  }
}
```
