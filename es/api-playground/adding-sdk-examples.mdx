---
- title: Agregar ejemplos de SDK
- description: Muestra ejemplos de código SDK en tu documentación de API.
---

Si tus usuarios interactúan con tu API usando un SDK en lugar de hacerlo directamente a través de una solicitud de red, puedes usar la `x-codeSamples` extensión para agregar ejemplos de código a tu documento OpenAPI y mostrarlos en tus páginas de OpenAPI.

Esta propiedad puede agregarse a cualquier método de solicitud y tiene el siguiente esquema.

<ParamField body="lang" type="string" required>
  El lenguaje del ejemplo de código.
</ParamField>

<ParamField body="label" type="string">
  La etiqueta para el ejemplo. Esto es útil cuando se proporcionan múltiples ejemplos para un solo endpoint.
</ParamField>

<ParamField body="source" type="string" required>
  El código fuente del ejemplo.
</ParamField>

Aquí hay un ejemplo de muestras de código para una aplicación de seguimiento de plantas, que tiene tanto una herramienta CLI de Bash como un SDK de JavaScript.

```yaml
paths:
  /plants:
    get:
      # ...
      x-codeSamples:
        - lang: bash
          label: List all unwatered plants
          source: |
            planter list -u
        - lang: javascript
          label: List all unwatered plants
          source: |
            const planter = require('planter');
            planter.list({ unwatered: true });
        - lang: bash
          label: List all potted plants
          source: |
            planter list -p
        - lang: javascript
          label: List all potted plants
          source: |
            const planter = require('planter');
            planter.list({ potted: true });
```
