Después de completar esta guía, tendrás un sitio de documentación en línea listo para personalizar y actualizar.

<div id="before-you-begin">
  ## Antes de comenzar
</div>

Mintlify utiliza un enfoque de documentación como código para administrar tu documentación. Cada página de tu sitio tiene un archivo correspondiente almacenado en tu <Tooltip headline="Repositorio" tip="El código fuente de tu documentación donde se almacenan todos los archivos y su historial. El editor web se conecta a tu repositorio de documentación para acceder y modificar el contenido, o puedes editar los archivos localmente en tu IDE preferido.">repositorio</Tooltip> de documentación. Puedes usar tu propio repositorio Git o un repositorio alojado en Mintlify.

Elige cómo quieres gestionar el repositorio.

<CardGroup cols={2}>
  <Card title="Alojado en Mintlify" icon="cloud">
    Mintlify crea un repositorio privado de GitHub conectado a tu implementación de Mintlify.

    Elige esta opción si:

    * Solo trabajarás en el editor web y no quieres usar Git localmente.
    * Quieres comenzar rápidamente sin conectar un repositorio.
    * Quieres que Mintlify administre tu repositorio de documentación por ti.
  </Card>

  <Card title="Tu propio repositorio" icon="github">
    Conecta tu propio repositorio Git a tu implementación de Mintlify.

    Elige esta opción si:

    * Trabajarás localmente y quieres sincronizar con tu repositorio remoto.
    * Ya tienes un repositorio para tu documentación.
    * Quieres que tu documentación esté bajo tu propia organización de GitHub.
  </Card>
</CardGroup>

<Tip>
  ¿No estás seguro de cuál elegir? Comienza con **Alojado en Mintlify**. Puedes migrar a tu propio repositorio en cualquier momento si se adapta mejor a tus necesidades.
</Tip>

<div id="deploy-your-documentation-site">
  ## Implementa tu sitio de documentación
</div>

<Tabs>
  <Tab title="Alojado en Mintlify">
    <Steps>
      <Step title="Crea tu cuenta">
        Ve a [mintlify.com/start](https://mintlify.com/start) y crea tu cuenta.
      </Step>

      <Step title="Omite la conexión con GitHub">
        Cuando se te pida conectar GitHub, haz clic en **Skip** para usar un repositorio alojado en Mintlify.
      </Step>

      <Step title="Completa la configuración inicial">
        Sigue las indicaciones de configuración inicial para preparar tu documentación. Mintlify crea automáticamente un repositorio privado en la organización `mintlify-community` y configura la Aplicación de GitHub por ti.
      </Step>

      <Step title="Tu sitio está implementado">
        Tu documentación se implementa de inmediato y está accesible en tu URL `.mintlify.app`.
      </Step>
    </Steps>
  </Tab>

  <Tab title="Tu propio repositorio">
    <Steps>
      <Step title="Crea tu cuenta">
        Ve a [mintlify.com/start](https://mintlify.com/start) y crea tu cuenta.
      </Step>

      <Step title="Conecta GitHub">
        Cuando se te pida, inicia sesión en tu cuenta de GitHub y autentícate.
      </Step>

      <Step title="Completa la configuración inicial">
        Sigue las indicaciones restantes de configuración inicial para crear un repositorio y continuar a tu dashboard.
      </Step>

      <Step title="Instala la Aplicación de GitHub">
        Instala la [Aplicación de GitHub](https://dashboard.mintlify.com/api/github/install-app) de Mintlify para habilitar implementaciones automáticas. Esto permite que Mintlify se sincronice con tu repositorio e implemente los cambios automáticamente cuando hagas push.
      </Step>

      <Step title="Opcional: selecciona un repositorio">
        Si quieres usar un repositorio diferente del que se creó durante la configuración inicial para tu documentación, selecciónalo en la página de [Git settings](https://dashboard.mintlify.com/settings/deployment/git-settings) de tu dashboard. Tu repositorio de documentación debe tener un archivo `docs.json` válido para poder implementar.
      </Step>

      <Step title="Tu sitio está implementado">
        Tu documentación se implementa y está accesible en tu URL `.mintlify.app`. Los cambios futuros que hagas push a tu repositorio desencadenarán automáticamente nuevas implementaciones.
      </Step>
    </Steps>

    <Info>
      Para obtener más información sobre la Aplicación de GitHub y sus permisos, consulta [GitHub](/es/deploy/github).
    </Info>
  </Tab>
</Tabs>

<div id="view-your-deployed-site">
  ## Consulta tu sitio desplegado
</div>

Tu sitio de documentación ya se ha desplegado en `https://<your-project-name>.mintlify.app`.

Encuentra tu URL exacta en la página **Overview** de tu [Dashboard de Mintlify](https://dashboard.mintlify.com/).

  Tu sitio ya está disponible para verse de inmediato. Usa esta URL para hacer pruebas y compartirla con tu equipo. Antes de compartirla con tus usuarios, quizá quieras añadir un [dominio personalizado](/es/customize/custom-domain).
</Tip>

<div id="make-your-first-change">
  ## Haz tu primer cambio
</div>

<Steps>
  <Step title="Abre el editor web">
    Navega al [editor web](https://dashboard.mintlify.com/editor) en tu dashboard.
  </Step>

  <Step title="Edita una página">
    Abre `index.mdx` y actualiza el campo `description` en el frontmatter:

    ```mdx
    ---
    title: "Introduction"
    description: "Your custom description here"
    ---
    ```
  </Step>

  <Step title="Publicar">
    Haz clic en el botón **Publish** en la esquina superior derecha de la barra de herramientas del editor web.
  </Step>

  <Step title="Ver en vivo">
    En la página de [Overview](https://dashboard.mintlify.com/) de tu dashboard, puedes ver el estado de implementación de tu sitio. Cuando la implementación haya finalizado, actualiza tu sitio de documentación para ver tus cambios en vivo.
  </Step>
</Steps>
