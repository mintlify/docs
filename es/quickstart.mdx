Después de completar esta guía, tendrás un sitio de documentación en línea listo para personalizar y actualizar.

<div id="before-you-begin">
  ## Antes de comenzar
</div>

Mintlify utiliza un enfoque de documentación como código para administrar tu documentación. Cada página de tu sitio tiene un archivo correspondiente almacenado en tu <Tooltip tip="El código fuente de tu documentación donde se almacenan todos los archivos y su historial. El editor web se conecta a tu repositorio de documentación para acceder y modificar el contenido, o puedes editar los archivos localmente en tu IDE preferido.">repositorio</Tooltip> de documentación.

Cuando conectas tu repositorio de documentación a tu implementación de Mintlify, puedes trabajar en tu documentación localmente o en el editor web y sincronizar cualquier cambio con tu repositorio remoto.

<div id="deploy-your-documentation-site">
  ## Implementa tu sitio de documentación
</div>

Ve a [mintlify.com/start](https://mintlify.com/start) y completa el proceso de configuración inicial. Durante la configuración, conectarás tu cuenta de GitHub, crearás o seleccionarás un repositorio para tu documentación e instalarás la Aplicación de GitHub para habilitar las implementaciones automáticas.

Después de la configuración inicial, tu sitio de documentación se implementa automáticamente y queda accesible en tu URL `.mintlify.app`.

<AccordionGroup>
  <Accordion title="Opcional: omitir la conexión con GitHub durante la configuración inicial">
    Si quieres comenzar rápidamente sin conectar tu propio repositorio, puedes omitir la conexión con GitHub durante la configuración inicial. Mintlify crea un repositorio privado en una organización privada y configura automáticamente la Aplicación de GitHub por ti.

    Esto te permite usar el editor web de inmediato y migrar a tu propio repositorio más adelante.
  </Accordion>
</AccordionGroup>

<div id="view-your-deployed-site">
  ## Consulta tu sitio desplegado
</div>

Tu sitio de documentación ya se ha desplegado en `https://<your-project-name>.mintlify.app`.

Encuentra tu URL exacta en la página **Overview** de tu [Dashboard de Mintlify](https://dashboard.mintlify.com/).

</Tip>

<div id="make-your-first-change">
  ## Haz tu primer cambio
</div>

<Tabs>
  <Tab title="CLI">
    <Steps>
      <Step title="Instala la CLI">
        La CLI requiere [Node.js](https://nodejs.org/en) v20.17.0 o superior. Usa una versión LTS para mayor estabilidad.

        <CodeGroup>
          ```bash npm
          npm i -g mint
          ```

          ```bash pnpm
          pnpm add -g mint
          ```
        </CodeGroup>

        Consulta [Instalar la CLI](/es/installation) para ver todos los detalles y la resolución de problemas.
      </Step>

      <Step title="Clona tu repositorio">
        Si aún no has clonado tu repositorio localmente, sigue las instrucciones en [Clonar tu repositorio](/es/installation#clone-your-repository).
      </Step>

      <Step title="Edita una página">
        Abre `index.mdx` en tu editor preferido y actualiza el campo `description` en el frontmatter:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>

      <Step title="Vista previa local">
        Ejecuta el siguiente comando desde tu directorio de documentación:

        ```bash
        mint dev
        ```

        Ve la vista previa en `http://localhost:3000`.
      </Step>

      <Step title="Haz push de tus cambios">
        Crea una confirmación (commit) y haz push de tus cambios para iniciar una implementación:

        ```bash
        git add .
        git commit -m "Update description"
        git push
        ```

        Mintlify implementa automáticamente tus cambios. Consulta el estado de implementación en la página de [Overview](https://dashboard.mintlify.com/) de tu dashboard.
      </Step>
    </Steps>
  </Tab>

  <Tab title="Editor web">
    <Steps>
      <Step title="Abre el editor web">
        Navega al [editor web](https://dashboard.mintlify.com/editor) en tu dashboard.
      </Step>

      <Step title="Edita una página">
        Abre `index.mdx` y actualiza el campo `description` en el frontmatter:

        ```mdx
        ---
        title: "Introduction"
        description: "Your custom description here"
        ---
        ```
      </Step>

      <Step title="Publicar">
        Haz clic en el botón **Publish** en la esquina superior derecha de la barra de herramientas del editor web.
      </Step>

      <Step title="Ver en vivo">
        En la página de [Overview](https://dashboard.mintlify.com/) de tu dashboard, puedes ver el estado de implementación de tu sitio. Cuando la implementación haya finalizado, actualiza tu sitio de documentación para ver tus cambios en vivo.
      </Step>
    </Steps>
  </Tab>
</Tabs>
