Encuentra soluciones a problemas comunes que puedes encontrar al usar el editor web.

<AccordionGroup>
  <Accordion title="Los cambios no aparecen después de publicar">
    **Posibles causas:**

    * La implementación aún está en curso
    * Problemas de caché del navegador
    * Errores de compilación o implementación

    **Soluciones:**

    1. Revisa el estado de la implementación en tu dashboard.
    2. Actualiza forzosamente tu navegador (<kbd>Ctrl</kbd> + <kbd>F5</kbd> o <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>)
    3. Borra la caché de tu navegador.
  </Accordion>

  <Accordion title="Errores de permisos al publicar">
    **Posibles causas:**

    * Permisos insuficientes en el repositorio de Git
    * Problemas de autenticación con tu proveedor de Git

    **Soluciones:**

    1. Verifica que tienes el acceso adecuado al repositorio.
    2. Comprueba si tu integración de Git está configurada correctamente.
    3. Revisa la documentación de [Permisos del editor](/es/advanced/dashboard/permissions).
  </Accordion>

  <Accordion title="Problemas al cargar el editor">
    **Posibles causas:**

    * Problemas de conectividad de red
    * Repositorios de documentación grandes

    **Soluciones:**

    1. Verifica tu conexión a internet.
    2. Actualiza la página.
    3. Contacta al soporte si el problema persiste.
  </Accordion>

  <Accordion title="Archivos que no cargan o muestran errores">
    **Posibles causas:**

    * Sintaxis MDX inválida en los archivos
    * Archivos faltantes o dañados
    * Archivos de gran tamaño que provocan timeouts

    **Soluciones:**

    1. Revisa la sintaxis del archivo en busca de errores de formato de MDX.
    2. Verifica que el archivo exista en tu repositorio.
  </Accordion>
</AccordionGroup>
