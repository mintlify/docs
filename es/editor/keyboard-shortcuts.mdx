Use atajos de teclado para navegar y editar más rápido en el editor web. El editor admite atajos comunes como copiar, pegar, deshacer y seleccionar todo, junto con los atajos que se enumeran a continuación.

<div id="editor-wide">

## En todo el editor

</div>

Estos atajos funcionan independientemente del modo en el que se encuentre.

| Comando | macOS | Windows |
| :--- | :--- | :--- |
| **Buscar archivos y contenido** | <kbd>Cmd</kbd> + <kbd>K</kbd> | <kbd>Ctrl</kbd> + <kbd>K</kbd> |
| **Alternar el asistente de IA** (cuando no se edita texto) | <kbd>Cmd</kbd> + <kbd>I</kbd> | <kbd>Ctrl</kbd> + <kbd>I</kbd> |
| **Alternar entre modo visual y source** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd> |
| **Alternar la vista de diff** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd> |
| **Alternar entre Navigation y árbol de Files** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> |
| **Ocultar/mostrar la barra lateral** | <kbd>Cmd</kbd> + <kbd>\\</kbd> | <kbd>Ctrl</kbd> + <kbd>\\</kbd> |

<div id="smart-substitutions">

### Sustituciones inteligentes

</div>

El editor convierte automáticamente ciertas secuencias de caracteres en símbolos tipográficos a medida que escribe. El editor omite las sustituciones dentro de bloques de código y código en línea.

| Tipo | Resultado |
| :--- | :--- |
| `->` | → |
| `<-` | ← |
| `=>` | ⇒ |
| `<=` | ≤ |
| `>=` | ≥ |
| `!=` | ≠ |
