---
title: "Atajos de teclado"
description: "Referencia de atajos de teclado para el editor web."
keywords: ["editor", "keys"]
---

Usa atajos de teclado para navegar y editar más rápido en el editor web. El editor permite usar atajos comunes como copiar, pegar, deshacer y seleccionar todo, junto con los atajos que se enumeran a continuación.

<div id="file-navigation">
  ## Navegación por archivos
</div>

| Comando | macOS | Windows |
| :--- | :--- | :--- |
| **Buscar archivos** | <kbd>Cmd</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>P</kbd> |

<div id="text-formatting">
  ## Formato de texto
</div>

Usa estos atajos de teclado en modo visual para dar formato al texto:

| Comando | macOS | Windows |
| :--- | :--- | :--- |
| **Negrita** | <kbd>Cmd</kbd> + <kbd>B</kbd> | <kbd>Ctrl</kbd> + <kbd>B</kbd> |
| **Subrayado** | <kbd>Cmd</kbd> + <kbd>U</kbd> | <kbd>Ctrl</kbd> + <kbd>U</kbd> |
| **Tachado** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> |
| **Código** | <kbd>Cmd</kbd> + <kbd>E</kbd> | <kbd>Ctrl</kbd> + <kbd>E</kbd> |
| **Subíndice** | <kbd>Cmd</kbd> + <kbd>,</kbd> | <kbd>Ctrl</kbd> + <kbd>,</kbd> |
| **Superíndice** | <kbd>Cmd</kbd> + <kbd>.</kbd> | <kbd>Ctrl</kbd> + <kbd>.</kbd> |
