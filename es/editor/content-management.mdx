<div id="navigating-your-files">
  ## Navegar por tus archivos
</div>

Usa el explorador de archivos de la barra lateral para recorrer tus archivos de documentación. Haz clic en cualquier archivo para abrirlo en el editor.

Pulsa <kbd>Command</kbd> + <kbd>P</kbd> (<kbd>Ctrl</kbd> + <kbd>P</kbd> en Windows) para buscar archivos por nombre.

<div id="organizing-your-navigation">
  ## Cómo organizar tu navigation
</div>

Agrega nuevas páginas a la navigation de tu sitio editando tu archivo `docs.json`.

<Steps>
  <Step title="Open docs.json">
    Abre el archivo `docs.json` en tu directorio raíz usando el explorador de archivos.
  </Step>
  <Step title="Locate the navigation section">
    Ubica la sección de `navigation` en el archivo `docs.json` donde quieres añadir tu nueva página.
  </Step>
  <Step title="Add your page">
    Inserta el nombre de archivo de tu página (sin la extensión `.mdx`) en el array `pages` del grupo correspondiente.
  </Step>
</Steps>

<div id="example-adding-a-themes-page">
  ### Ejemplo: Agregar una página de temas
</div>

```json title="Adding a themes page to the Profile group" highlight="18"
{
    "navigation": {
        "groups": [
            {
                "group": "Primeros pasos",
                "pages": [
                    "index",
                    "quickstart", 
                    "installation"
                ]
            },
            {
                "group": "Perfil",
                "pages": [
                    "settings",
                    "account-types", 
                    "dashboard",
                    "themes"
                ]
            }
        ]
    }
}
```


<div id="editing-content">
  ## Edición de contenido
</div>

Haz cambios en tus páginas usando el modo visual o el Modo Markdown en el editor.

En el modo visual, presiona <kbd>/</kbd> para abrir el menú de componentes. Añade bloques de contenido, avisos, bloques de código y otros componentes para personalizar tu documentación.

En el Modo Markdown, puedes editar directamente el `MDX` de tus páginas. Esto es útil cuando necesitas:

- Definir propiedades específicas de componentes
- Trabajar con componentes anidados complejos
- Copiar y pegar contenido `MDX` desde otras fuentes
