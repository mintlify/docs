Algunas funciones de Mintlify consumen créditos.

* Respuestas del Assistant en tu sitio de documentación
* Respuestas del Assistant API y del chat de discovery
* Ejecuciones del agente en el editor o en Slack
* Ejecuciones de workflows

Para obtener la información de precios más actualizada, consulta la [página de Precios](https://mintlify.com/pricing) o revisa la página de [Uso](https://app.mintlify.com/settings/organization/usage) en tu panel.

**Los excedentes** se cobran por cada crédito adicional a $0.01 en lugar de activar una actualización automática de nivel. Puedes configurar alertas de uso para recibir un correo electrónico cuando alcances cierto porcentaje del límite de tu nivel. Permitir excedentes puede ser más barato que subir de nivel según tus patrones de uso.

<Note>
  Los excedentes están desactivados de forma predeterminada. Debes activarlos en la página de [Uso](https://app.mintlify.com/settings/organization/usage) de tu panel.
</Note>

**Los acumulados** trasladan al mes siguiente hasta el 50 % de los créditos no utilizados, por lo que el máximo de créditos disponibles en un mes es 1.5 veces tu límite mensual. Los acumulados se reinician si cambias de nivel.

Los niveles más altos ofrecen un costo efectivo por crédito más bajo. Si tu uso alcanza de forma constante el tope de tu nivel actual, actualizar es más económico que pagar cargos continuos por excedente.

Las actualizaciones y reducciones de plan entran en vigor de inmediato y se prorratean para el ciclo de facturación actual.

Para un uso de alto volumen o paquetes de créditos personalizados, [contacta con ventas](mailto:sales@mintlify.com). Los paquetes de créditos personalizados solo están disponibles en los planes empresariales.

<div id="how-credits-work">
  ## Cómo funcionan los créditos
</div>

Las distintas funciones consumen diferentes cantidades de créditos por interacción:

| Función | Promedio de créditos por interacción |
|:--------|:-------------------------------------|
| Respuesta del Assistant | 23 |
| Respuesta del Assistant API | 23 |
| Ejecución del agente en el editor | 115 |
| Ejecución del agente en Slack | 110 |

La página de [Uso](https://app.mintlify.com/settings/organization/usage) divide el consumo de créditos en categorías de producto distintas para que puedas ver exactamente a dónde van los créditos:

| Categoría | Qué incluye |
|:----------|:------------|
| Assistant | Respuestas del Assistant web integrado en tu sitio de documentación, incluido el bot de Slack del Assistant. |
| Assistant API | Respuestas del [Assistant API](/es/api/assistant/create-assistant-message-v2) y de las superficies de chat de discovery. |
| Agent | Ejecuciones del agente en el editor y del agente de Slack. |
| Editor agent | Ejecuciones del agente en el editor. |
| Workflows | Ejecuciones de workflows programados y activados. |

Usa el filtro de producto en el gráfico de uso para ver el consumo de una sola categoría a lo largo del tiempo.

Los workflows también consumen créditos cuando se ejecutan:

| Workflow | Promedio de créditos por ejecución |
|:---------|:-----------------------------------|
| Actualización a partir de cambios de código | 180 |
| Actualización a partir de conversaciones con el assistant | 212 |
| Detección de enlaces rotos | 285 |
| Changelog | 223 |
| Auditoría SEO | 422 |
| Traducciones | 913 |
| Revisión de erratas | 330 |
| Estilo de redacción | 235 |

<div id="estimating-your-usage">
  ## Estimar tu uso
</div>

Utiliza los promedios de la sección [Cómo funcionan los créditos](/es/credits#how-credits-work) para estimar tus necesidades mensuales de créditos.

Por ejemplo, si tu sitio de documentación maneja 500 respuestas del Assistant por mes, eso equivale aproximadamente a 11,500 créditos (500 × 23). Agregar un workflow semanal de detección de enlaces rotos añade unos 1,140 créditos al mes (4 × 285).

Después de usar las funciones que consumen créditos durante un mes, revisa tus patrones de uso para ver si debes ajustar tu nivel.

<div id="cost-optimization">
  ## Optimización de costos
</div>

**Programa los workflows en lugar de ejecutarlos en cada push.** Los workflows como las auditorías SEO, las verificaciones de estilo de redacción y la detección de enlaces rotos no necesitan ejecutarse en cada cambio de código. Ejecutarlos en un cron diario o semanal en vez de en cada push reduce significativamente el consumo de créditos sin un impacto relevante en la calidad del contenido.

**Monitorea tus patrones de uso.** La página de [Uso](https://app.mintlify.com/settings/organization/usage) en tu panel muestra un desglose por categoría de función, con totales separados para el Assistant integrado en el producto y el Assistant API. Si un producto o workflow en particular está consumiendo más créditos de lo esperado, revisa su disparador o cualquier instrucción personalizada.
