<Info>
  Las exportaciones en PDF están disponibles en los [planes Enterprise](https://mintlify.com/pricing).
</Info>

Puedes exportar tu documentación como un único archivo PDF para verla sin conexión. El PDF incluye todo el contenido de la documentación, incluidas las imágenes y los enlaces, y un índice navegable.

Las exportaciones en PDF de todo el sitio incluyen todas las páginas independientemente de las restricciones de grupo, por lo que cualquier persona con acceso al PDF exportado puede leer contenido que, de otro modo, podría estar restringido a grupos de usuarios.

<Tip>
  Para permitir que los usuarios descarguen páginas individuales como PDF, usa la opción `download-pdf` del [menú contextual](/es/ai/contextual-menu).
</Tip>

<div id="export-all-content">
  ## Exportar todo el contenido
</div>

1. Ve a la página [Exports](https://app.mintlify.com/settings/deployment/export-docs) en tu dashboard.
2. Selecciona **Export all content**.
3. Opcionalmente, personaliza las opciones de exportación:
    - **Page format**: Elige el tamaño de página del PDF.
    - **Scale percent**: Ajusta la escala del PDF.
    - **Include footer**: Incluye un pie de página con el número de página y el total de páginas.
    - **Language**: Si tu contenido está disponible en varios idiomas, elige el idioma que quieres exportar.
4. Selecciona **Export**.
5. Cuando se complete la exportación, Mintlify te envía un correo electrónico con un enlace para descargar el archivo PDF.

<div id="export-select-pages">
  ## Exportar páginas específicas
</div>

1. Ve a la página [Exports](https://app.mintlify.com/settings/deployment/export-docs) en tu dashboard.
2. Selecciona **New export**.
3. Selecciona las páginas que quieres exportar.
4. Opcionalmente, personaliza las opciones de exportación:
    - **Page format**: Elige el tamaño de página del PDF.
    - **Scale percent**: Ajusta la escala del PDF.
    - **Include footer**: Incluye un pie de página con el número de página y el total de páginas.
    - **Language**: Si tu contenido está disponible en varios idiomas, elige el idioma que quieres exportar.
5. Selecciona **Export**.
6. Cuando se complete la exportación, Mintlify te envía un correo electrónico con un enlace para descargar el archivo PDF.

<Note>
  Las exportaciones normalmente tardan unos minutos para sitios pequeños y más para sitios grandes. Si no recibes el correo electrónico, consulta la página [Exports](https://app.mintlify.com/settings/deployment/export-docs) en tu dashboard para volver a descargar exportaciones anteriores o reintentar una exportación fallida.

  Las exportaciones en PDF se activan manualmente desde el dashboard. No hay una opción de API ni de exportación programada.
</Note>
