Mintlify maneja automáticamente muchas prácticas recomendadas de SEO, incluyendo:

* Generación de meta etiquetas
* Sitemap y `robots.txt` generación de archivo
* Estructura HTML semántica
* Optimización para móviles

Puedes personalizar completamente las meta etiquetas de tu sitio añadiendo el campo `metatags` a tu `docs.json` o al frontmatter de una página.

## Meta etiquetas globales

Para establecer meta etiquetas predeterminadas para todas las páginas, añade el campo `metatags` a tu `docs.json`.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Meta etiquetas específicas de página

Para establecer meta etiquetas específicas de página, añádelas al frontmatter de la página.

```mdx
---
title: "Your example page title"
"og:image": "link to your meta tag image"
---
```

<Note>
  Las meta etiquetas con dos puntos deben estar entre comillas.
</Note>

## Sitemaps y archivos robots.txt

Mintlify genera automáticamente un archivo `sitemap.xml` y un archivo `robots.txt`. Puedes ver tu sitemap añadiendo `/sitemap.xml` a la URL de tu sitio de documentación.

Solo las páginas incluidas en tu `docs.json` se incluyen por defecto. Para incluir enlaces ocultos, añade `seo.indexing` a tu `docs.json`:

```json
"seo": {
    "indexing": all
}
```

### Sitemaps personalizados y archivos robots.txt

Para añadir un archivo `sitemap.xml` o `robots.txt` personalizado, crea un archivo `sitemap.xml` o `robots.txt` en la raíz de tu proyecto. Añadir un archivo personalizado anulará el archivo generado automáticamente del mismo nombre. Si eliminas un archivo personalizado, se volverá a utilizar el archivo predeterminado.

## Desactivar la indexación

Si quieres evitar que una página sea indexada por los motores de búsqueda, puedes incluir lo siguiente en el [frontmatter](pages) de tu página:

```
---
noindex: true
---
```

También puedes especificar `noindex` para todas las páginas en tu documentación estableciendo el campo `metatags.robots` a `"noindex"` en tu `docs.json`:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```

## Prácticas recomendadas de SEO

<AccordionGroup>
  <Accordion title="Escribe títulos y descripciones descriptivos">
    * Usa títulos de página claros y descriptivos (50-60 caracteres)
    * Escribe descripciones convincentes (150-160 caracteres)
    * Incluye palabras clave relevantes
    * Haz que cada título y descripción de página sea único
  </Accordion>

  <Accordion title="Optimiza la estructura de tu contenido">
    * Usa una jerarquía de encabezados adecuada (H1 → H2 → H3)
    * Escribe primero para humanos, después para motores de búsqueda
    * Incluye palabras clave relevantes en encabezados y contenido
    * Mantén las URLs cortas, descriptivas y organizadas jerárquicamente
    * Divide el contenido largo con subtítulos y listas
  </Accordion>

  <Accordion title="Estrategia de enlaces internos">
    * Enlaza a páginas relacionadas dentro de tu documentación
    * Usa texto de anclaje descriptivo en lugar de "haz clic aquí"
    * Crea grupos temáticos enlazando conceptos relacionados
    * Utiliza las funciones de referencias cruzadas automáticas
  </Accordion>

  <Accordion title="SEO de imágenes">
    * Usa nombres de archivo descriptivos para las imágenes
    * Incluye siempre texto alternativo para accesibilidad y SEO
    * Optimiza los tamaños de archivo de las imágenes para una carga más rápida
    * Usa imágenes relevantes que apoyen tu contenido
  </Accordion>
</AccordionGroup>
