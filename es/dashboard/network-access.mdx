<Info>
  Los controles de acceso a la red requieren un [plan Enterprise](https://mintlify.com/pricing?ref=network-access).
</Info>

Usa una lista de IP permitidas para restringir el acceso al dashboard y al editor a direcciones IP y rangos CIDR de confianza. Cuando activas la lista de permitidos, los miembros de la organización solo pueden iniciar sesión y usar el dashboard y el editor web desde una dirección IP permitida.

La lista de IP permitidas se aplica a tu dashboard y editor de Mintlify. No afecta a tu sitio de documentación publicado. Las solicitudes de API usan controles de IP independientes.

Puedes permitir el acceso de dos formas:

- **Dirección IP individual**: permite una única dirección, como `203.0.113.7`.
- **Rango CIDR**: permite un bloque de direcciones, como `10.0.0.0/8`. Usa la notación CIDR para conceder acceso a toda una oficina o red VPN sin listar cada dirección de forma individual.

<div id="configure-the-ip-allowlist">
  ## Configurar la lista de IP permitidas
</div>

<Warning>
  Añade tu dirección IP actual a la lista de permitidos antes de activarla. Si tu dirección no está en la lista, puedes bloquearte el acceso al dashboard. La página [Network access](https://app.mintlify.com/settings/organization/network-access) muestra tu dirección IP actual para ayudarte a evitarlo.
</Warning>

1. En tu dashboard de Mintlify, ve a la página [Network access](https://app.mintlify.com/settings/organization/network-access).
2. En **Allowed IP addresses**, introduce una dirección IP individual (por ejemplo, `203.0.113.7`) o un rango CIDR (por ejemplo, `10.0.0.0/8`).
3. Haz clic en **Add**.
4. Repite el proceso para cada dirección o rango que quieras permitir.
5. Activa la lista de permitidos para empezar a aplicarla.
