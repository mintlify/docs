---
title: "Roles"
description: "Asigna roles de Propietario, Administrador o Editor para gestionar el acceso y los permisos del equipo."
---

<Info>
  La funcionalidad de RBAC está disponible en los [planes personalizados](https://mintlify.com/pricing?ref=rbac).
</Info>

Mintlify ofrece dos niveles de acceso al dashboard: Editor y Administrador.

A continuación se describen las acciones limitadas al rol de Administrador:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Update user roles       |   ❌   |  ✅   |
| Delete users            |   ❌   |  ✅   |
| Invite admin users      |   ❌   |  ✅   |
| Manage & update billing |   ❌   |  ✅   |
| Update custom domain    |   ❌   |  ✅   |
| Update Git source       |   ❌   |  ✅   |
| Delete org              |   ❌   |  ✅   |

El resto de acciones en el dashboard están disponibles para ambos roles.

Puedes invitar a tantos administradores como quieras, pero recomendamos limitar el acceso de administrador a quienes realmente lo necesiten.
