<Info>
  El control de acceso basado en roles (RBAC) está disponible en [Enterprise plans](https://mintlify.com/pricing?ref=rbac).
</Info>

Mintlify ofrece tres niveles de acceso al dashboard: Viewer, Editor y Admin.

<Note>
  El rol de Viewer está actualmente detrás de una feature flag y no está disponible para todas las organizaciones.
</Note>

A continuación se describen las acciones limitadas por rol:

|                         | Viewer | Editor | Admin |
| ----------------------- | :----: | :----: | :---: |
| Actualizar roles de usuario |   ❌   |   ❌   |  ✅   |
| Eliminar usuarios       |   ❌   |   ❌   |  ✅   |
| Invitar Admins          |   ❌   |   ❌   |  ✅   |
| Invitar Editors         |   ❌   |   ✅   |  ✅   |
| Invitar Viewers         |   ❌   |   ✅   |  ✅   |
| Gestionar y actualizar la facturación |   ❌   |   ❌   |  ✅   |
| Actualizar el domain personalizado |   ❌   |   ❌   |  ✅   |
| Actualizar Git source   |   ❌   |   ❌   |  ✅   |
| Eliminar la organización |   ❌   |   ❌   |  ✅   |

Otras acciones en el dashboard están disponibles para ambos roles.

Puedes invitar a tantos admins como quieras, pero recomendamos limitar el acceso de admin a quienes realmente lo necesiten.
