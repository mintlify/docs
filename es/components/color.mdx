Usa el componente Color para mostrar paletas de colores en tu documentación. Los colores se pueden mostrar en una cuadrícula compacta u organizados en una tabla con filas etiquetadas.

## Variante compacta

Muestra colores en un diseño de cuadrícula simple con nombres y valores de colores.

<Color variant="compact">
  <Color.Item name="blue-500" value="#3B82F6" />
  <Color.Item name="blue-600" value="#2563EB" />
  <Color.Item name="blue-700" value="#1D4ED8" />
  <Color.Item name="blue-800" value="#1E40AF" />
</Color>

```mdx Ejemplo compacto
<Color variant="compact">
  <Color.Item name="blue-500" value="#3B82F6" />
  <Color.Item name="blue-600" value="#2563EB" />
  <Color.Item name="blue-700" value="#1D4ED8" />
  <Color.Item name="blue-800" value="#1E40AF" />
</Color>
```

## Variante tabla

Organiza colores en filas etiquetadas para documentación de sistemas de diseño.

<Color variant="table">
  <Color.Row title="Primario">
    <Color.Item name="primary-500" value="#3B82F6" />
    <Color.Item name="primary-600" value="#2563EB" />
    <Color.Item name="primary-700" value="#1D4ED8" />
  </Color.Row>
  <Color.Row title="Secundario">
    <Color.Item name="secondary-500" value="#8B5CF6" />
    <Color.Item name="secondary-600" value="#7C3AED" />
  </Color.Row>
</Color>

```mdx Ejemplo tabla
<Color variant="table">
```

## Formatos de color

El componente soporta todos los formatos de color CSS incluyendo hex, rgb, rgba, hsl y oklch.

<Color variant="compact">
  <Color.Item name="hex" value="#FF5733" />
  <Color.Item name="rgb" value="rgb(51, 255, 87)" />
  <Color.Item name="rgba" value="rgba(51, 87, 255, 0.7)" />
  <Color.Item name="hsl" value="hsl(180, 70%, 55%)" />
  <Color.Item name="oklch" value="oklch(70% 0.2 145)" />
</Color>

```mdx Ejemplo formatos de color
<Color variant="compact">
  <Color.Item name="hex" value="#FF5733" />
  <Color.Item name="rgb" value="rgb(51, 255, 87)" />
  <Color.Item name="rgba" value="rgba(51, 87, 255, 0.7)" />
  <Color.Item name="hsl" value="hsl(180, 70%, 55%)" />
  <Color.Item name="oklch" value="oklch(70% 0.2 145)" />
</Color>
```

## Colores adaptados al tema

Define diferentes colores para los modos claro y oscuro usando un objeto con propiedades `light` y `dark`.

<Color variant="compact">
  <Color.Item name="bg-primary" value={{ light: "#FFFFFF", dark: "#000000" }} />
  <Color.Item name="bg-secondary" value={{ light: "#F9FAFB", dark: "#0A0A0A" }} />
  <Color.Item name="text-primary" value={{ light: "#111827", dark: "#F9FAFB" }} />
</Color>

```mdx Ejemplo adaptado al tema
<Color variant="compact">
  <Color.Item name="bg-primary" value={{ light: "#FFFFFF", dark: "#000000" }} />
  <Color.Item name="bg-secondary" value={{ light: "#F9FAFB", dark: "#0A0A0A" }} />
  <Color.Item name="text-primary" value={{ light: "#111827", dark: "#F9FAFB" }} />
</Color>
```

## Propiedades

### Color

<ResponseField name="variant" type="string" required>
  Estilo de visualización para la paleta de colores. Opciones: `compact` o `table`.
</ResponseField>

<ResponseField name="children" type="Color.Item | Color.Row" required>
  Elementos de color o filas a mostrar.
</ResponseField>

### Color.Row

<ResponseField name="title" type="string">
  Etiqueta para la fila de colores.
</ResponseField>

<ResponseField name="children" type="Color.Item" required>
  Elementos de color a mostrar en la fila.
</ResponseField>

### Color.Item

<ResponseField name="name" type="string">
  Nombre o etiqueta para el color.
</ResponseField>

<ResponseField name="value" type="string | { light: string, dark: string }" required>
  Valor de color en cualquier formato CSS, o un objeto con valores para modos claro y oscuro.
</ResponseField>
