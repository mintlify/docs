Usa desplegables para mostrar y ocultar contenido anidado dentro de los campos de respuesta. Son especialmente útiles para mostrar propiedades de objetos complejas en la documentación de la API.

<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      El nombre completo del usuario
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Indica si el usuario tiene más de 21 años
    </ResponseField>
  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">

    </ResponseField>
  </Expandable>
</ResponseField>
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="title" type="string">
  El nombre del objeto que se muestra.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Configúralo en `true` para que el elemento expandible se abra al cargar la página.
</ResponseField>
