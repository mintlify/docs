---
- title: Mosaicos
- description: Muestra vistas previas visuales con títulos y descripciones en un
    diseño de cuadrícula.
---

Utilice mosaicos para crear elementos de exhibición visual con un fondo estampado, título y descripción. Los mosaicos son ideales para mostrar vistas previas de componentes, destacar características o elementos de navegación en un diseño de cuadrícula.

## Propiedades

<ResponseField name="href" type="string" required>
  URL para navegar cuando se hace clic en el mosaico.
</ResponseField>

<ResponseField name="title" type="string">
  El título que se muestra debajo de la vista previa del mosaico.
</ResponseField>

<ResponseField name="description" type="string">
  Una breve descripción que se muestra debajo del título.
</ResponseField>

<ResponseField name="children" type="React.ReactNode" required>
  Contenido mostrado dentro del área de vista previa del mosaico, típicamente imágenes o SVGs.
</ResponseField>
