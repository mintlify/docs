Usa el componente `<Graph>` para crear gráficos interactivos y visualizaciones de datos en tu documentación. Los gráficos ayudan a los usuarios a comprender tendencias, comparaciones y distribuciones de datos de un vistazo.

## Uso básico

Envuelve tu configuración de datos en un componente `<Graph>` y especifica el tipo de gráfico.

<Graph type="line" title="Ingresos mensuales">
  {[
    { month: "Ene", revenue: 4000 },
    { month: "Feb", revenue: 3000 },
    { month: "Mar", revenue: 5000 },
    { month: "Abr", revenue: 4500 },
    { month: "May", revenue: 6000 },
    { month: "Jun", revenue: 5500 }
  ]}
</Graph>

```jsx
<Graph type="line" title="Ingresos mensuales">
  {[
    { month: "Ene", revenue: 4000 },
    { month: "Feb", revenue: 3000 },
    { month: "Mar", revenue: 5000 },
    { month: "Abr", revenue: 4500 },
    { month: "May", revenue: 6000 },
    { month: "Jun", revenue: 5500 }

</Graph>
```

## Tipos de gráficos

### Gráfico de líneas

Muestra tendencias a lo largo del tiempo o datos continuos.

<Graph type="line" title="Crecimiento de usuarios" xKey="month" yKey="users">
  {[
    { month: "Ene", users: 1200 },
    { month: "Feb", users: 1900 },
    { month: "Mar", users: 2400 },
    { month: "Abr", users: 3100 },
    { month: "May", users: 4200 },
    { month: "Jun", users: 5100 }
  ]}
</Graph>

<Graph type="line" title="Crecimiento de usuarios" xKey="month" yKey="users">

## Series múltiples

Muestra múltiples series de datos en el mismo gráfico para comparación.

<Graph type="line" title="Ingresos vs Gastos" xKey="month">
  {{
    data: [
      { month: "Ene", revenue: 4000, expenses: 2400 },
      { month: "Feb", revenue: 3000, expenses: 1398 },
      { month: "Mar", revenue: 5000, expenses: 3800 },
      { month: "Abr", revenue: 4500, expenses: 3908 },
      { month: "May", revenue: 6000, expenses: 4800 },
      { month: "Jun", revenue: 5500, expenses: 3800 }
    ],
    series: [
      { key: "revenue", label: "Ingresos", color: "#22c55e" },
      { key: "expenses", label: "Gastos", color: "#ef4444" }
    ]
  }}
</Graph>

```jsx
<Graph type="line" title="Ingresos vs Gastos" xKey="month">
  {{
    data: [
      { month: "Ene", revenue: 4000, expenses: 2400 },
      { month: "Feb", revenue: 3000, expenses: 1398 },
      { month: "Mar", revenue: 5000, expenses: 3800 },
      { month: "Abr", revenue: 4500, expenses: 3908 },

      { month: "Jun", revenue: 5500, expenses: 3800 }

```

## Props

<ResponseField name="type" type="string" required>
  El tipo de gráfico. Opciones: `line`, `bar`, `area`, `pie`, `donut`.
</ResponseField>

<ResponseField name="title" type="string">
  El título del gráfico que se muestra sobre la visualización.
</ResponseField>

<ResponseField name="xKey" type="string">
  La clave de datos a usar para el eje x (para gráficos de líneas, barras y área).
</ResponseField>

<ResponseField name="yKey" type="string">
  La clave de datos a usar para el eje y (para gráficos de serie única).
</ResponseField>

<ResponseField name="height" type="number" default="300">
  La altura del gráfico en píxeles.
</ResponseField>

<ResponseField name="color" type="string">
  El color principal para gráficos de serie única. Acepta hex, RGB o nombres de colores CSS.
</ResponseField>

<ResponseField name="stacked" type="boolean" default="false">
  Apilar múltiples series una encima de otra (para gráficos de barras y área).
</ResponseField>

<ResponseField name="showLegend" type="boolean" default="true">
  Mostrar u ocultar la leyenda del gráfico.
</ResponseField>

<ResponseField name="showGrid" type="boolean" default="true">
  Mostrar u ocultar las líneas de cuadrícula.
</ResponseField>

<ResponseField name="children" type="array | object" required>
  Los datos del gráfico. Pasa un array para serie única o un objeto con `data` y `series` para múltiples series.
</ResponseField>
