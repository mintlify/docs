Usa el componente `Visibility` para mostrar u ocultar contenido según la audiencia. Puedes dirigir contenido específicamente a lectores humanos o agentes de IA.

<Visibility for="humans">
  Este contenido solo es visible para lectores humanos en el navegador.
</Visibility>

<Visibility for="agents">
  Este contenido solo es visible para agentes de IA que consumen la exportación Markdown.
</Visibility>

## Ejemplo

````mdx Contenido solo para humanos y solo para agentes
<Visibility for="humans">
  Este párrafo solo se muestra a los lectores humanos que
  visitan la página en un navegador.
</Visibility>

<Visibility for="agents">
  Este párrafo solo se incluye cuando los agentes de IA
  solicitan la página a través de la exportación Markdown.
</Visibility>
````

## Cómo funciona

Mintlify detecta si el visitante actual es un humano o un agente de IA y muestra el contenido en consecuencia:

- **En el navegador** — Mintlify evalúa el user agent del visitante en el servidor. El contenido dentro de `<Visibility for="humans">` se muestra a los visitantes normales, mientras que el contenido de `<Visibility for="agents">` se muestra cuando se detecta un agente de IA.
- **En la [exportación Markdown](/es/ai/markdown-export)** — cuando se solicita una página como Markdown (mediante la extensión `.md` o el encabezado `Accept: text/markdown`), solo se incluye el contenido marcado con `for="agents"`. El contenido marcado con `for="humans"` se elimina de la salida.

El contenido fuera de cualquier componente `Visibility` siempre se muestra a ambas audiencias.

## Propiedades

<ResponseField name="for" type='"agents" | "humans"' required>
  La audiencia objetivo del contenido envuelto. También acepta las formas en singular `"agent"` y `"human"`.
</ResponseField>
