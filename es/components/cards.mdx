Usa tarjetas para crear contenedores visuales para el contenido. Las tarjetas son contenedores flexibles que pueden incluir texto, iconos, imágenes y enlaces.

<div id="card-variations">
  ## Variantes de tarjetas
</div>

Las tarjetas admiten varias opciones de diseño y estilo para adaptarse a diferentes necesidades de contenido.

<div id="horizontal-layout">
  ### Diseño horizontal
</div>

Agrega la propiedad `horizontal` para mostrar las tarjetas en un diseño horizontal más compacto.

</Card>

```mdx Horizontal card example
<Card title="Tarjeta horizontal" icon="text-align-start" horizontal>

```

</div>

<div id="feature-icon-placement">
  ### Ubicación de icono destacado
</div>

Establece `iconPlacement="feature"` para mostrar el icono centrado sobre un área destacada en la parte superior de la tarjeta. El área destacada incluye una banda de color detrás del icono, y al combinarlo con `img`, la imagen rellena el área con el icono superpuesto.

<Columns cols={2}>
  <Card title="Icono destacado" icon="sparkles" iconPlacement="feature">
    Una tarjeta con el icono mostrado en un área destacada.
  </Card>
  <Card title="Destacado con imagen" icon="sparkles" iconPlacement="feature" img="https://mintlify-assets.b-cdn.net/yosemite.jpg">
    Una tarjeta con un icono superpuesto sobre una imagen destacada.
  </Card>
</Columns>

```mdx Feature icon placement examples
<Card title="Icono destacado" icon="sparkles" iconPlacement="feature">
  Una tarjeta con el icono mostrado en un área destacada.
</Card>

<Card
```

<div id="dynamic-cards">
  ### Tarjetas dinámicas
</div>

Agrega la propiedad `dynamic` para rellenar automáticamente el título, icono, descripción y CTA de una tarjeta a partir de los metadatos de navegación de la página enlazada. Las propiedades establecidas explícitamente tienen prioridad sobre los valores resueltos.

```mdx Dynamic card example
<Card dynamic href="/components/accordion" />
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="title" type="string">
  El título que se muestra en la tarjeta.
</ResponseField>

<IconsOptional />

<ResponseField name="color" type="string">
  Color del ícono como código hexadecimal (por ejemplo, `#FF6B6B`).
</ResponseField>

<ResponseField name="href" type="string">
  URL a la que se navega cuando se hace clic en la tarjeta.
</ResponseField>

<ResponseField name="iconPlacement" type="string" default="default">
  Controla dónde se muestra el icono. Establece `"default"` para mostrarlo en la fila del título, o `"feature"` para mostrarlo centrado sobre un área destacada en la parte superior de la tarjeta.
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  Muestra la tarjeta en un diseño horizontal compacto.
</ResponseField>

<ResponseField name="img" type="string">
  URL o ruta local a una imagen que se muestra en la parte superior de la tarjeta. Cuando `iconPlacement` es `"feature"`, la imagen rellena el área destacada con el icono superpuesto.
</ResponseField>

<ResponseField name="dynamic" type="boolean">
  Rellena automáticamente el título, icono, descripción y CTA de la tarjeta a partir de los metadatos de navegación de la página enlazada. Requiere que `href` apunte a una ruta de página interna. Las propiedades establecidas explícitamente tienen prioridad.
</ResponseField>

<ResponseField name="cta" type="string">
  Texto personalizado para el botón de acción.
</ResponseField>

<ResponseField name="arrow" type="boolean">
  Mostrar u ocultar el ícono de flecha del enlace.
</ResponseField>
