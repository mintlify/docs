---
title: "Tooltips"
description: "Muestra una definición al pasar el cursor sobre el texto"
---

Usa los tooltips para ofrecer contexto adicional o definiciones cuando un usuario pasa el cursor sobre una cadena de texto. Los tooltips pueden incluir enlaces opcionales de llamada a la acción.

**Ejemplo**: <Tooltip tip="Interfaz de programación de aplicaciones: un conjunto de protocolos para que las aplicaciones de software puedan comunicarse." cta="Lee nuestra guía de API" href="/es/api-reference">API</Tooltip> La documentación ayuda a los desarrolladores a entender cómo integrarse con tu servicio.

```mdx Tooltip example wrap
<Tooltip tip="Interfaz de Programación de Aplicaciones: un conjunto de protocolos para que las aplicaciones de software se comuniquen." cta="Lee nuestra guía de API" href="/api-reference">API</Tooltip> La documentación de API ayuda a los desarrolladores a entender cómo integrar con tu servicio.
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="tip" type="string" required>
  El texto que se muestra en el tooltip.
</ResponseField>

<ResponseField name="cta" type="string">
  El texto de la llamada a la acción para un enlace dentro del tooltip.
</ResponseField>

<ResponseField name="href" type="string">
  URL del enlace de la llamada a la acción. Obligatorio cuando se usa `cta`.
</ResponseField>
