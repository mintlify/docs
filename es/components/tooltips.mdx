---
title: "Tooltips"
description: "Añade ayuda contextual con tooltips al pasar el cursor sobre términos y conceptos."
---

Usa los tooltips para proporcionar contexto adicional o definiciones cuando el usuario pasa el cursor sobre un texto. Los tooltips pueden incluir enlaces de llamada a la acción opcionales.

**Ejemplo**: <Tooltip tip="Interfaz de programación de aplicaciones: conjunto de protocolos que permiten la comunicación entre aplicaciones de software." cta="Lee nuestra guía de API" href="/es/api-reference">API</Tooltip> La documentación ayuda a los desarrolladores a entender cómo integrarse con tu servicio.

```mdx Tooltip example wrap
<Tooltip tip="Interfaz de Programación de Aplicaciones: un conjunto de protocolos para que las aplicaciones de software se comuniquen." cta="Lee nuestra guía de API" href="/api-reference">API</Tooltip> La documentación de API ayuda a los desarrolladores a comprender cómo integrarse con tu servicio.
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="tip" type="string" required>
  El texto que se muestra en el tooltip.
</ResponseField>

<ResponseField name="cta" type="string">
  El texto de la llamada a la acción para un enlace dentro del tooltip.
</ResponseField>

<ResponseField name="href" type="string">
  URL del enlace de la llamada a la acción. Obligatorio cuando se usa `cta`.
</ResponseField>
