---
title: "Tooltips"
description: "Muestra una definición al pasar el cursor sobre el texto"
icon: "message-square"
---

Usa tooltips para proporcionar contexto adicional o definiciones cuando un usuario pasa el cursor sobre un fragmento de texto. Los tooltips pueden incluir enlaces opcionales con llamada a la acción.

**Ejemplo**: <Tooltip tip="Interfaz de programación de aplicaciones: un conjunto de protocolos que permite que las aplicaciones de software se comuniquen." cta="Lee nuestra guía de API" href="/es/api-reference">API</Tooltip> La documentación ayuda a los desarrolladores a entender cómo integrarse con tu servicio.

```mdx Envoltura de ejemplo de tooltip
<Tooltip tip="Interfaz de programación de aplicaciones: conjunto de protocolos que permite la comunicación entre aplicaciones de software." cta="Lee nuestra guía de API" href="/api-reference">API</Tooltip> la documentación ayuda a los desarrolladores a entender cómo integrarse con tu servicio.
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="tip" type="string" required>
  El texto que se muestra en el indicador (tooltip).
</ResponseField>

<ResponseField name="cta" type="string">
  El texto del llamado a la acción para un enlace dentro del indicador (tooltip).
</ResponseField>

<ResponseField name="href" type="string">
  URL del enlace del llamado a la acción. Obligatorio cuando se usa `cta`.
</ResponseField>
