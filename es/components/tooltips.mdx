Usa los tooltips para proporcionar contexto adicional o definiciones cuando el usuario pasa el cursor sobre un fragmento de texto. Los tooltips pueden incluir enlaces de llamada a la acción opcionales.

**Ejemplo**: <Tooltip headline="API" tip="API (Interfaz de programación de aplicaciones): un conjunto de protocolos para que las aplicaciones de software se comuniquen entre sí." cta="Lee nuestra guía de API" href="/es/api-reference">API</Tooltip> La documentación de API ayuda a los desarrolladores a entender cómo integrarse con tu servicio.

```mdx Tooltip example wrap
<Tooltip headline="API" tip="API (Interfaz de programación de aplicaciones): un conjunto de protocolos para que las aplicaciones de software se comuniquen entre sí." cta="Lee nuestra guía de API" href="/api-reference">API</Tooltip> La documentación ayuda a los desarrolladores a entender cómo integrarse con tu servicio.
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="tip" type="string" required>
  El texto que se muestra en el tooltip.
</ResponseField>

<ResponseField name="headline" type="string">
  Texto que se muestra antes del texto de `tip`.
</ResponseField>

<ResponseField name="cta" type="string">
  El texto de la llamada a la acción para un enlace dentro del tooltip.
</ResponseField>

<ResponseField name="href" type="string">
  URL del enlace de la llamada a la acción. Obligatorio cuando se usa `cta`.
</ResponseField>
