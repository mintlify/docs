---

El componente `Columns` te permite agrupar varios componentes `Card`. Se utiliza con mayor frecuencia para disponer tarjetas en una cuadrícula, especificando el número de columnas.

<Columns cols={2}>
  <Card title="Get started" icon="rocket">
    Configura tu proyecto con nuestra guía de inicio rápido.
  </Card>

  <Card title="API reference" icon="code">
    Explora endpoints, parámetros y ejemplos de tu API.
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="Comenzar">
    Configure su proyecto con nuestra guía de inicio rápido.
  </Card>
  <Card title="Referencia de API">
    Explore endpoints, parámetros y ejemplos para su API.
  </Card>
</Columns>
```


<div id="properties">
  ## Propiedades
</div>

<ResponseField name="cols" default={2}>
  El número de columnas por fila.
</ResponseField>
