---
title: 'Columnas'
description: 'Muestra tarjetas en paralelo en una cuadrícula'
keywords: ['grupos de tarjetas']
---

El componente `Columns` te permite agrupar varios componentes `Card`. Se usa con mayor frecuencia para colocar tarjetas en una cuadrícula, especificando el número de columnas.

<Columns cols={2}>
  <Card title="Get started" icon="rocket">
    Configura tu proyecto con nuestra guía de inicio rápido.
  </Card>

  <Card title="API reference" icon="code">
    Explora endpoints, parámetros y ejemplos para tu API.
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="Comenzar">
    Configura tu proyecto con nuestra guía de inicio rápido.
  </Card>
  <Card title="Referencia de API">
    Explora endpoints, parámetros y ejemplos para tu API.
  </Card>
</Columns>
```


<div id="properties">
  ## Propiedades
</div>

<ResponseField name="cols" default={2}>
  El número de columnas por fila.
</ResponseField>
