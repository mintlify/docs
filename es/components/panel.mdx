title: "Panel"
description: "Personaliza el contenido del panel lateral derecho en las páginas de documentación para mostrar información complementaria, ejemplos o elementos de navegación."
keywords: ["sidebar"]
---

Puedes usar el componente `<Panel>` para personalizar el panel del lado derecho de una página con cualquier componente que desees.

Si una página tiene un componente `<Panel>`, los componentes [RequestExample](/es/components/examples#requestexample) y [ResponseExample](/es/components/examples#responseexample) deben estar dentro de `<Panel>`.

Los componentes en un `<Panel>` reemplazan la tabla de contenido de una página.

```mdx
<Panel>
  <Info>Ancla información al panel lateral o agrega cualquier otro componente.</Info>
</Panel>
```
