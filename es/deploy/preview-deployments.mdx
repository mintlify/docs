title: "Implementaciones de vista previa"
description: "Previsualiza cambios en tu documentación en una implementación en vivo"
---

<Info>
  Las implementaciones de vista previa están disponibles en los [planes Pro y Enterprise](https://mintlify.com/pricing?ref=preview-deployments).
</Info>

Las implementaciones de vista previa te permiten ver cómo se verán los cambios en tu documentación antes de fusionarlos en producción. Cada vista previa crea una URL para compartir que se actualiza automáticamente a medida que haces push de nuevos cambios.

<div id="creating-preview-deployments">
  ## Creación de implementaciones de vista previa
</div>

Las implementaciones de vista previa se crean automáticamente mediante solicitudes de extracción o manualmente desde tu dashboard.

<div id="automatic-previews">
  ### Vistas previas automáticas
</div>

Cuando creas una solicitud de extracción, el bot de Mintlify agrega automáticamente un enlace para ver la implementación de vista previa en tu solicitud de extracción. La vista previa se actualiza cada vez que haces push de nuevas confirmaciones a la branch.

<div id="manual-previews">
  ### Vistas previas manuales
</div>

Puedes crear manualmente una vista previa para cualquier branch.

1. Ve a tu [dashboard](https://dashboard.mintlify.com/).
2. Selecciona **Previews**.
3. Selecciona **Create preview**.
4. Ingresa el nombre de tu branch.
5. Selecciona **Create deployment**.

<div id="redeploying-a-preview">
  ## Volver a implementar una vista previa
</div>

Vuelve a implementar una vista previa para actualizar el contenido o volver a intentarlo tras una implementación fallida.

1. Selecciona la vista previa en tu [dashboard](https://dashboard.mintlify.com/).
2. Selecciona **Volver a implementar**.

<div id="sharing-preview-deployments">
  ## Compartir implementaciones de vista previa
</div>

Las implementaciones de vista previa pueden ser de acceso público o estar restringidas a usuarios autenticados.

**Acceso público**: Las URL de vista previa son públicas de forma predeterminada. Comparte un enlace de vista previa con cualquiera que necesite revisar tus cambios.

**Acceso restringido**: Activa la autenticación para implementaciones de vista previa para limitar el acceso únicamente a usuarios autenticados de Mintlify. Consulta [Configuración de Autenticación](/es/deploy/authentication-setup) para obtener más información.
