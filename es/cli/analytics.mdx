El comando `mint analytics` te da acceso a las [analiticas de tu documentacion](/es/optimize/analytics) desde la terminal. Esto es util para crear informes de analiticas mediante scripts, canalizar datos hacia otras herramientas y realizar consultas rapidas sin abrir el panel de control.

<div id="prerequisites">

## Requisitos previos

</div>

Debes iniciar sesion para usar los comandos de analiticas. Ejecuta `mint login` para autenticarte con tu cuenta de Mintlify.

Consulta [Vista previa local](/es/cli/preview#log-in-for-search-and-assistant) para mas detalles sobre la autenticacion.

<div id="view-key-metrics">

## Ver metricas clave

</div>

Muestra un resumen de vistas, visitantes, busquedas, comentarios y uso del asistente:

```bash
mint analytics stats
```

Usa flags para filtrar por rango de fechas, pagina u origen del trafico:

- `--from`: Fecha de inicio en formato `YYYY-MM-DD`. Por defecto, hace 7 dias.
- `--to`: Fecha de fin en formato `YYYY-MM-DD`. Por defecto, hoy.
- `--page`: Filtrar por una ruta de pagina especifica.
- `--humans`: Mostrar solo trafico humano.
- `--agents`: Mostrar solo trafico de IA.

```bash Example analytics stats flags
mint analytics stats --from 2025-01-01 --to 2025-01-31
mint analytics stats --page /quickstart
mint analytics stats --humans
mint analytics stats --agents
```

<div id="view-search-analytics">

## Ver analiticas de busqueda

</div>

Consulta lo que tus usuarios estan buscando, con recuentos de resultados y tasas de clics:

```bash
mint analytics search
```

Usa flags para filtrar por cadena de consulta o pagina:

- `--query`: Filtrar por subcadena de consulta de busqueda.
- `--page`: Filtrar por una ruta de pagina especifica.

```bash Example analytics search flags
mint analytics search --query "authentication"

mint analytics search --page /api-reference
```

<div id="view-feedback">

## Ver comentarios

</div>

Revisa los comentarios enviados por los usuarios en las paginas de tu documentacion:

```bash
mint analytics feedback
```

Usa flags para agrupar comentarios por pagina o filtrar por comentarios de fragmentos de codigo:

- `--type`: Agrupar comentarios por pagina o fragmento de codigo. Por defecto, pagina.
- `--page`: Filtrar por una ruta de pagina especifica.

```bash Example analytics feedback flags
mint analytics feedback --type page
mint analytics feedback --type code
```

<div id="view-assistant-conversations">

## Ver conversaciones del asistente

</div>

Lista las conversaciones recientes y las categorias de conversaciones del asistente de IA:

```bash
mint analytics conversation list
mint analytics conversation buckets list
```

Cada salida de lista incluye un ID para cada entrada. Pasa ese ID al comando `view` correspondiente para ver los detalles completos:

```bash
mint analytics conversation view <conversation-id>
mint analytics conversation buckets view <bucket-id>
```

<div id="choose-an-output-format">

## Elegir un formato de salida

</div>

Todos los comandos de analiticas usan por defecto la salida `plain`, que esta separada por tabulaciones y es adecuada para canalizar hacia otras herramientas. Usa `--format` para cambiar la salida:

```bash
mint analytics stats --format table    # Pretty-printed table with colors
mint analytics stats --format json     # Raw JSON
mint analytics stats --format graph    # Horizontal bar chart
```

Usa el flag `--agent` para salida en JSON. La CLI detecta automaticamente los entornos de agentes cuando la variable de entorno `CLAUDECODE` es `1`.

<div id="set-defaults-with-config">

## Establecer valores predeterminados con config

</div>

Para evitar repetir flags comunes, establece valores predeterminados con `mint config`. La configuracion se guarda en `~/.config/mintlify/config.json`.

Claves de configuracion disponibles:

| Key | Description |
| --- | --- |
| `subdomain` | Subdominio predeterminado para los comandos de analiticas. |
| `dateFrom` | Fecha de inicio predeterminada para las consultas de analiticas (`YYYY-MM-DD`). |
| `dateTo` | Fecha de fin predeterminada para las consultas de analiticas (`YYYY-MM-DD`). |

```bash Example configs
mint config set subdomain my-docs
mint config set dateFrom 2025-01-01
```

Una vez establecidos, los comandos de analiticas usan estos valores automaticamente:

```bash
# Uses the configured subdomain and dateFrom without flags
mint analytics stats
```

Para ver o borrar un valor, ejecuta:

```bash
mint config get subdomain
mint config clear subdomain
```
