<div id="about-the-cli">
## Acerca del CLI
</div>

Usa el [CLI](https://www.npmjs.com/package/mint) para previsualizar tu documentación localmente, probar cambios en tiempo real, analizar el rendimiento del sitio y ejecutar verificaciones de calidad antes de desplegar a producción.

El CLI se ejecuta en tu máquina y se conecta a tu despliegue de Mintlify cuando necesitas datos en vivo.

<div id="preview-locally">
### Previsualización local
</div>

Ejecuta `mint dev` para previsualizar tu documentación en `localhost:3000`. A medida que escribes y actualizas tu contenido, los cambios aparecen en tiempo real sin necesidad de desplegar. Inicia sesión para habilitar la búsqueda y el asistente de IA en tu previsualización local.

<div id="analyze-your-documentation">
### Analiza tu documentación
</div>

Ejecuta `mint analytics` para ver el tráfico, consultas de búsqueda, comentarios de usuarios y conversaciones del asistente de IA directamente desde la terminal. Útil para verificaciones rápidas, scripting y canalizar datos hacia otras herramientas.

<div id="run-quality-checks">
### Ejecuta verificaciones de calidad
</div>

Ejecuta `mint broken-links` para encontrar enlaces rotos, `mint a11y` para verificar la accesibilidad y `mint validate` para validar la compilación de tu documentación antes de que llegue a producción.

<div id="manage-configuration">
### Gestiona la configuración
</div>

Ejecuta `mint config` para establecer valores predeterminados persistentes como el subdominio de tu documentación y los rangos de fechas para los comandos de analítica.
