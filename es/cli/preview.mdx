<div id="start-the-local-preview">
  ## Iniciar la previsualización local
</div>

Navega al directorio de tu documentación que contiene el archivo `docs.json` y ejecuta:

```bash
mint dev
```

La previsualización local se inicia en `http://localhost:3000`. A medida que editas, los cambios aparecen en tiempo real.

Por defecto, la previsualización se abre automáticamente en tu navegador. Para evitar que el navegador se abra, usa el flag `--no-open`:

```bash
mint dev --no-open
```

Para generar una previsualización sin instalar la CLI globalmente, ejecuta:

```bash
npx mint dev
```

<div id="log-in-for-search-and-assistant">
  ## Iniciar sesión para búsqueda y asistente
</div>

Debes autenticar la CLI con tu cuenta de Mintlify para habilitar la búsqueda y el [asistente](/es/assistant/index).

```bash
mint login
```

Después de autenticarte con el comando `mint login`, la CLI almacena tus credenciales en `~/.config/mintlify/config.json` para que permanezcas conectado entre sesiones. La CLI también te solicita que selecciones un proyecto predeterminado, que comandos como `mint analytics` utilizan.

Una vez que hayas iniciado sesión, ejecuta `mint dev` para iniciar la previsualización local con la búsqueda y el asistente habilitados. El asistente usa el mismo contenido indexado que tu sitio de documentación desplegado.

Para verificar el estado de tu autenticación, ejecuta:

```bash
mint status
```

Para cerrar sesión, ejecuta:

```bash
mint logout
```

<div id="custom-ports">
  ## Puertos personalizados
</div>

Por defecto, la CLI usa el puerto 3000. Para usar un puerto diferente, usa el flag `--port`:

```bash
mint dev --port 3333
```

Si el puerto ya está en uso, la CLI intenta automáticamente el siguiente puerto disponible.

<div id="skip-openapi-processing">
  ## Omitir el procesamiento de OpenAPI
</div>

Si tienes muchos archivos OpenAPI, omite el procesamiento de OpenAPI durante el desarrollo local para mejorar el rendimiento con el flag `--disable-openapi`:

```bash
mint dev --disable-openapi
```

<div id="preview-as-a-specific-group">
  ## Previsualizar como un grupo específico
</div>

Si usas control de acceso basado en grupos, previsualiza como un grupo de autenticación específico con el flag `--groups`:

```bash
mint dev --groups admin
```

Consulta [Configuración de autenticación](/es/deploy/authentication-setup#control-access-with-groups) para más información sobre grupos.
