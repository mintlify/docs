<div id="mint-automations">

</div>

Crea, lista y elimina [automatizaciones](/es/automations) desde la terminal. Requiere autenticaciﾃｳn con `mint login`.

Todos los subcomandos aceptan estos flags compartidos:

| Flag | Descripciﾃｳn |

<div id="mint-automations-create">
  ### `mint automations create`
</div>

```bash

```

| Flag | Descripciﾃｳn |
| --- | --- |
| `--name` | Nombre de la automatizaciﾃｳn. Obligatorio salvo que se proporcione `--file`. |
| `--prompt` | Instrucciones que se aﾃｱaden al prompt base de la automatizaciﾃｳn en cada ejecuciﾃｳn. |

| `--cron` | Expresiﾃｳn cron para un trigger programado. Mutuamente excluyente con `--push-repo`. |
| `--push-repo` | Repositorio (`owner/repo`) para un trigger de push. Repetible para escuchar varios repositorios. Mutuamente excluyente con `--cron`. |
| `--context-repo` | Repositorio de contexto adicional (`owner/repo`) que el agente lee cuando se ejecuta la automatizaciﾃｳn. Repetible, hasta 10 en total. |
| `--automerge` | Fusiona automﾃ｡ticamente las pull requests abiertas por esta automatizaciﾃｳn. Consulta [Configurar automerge](/es/guides/configure-automerge) para los requisitos de configuraciﾃｳn. |
| `--file` | Ruta a un archivo JSON o YAML con el cuerpo completo de la automatizaciﾃｳn. Anula los flags en lﾃｭnea. |

Proporciona exactamente un trigger: pasa `--cron` para una automatizaciﾃｳn programada o uno o mﾃ｡s flags `--push-repo` para una automatizaciﾃｳn activada por push.

<div id="examples">
  #### Ejemplos
</div>

```bash

mint automations create \
  --name "Translate content" \
  --type translations \
  --cron "0 6 * * *"

# Automatizaciﾃｳn activada por push con contexto adicional
mint automations create \
  --name "Sync API reference" \
  --type source-code-agent \
  --push-repo my-org/api \
  --context-repo my-org/shared-types \
  --automerge

# Crear desde un archivo
mint automations create --file automation.yaml
```

Un archivo de automatizaciﾃｳn usa la misma forma que los flags en lﾃｭnea. El campo `on` contiene el trigger:

```yaml
name: Translate content
type: translations
on:
  cron: "0 6 * * *"
prompt: Prefer formal tone in French translations.
automerge: false
context:
  - repo: my-org/shared-content
```

<div id="telemetry">
  ## Telemetrﾃｭa
</div>

La CLI recopila telemetrﾃｭa anﾃｳnima de uso para ayudar a mejorar Mintlify. Los datos de telemetrﾃｭa incluyen el nombre del comando, la versiﾃｳn de la CLI, el sistema operativo y la arquitectura. ```bash

| Variable | Valor | Descripciﾃｳn |
| --- | --- | --- |
| `MINTLIFY_TELEMETRY_DISABLED` | `1` | Deshabilitar la telemetrﾃｭa de la CLI de Mintlify. |
| `DO_NOT_TRACK` | `1` | Deshabilitar la telemetrﾃｭa usando el estﾃ｡ndar [Console Do Not Track](https://consoledonottrack.com/). |

Tu preferencia se guarda en `~/.config/mintlify/config.json` y persiste entre sesiones de la CLI.
