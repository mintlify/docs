## Acerca de los servidores MCP

El Protocolo de Contexto de Modelo (MCP) es un protocolo abierto que crea conexiones estandarizadas entre aplicaciones de IA y servicios externos, como documentación. Mintlify genera un servidor MCP a partir de tu documentación, preparando tu contenido para el ecosistema más amplio de IA donde cualquier cliente MCP (como Claude, Cursor, Goose y otros) puede conectarse a tu documentación.

Tu servidor MCP expone una herramienta de búsqueda para que las aplicaciones de IA consulten tu documentación.

### Cómo funcionan los servidores MCP

Cuando un servidor MCP está conectado a una herramienta de IA, el LLM puede decidir utilizar tu herramienta de búsqueda de documentación durante la generación de respuestas.

* El LLM puede buscar proactivamente en tu documentación mientras genera una respuesta, no solo cuando se le pregunta explícitamente.
* El LLM determina cuándo usar la herramienta de búsqueda basándose en el contexto de la conversación y la relevancia de tu documentación.
* Cada llamada a la herramienta ocurre durante el proceso de generación, permitiendo que el LLM incorpore información en tiempo real de tu documentación en su respuesta.

Por ejemplo, si un usuario hace una pregunta sobre programación y el LLM determina que tu documentación es relevante, puede buscar en tu documentación e incluir esa información en la respuesta sin que el usuario pregunte explícitamente sobre tu documentación.

## Accediendo a tu servidor MCP

<Note>
  Los servidores MCP solo pueden generarse para documentación pública. La documentación protegida por autenticación de usuario final no puede ser accedida para la generación del servidor.
</Note>

Mintlify genera automáticamente un servidor MCP para tu documentación y lo aloja en la URL de tu documentación con la `/mcp` ruta. Por ejemplo, el servidor MCP de Mintlify está disponible en `https://mintlify.com/docs/mcp`.

Puedes ver y copiar la URL de tu servidor MCP en tu [dashboard](https://dashboard.mintlify.com/products/mcp).

La `/mcp` ruta está reservada para servidores MCP alojados y no puede utilizarse para otros elementos de navegación.

## Usando tu servidor MCP

Tus usuarios deben conectar tu servidor MCP a sus herramientas de IA preferidas.

1. Haz que la URL de tu servidor MCP esté disponible públicamente.
2. Los usuarios copian la URL de tu servidor MCP y la añaden a sus herramientas.
3. Los usuarios acceden a tu documentación a través de sus herramientas.

Estas son algunas de las formas en que puedes ayudar a tus usuarios a conectarse a tu servidor MCP:

<Tabs>
  <Tab title="Menú contextual">
    Añade opciones en el [menú contextual](/es/ai/contextual-menu) para que tus usuarios se conecten a tu servidor MCP desde cualquier página de tu documentación.

    | Opción                  | Identificador | Descripción                                                  |
    | :---------------------- | :------------ | :----------------------------------------------------------- |
    | **Copy MCP server URL** | `mcp`         | Copia la URL de tu servidor MCP al portapapeles del usuario. |
    | **Connect to Cursor**   | `cursor`      | Instala tu servidor MCP en Cursor.                           |
    | **Connect to VS Code**  | `vscode`      | Instala tu servidor MCP en VS Code.                          |
  </Tab>

  <Tab title="Claude">
    <Steps>
      <Step title="Obtén la URL de tu servidor MCP.">
        Navega a tu [dashboard](https://dashboard.mintlify.com/products/mcp) y encuentra la URL de tu servidor MCP.
      </Step>

      <Step title="Publica la URL de tu servidor MCP para tus usuarios.">
        Crea una guía para tus usuarios que incluya la URL de tu servidor MCP y los pasos para conectarlo a Claude.

        1. Navega a la página [Connectors](https://claude.ai/settings/connectors) en la configuración de Claude.
        2. Selecciona **Add custom connector**.
        3. Añade el nombre y la URL de tu servidor MCP.
        4. Selecciona **Add**.
        5. Al usar Claude, selecciona el botón de adjuntos (el icono de más).
        6. Selecciona tu servidor MCP.
      </Step>
    </Steps>

    Consulta la [Model Context Protocol documentation](https://modelcontextprotocol.io/docs/tutorials/use-remote-mcp-server#connecting-to-a-remote-mcp-server) para más detalles.
  </Tab>

  <Tab title="Claude Code">
    <Steps>
      <Step title="Obtén la URL de tu servidor MCP.">
        Navega a tu [dashboard](https://dashboard.mintlify.com/products/mcp) y encuentra la URL de tu servidor MCP.
      </Step>

      <Step title="Publica la URL de tu servidor MCP para tus usuarios.">
        Crea una guía para tus usuarios que incluya la URL de tu servidor MCP y el comando para conectarlo a Claude Code.

        ```bash
        claude mcp add --transport http <name> <url>
        ```
      </Step>
    </Steps>

    Consulta la [Claude Code documentation](https://docs.anthropic.com/en/docs/claude-code/mcp#installing-mcp-servers) para más detalles.
  </Tab>

  <Tab title="Cursor">
    <Steps>
      <Step title="Obtén la URL de tu servidor MCP.">
        Navega a tu [dashboard](https://dashboard.mintlify.com/products/mcp) y encuentra la URL de tu servidor MCP.
      </Step>

      <Step title="Publica la URL de tu servidor MCP para tus usuarios.">
        Crea una guía para tus usuarios que incluya la URL de tu servidor MCP y los pasos para conectarlo a Cursor.

        1. Usa <kbd>Command</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> (<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> en Windows) para abrir la paleta de comandos.
        2. Busca "Open MCP settings".
        3. Selecciona **Add custom MCP**. Esto abrirá el archivo `mcp.json`.
        4. En `mcp.json`, configura tu servidor:

        ```json
        {
          "mcpServers": {
            "<your-mcp-server-name>": {
              "url": "<your-mcp-server-url>"
            }
          }
        }
        ```
      </Step>
    </Steps>

    Consulta la [Cursor documentation](https://docs.cursor.com/en/context/mcp#installing-mcp-servers) para más detalles.
  </Tab>

  <Tab title="VS Code">
    <Steps>
      <Step title="Obtén la URL de tu servidor MCP.">
        Navega a tu [dashboard](https://dashboard.mintlify.com/products/mcp) y encuentra la URL de tu servidor MCP.
      </Step>

      <Step title="Publica la URL de tu servidor MCP para tus usuarios.">
        Crea una guía para tus usuarios que incluya la URL de tu servidor MCP y los pasos para conectarlo a VS Code.

        1. Crea un archivo `.vscode/mcp.json`.
        2. En `mcp.json`, configura tu servidor:

        ```json
        {
          "servers": {
            "<your-mcp-server-name>": {
              "type": "http",
              "url": "<your-mcp-server-url>"
            }
          }
        }
        ```
      </Step>
    </Steps>

    Consulta la [VS Code documentation](https://code.visualstudio.com/docs/copilot/chat/mcp-servers) para más detalles.
  </Tab>
</Tabs>

### Example: Connecting to the Mintlify MCP server

Conéctate al servidor MCP de Mintlify para buscar en nuestra documentación. Esto te dará respuestas más precisas sobre cómo usar Mintlify en tu entorno local y demuestra cómo puedes ayudar a tus usuarios a conectarse a tu servidor MCP.

<Tabs>
  <Tab title="Menú contextual">
    En la parte superior de esta página, selecciona el menú contextual y elige **Connect to Cursor** o **Connect to VS Code** para conectar el servidor MCP de Mintlify al IDE de tu elección.
  </Tab>

  <Tab title="Claude">
    Para usar el servidor MCP de Mintlify con Claude:

    <Steps>
      <Step title="Añade el servidor MCP de Mintlify a Claude">
        1. Navega a la página [Connectors](https://claude.ai/settings/connectors) en la configuración de Claude.
        2. Selecciona **Agregar conector personalizado**.
        3. Agregar el servidor MCP de Mintlify:

        * Name: `Mintlify`
        * URL: `https://mintlify.com/docs/mcp`

        4. Seleccionar**Agregar**.
      </Step>

      <Step title="Acceder al servidor MCP en tu chat">
        1. Cuando uses Claude, selecciona el botón de adjuntos (el icono de más).
        2. Selecciona el servidor MCP de Mintlify.
        3. Hazle una pregunta a Claude sobre Mintlify.
      </Step>
    </Steps>

    Consulta la[documentación del Model Context Protocol](https://modelcontextprotocol.io/docs/tutorials/use-remote-mcp-server#connecting-to-a-remote-mcp-server) para más detalles.
  </Tab>

  <Tab title="Claude Code">
    Para usar el servidor MCP de Mintlify con Claude Code, ejecuta el siguiente comando:

    ```bash
    claude mcp add --transport http Mintlify https://mintlify.com/docs/mcp
    ```

    Prueba la conexión ejecutando:

    ```bash
    claude mcp list
    ```

    Consulta la[documentación de Claude Code](https://docs.anthropic.com/en/docs/claude-code/mcp#installing-mcp-servers) para más detalles.
  </Tab>

  <Tab title="Cursor">
    <PreviewButton href="cursor://anysphere.cursor-deeplink/mcp/install?name=mintlify&config=eyJ1cmwiOiJodHRwczovL21pbnRsaWZ5LmNvbS9kb2NzL21jcCJ9">Instalar en Cursor</PreviewButton>

    Para conectar el servidor MCP de Mintlify a Cursor, haz clic en el botón**Instalar en Cursor**. O para conectar manualmente el servidor MCP, sigue estos pasos:

    <Steps>
      <Step title="Abrir configuración de MCP">
        1. Usa<kbd>Command</kbd> +<kbd>Shift</kbd> +<kbd>P</kbd> (<kbd>Ctrl</kbd> +<kbd>Shift</kbd> +<kbd>P</kbd> en Windows) para abrir la paleta de comandos.
        2. Busca "Open MCP settings".
        3. Selecciona**Add custom MCP**. Esto abrirá el archivo`mcp.json`.
      </Step>

      <Step title="Configurar el servidor MCP de Mintlify">
        En`mcp.json`, agrega:

        ```json
        {
          "mcpServers": {
            "Mintlify": {
              "url": "https://mintlify.com/docs/mcp"
            }
          }
        }
        ```
      </Step>

      <Step title="Probar la conexión">
        En el chat de Cursor, pregunta "¿Qué herramientas tienes disponibles?" Cursor debería mostrar el servidor MCP de Mintlify como una herramienta disponible.
      </Step>
    </Steps>

    Consulta[Installing MCP servers](https://docs.cursor.com/en/context/mcp#installing-mcp-servers) en la documentación de Cursor para más detalles.
  </Tab>

  <Tab title="VS Code">

    Para conectar el servidor MCP de Mintlify a VS Code, haz clic en el botón**Instalar en VS Code**. O para conectar manualmente el servidor MCP, crea un archivo`.vscode/mcp.json` y agrega:

    {
      "servers": {
