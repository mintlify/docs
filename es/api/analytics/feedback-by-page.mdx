<div id="usage">
  ## Uso
</div>

Usa este endpoint para obtener conteos agregados de comentarios para una página específica de tu documentación. La respuesta incluye el número de valoraciones positivas, valoraciones negativas, comentarios sobre fragmentos de código y el conteo total.

A diferencia del endpoint [Obtener comentarios](/es/api/analytics/feedback) que devuelve entradas individuales de comentarios, este endpoint devuelve un resumen de todos los comentarios para una ruta de página específica.

<div id="filtering">
  ## Filtrado
</div>

Filtra los comentarios por:
- **Ruta de la página** (obligatorio): Especifica el parámetro `path` para apuntar a una página de documentación específica
- **Rango de fechas**: Utiliza `dateFrom` y `dateTo` para limitar los resultados a un período de tiempo específico
