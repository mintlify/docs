<div id="usage">
  ## Uso
</div>

Usa este endpoint para obtener los contadores de comentarios agregados para una página específica de tu documentación. La respuesta incluye el número de votos positivos, negativos y comentarios sobre fragmentos de código junto con un total.

Esto es útil para crear paneles o informes que muestren las tendencias de comentarios por página sin recuperar las entradas de comentarios individuales.

<div id="filtering">
  ## Filtrado
</div>

Filtra los contadores de comentarios por:
- **Ruta de la página** (obligatorio): la ruta de la página de documentación para la cual agregar los comentarios
- **Rango de fechas**: usa `dateFrom` y `dateTo` para limitar la agregación a un período de tiempo específico
