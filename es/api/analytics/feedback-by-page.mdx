<div id="usage">
  ## Uso
</div>

Usa este endpoint para exportar comentarios agregados por página de documentación. Cada entrada muestra el número total de pulgares arriba, pulgares abajo y comentarios sobre fragmentos de código para una ruta de página determinada.

Pagina los resultados aumentando el parámetro `limit`. Continúa obteniendo resultados mientras `hasMore` sea `true`.

<div id="filtering">
  ## Filtrado
</div>

Filtra los resultados por:

- **Rango de fechas**: Utiliza `dateFrom` y `dateTo` para limitar los resultados a un período de tiempo específico
- **Fuente**: Filtra por tipos de feedback `code_snippet` o `contextual`
- **Estado**: Filtra por valores de estado como `pending`, `in_progress`, `resolved` o `dismissed`

<div id="response-fields">
  ## Campos de respuesta
</div>

Cada elemento del array `feedback` incluye:

- **path**: La ruta de la página de documentación
- **thumbsUp**: Cantidad de comentarios contextuales positivos
- **thumbsDown**: Cantidad de comentarios contextuales negativos
- **code**: Cantidad de comentarios sobre fragmentos de código
- **total**: Cantidad total de comentarios para la página
