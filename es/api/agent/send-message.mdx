## Uso

Este endpoint envía un mensaje de seguimiento a una sesión de trabajo del agente existente. Úsalo para continuar una conversación con el agente después de la creación inicial del trabajo, como solicitar cambios adicionales o proporcionar aclaraciones.

## Ejemplo

```javascript
const response = await fetch(
  `https://api.mintlify.com/v2/agent/${projectId}/job/${jobId}/message`,
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      prompt: 'También agrega un ejemplo de manejo de errores en la documentación de la API'
    })
  }
);

const job = await response.json();
```
