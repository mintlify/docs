Este endpoint crea un trabajo del agente a partir de un prompt proporcionado. El trabajo se ejecuta de forma asíncrona en segundo plano y devuelve inmediatamente los detalles del trabajo. Consulta el [endpoint de obtención](/es/api/agent/get-agent-job) para verificar el estado y obtener el enlace del pull request una vez que el trabajo se complete.

Si los archivos se editan correctamente, al finalizar el trabajo se crea automáticamente un pull request.

## Límites de tasa

La API del agente tiene los siguientes límites:

- 100 usos por proyecto de Mintlify por hora

## Sondeo para completar

Después de crear un trabajo, consulta el endpoint [obtener trabajo del agente](/es/api/agent/get-agent-job) para verificar cuándo se completa el trabajo:

```javascript
const jobId = response.id;

// Sondear hasta que el trabajo se complete
let job;
do {
  await new Promise(resolve => setTimeout(resolve, 5000)); // Esperar 5 segundos
  const statusResponse = await fetch(
    `https://api.mintlify.com/v2/agent/${projectId}/job/${jobId}`,
    { headers: { 'Authorization': `Bearer ${apiKey}` } }
  );
  job = await statusResponse.json();
} while (job.status === 'pending' || job.status === 'running');

if (job.prLink) {
  console.log('Pull request creado:', job.prLink);
}
```

## Continuar una conversación

Usa el endpoint [enviar mensaje](/es/api/agent/send-message) para enviar instrucciones adicionales a un trabajo existente.
