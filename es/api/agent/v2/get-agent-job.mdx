---
title: "Obtener trabajo del agente"
description: "Recupera el estado y los detalles de un trabajo del agente, incluido el progreso y el enlace a la solicitud de extracción."
openapi: /es/admin-openapi.json GET /v2/agent/{projectId}/job/{id}
keywords: [ "trabajo del agente", "estado", "obtener", "detalles", "consultar" ]
---

Consulta este endpoint para seguir el progreso de un trabajo del agente. El campo `status` transita por `active` → `completed` o `failed`.

Una vez que el agente crea una solicitud de extracción, el campo `prLink` se rellena.
