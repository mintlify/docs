---
title: "Crear trabajo del agente"
description: "Crea un trabajo del agente que se ejecuta en segundo plano y crea automáticamente una solicitud de extracción con los cambios de documentación."
openapi: /es/admin-openapi.json POST /v2/agent/{projectId}/job
keywords: [ "trabajo del agente", "crear", "automatización", "automatizar" ]
---

Este endpoint crea un trabajo del agente en segundo plano. El trabajo se ejecuta de forma asíncrona — utiliza el endpoint [get agent job](/es/api/agent/v2/get-agent-job) para consultar el estado.

Si el agente edita archivos correctamente, se crea automáticamente una solicitud de extracción y el campo `prLink` se rellena en la respuesta del trabajo.

<div id="rate-limits">
  ## Límites de tasa
</div>

- 100 usos por proyecto de Mintlify por hora

<div id="usage">
  ## Uso
</div>

```bash
curl -X POST https://api.mintlify.com/v2/agent/{projectId}/job \
  -H "Authorization: Bearer mint_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Add a quickstart guide for the Python SDK"}'
```
