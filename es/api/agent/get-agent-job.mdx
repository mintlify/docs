## Uso

Este endpoint obtiene los detalles y el estado de un trabajo de agente específico mediante su identificador único. Úsalo para consultar el estado de finalización del trabajo y obtener el enlace del pull request.

## Estado del trabajo

El campo `status` indica el estado actual del trabajo:

| Estado | Descripción |
|--------|-------------|
| `pending` | El trabajo está en cola esperando para iniciar |
| `running` | El trabajo está procesando activamente |
| `completed` | El trabajo finalizó correctamente |
| `failed` | El trabajo encontró un error |

## Campos de respuesta

| Campo | Descripción |
|-------|-------------|
| `id` | Identificador único del trabajo |
| `status` | Estado actual del trabajo |
| `source.repository` | URL del repositorio de GitHub |
| `source.ref` | Nombre de la branch donde se realizan los cambios |
| `model` | Modelo de IA utilizado para el trabajo |
| `prLink` | URL del pull request (disponible después de completarse) |
| `createdAt` | Marca de tiempo de creación del trabajo |
| `archivedAt` | Marca de tiempo de finalización del trabajo |
