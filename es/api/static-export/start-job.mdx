<Info>
  La exportación estática requiere un [plan Enterprise](https://mintlify.com/pricing?ref=static-export).
</Info>

El campo `domain` es el nombre de host donde se sirve tu documentación: el dominio personalizado configurado para tu proyecto en el dashboard de Mintlify (por ejemplo, `docs.example.com`), o tu subdominio de Mintlify (por ejemplo, `acme.mintlify.app`) si no has configurado un dominio personalizado. Pasa únicamente el nombre de host, sin el protocolo ni ningún prefijo de ruta.
