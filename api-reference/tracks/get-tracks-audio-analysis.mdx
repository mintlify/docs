# Get Track's Audio Analysis

Get a low-level audio analysis for a track


## Endpoint

```
GET https://api.spotify.com/v1/audio-analysis/{id}
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/audio-analysis/{id}" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

  // Response data
