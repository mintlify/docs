# Get Several Chapters

Get Spotify catalog information for several chapters


## Endpoint

```
GET https://api.spotify.com/v1/chapters
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/chapters" \
  -H "Authorization: Bearer {access_token}"
```

Returns JSON with the requested data.
