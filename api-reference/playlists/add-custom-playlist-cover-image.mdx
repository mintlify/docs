# Add Custom Playlist Cover Image

Replace the image used to represent a specific playlist

<Note>
This endpoint requires user authentication with the following scope(s): `ugc-image-upload playlist-modify-public playlist-modify-private`
</Note>


## Endpoint

```
PUT https://api.spotify.com/v1/playlists/{playlist_id}/images
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X PUT "https://api.spotify.com/v1/playlists/{playlist_id}/images" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

  // Response data
}
