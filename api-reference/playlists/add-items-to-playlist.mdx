# Add Items to Playlist

Add one or more items to a user's playlist

<Note>
This endpoint requires user authentication with the following scope(s): `playlist-modify-public playlist-modify-private`
</Note>


## Endpoint

```
POST https://api.spotify.com/v1/playlists/{playlist_id}/tracks
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X POST "https://api.spotify.com/v1/playlists/{playlist_id}/tracks" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

### Example response

```json
{
  // Response data
}
```
