# Get Category's Playlists

Get a list of Spotify playlists tagged with a particular category


## Endpoint

```
GET https://api.spotify.com/v1/browse/categories/{category_id}/playlists
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/browse/categories/{category_id}/playlists" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

  // Response data
