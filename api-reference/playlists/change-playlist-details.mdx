# Change Playlist Details

Change a playlist's name and public/private state

<Note>
This endpoint requires user authentication with the following scope(s): `playlist-modify-public playlist-modify-private`
</Note>


## Endpoint

```
PUT https://api.spotify.com/v1/playlists/{playlist_id}
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X PUT "https://api.spotify.com/v1/playlists/{playlist_id}" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

### Example response

```json
{
  // Response data
}
```
