# Create Playlist

Create a playlist for a Spotify user

<Note>
This endpoint requires user authentication with the following scope(s): `playlist-modify-public playlist-modify-private`
</Note>


## Endpoint

```
POST https://api.spotify.com/v1/users/{user_id}/playlists
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X POST "https://api.spotify.com/v1/users/{user_id}/playlists" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

  // Response data
}
