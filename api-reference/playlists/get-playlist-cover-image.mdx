# Get Playlist Cover Image

Get the current image associated with a specific playlist


## Endpoint

```
GET https://api.spotify.com/v1/playlists/{playlist_id}/images
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/playlists/{playlist_id}/images" \
  -H "Authorization: Bearer {access_token}"
```

Returns JSON with the requested data.
