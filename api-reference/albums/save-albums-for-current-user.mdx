# Save Albums for Current User

Save one or more albums to the current user's library

<Note>
This endpoint requires user authentication.
</Note>


## Endpoint

```
PUT https://api.spotify.com/v1/me/albums
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X PUT "https://api.spotify.com/v1/me/albums" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
