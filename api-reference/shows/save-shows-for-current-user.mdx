# Save Shows for Current User

Save one or more shows to current user's library

<Note>
This endpoint requires user authentication with the following scope(s): `user-library-modify`
</Note>


## Endpoint

```
PUT https://api.spotify.com/v1/me/shows
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X PUT "https://api.spotify.com/v1/me/shows" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
