# Get User's Saved Audiobooks

Get a list of the audiobooks saved in the current user's library

<Note>
This endpoint requires user authentication.
</Note>


## Endpoint

```
GET https://api.spotify.com/v1/me/audiobooks
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/me/audiobooks" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
