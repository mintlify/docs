# Get Playback State

Get information about the user's current playback state

<Note>
This endpoint requires user authentication with the following scope(s): `user-read-playback-state`
</Note>


## Endpoint

```
GET https://api.spotify.com/v1/me/player
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/me/player" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
