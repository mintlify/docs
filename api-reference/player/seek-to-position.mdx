# Seek To Position

Seeks to the given position in the user's currently playing track

<Note>
This endpoint requires user authentication with the following scope(s): `user-modify-playback-state`
</Note>


## Endpoint

```
PUT https://api.spotify.com/v1/me/player/seek
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X PUT "https://api.spotify.com/v1/me/player/seek" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
