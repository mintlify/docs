# Skip To Next

Skips to next track in the user's queue

<Note>
This endpoint requires user authentication with the following scope(s): `user-modify-playback-state`
</Note>


## Endpoint

```
POST https://api.spotify.com/v1/me/player/next
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X POST "https://api.spotify.com/v1/me/player/next" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
