# Get Available Devices

Get information about a user's available devices

<Note>
This endpoint requires user authentication with the following scope(s): `user-read-playback-state`
</Note>


## Endpoint

```
GET https://api.spotify.com/v1/me/player/devices
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/me/player/devices" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

### Example response

```json
{
  // Response data
}
```
