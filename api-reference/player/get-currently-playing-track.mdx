# Get Currently Playing Track

Get the object currently being played

<Note>
This endpoint requires user authentication with the following scope(s): `user-read-currently-playing`
</Note>


## Endpoint

```
GET https://api.spotify.com/v1/me/player/currently-playing
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/me/player/currently-playing" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
