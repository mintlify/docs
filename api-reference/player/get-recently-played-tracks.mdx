# Get Recently Played Tracks

Get tracks from the current user's recently played tracks

<Note>
This endpoint requires user authentication with the following scope(s): `user-read-recently-played`
</Note>


## Endpoint

```
GET https://api.spotify.com/v1/me/player/recently-played
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/me/player/recently-played" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
{
  // Response data
}
```
