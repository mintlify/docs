# Set Playback Volume

Set the volume for the user's current playback device

<Note>
This endpoint requires user authentication with the following scope(s): `user-modify-playback-state`
</Note>


## Endpoint

```
PUT https://api.spotify.com/v1/me/player/volume
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X PUT "https://api.spotify.com/v1/me/player/volume" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

### Example response

```json
{
  // Response data
}
```
