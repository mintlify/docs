# Get Single Browse Category

Get a single category used to tag items in Spotify


## Endpoint

```
GET https://api.spotify.com/v1/browse/categories/{category_id}
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/browse/categories/{category_id}" \
  -H "Authorization: Bearer {access_token}"
```

Returns JSON with the requested data.
