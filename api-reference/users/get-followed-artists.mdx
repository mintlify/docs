# Get Followed Artists

Get the current user's followed artists

<Note>
This endpoint requires user authentication with the following scope(s): `user-follow-read`
</Note>


## Endpoint

```
GET https://api.spotify.com/v1/me/following
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X GET "https://api.spotify.com/v1/me/following" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

```json
  // Response data
}
```
