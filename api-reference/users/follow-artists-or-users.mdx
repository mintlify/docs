# Follow Artists or Users

Add the current user as a follower of one or more artists or other Spotify users

<Note>
This endpoint requires user authentication with the following scope(s): `user-follow-modify`
</Note>


## Endpoint

```
PUT https://api.spotify.com/v1/me/following
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X PUT "https://api.spotify.com/v1/me/following" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

### Example response

```json
{
  // Response data
}
```
