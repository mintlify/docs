# Unfollow Playlist

Remove the current user as a follower of a playlist

<Note>
This endpoint requires user authentication with the following scope(s): `playlist-modify-public playlist-modify-private`
</Note>


## Endpoint

```
DELETE https://api.spotify.com/v1/playlists/{playlist_id}/followers
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication: `Bearer {access_token}`
</ParamField>

## Example request

```bash
curl -X DELETE "https://api.spotify.com/v1/playlists/{playlist_id}/followers" \
  -H "Authorization: Bearer {access_token}"
```

## Response

Returns JSON with the requested data.

{
  // Response data
}
