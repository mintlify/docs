List contact emails in your dashboard so your team gets the right notifications from Mintlify. Configure both from the [Notifications](https://dashboard.mintlify.com/settings/organization/notifications) page in your dashboard.

## Security contact

<Info>
  Security contact is available on [Pro and Enterprise plans](https://mintlify.com/pricing?ref=security-contact).
</Info>

The security contact receives security-related communications about your organization, including vulnerability reports, security alerts, and compliance updates.

To add, update, or remove a security contact, go to the [Notifications](https://dashboard.mintlify.com/settings/organization/notifications) page.

## Admin contacts

Admin contacts receive operational notifications about your organization, including:

- Billing notices like usage overages.
- Deployment lifecycle notifications, like when a deployment is queued for deletion.

If you don't set any admin contacts, the oldest member of the organization receives these notifications. Add one or more admin contact emails to route notifications to the right people on your team.

To add, update, or remove an admin contact, go to the [Notifications](https://dashboard.mintlify.com/settings/organization/notifications) page, enter an email address, and save your changes. Repeat to add more recipients.
