Mintlify provides three dashboard access levels: viewer, editor, and admin.

| Suggest changes in the web editor |   ✅   |   ✅   |  ✅   |
| Edit content in the web editor |   ❌   |   ✅   |  ✅   |
| Upload media in the web editor |   ❌   |   ✅   |  ✅   |
| Trigger deployments            |   ❌   |   ✅   |  ✅   |
| Create and delete API keys     |   ❌   |   ✅   |  ✅   |
| Configure integrations         |   ❌   |   ✅   |  ✅   |
| Configure assistant            |   ❌   |   ✅   |  ✅   |
| Configure add-ons              |   ❌   |   ✅   |  ✅   |
| Invite editors and viewers     |   ❌   |   ✅   |  ✅   |
| Update user roles              |   ❌   |   ❌   |  ✅   |
| Invite admins                  |   ❌   |   ❌   |  ✅   |

<Tip>
  Limit admin access to only the people who need to perform admin tasks.
</Tip>

## Viewer editor access

Viewers can open the web editor in a read-only suggesting mode. When a viewer opens the editor, they are automatically placed in suggesting mode and cannot switch to direct editing. This allows viewers to propose changes without modifying the live content.

In suggesting mode, viewers can:

- Browse and read all documentation files
- Leave comments on content
- Propose text suggestions for editors and admins to review

Viewers cannot:

- Directly edit content or format text
- Upload images or videos
- Publish changes

## Add members to your organization

By default, the person who creates your Mintlify organization has admin access. Add additional editors in the [Members](https://dashboard.mintlify.com/settings/organization/members) page of your dashboard.

You can have as many members as you need in your organization. Seats can incur additional costs. See the [pricing page](https://mintlify.com/pricing) for more details.
