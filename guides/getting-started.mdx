Welcome to the Mintlify guides. This section helps you create documentation that serves your users' needs and goals, whether you're starting from scratch or improving existing docs.

## What you'll find here

The guides cover everything you need to plan, write, and maintain effective documentation:

- **AI tools**: Accelerate your workflow with AI-powered documentation tools
- **API documentation**: Create maintainable API docs that help users test and learn
- **Best practices**: Make your documentation accessible, well-organized, and easy to maintain
- **Git workflows**: Understand Git fundamentals for docs-as-code
- **Use cases**: Find solutions for developer docs, knowledge bases, support centers, and custom frontends
