title: Format currency
description: Use formatCurrency to display monetary values with proper locale formatting
---

The `formatCurrency` utility formats numbers as currency strings with locale-aware formatting, including thousands separators and decimal places.

## Usage

```typescript
import { formatCurrency } from '#/lib/utils';

// Format with default 2 decimal places
const price = formatCurrency(1234.56);
// Returns: "$1,234.56"

// Format with custom decimal places
const precise = formatCurrency(1234.5678, 4);
// Returns: "$1,234.5678"

// Format whole numbers
const rounded = formatCurrency(10648);
// Returns: "$10,648.00"
```

## Parameters

<ParamField path="num" type="number" required>
  The numeric value to format as currency.
</ParamField>

<ParamField path="decimalPlaces" type="number" default={2}>
  The number of decimal places to display. Defaults to 2.
</ParamField>

## Implementation details

The utility uses `Intl.NumberFormat` with the following configuration:
- Locale: `en-US`
- Currency: `USD`
- Style: `currency`

Formatters are cached by decimal precision to avoid creating new instances on every call, improving performance when formatting multiple values.

## Example

The Assistant dashboard uses `formatCurrency` to display monthly spend with precise decimal values:

```tsx
<StatsCard
  label="Current monthly spend"
  value={formatCurrency(currentSpend)}
  subtext={`${nextBillingFormatted} next renewal`}
  variant="renewal"
/>
```

This ensures monetary values display consistently with proper formatting, including the dollar sign, thousands separators, and decimal places.
