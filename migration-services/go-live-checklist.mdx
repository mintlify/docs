---
title: "Go-live checklist"
description: "A checklist to verify your Mintlify site is ready before going live."
---

- \[ \] **Connect Github docs repo to Mintlify.** Install Github app. [Docs](https://www.mintlify.com/docs/deploy/github). Note if the Github app is already installed, you will have uninstall and re-install the Github app. To verify this step works, try making a change to the docs locally on computer, push to github, and check that the Mintlify docs site redeploys on the [dashboard home page](https://www.notion.so/Mintlify-Referral-Policy-335aa841023480d2b239f9014cd12653?pvs=21).
  - \[ \] Determine if you need to configure multi-repository structure to build a Mintlify site from multiple repos. [Link here](https://www.mintlify.com/docs/deploy/multi-repo).
- \[ \] **Invite team members**. Invite the emails of all team members that will need access to Mintlify in Settings \> Workspace \> Members. Guide to member [permissions](https://www.mintlify.com/docs/dashboard/roles).
- \[ \] **Choose a [theme](https://www.mintlify.com/docs/customize/themes).** If you have trouble choosing, I recommend luma.
- \[ \] **Figure out your information architecture.** How do you want information to be laid out based on the goals of your users? [Reference here](https://www.mintlify.com/docs/guides/content-types).
- \[ \] **Navigate to the assistant tab.** Turn it on. [Configure Assistant Deflection and Search Domains](https://www.mintlify.com/docs/assistant/configure). [Add assistant skills](https://www.mintlify.com/docs/assistant/skills).
- \[ \] **Enable relevant add-ons.** Go to Settings \> Add-ons. Enable relevant add-ons such as agent feedback, [CI checks](https://www.mintlify.com/docs/deploy/ci), related topics, etc.
- \[ \] **Set up the slack agent.** This lets you make changes to the docs directly in slack. [Reference here](https://www.mintlify.com/docs/agent/slack#add-the-agent-to-slack).
- \[ \] **Set up automations.** [How to do this here](https://www.mintlify.com/docs/automations/manage#enable-an-automation).
- \[ \] **Add a [skill.md](http://skill.md) file.** Add to give agents specific directions when surfacing your docs. [link here](https://www.mintlify.com/docs/ai/skillmd).
- \[ \] **Install relevant Analytics plugins.** See [here](https://www.mintlify.com/docs/integrations/analytics/overview) for list of analytics integrations.
- \[ \] **Attach your custom domain.** Directions [here](https://www.mintlify.com/docs/customize/custom-domain).
