---
title: "Go Live Checklist"
---

**Connect Github docs repo to Mintlify.** Install Github app. [<u>Docs</u>](https://www.mintlify.com/docs/deploy/github). Note if the Github app is already installed, you will have uninstall and re-install the Github app. To verify this step works, try making a change to the docs locally on computer, push to github, and check that the Mintlify docs site redeploys on the dashboard.

**Invite team members**. Invite the emails of all team members that will need access to Mintlify in Settings \> Workspace \> Members. Guide to member [<u>permissions</u>](https://www.mintlify.com/docs/dashboard/roles).

**Verify your information architecture.** How do you want information to be laid out based on the goals of your users? [<u>Reference here</u>](https://www.mintlify.com/docs/guides/content-types).

**Navigate to the assistant tab.** Configure the assistant. 

- [<u>Configure Assistant Deflection and Search Domains</u>](https://www.mintlify.com/docs/assistant/configure). 
- [<u>Add assistant skills</u>](https://www.mintlify.com/docs/assistant/skills).
