## Usage

Send additional instructions to an existing agent session. This allows you to:

- Request changes to the agent's work
- Provide clarifications or corrections
- Add more tasks after reviewing the initial PR

The agent continues from where it left off, maintaining context from previous messages.

## Example

```bash
# After reviewing the initial PR, request additional changes
curl -X POST "https://api.mintlify.com/v2/agent/{projectId}/job/{jobId}/message" \
  -H "Authorization: Bearer $MINT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Also add code examples for each troubleshooting step"
  }'
```

## Best practices

- Wait for the previous operation to complete before sending new messages
- Be specific about what changes you want
- Reference specific sections or files when requesting modifications

## Rate limits

- 100 requests per project per hour (shared with job creation)
