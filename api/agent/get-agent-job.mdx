## Usage

This endpoint retrieves the details and status of a specific Mintie bot job by its unique identifier. Use this to check the progress, status, and results of a previously created Mintie bot job.

## Job details

The response includes information such as:
- Job execution status and completion state
- Branch information and pull request details
- Session metadata and timestamps
