## Usage

This endpoint retrieves all Mintie bot jobs for the specified domain, providing an overview of all Mintie bot activities and their current status. This is useful for monitoring and managing multiple concurrent or historical Mintie bot jobs.

## Response

Use this endpoint to get a comprehensive view of all previous Mintie bot sessions.
