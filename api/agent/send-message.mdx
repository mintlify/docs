## Usage

This endpoint sends a follow-up message to an existing agent job session. Use this to continue a conversation with the agent after the initial job creation, such as requesting additional changes or providing clarification.

## Example

```javascript
const response = await fetch(
  `https://api.mintlify.com/v2/agent/${projectId}/job/${jobId}/message`,
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      prompt: 'Also add an example for error handling in the API docs'
    })
  }
);

const job = await response.json();
```

## Rate limits

The agent API has the following limits:

- 100 uses per Mintlify project per hour
