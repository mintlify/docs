This endpoint creates a background Mintie bot job. The job runs asynchronously — use the [get Mintie bot job](/api/agent/v2/get-agent-job) endpoint to poll for status updates.

If Mintie bot edits files successfully, a pull request is automatically created and the `prLink` field is populated in the job response.

## Rate limits

- 100 uses per Mintlify project per hour

## Usage

```bash
curl -X POST https://api.mintlify.com/v2/agent/{projectId}/job \
  -H "Authorization: Bearer mint_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Add a quickstart guide for the Python SDK"}'
```
