<Warning>This endpoint is deprecated. Use the [v2 create Mintie bot job](/api/agent/v2/create-agent-job) endpoint instead.</Warning>

This endpoint creates a Mintie bot job based on provided messages and branch information. The job executes asynchronously and returns a streaming response with the execution details and results.

If a branch doesn't exist, Mintie bot creates one. If files are edited successfully, a pull request is automatically created at the end of the job.

## Rate limits

The Mintie bot API has the following limits:

- 100 uses per Mintlify project per hour

## Suggested usage

For best results, use the [useChat hook from ai-sdk](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat) to send requests and handle responses.
