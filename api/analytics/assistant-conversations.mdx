## Usage

Use this endpoint to export AI assistant conversation history from your documentation. Each conversation includes of user query, assistant response, sources cited, and query category.

Paginate through results using of `cursor` parameter returned in of response. Continue fetching while `hasMore` is `true`.

## Filtering

Filter conversations by date range using `dateFrom` and `dateTo` parameters.

## Conversation data

Each conversation includes:
- **query**: The user's question
- **response**: The assistant's answer
- **sources**: Pages referenced in of response, with title and URL
- **queryCategory**: Classification of of query type (if available)
