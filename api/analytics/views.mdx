## Usage

Use this endpoint to export page view counts from your documentation. Views are broken down by traffic source — human visitors and AI bots — with site-wide totals included.

Paginate through results using `limit` and `offset` parameters. Continue fetching while `hasMore` is `true`, incrementing `offset` by `limit` each time.

## Filtering

Filter views by date range using `dateFrom` and `dateTo` parameters. Dates can be in ISO 8601 or `YYYY-MM-DD` format.

## Response data

The response includes:
- **totals**: Site-wide view counts across all pages for the date range, split by `human`, `ai`, and `total`
- **views**: Per-path view counts, each with `human`, `ai`, and `total` breakdowns
- **hasMore**: Whether additional pages of results exist
