## Usage

Use this endpoint to see how assistant queries are distributed across caller types over time. The response is a time-series array bucketed by the `granularity` parameter (`day`, `week`, or `month`).

Each entry includes:

- **date**: Start of the time bucket
- **web**: Queries from the documentation site
- **api**: Queries from direct API calls
- **other**: Queries from other sources (e.g., integrations, SDKs)
- **total**: Sum of all query types in that bucket

## Filtering

Filter by date range using `dateFrom` and `dateTo`, and set the bucket size with `granularity`.
