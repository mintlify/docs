## Usage

Use this endpoint to export page view counts from your documentation. Each entry shows how many content view events a page received, split by human and AI bot traffic.

Paginate through results using `offset` and `limit`. Increment `offset` by `limit` while `hasMore` is `true`.

## Filtering

Filter views by date range using `dateFrom` and `dateTo` parameters.

## Response data

The response includes:
- **totals**: Site-wide view counts across all pages for the date range, split by `human`, `ai`, and `total`
- **views**: Per-path view counts, each with `human`, `ai`, and `total` fields
- **hasMore**: Whether additional pages of results exist
