## Usage

Use this endpoint to get aggregated feedback counts for a specific page in your documentation. The response includes the number of thumbs up, thumbs down, and code snippet feedback entries along with a total count.

This is useful for building dashboards or reports that show feedback trends per page without retrieving individual feedback entries.

## Filtering

Filter feedback counts by:
- **Page path** (required): The path of the documentation page to aggregate feedback for
- **Date range**: Use `dateFrom` and `dateTo` to limit the aggregation to a specific time period
