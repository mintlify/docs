## Usage

Use this endpoint to get a high-level overview of feedback across your documentation pages. Results are aggregated by page path and sorted by total feedback count in descending order.

## Filtering

Filter feedback by:
- **Date range**: Use `dateFrom` and `dateTo` to limit results to a specific time period
- **Source**: Filter by `code_snippet` or `contextual` feedback types
- **Status**: Filter by status values like `pending`, `in_progress`, `resolved`, or `dismissed`

## Response

Each entry in the response represents a single page and includes:

- **thumbsUp**: Number of positive contextual ratings
- **thumbsDown**: Number of negative contextual ratings
- **code**: Number of code snippet feedback entries
- **total**: Sum of all feedback for the page
