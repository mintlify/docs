## Usage

Use this endpoint to retrieve aggregated feedback counts for a specific page in your documentation. The response includes the number of thumbs up ratings, thumbs down ratings, code snippet feedback entries, and the total count.

Unlike the [Get feedback](/api/analytics/feedback) endpoint which returns individual feedback entries, this endpoint returns a summary of all feedback for a given page path.

## Filtering

Filter feedback by:
- **Page path** (required): Specify the `path` query parameter to target a specific documentation page
- **Date range**: Use `dateFrom` and `dateTo` to limit results to a specific time period
