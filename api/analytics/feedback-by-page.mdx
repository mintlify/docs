## Usage

Use this endpoint to get a summary of all feedback submitted on a specific documentation page. The response includes counts of thumbs-up ratings, thumbs-down ratings, and code snippet feedback.

This is useful for identifying which pages need improvement or which pages are performing well based on reader sentiment.

## Filtering

Optionally filter by date range:
- **`dateFrom`**: Only include feedback submitted on or after this date
- **`dateTo`**: Only include feedback submitted before this date

## Response fields

The `feedback` object contains:

- **`thumbsUp`**: Number of page ratings where the reader found the content helpful
- **`thumbsDown`**: Number of page ratings where the reader did not find the content helpful
- **`code`**: Number of code snippet feedback submissions on the page
- **`total`**: Sum of all feedback counts (`thumbsUp` + `thumbsDown` + `code`)
