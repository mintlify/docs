## Usage

Use this endpoint to export user feedback collected from your documentation. Feedback includes contextual page feedback, thumbs-only votes, and code snippet feedback.

Paginate through results using the `cursor` parameter returned in the response. Continue fetching while `hasMore` is `true`.

## Filtering

Filter feedback by:
- **Date range**: Use `dateFrom` and `dateTo` to limit results to a specific time period
- **Source**: Filter by `code_snippet`, `contextual`, or `thumbs_only` feedback types
- **Status**: Filter by status values like `pending`, `in_progress`, `resolved`, or `dismissed`

## Response types

The response contains different feedback types based on the source:

- **Contextual feedback**: Page-level feedback with a text comment. Includes `helpful` boolean and optional `contact` email.
- **Thumbs-only feedback**: A thumbs up or thumbs down vote without a text comment. Shares the same schema as contextual feedback.
- **Code snippet feedback**: Feedback on a specific code block. Includes `code`, `filename`, and `lang` fields.
