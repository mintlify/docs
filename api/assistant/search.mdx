Use this endpoint to search your documentation using AI-powered semantic search. Results are ranked by relevance and include context snippets from each matching page.

## Rate limits

- 10,000 requests per Mintlify organization per hour

## Use cases

- Build custom search interfaces for your documentation
- Integrate documentation search into your application
- Power context-aware help systems that surface relevant content
