## Summary

The Notion (and Linear) connector cards were removed from the agent settings page in the dashboard, so the standalone "Connect Notion" doc page no longer reflects the available UI. This PR removes the now-stale Notion setup docs and cleans up references across English, Spanish, and French.

## Changes

- Deleted `agent/notion.mdx` and its translations (`es/agent/notion.mdx`, `fr/agent/notion.mdx`).
- Removed the Notion card and `/agent/notion` entry from the agent overview pages and `docs.json` navigation.
- Updated the audit logs integration row to list GitHub and Slack only.
- Re-pointed legacy changelog references to `/agent` to fix broken links.

## Context

Triggered by the upstream change that removes the Linear and Notion install/disconnect flows from the agent settings page. Slack and GitHub integrations are unchanged, and users can still ask the agent in Slack to set up third-party integrations.
