</Info>

将 Mintlify agent 连接到 Jira，让你的团队可以直接从 Jira issue 和评论中触发文档更新。

## 连接你的 Jira 工作区

1. 在[控制台](https://dashboard.mintlify.com/products/agent)中打开 agent 面板。
2. 点击**设置**按钮。
3. 在 Jira 集成部分，点击**连接**。
4. 按照提示在你的 Jira 工作区中授权 Mintlify。

## 在 Jira 中使用 agent

连接后，在 Jira issue 评论中提及 agent 即可触发文档更新。Agent 会读取 issue 的上下文，包括摘要、描述和评论，然后创建包含建议更改的 pull request。

你可以：

- 在 issue 评论中提及 agent 并说明需要记录的内容。
- 引用需要更新的特定页面或章节。
- 附加相关 issue 或 epic 的上下文供 agent 使用。

## 提示词示例

以下是一些可以在 Jira issue 评论中使用的提示词示例：

- `@mintlify 在 API 参考中记录这个新功能。`
- `@mintlify 更新快速入门指南以反映此 issue 中的更改。`
- `@mintlify 根据此处描述的 bug 添加故障排除章节。`

## 最佳实践

- **具体明确**：告诉 agent 你想记录什么以及应该放在哪里。
- **添加上下文**：如果 issue 不包含所有必要信息，请在评论中向 agent 提供额外的详细信息。
- **仔细审查**：在合并之前，务必审查 agent 创建的 pull request。
