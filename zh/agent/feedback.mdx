当 AI agent 通过 [Markdown 页面](/zh/ai-native#discovering)浏览你的文档时，它们可以提交关于所遇到问题的反馈。这些反馈会与用户反馈一起显示在你的控制台中，以便你可以跟踪和解决由人类和 AI agent 报告的文档问题。

<div id="how-agent-feedback-works">
  ## Agent 反馈的工作原理
</div>

你的 Markdown 页面包含指示 agent 如何报告问题的说明。Agent 通过 POST 请求提交反馈，其中包含页面路径和问题描述。

Agent 反馈会显示在[反馈面板](https://dashboard.mintlify.com/products/analytics/v2/feedback)中，并带有 <Badge color="blue">Agent</Badge> 徽章，以便与人类反馈区分开来。你可以像管理其他类型的反馈一样筛选、管理和解决 agent 反馈。

Agent 反馈默认对所有文档站点启用。无需任何配置。

<div id="view-agent-feedback">
  ## 查看 agent 反馈
</div>

1. 在控制台中打开[反馈面板](https://dashboard.mintlify.com/products/analytics/v2/feedback)。
2. 选择 **Ratings by page** 查看概览，或选择 **Detailed feedback** 查看单条反馈详情。
3. 查找 <Badge color="blue">Agent</Badge> 徽章以识别由 AI agent 提交的反馈。

使用筛选菜单中的 **Show agent feedback** 复选框来在视图中包含或排除 agent 反馈。

<div id="manage-agent-feedback">
  ## 管理 agent 反馈
</div>

Agent 反馈支持与其他反馈类型相同的管理功能：

- **更改状态**：将反馈标记为待处理、进行中、已解决或已忽略。
- **添加内部备注**：点击某条反馈，为你的团队添加备注。
- **批量操作**：选择多个条目，批量更改状态、标记或删除。

有关管理反馈的更多信息，请参阅[反馈](/zh/optimize/feedback)。
