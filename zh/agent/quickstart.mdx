<div id="open-the-agent-panel">
  ## 打开 Agent 面板
</div>

在桌面设备上，Agent 面板可以调整大小。在移动设备上，Agent 会以全屏方式打开。

要打开 Agent 面板：

- 在控制台的任意页面，使用快捷键 <kbd>⌘</kbd>+<kbd>I</kbd>（macOS）或 <kbd>Ctrl</kbd>+<kbd>I</kbd>（Windows/Linux）。
- 在 [Overview](https://dashboard.mintlify.com/) 页面，点击 **Ask agent**。

- **Chat**：点击加号 icon，<Icon icon="plus" />，开始新的聊天。在聊天视图中，向 agent 发送提示词，以更新你的文档或提出问题。Agent 会根据你的指令创建拉取请求（PR；亦称“合并请求”/Merge Request），并显示链接以查看这些拉取请求，或在 web 编辑器中打开变更。
- **Suggestions**：如果启用，建议面板会基于拉取请求中的更改以及用户与 AI 助手的对话，显示对文档的推荐更新。
- **History**：点击历史记录 icon，<Icon icon="clock" />，浏览过去的对话并继续处理先前的请求。点击任意对话即可在聊天视图中加载。
- **Settings**：点击设置 icon，<Icon icon="settings" />，配置 agent 的 integrations 和存储库访问权限。

<Tip>
  为每个任务与 agent 开启一段新的对话。这样可以让 agent 的上下文更聚焦，并帮助你将对话与特定项目关联起来。
</Tip>

<div id="connect-your-github-account">
  ## 连接你的 GitHub 账号
</div>

默认情况下，代理会以 Mintlify 机器人身份发起拉取请求（PR）。要让拉取请求归属到你本人，请在控制台的 [My profile](https://dashboard.mintlify.com/settings/account) 页面连接你的 GitHub 账号。

<div id="connect-repositories-as-context">
  ## 将代码仓库连接为 context
</div>

此 agent 只能访问你通过 Mintlify GitHub 应用连接的代码仓库。你可以在 agent 面板的 **Settings**，或在 [GitHub 应用设置](https://github.com/apps/mintlify/installations/new) 中配置 agent 可访问的代码仓库。
