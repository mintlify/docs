折叠面板允许用户展开和折叠内容部分。使用折叠面板实现渐进式披露，并用于组织信息。


<div id="single-accordion">
  ## 单个折叠面板
</div>

<Accordion title="I am an Accordion.">
  你可以在这里放入任意内容，包括其他组件，比如代码：

  ```java HelloWorld.java
   class HelloWorld {
       public static void main(String[] args) {
           System.out.println("Hello, World!");
       }
   }
  ```
</Accordion>

<div id="accordion-groups">
  ## 折叠面板组
</div>

使用 `<AccordionGroup>` 将相关的折叠面板分组。这会创建一个连贯的折叠面板区域，用户可以分别展开或折叠各个面板。

<AccordionGroup>
  <Accordion title="入门">
    你可以在折叠面板中放入其他组件。

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="高级功能" icon="bot">
    添加图标可让折叠面板在视觉上更易区分、便于快速扫描。
  </Accordion>

  <Accordion title="故障排除">
    将相关内容组织成分组，便于管理。
  </Accordion>
</AccordionGroup>

<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string" required>
  折叠面板预览中的标题。
</ResponseField>

<ResponseField name="description" type="string">
  折叠面板预览中标题下方的说明。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  是否默认展开折叠面板。
</ResponseField>
