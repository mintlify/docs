折叠面板允许用户展开和收起内容区块。使用折叠面板进行渐进式展示，并组织信息。


<div id="single-accordion">
  ## 单个折叠面板
</div>

<Accordion title="我是折叠面板。">
  你可以在这里放入任何内容，包括其他组件，比如代码：

  ```java HelloWorld.java
   class HelloWorld {
       public static void main(String[] args) {
           System.out.println("Hello, World!");
       }
   }
  ```
</Accordion>

<div id="accordion-groups">
  ## 折叠面板分组
</div>

使用 `<AccordionGroup>` 将相关的折叠面板归为一组。这样可以创建一个连贯的折叠面板区域，用户可单独展开或收起每一项。

<AccordionGroup>
  <Accordion title="入门">
    你可以在折叠面板中嵌入其他组件。

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="高级功能" icon="bot">
    可添加图标，使折叠面板更易于区分和快速浏览。
  </Accordion>

  <Accordion title="故障排查">
    将相关内容按组进行组织。
  </Accordion>
</AccordionGroup>

````mdx Accordion Group Example
<AccordionGroup>
````

<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string" required>
  折叠面板预览中的标题。
</ResponseField>

<ResponseField name="description" type="string">
  折叠面板预览中标题下方的说明。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  是否默认展开折叠面板。
</ResponseField>
