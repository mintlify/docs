折叠面板允许用户展开或收起内容区块。使用折叠面板进行渐进式披露，并帮助组织信息。



<div id="single-accordion">
  ## 单个折叠面板
</div>

<Accordion title="I am an Accordion.">
  这里可以放入任意内容，包括其他组件，例如代码：
   ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
</Accordion>

<div id="accordion-groups">
  ## 手风琴分组
</div>

使用 `<AccordionGroup>` 将相关手风琴组合在一起。这样可以创建一个连贯的手风琴区域，每个手风琴都可单独展开或折叠。

<AccordionGroup>
  <Accordion title="快速开始">
    你可以在手风琴内放入其他组件。

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="高级功能" icon="bot">
    添加图标可使手风琴在视觉上更易区分、便于快速浏览。
  </Accordion>
  
  <Accordion title="故障排查">
    将相关内容组织成分组。
  </Accordion>
</AccordionGroup>

<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string" required>
  Accordion 预览中的标题。
</ResponseField>

<ResponseField name="description" type="string">
  Accordion 预览中标题下方的说明。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Accordion 是否默认展开。
</ResponseField>
