
<div id="inline-icons">
  ## 行内图标
</div>

当在段落中使用时，图标将以内联形式呈现。

<Icon icon="flag" iconType="solid" /> 轻松获得你想要的文档。

```markdown Inline icon example
<Icon icon="flag" iconType="solid" /> 您想要的文档，轻松搞定。
```


<div id="properties">
  ## 属性
</div>

<IconsRequired />

<ResponseField name="color" type="string">
  icon 的颜色，使用十六进制色值（例如 `#FF5733`）。
</ResponseField>

<ResponseField name="size" type="number">
  icon 的大小（像素）。
</ResponseField>
