<div id="inline-icons">
  ## 内联图标
</div>

当图标用于段落中时，会以内联形式显示。

<Icon icon="flag" iconType="solid" /> 你想要的文档，轻松获得。

```markdown Inline icon example
<Icon icon="flag" iconType="solid" /> 您想要的文档，轻松搞定。
```


<div id="properties">
  ## 属性
</div>

<IconsRequired />

<ResponseField name="color" type="string">
  icon 的颜色，使用十六进制色值（例如 `#FF5733`）。
</ResponseField>

<ResponseField name="size" type="number">
  icon 的尺寸（像素）。
</ResponseField>
