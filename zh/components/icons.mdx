<div id="inline-icons">
  ## 内联图标
</div>

当在段落中使用时，图标会以内联方式呈现。

<Icon icon="flag" iconType="solid" /> 轻松获取所需的文档。

```markdown Inline icon example
<Icon icon="flag" iconType="solid" /> 您想要的文档，轻松搞定。
```


<div id="properties">
  ## 属性
</div>

<IconsRequired />

<ResponseField name="color" type="string">
  icon 的颜色，使用十六进制色值（例如 `#FF5733`）。
</ResponseField>

<ResponseField name="size" type="number">
  icon 的大小（以像素为单位）。
</ResponseField>
