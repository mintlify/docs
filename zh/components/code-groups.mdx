使用 `CodeGroup` 组件在选项卡式界面中展示多个代码块，便于用户对比不同编程语言的实现，或查看完成同一任务的其他方案。

<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>

## 创建代码组

要创建代码组，请用 `<CodeGroup>` 标签包裹多个代码块。每个代码块必须包含一个标题，该标题会显示为选项卡的标签。

## 语言下拉菜单

你可以使用 `dropdown` 属性将代码组中的标签页替换为下拉菜单，以在不同语言之间切换。

<CodeGroup dropdown>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>
