`Columns` 组件用于将多个 `Card` 组件分组。最常见的用法是通过指定网格列数，将卡片排成网格。

<Columns cols={2}>
  <Card title="Get started" icon="rocket">
    使用我们的快速入门指南配置你的项目。
  </Card>

  <Card title="API reference" icon="code">
    查看你的 API 的端点、参数和示例。
  </Card>
</Columns>

```mdx Columns 示例
<Columns cols={2}>
  <Card title="快速上手">
    按照我们的快速入门指南配置你的项目。
  </Card>
  <Card title="API 参考">
    浏览你的 API 的端点、参数和示例。
  </Card>
</Columns>
```

<div id="properties">
  ## 属性
</div>

<ResponseField name="cols" default={2}>
  每行的列数。
</ResponseField>
