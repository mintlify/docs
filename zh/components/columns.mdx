---
title: '列'
description: '以网格形式并排显示卡片'
keywords: ['card groups']
---

`Columns` 组件可将多个 `Card` 组件归为一组。它通常用于通过指定网格列数，将卡片排列成网格。

<Columns cols={2}>
  <Card title="开始使用" icon="rocket">
    按照我们的快速入门指南设置你的项目。
  </Card>

  <Card title="API 参考" icon="code">
    了解你的 API 的端点、参数和示例。
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="开始使用">
    通过我们的快速入门指南设置您的项目。
  </Card>
  <Card title="API 参考">
    探索您的 API 的端点、参数和示例。
  </Card>
</Columns>
```


<div id="properties">
  ## 属性
</div>

<ResponseField name="cols" default={2}>
  每行包含的列数。
</ResponseField>
