`Columns` 组件用于将多个 `Card` 组件分组。它通常通过指定网格列数，将卡片排布为网格。

<Columns cols={2}>
  <Card title="Get started" icon="rocket">
    按照我们的快速入门指南设置项目。
  </Card>

  <Card title="API reference" icon="code">
    浏览你 API 的端点、参数和示例。
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="开始使用">
    通过我们的快速入门指南设置您的项目。
  </Card>
  <Card title="API 参考">
    探索您的 API 的端点、参数和示例。
  </Card>
</Columns>
```


<div id="properties">
  ## 属性
</div>

<ResponseField name="cols" default={2}>
  每行包含的列数。
</ResponseField>
