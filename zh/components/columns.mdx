---

`Columns` 组件可让你将多个 `Card` 组件分组。它通常用于通过指定网格列数，将卡片排成网格。

<Columns cols={2}>
  <Card title="开始使用" icon="rocket">
    使用我们的快速入门指南快速设置你的项目。
  </Card>

  <Card title="API 参考" icon="code">
    浏览 API 的端点、参数和示例。
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="开始使用">
    使用我们的快速入门指南设置您的项目。
  </Card>
  <Card title="API 参考">
    浏览您的 API 端点、参数和示例。
  </Card>
</Columns>
```


<div id="properties">
  ## 属性
</div>

<ResponseField name="cols" default={2}>
  每行包含的列数。
</ResponseField>
