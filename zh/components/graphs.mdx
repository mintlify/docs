使用 `<Graph>` 组件在文档中创建交互式图表和数据可视化。图表帮助用户一目了然地理解数据趋势、比较和分布。

## 基本用法

将数据配置包装在 `<Graph>` 组件中并指定图表类型。

<Graph type="line" title="月度收入">
  {[
    { month: "1月", revenue: 4000 },
    { month: "2月", revenue: 3000 },
    { month: "3月", revenue: 5000 },
    { month: "4月", revenue: 4500 },
    { month: "5月", revenue: 6000 },
    { month: "6月", revenue: 5500 }
  ]}
</Graph>

```jsx
<Graph type="line" title="月度收入">
  {[
    { month: "1月", revenue: 4000 },
    { month: "2月", revenue: 3000 },
    { month: "3月", revenue: 5000 },
    { month: "4月", revenue: 4500 },
    { month: "5月", revenue: 6000 },
    { month: "6月", revenue: 5500 }
  ]}
</Graph>
```

## 图表类型

### 折线图

显示随时间变化的趋势或连续数据。

<Graph type="line" title="用户增长" xKey="month" yKey="users">
  {[
    { month: "1月", users: 1200 },
    { month: "2月", users: 1900 },
    { month: "3月", users: 2400 },
    { month: "4月", users: 3100 },
    { month: "5月", users: 4200 },
    { month: "6月", users: 5100 }
  ]}
</Graph>

```jsx
<Graph type="line" title="用户增长" xKey="month" yKey="users">
```

## 多系列

在同一图表上显示多个数据系列进行比较。

<Graph type="line" title="收入 vs 支出" xKey="month">
  {{
    data: [
      { month: "1月", revenue: 4000, expenses: 2400 },
      { month: "2月", revenue: 3000, expenses: 1398 },
      { month: "3月", revenue: 5000, expenses: 3800 },
      { month: "4月", revenue: 4500, expenses: 3908 },
      { month: "5月", revenue: 6000, expenses: 4800 },
      { month: "6月", revenue: 5500, expenses: 3800 }
    ],
    series: [
      { key: "revenue", label: "收入", color: "#22c55e" },
      { key: "expenses", label: "支出", color: "#ef4444" }
    ]
  }}
</Graph>

```jsx
<Graph type="line" title="收入 vs 支出" xKey="month">
```

## 属性

<ResponseField name="type" type="string" required>
  图表类型。选项：`line`、`bar`、`area`、`pie`、`donut`。
</ResponseField>

<ResponseField name="title" type="string">
  显示在可视化上方的图表标题。
</ResponseField>

<ResponseField name="xKey" type="string">
  用于 x 轴的数据键（适用于折线图、柱状图和面积图）。
</ResponseField>

<ResponseField name="yKey" type="string">
  用于 y 轴的数据键（适用于单系列图表）。
</ResponseField>

<ResponseField name="height" type="number" default="300">
  图表高度（像素）。
</ResponseField>

<ResponseField name="color" type="string">
  单系列图表的主色。接受十六进制、RGB 或 CSS 颜色名称。
</ResponseField>

<ResponseField name="stacked" type="boolean" default="false">
  将多个系列堆叠在一起（适用于柱状图和面积图）。
</ResponseField>

<ResponseField name="showLegend" type="boolean" default="true">
  显示或隐藏图表图例。
</ResponseField>

<ResponseField name="showGrid" type="boolean" default="true">
  显示或隐藏网格线。
</ResponseField>

<ResponseField name="children" type="array | object" required>
  图表数据。传递数组用于单系列，或传递包含 `data` 和 `series` 的对象用于多系列。
</ResponseField>
