---
title: "可展开内容"
description: "切换以显示嵌套属性"
icon: "list-tree"
---

使用可展开内容在响应字段中显示或隐藏嵌套内容。可展开内容在 API 文档中展示复杂对象的属性时尤其有用。

<ResponseField name="user" type="User Object">
  <Expandable title="properties">

<ResponseField name="full_name" type="string">
  用户的全名
</ResponseField>

<ResponseField name="is_over_21" type="boolean">
  用户是否年满 21 岁
</ResponseField>

  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      用户的全名
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      用户是否年满 21 岁
    </ResponseField>
  </Expandable>
</ResponseField>
```

## 属性

<ResponseField name="title" type="string">
  要显示的对象名称。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  设为 `true` 时，页面加载后默认展开。
</ResponseField>
