---
title: "可展开项"
description: "切换显示嵌套属性"
icon: "list-tree"
---

使用可展开项在响应字段中显示或隐藏嵌套内容。它们在 API 文档中展示复杂对象的属性时尤其有用。

<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      用户的全名
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      用户是否年满 21 岁
    </ResponseField>
  </Expandable>
</ResponseField>

```mdx 可展开示例
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      用户的完整姓名
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      该用户是否已年满 21 岁
    </ResponseField>
  </Expandable>
</ResponseField>
```


<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string">
  要显示对象的名称。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  设为 `true` 以在页面加载时自动展开。
</ResponseField>
