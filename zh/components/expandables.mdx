---
title: "可展开项"
description: "切换以显示嵌套属性"
---

使用可展开项在响应字段中显示和隐藏嵌套内容。可展开项在 API 文档中展示复杂对象属性时尤为实用。

<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      用户的全名
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      用户是否已年满 21 岁
    </ResponseField>
  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="属性">
    <ResponseField name="full_name" type="string">
      用户的全名
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      用户是否超过21岁
    </ResponseField>
  </Expandable>
</ResponseField>
```


<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string">
  正在显示的对象名称。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  设为 `true`，使展开项在页面加载时默认打开。
</ResponseField>
