---
title: '标注'
description: '使用标注为你的内容添加醒目的上下文'
---

标注可以设置为 Note、Warning、Info、提示、Check、Danger，或创建你自己的标注：

<Note>这会在内容中添加一条 Note</Note>

```mdx
<Note>这在内容中添加了一个注释</Note>
```

<Warning>这会显示一条 Warning，提示你留意</Warning>

```mdx
<Warning>这是一个需要注意的警告提示</Warning>
```

<Info>用于强调重要信息</Info>

```mdx
<Info>这会突出显示重要信息</Info>
```

<Tip>这是一个有用的提示</Tip>

```mdx
<Tip>这是一个有用的提示</Tip>
```

<Check>这会使其处于“已勾选”状态</Check>

```mdx
<Check>这为我们提供了选中状态</Check>
```

<Danger>这是一个危险警示</Danger>

```mdx
<Danger>这是一个危险提示</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">这是一个自定义公告</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">这是一个自定义提示框</Callout>
```
