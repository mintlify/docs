---
title: '提示框'
description: '使用提示框为内容添加醒目的说明'
icon: 'info'
---

提示框可设为 Note、Warning、Info、Tip、Check、Danger，或自定义你自己的提示框：

<Note>这会在内容中添加一条说明</Note>

```mdx
<Note>这在对象中添加了一个注释</Note>
```

<Warning>这会触发一个 Warning，提醒你留意</Warning>

```mdx
<Warning>这会发出一个需要注意的警告</Warning>
```

<Info>用于提示关键信息</Info>

```mdx
<Info>这会突出显示重要信息</Info>
```

<Tip>这是一条有用的提示</Tip>

```mdx
<Tip>这是一个有用的提示</Tip>
```

<Check>这会使其进入“已选中”状态</Check>

```mdx
<Check>这为我们提供了选中状态</Check>
```

<Danger>这是一个危险警示</Danger>

```mdx
<Danger>这是一个危险提示</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> 自定义标注</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">这是一个自定义提示框</Callout>
```
