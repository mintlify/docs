---
title: "工具提示"
description: "当用户将鼠标悬停在文本上时显示定义"
icon: "message-square"
---

使用工具提示，在用户将鼠标悬停于一段文本时提供额外的上下文或定义。工具提示可以包含可选的行动呼吁链接。

**示例**：<Tooltip tip="Application Programming Interface: a set of protocols for software applications to communicate." cta="阅读我们的 API 指南" href="/zh/api-reference">API</Tooltip> 文档可帮助开发者了解如何与你的服务集成。

```mdx Tooltip example wrap
<Tooltip tip="应用程序编程接口：软件应用程序之间通信的协议集合。" cta="阅读我们的 API 指南" href="/api-reference">API</Tooltip> 文档帮助开发者了解如何与您的服务进行集成。
```

<div id="properties">
  ## 属性
</div>

<ResponseField name="tip" type="string" required>
  工具提示中显示的文本。
</ResponseField>

<ResponseField name="cta" type="string">
  工具提示内链接的号召性用语文本。
</ResponseField>

<ResponseField name="href" type="string">
  号召性用语链接的 URL。使用 `cta` 时为必填。
</ResponseField>
