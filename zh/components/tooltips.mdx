---
title: "工具提示"
description: "当鼠标悬停在文本上时显示定义"
---

当用户将鼠标悬停在一段文本上时，使用工具提示提供额外的上下文或定义。工具提示可以包含可选的行动呼吁链接。

**示例**: <Tooltip tip="Application Programming Interface: a set of protocols for software applications to communicate." cta="阅读我们的 API 指南" href="/zh/api-reference">API</Tooltip> 文档有助于开发者了解如何将您的服务集成到他们的应用中。

```mdx Tooltip example wrap
<Tooltip tip="应用程序编程接口：软件应用程序之间通信的协议集合。" cta="阅读我们的 API 指南" href="/api-reference">API</Tooltip> 文档帮助开发者了解如何与您的服务进行集成。
```

<div id="properties">
  ## 属性
</div>

<ResponseField name="tip" type="string" required>
  悬浮提示中显示的文本。
</ResponseField>

<ResponseField name="cta" type="string">
  悬浮提示内链接的行动呼吁文本。
</ResponseField>

<ResponseField name="href" type="string">
  行动呼吁链接的URL。使用 `cta` 时为必填。
</ResponseField>
