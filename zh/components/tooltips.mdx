使用工具提示可在用户将鼠标悬停在文本上时提供额外的上下文或定义。工具提示还可以包含可选的行动呼吁链接。

**示例**: <Tooltip tip="应用程序编程接口：一组用于使软件应用彼此通信的协议。" cta="阅读我们的 API 指南" href="/zh/api-reference">API</Tooltip> 文档可帮助开发者了解如何将您的服务集成到他们的系统中。

```mdx Tooltip example wrap
<Tooltip tip="应用程序编程接口：一组用于软件应用程序通信的协议。" cta="阅读我们的 API 指南" href="/api-reference">API</Tooltip> 文档可帮助开发者了解如何与您的服务集成。
```

<div id="properties">
  ## 属性
</div>

<ResponseField name="tip" type="string" required>
  悬浮提示中显示的文本。
</ResponseField>

<ResponseField name="cta" type="string">
  悬浮提示内链接的行动呼吁文本。
</ResponseField>

<ResponseField name="href" type="string">
  行动呼吁链接的URL。使用 `cta` 时为必填。
</ResponseField>
