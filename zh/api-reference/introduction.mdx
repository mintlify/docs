Mintlify 的 REST API 使你能够通过编程与文档交互、触发更新，并嵌入由 AI 驱动的聊天体验。

<div id="endpoints">
  ## 端点
</div>

* [触发更新](/zh/api-reference/update/trigger): 按需触发站点更新。
* [获取更新状态](/zh/api-reference/update/status): 获取更新状态及文档的其他详情。
* [生成 AI 助手消息](/zh/api-reference/assistant/create-assistant-message): 将基于你的文档训练的 AI 助手嵌入到任意应用中。
* [搜索文档](/zh/api-reference/assistant/search): 在你的文档中搜索。

<div id="authentication">
  ## 认证
</div>

你可以通过[控制台](https://dashboard.mintlify.com/settings/organization/api-keys)生成 API key。API key 与整个组织关联，可用于多个部署。

<div id="admin-api-key">
  ### 管理员 API key
</div>

管理员 API key 用于 [Trigger update](/zh/api-reference/update/trigger) 和 [Get update status](/zh/api-reference/update/status) 端点。

管理员 API key 以 `mint_` 前缀开头。请妥善保管你的管理员 API key。

<div id="assistant-api-key">
  ### AI 助手 API key
</div>

AI 助手 API key 用于 [Generate assistant message](/zh/api-reference/assistant/create-assistant-message) 和 [Search documentation](/zh/api-reference/assistant/search) 端点。

AI 助手 API key 以 `mint_dsc_` 前缀开头。

AI 助手 API 的 **key** 是应在服务端保存且需保密的 token。

AI 助手 API 的 **token** 是公开令牌，可在前端代码中引用。

<Note>
  使用 AI 助手 API token 的调用可能产生费用：要么消耗你的 AI 助手额度，要么产生超额费用。
</Note>
