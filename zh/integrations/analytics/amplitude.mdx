将以下内容添加到你的 `docs.json` 文件中，以将 Analytics 数据发送到 Amplitude。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json 示例
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
