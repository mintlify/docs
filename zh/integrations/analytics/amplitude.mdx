---
title: "Amplitude"
---

在你的 `docs.json` 文件中添加以下内容，将分析数据发送到 Amplitude。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Example
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
