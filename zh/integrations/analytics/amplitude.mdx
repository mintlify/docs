---
title: "Amplitude"
---

在你的 `docs.json` 文件中添加以下内容以将 Analytics 数据发送到 Amplitude。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json 示例
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
