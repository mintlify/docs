在你的 `docs.json` 文件中添加以下内容，将 Analytics 数据发送到 Common Room。

你可以从 Common Room Signals 设置中获取 `siteId`。它必须是有效的 UUID。

<CodeGroup>

  ```json docs.json 中的 Analytics 选项
  "integrations": {
      "commonRoom": {
          "siteId": "required"
      }
  }
  ```

  ```json 示例
  "integrations": {
      "commonRoom": {
          "siteId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
      }
  }
  ```
</CodeGroup>
