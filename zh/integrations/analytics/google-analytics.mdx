您需要新建一个用于 Mintlify 的 <Tooltip tip="Google Analytics 4">GA4</Tooltip> 属性。收集到的数据会归入与您其他 Google Analytics 数据相同的项目。

如果您仍在使用旧版 Google Analytics（Universal Analytics），依然可以创建 <Tooltip tip="Google Analytics 4">GA4</Tooltip> 属性。<Tooltip tip="Google Analytics 4">GA4</Tooltip> 数据与 UA 数据略有不同，但仍会汇总到同一个项目中。

<div id="how-to-connect-ga4-to-mintlify">
  ## 如何将 GA4 连接到 Mintlify
</div>

<div id="create-a-web-stream">
  ### 创建 Web 数据流
</div>

你需要先创建一个 Web 数据流以获取 Measurement ID，然后将其填入 Mintlify。

点击 Google Analytics 界面左下角的齿轮图标，进入设置。接着点击 Data Streams（数据流）。

<Frame>![](/images/ga4-web-streams.png)</Frame>

创建一个 Web 数据流，并将你的 Mintlify 文档站点 URL 填入为数据流的 URL。

你的 Measurement ID 格式为 `G-XXXXXXX`，创建 Web 数据流后会立即显示在 Stream Details（数据流详情）中。

<div id="put-measurement-id-in-docsjson">
  ### 在 docs.json 中填写 Measurement ID
</div>

将你的 Measurement ID 添加到 `docs.json` 文件中，如下所示：

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<div id="wait">
  ### 请稍候
</div>

Google Analytics 需要两到三天才会显示你的数据。

你可以使用 [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) 检查分析功能是否已正确启用。每当 GA4 发出请求时，该扩展都会在你的浏览器控制台中输出日志。
