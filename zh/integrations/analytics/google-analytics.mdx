您需要创建一个新的<Tooltip tip="Google Analytics 4">GA4</Tooltip>属性以在 Mintlify 中使用。收集的数据会归入与您其他 Google Analytics 数据相同的项目。

如果您使用的是旧版 Google Analytics（Universal Analytics），仍可创建<Tooltip tip="Google Analytics 4">GA4</Tooltip>属性。<Tooltip tip="Google Analytics 4">GA4</Tooltip>数据与 UA 数据略有不同，但仍会被收集到同一项目中。

## 如何将 GA4 连接到 Mintlify

### 创建 Web 数据流

你需要先创建一个 Web 数据流，以获取 Measurement ID 并将其填入 Mintlify。

点击 Google Analytics 界面左下角的齿轮图标，然后点击 Data Streams。

<Frame>![](/images/ga4-web-streams.png)</Frame>

创建一个 Web 数据流，并将你的 Mintlify 文档站点 URL 填入为数据流的 URL。

你的 Measurement ID 形如 `G-XXXXXXX`，创建 Web 数据流后会立即在 Stream Details 中显示。

### 在 docs.json 中填写 Measurement ID

将你的 Measurement ID 添加到 `docs.json` 文件中，如下所示：

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

### 等待

Google Analytics 通常需要两到三天才会显示你的数据。

你可以使用 [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) 检查分析是否已正确启用。每当 GA4 发出请求时，该扩展都会在浏览器控制台中记录日志。
