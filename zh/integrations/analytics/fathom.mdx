在你的 `docs.json` 文件中添加以下内容，将分析数据发送到 Fathom。

你可以在脚本设置中获取 `siteId`。

<CodeGroup>
  ```json docs.json 中的 Analytics 选项
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json 示例
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
