---
title: "Fathom"
---

在你的 `docs.json` 文件中添加以下内容，将 Analytics 数据发送到 Fathom。

你可以在脚本设置中找到 `siteId`。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "fathom": {
        "siteId": "必填"
    }
}
```

```json 示例
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
