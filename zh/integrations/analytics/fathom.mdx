---
title: "Fathom"
description: "使用 Fathom 跟踪文档分析数据。"
---

在你的 `docs.json` 文件中添加以下配置，将分析数据发送到 Fathom。

你可以在脚本设置中找到 `siteId`。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "fathom": {
        "siteId": "必填"
    }
}
```

```json 示例
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
