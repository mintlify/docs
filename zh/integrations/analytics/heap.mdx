---
title: "Heap"
description: "使用 Heap 分析跟踪用户交互。"
---

将以下内容添加到你的 `docs.json` 文件中，以将分析数据发送到 Heap。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "heap": {
        "appId": "必填"
    }
}
```

```json 示例
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
