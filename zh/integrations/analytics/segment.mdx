title: "Segment"

keywords: ["Segment integration", "write key", "event streaming", "CDP integration"]
---

将你的 Segment write key 添加到 `docs.json` 文件，以将分析数据发送到 Segment。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json 示例
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
