---
title: "Segment"
---

在 `docs.json` 文件中添加你的 Segment 写入密钥（write key），即可将分析数据发送到 Segment。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Example
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
