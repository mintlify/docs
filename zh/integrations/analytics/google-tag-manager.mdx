---
title: "Google Tag Manager"
description: "使用 Google Tag Manager 管理 Analytics 标签和事件。"
keywords: ["GTM integration", "tag management", "tag ID", "cookie consent"]
---

将你的 tag ID 添加到 `docs.json` 文件，我们会将 Google Tag Manager 脚本注入到你所有页面中。

如有需要，你需要通过 Google Tag Manager 配置 Cookie 同意横幅。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json 示例
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
