---
title: "Google Tag Manager"
description: "使用 Google Tag Manager 管理分析标签和事件。"
---

将你的标签 ID 添加到 `docs.json` 文件中，我们会把 Google Tag Manager 脚本注入到你所有页面。

如有需要，你需通过 Google Tag Manager 自行设置 Cookie 同意横幅。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json 示例
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
