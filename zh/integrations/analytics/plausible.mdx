---
title: "Plausible"
description: "使用 Plausible 跟踪简洁且注重隐私的分析数据。"
---

在 `docs.json` 中添加站点的 domain，以将 Analytics 发送到 Plausible。

<Info>

    不要在 domain 或 server 中包含 `https://`。

</Info>

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json 示例
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
