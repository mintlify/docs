---
title: "Plausible"
---

将你站点的 domain 添加到 `docs.json` 中，以将 Analytics 数据发送到 Plausible。

<Info>

    不要在 domain 或 server 中包含 `https://`。

</Info>

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json 示例
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
