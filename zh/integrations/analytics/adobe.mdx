将以下内容添加到你的 `docs.json` 文件中，以将分析数据发送到 Adobe Analytics。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "adobe": {
        "launchUrl": "required"
    }
}
```

```json Example
"integrations": {
    "adobe": {
        "launchUrl": "https://assets.adobedtm.com/launch-EN93497c30fdf0424eb678d5f4ffac66dc.min.js"
    }
}
```

</CodeGroup>

<div id="get-your-launch-url">
  ## 获取你的 Launch URL
</div>

1. 登录 [Adobe Experience Platform Data Collection](https://experience.adobe.com/#/data-collection/)。
2. 打开你的 Launch 属性，然后转到 **Environments** 选项卡。
3. 从生产环境中复制脚本 URL。
