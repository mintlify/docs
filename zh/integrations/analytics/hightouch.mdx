---
title: "Hightouch"
---

在你的 `docs.json` 文件中添加以下配置，以将 Analytics 数据发送到 Hightouch。

<Info>

  `apiHost` 中不要包含 `https://`。

</Info>

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "hightouch": {
        "writeKey": "必填"
        "apiHost": "可选"
    }
}
```

```json 示例
"integrations": {
    "hightouch": {
        "writeKey": "9132c81do371p76sr11da0075469b54f77649c9a067dp0303p56q0q64n072336"
        "apiHost": "us-east-1.hightouch-events.com" # 可选，默认值为 `us-east-1.hightouch-events.com`
    }
}
```

</CodeGroup>
