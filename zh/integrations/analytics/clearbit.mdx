---
title: "Clearbit"
---

在你的 `docs.json` 文件中添加以下内容，将分析数据发送到 Clearbit。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
