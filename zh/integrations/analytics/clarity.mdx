在你的 `docs.json` 文件中添加以下内容，以将分析数据发送到 Microsoft Clarity。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "clarity": {
        "projectId": "必填"
    }
}
```

```json 示例
"integrations": {
    "clarity": {
        "projectId": "abc123xyz"
    }
}
```

</CodeGroup>

<div id="get-your-project-id">
  ## 获取项目 ID
</div>

1. 创建一个 [Clarity 帐户](https://clarity.microsoft.com/projects)。
2. 点击 **Get tracking code**。
3. 从跟踪代码中复制项目 ID。
