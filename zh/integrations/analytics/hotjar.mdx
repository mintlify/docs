在你的 `docs.json` 文件中添加以下配置，将数据发送到 Hotjar 进行 Analytics。

```json Analytics options in docs.json
"integrations": {
    "hotjar": {
        "hjid": "required",
        "hjsv": "required"
    }
}
```
