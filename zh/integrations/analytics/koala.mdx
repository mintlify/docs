---
title: "Koala"
---

在你的 `docs.json` 文件中添加如下内容，以将分析数据发送到 Koala。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
