---
title: "Koala"
---

在你的 `docs.json` 文件中添加以下内容，以将 Analytics 数据发送到 Koala。

<CodeGroup>

```json docs.json 中的 Analytics 选项
"integrations": {
    "koala": {
        "publicApiKey": "必填"
    }
}
```

```json 示例
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
