title: "Koala"
description: "使用 Koala 识别访问你文档站点的公司和访客。"
keywords: ["Koala 集成", "访客识别", "账户评分", "public API key"]
---

在你的 `docs.json` 文件中添加以下内容，启用 Koala 访客识别。

<CodeGroup>
  ```json docs.json 中的 Analytics 选项
  "integrations": {
      "koala": {
          "publicApiKey": "required"
      }
  }
  ```

  ```json 示例
  "integrations": {
      "koala": {
          "publicApiKey": "pk_abc123def456"
      }
  }
  ```
</CodeGroup>
