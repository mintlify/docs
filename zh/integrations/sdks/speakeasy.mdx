由 Speakeasy SDK 自动生成的代码片段可直接集成到 Mintlify 的 API 参考文档中。这些 SDK 使用代码片段将显示在由 Mintlify 驱动的文档站点的[API交互测试台](/zh/api-playground/overview)中。

<Frame>
  ![带有 Speakeasy 代码片段的 Mintlify API 交互测试台。](/images/speakeasy/mintlify-with-speakeasy-openapi.png)
</Frame>

## 前置条件

要将 Mintlify 与 Speakeasy 集成，你需要准备：

- 一个 [Mintlify 文档仓库](/zh/quickstart#creating-the-repository)。
- 一个由 Speakeasy 生成并已配置[自动化代码示例 URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls)的 SDK。

## 设置集成

要将 Speakeasy 与 Mintlify 集成，您需要从注册表获取该 API 的合并规范公共 URL，并更新您的 `docs.json` 配置文件。

### 从注册表获取该 API 的合并规范公共 URL

前往 [Speakeasy 仪表盘](https://app.speakeasy.com) 并打开 **API Registry** 选项卡。打开该 API 的 `*-with-code-samples` 条目。

<Frame>
  ![Speakeasy API Registry 页面截图。API Registry 选项卡用红色方框和数字 1 标出，API 的条目用红色方框和数字 2 标出。](/images/speakeasy/openapi-registry-and-combined-spec.png)
</Frame>

<Note>
  如果该条目未标记为 **Combined Spec**，请确保该 API 已配置 [自动代码示例 URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls)。
</Note>

在该注册表条目的页面中，复制提供的公共 URL。

<Frame>
  ![显示合并规范注册表条目，并用红色方框标出复制 URL 功能的截图。](/images/speakeasy/copy-combined-spec-url.png)
</Frame>

### 更新您的 `docs.json` 配置文件

将合并规范 URL 添加到 `docs.json` 文件中的 **Anchors** 或 **Tabs** 部分。

通过如下方式更新 `docs.json` 文件中的 `anchor` 字段，将合并规范 URL 添加到锚点：

```json docs.json
{
  "anchors": [
    {
      "name": "API Reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL",
      "url": "api-reference",
      "icon": "square-terminal"
    }
  ]
}
```

通过如下方式更新 `docs.json` 文件中的 `tab` 字段，将合并规范 URL 添加到选项卡：

```json docs.json
{
  "tabs": [
    {
      "name": "API Reference",
      "url": "api-reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL"
    }
  ]
}
```

现在，您可以在 API 文档中查看由 Speakeasy 生成的代码片段，并在 API交互测试台中进行尝试。
