---
title: "Intercom"
description: "连接 Intercom，用于应用内客户消息沟通。"
---

在你的 `docs.json` 文件中添加以下内容，以接入 [Intercom](https://www.intercom.com) 小组件。

<CodeGroup>

```json Integration options in docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json 示例
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

</CodeGroup>
