---
title: "Intercom"
---

在你的 `docs.json` 文件中添加以下配置以启用 [Intercom](https://www.intercom.com) 小部件。

<CodeGroup>

```json Integration options in docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json 示例
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

</CodeGroup>
