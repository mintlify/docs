---
title: "Intercom"
---

在你的 `docs.json` 文件中添加以下内容，以集成 [Intercom](https://www.intercom.com) 小组件。

<CodeGroup>

```json Integration options in docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json Example
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

</CodeGroup>
