---
title: "Front"
---

在你的 `docs.json` 文件中添加以下内容以集成 [Front Chat](https://front.com) 小组件。

<CodeGroup>

```json Integration options in docs.json
"integrations": {
    "frontchat": "CHAT_ID"
}
```

```json Example
"integrations": {
    "frontchat": "1365d046d7c023e9b030ce90d02d093a"
}
```

</CodeGroup>
