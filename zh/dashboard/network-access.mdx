<Info>
  网络访问控制需要 [Enterprise 计划](https://mintlify.com/pricing?ref=network-access)。
</Info>

使用 IP 允许列表将仪表板和编辑器的访问限制为受信任的 IP 地址和 CIDR 范围。启用允许列表后,组织成员只能从允许的 IP 地址登录并使用仪表板和 Web 编辑器。

IP 允许列表适用于你的 Mintlify 仪表板和编辑器。它不会影响你已发布的文档站点。API 请求使用单独的 IP 控制。

你可以通过两种方式允许访问:

- **单个 IP 地址**:允许单个地址,例如 `203.0.113.7`。
- **CIDR 范围**:允许一段地址,例如 `10.0.0.0/8`。使用 CIDR 表示法向整个办公室或 VPN 网络授予访问权限,而无需逐一列出每个地址。

<div id="configure-the-ip-allowlist">
  ## 配置 IP 允许列表
</div>

<Warning>
  在启用允许列表之前,请将你当前的 IP 地址添加到列表中。如果你的地址不在列表中,你可能会将自己锁定在仪表板之外。[Network access](https://app.mintlify.com/settings/organization/network-access) 页面会显示你当前的 IP 地址,以帮助你避免这种情况。
</Warning>

1. 在 Mintlify 仪表板中,前往 [Network access](https://app.mintlify.com/settings/organization/network-access) 页面。
2. 在 **Allowed IP addresses** 下,输入单个 IP 地址(例如 `203.0.113.7`)或 CIDR 范围(例如 `10.0.0.0/8`)。
3. 点击 **Add**。
4. 对你想要允许的每个地址或范围重复此操作。
5. 启用允许列表以开始执行。
