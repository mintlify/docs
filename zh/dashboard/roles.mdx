<Info>
  RBAC 功能适用于[自定义方案](https://mintlify.com/pricing?ref=rbac)。
</Info>

Mintlify 提供两种控制台访问级别：Editor 和 Admin。

以下为仅限 Admin 角色执行的操作：

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| 更新用户角色            |   ❌   |  ✅   |
| 删除用户                |   ❌   |  ✅   |
| 邀请管理员              |   ❌   |  ✅   |
| 管理并更新账单          |   ❌   |  ✅   |
| 更新自定义 domain       |   ❌   |  ✅   |
| 更新 Git source         |   ❌   |  ✅   |
| 删除组织                |   ❌   |  ✅   |

控制台中的其他操作对两种角色均可用。

你可以邀请任意数量的管理员，但建议仅将管理员访问权限授予确有需要的用户。
