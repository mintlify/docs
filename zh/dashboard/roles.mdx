<Info>
  RBAC 功能适用于 [Enterprise 方案](https://mintlify.com/pricing?ref=rbac)。
</Info>

Mintlify 提供两种控制台访问级别：Editor 和 Admin。

以下为仅限 Admin 角色执行的操作：

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| 更新用户角色            |   ❌   |  ✅   |
| 删除用户                |   ❌   |  ✅   |
| 邀请管理员              |   ❌   |  ✅   |
| 管理与更新计费          |   ❌   |  ✅   |
| 更新自定义 domain       |   ❌   |  ✅   |
| 更新 Git source         |   ❌   |  ✅   |
| 删除 org                |   ❌   |  ✅   |

控制台上的其他操作两种角色均可执行。

你可以邀请任意数量的管理员，但我们建议将管理员访问权限仅授予确有需要的用户。
