<div id="usage">
  ## 使用方法
</div>

使用此端点导出文档搜索分析数据。结果按命中次数从高到低排序，显示用户最常搜索的词语。

使用响应中返回的 `nextCursor` 参数对结果进行分页。当 `nextCursor` 不为 null 时继续获取数据。

<div id="filtering">
  ## 筛选
</div>

使用 `dateFrom` 和 `dateTo` 参数按日期范围过滤搜索数据。
