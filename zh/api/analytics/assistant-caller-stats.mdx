<div id="usage">
  ## 使用方法
</div>

使用此端点查看助手查询在不同调用方类型之间的分布情况。响应将总查询次数细分为：

- **web**: 来自文档站点的查询
- **api**: 来自直接 API 调用的查询
- **other**: 来自其他来源的查询（例如集成、SDK）
- **total**: 所有查询类型的总和

<div id="filtering">
  ## 筛选
</div>

使用 `dateFrom` 和 `dateTo` 参数按日期范围筛选。
