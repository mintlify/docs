<div id="usage">
  ## 用法
</div>

使用此端点导出文档的页面浏览计数。浏览量按流量来源分类——人类访客和 AI 机器人——并包含全站总计。

使用 `limit` 和 `offset` 参数对结果进行分页。当 `hasMore` 为 `true` 时继续获取数据，每次将 `offset` 增加 `limit`。

<div id="filtering">
  ## 筛选
</div>

使用 `dateFrom` 和 `dateTo` 参数按日期范围筛选浏览量。日期格式可以是 ISO 8601 或 `YYYY-MM-DD`。

<div id="response-data">
  ## 响应数据
</div>

响应包含：
- **totals**：日期范围内全站浏览计数，分为 `human`、`ai` 和 `total`
- **views**：按路径的浏览计数，每个路径包含 `human`、`ai` 和 `total` 分类
- **hasMore**：是否存在更多结果页
