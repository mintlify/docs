<div id="usage">
  ## 用法
</div>

使用此端点获取文档页面反馈的概览。结果按页面路径聚合，并按反馈总数降序排列。

<div id="filtering">
  ## 筛选
</div>

按以下条件筛选反馈：

- **日期范围**：使用 `dateFrom` 和 `dateTo` 限制结果到特定时间段
- **来源**：按 `code_snippet` 或 `contextual` 反馈类型筛选
- **状态**：按 `pending`、`in_progress`、`resolved` 或 `dismissed` 等状态值筛选

<div id="response">
  ## 响应
</div>

响应中的每个条目代表一个页面，包含：

- **thumbsUp**：正面上下文评价的数量
- **thumbsDown**：负面上下文评价的数量
- **code**：代码片段反馈条目的数量
- **total**：该页面所有反馈的总和
