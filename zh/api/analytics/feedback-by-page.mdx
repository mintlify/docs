<div id="usage">
  ## 用法
</div>

使用此端点获取文档中特定页面的反馈汇总计数。响应包括点赞数量、踩数量、代码片段反馈数量以及总数。

与返回单条反馈记录的[获取反馈](/zh/api/analytics/feedback)端点不同，此端点返回指定页面路径下所有反馈的汇总摘要。

<div id="filtering">
  ## 筛选
</div>

可以按以下维度筛选反馈：
- **页面路径**（必填）：指定 `path` 查询参数以定位特定的文档页面
- **日期范围**：使用 `dateFrom` 和 `dateTo` 将结果限定在特定时间段内
