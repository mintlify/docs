<div id="usage">
  ## 用法
</div>

使用此端点获取文档中特定页面的聚合反馈计数。响应包括点赞数、点踩数和代码片段反馈数以及总计。

这对于构建仪表板或报告非常有用，可以按页面显示反馈趋势，而无需检索单独的反馈条目。

<div id="filtering">
  ## 筛选
</div>

可以按以下维度筛选反馈计数：
- **页面路径**（必填）：要聚合反馈的文档页面路径
- **日期范围**：使用 `dateFrom` 和 `dateTo` 将聚合限定在特定时间段内
