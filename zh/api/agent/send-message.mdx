## 用法

此端点向现有代理作业会话发送后续消息。用于在初始作业创建后继续与代理的对话，例如请求额外的更改或提供澄清说明。

## 示例

```javascript
const response = await fetch(
  `https://api.mintlify.com/v2/agent/${projectId}/job/${jobId}/message`,
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      prompt: '也请在 API 文档中添加错误处理的示例'
    })
  }
);

const job = await response.json();
```

## 速率限制

代理 API 的限制如下：

- 每个 Mintlify 项目每小时最多可调用 100 次
