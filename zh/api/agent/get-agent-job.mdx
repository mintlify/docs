## 用法

此端点通过代理任务的唯一标识符获取该任务的详细信息和状态。可用于轮询任务完成状态并获取拉取请求链接。

## 作业状态

`status` 字段表示作业的当前状态：

| 状态 | 描述 |
|------|------|
| `pending` | 作业已排队等待开始 |
| `running` | 作业正在处理中 |
| `completed` | 作业已成功完成 |
| `failed` | 作业遇到错误 |

## 响应字段

| 字段 | 描述 |
|------|------|
| `id` | 作业的唯一标识符 |
| `status` | 当前作业状态 |
| `source.repository` | GitHub 仓库 URL |
| `source.ref` | 进行更改的分支名称 |
| `model` | 用于作业的 AI 模型 |
| `prLink` | 拉取请求 URL（完成后可用） |
| `createdAt` | 作业创建时间戳 |
| `archivedAt` | 作业完成时间戳 |
