<div id="usage">
  ## 用法
</div>

此端点通过代理任务的唯一标识符获取该任务的详细信息和状态。可用于查看先前创建的代理任务的进度、状态和结果。

<div id="job-details">
  ## 作业详情
</div>

响应包含以下信息：

- 作业执行状态与完成情况
- branch 信息和拉取请求（PR；亦称"合并请求"/Merge Request）详情
- 会话 metadata 与时间戳
