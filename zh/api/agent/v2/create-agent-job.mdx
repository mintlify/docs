---
title: "创建 agent 任务"
description: "创建一个在后台运行的 agent 任务，并自动创建包含文档更改的拉取请求。"
openapi: /zh/admin-openapi.json POST /v2/agent/{projectId}/job
keywords: [ "agent 任务", "创建", "自动化", "自动" ]
---

此端点创建一个后台 agent 任务。任务异步执行 — 使用 [get agent job](/zh/api/agent/v2/get-agent-job) 端点轮询状态更新。

如果 agent 成功编辑文件，将自动创建拉取请求（PR；亦称"合并请求"/Merge Request），并在任务响应的 `prLink` 字段中填充相应链接。

<div id="rate-limits">
  ## 速率限制
</div>

- 每个 Mintlify 项目每小时最多可调用 100 次

<div id="usage">
  ## 用法
</div>

```bash
curl -X POST https://api.mintlify.com/v2/agent/{projectId}/job \
  -H "Authorization: Bearer mint_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Add a quickstart guide for the Python SDK"}'
```
