---
title: "获取 agent 任务"
description: "获取 agent 任务的当前状态和详细信息，包括进度和拉取请求链接。"
openapi: /zh/admin-openapi.json GET /v2/agent/{projectId}/job/{id}
keywords: [ "agent 任务", "状态", "获取", "详情", "轮询" ]
---

轮询此端点以跟踪 agent 任务的进度。`status` 字段的状态转换为 `active` → `completed` 或 `failed`。

一旦 agent 创建拉取请求（PR；亦称"合并请求"/Merge Request），`prLink` 字段即会填充。
