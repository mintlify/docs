<Info>
  静态导出需要 [Enterprise 套餐](https://mintlify.com/pricing?ref=static-export)。
</Info>

`domain` 字段是提供你的文档服务的主机名——即你在 Mintlify 控制台中为项目配置的自定义域名（例如 `docs.example.com`）；如果你尚未设置自定义域名，则使用你的 Mintlify 子域名（例如 `acme.mintlify.app`）。仅传入主机名，不要包含协议或任何路径前缀。
