要在自定义域名上托管你的文档，你需要先在 Mintlify 设置中填写目标自定义域名，并在你的域名服务商处配置相应的 DNS 记录。

<Info>
  想要设置像 `mintlify.com/docs` 这样的自定义子目录？请参阅[自定义子目录](/zh/advanced/subpath/cloudflare)文档。
</Info>

## 控制台设置

1. 前往你的[控制台](https://dashboard.mintlify.com)
2. 选择**Settings**
3. 选择**Domain Setup**
4. 输入你要使用的自定义域名，例如 `docs.mintlify.com`

## 配置 DNS

1. 前往你的域名注册商网站，打开该域名的 DNS 设置。
2. 新增一条 DNS 记录，填写如下值：

```bash
CNAME | docs | cname.vercel-dns.com.
```

## 按提供商配置

<AccordionGroup>
  <Accordion title="Vercel 验证">
    如果 Vercel 是你的域名提供商，你需要添加一个用于验证的 `TXT` 记录。提交自定义域名后，这些信息会显示在你的控制面板中，并通过电子邮件发送给你。
  </Accordion>

  <Accordion title="Cloudflare 安全设置">
    如果 Cloudflare 是你的 DNS 提供商，你需要在 https 设置中启用“完全严格（full strict）”安全选项。
  </Accordion>
</AccordionGroup>
