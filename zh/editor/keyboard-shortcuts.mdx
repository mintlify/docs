在 Web 编辑器中使用键盘快捷键可以更快速地进行导航和编辑。编辑器支持常见的快捷键，如复制、粘贴、撤销和全选，以及下方列出的其他快捷键。

<div id="visual-mode">
  ## 可视化模式
</div>

在可视化模式下，你可以使用以下快捷键进行编辑。

<div id="search-and-navigation">
  ### 搜索和导航
</div>

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **搜索文件和内容** | <kbd>Cmd</kbd> + <kbd>K</kbd> | <kbd>Ctrl</kbd> + <kbd>K</kbd> |
| **打开助手** | <kbd>Cmd</kbd> + <kbd>I</kbd> | <kbd>Ctrl</kbd> + <kbd>I</kbd> |

<div id="text-formatting">
  ### 文本格式化
</div>

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **加粗** | <kbd>Cmd</kbd> + <kbd>B</kbd> | <kbd>Ctrl</kbd> + <kbd>B</kbd> |
| **下划线** | <kbd>Cmd</kbd> + <kbd>U</kbd> | <kbd>Ctrl</kbd> + <kbd>U</kbd> |
| **删除线** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> |
| **代码** | <kbd>Cmd</kbd> + <kbd>E</kbd> | <kbd>Ctrl</kbd> + <kbd>E</kbd> |
| **下标** | <kbd>Cmd</kbd> + <kbd>,</kbd> | <kbd>Ctrl</kbd> + <kbd>,</kbd> |
| **上标** | <kbd>Cmd</kbd> + <kbd>.</kbd> | <kbd>Ctrl</kbd> + <kbd>.</kbd> |

<div id="headings">
  ### 标题
</div>

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **正文** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>0</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>0</kbd> |
| **一级标题** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>1</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>1</kbd> |
| **二级标题** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>2</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>2</kbd> |
| **三级标题** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>3</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>3</kbd> |
| **四级标题** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>4</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>4</kbd> |

<div id="lists-and-blocks">
  ### 列表和区块
</div>

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **无序列表** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>8</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>8</kbd> |
| **引用块** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> |

<div id="content-actions">
  ### 内容操作
</div>

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **添加链接**（选中文本时） | <kbd>Cmd</kbd> + <kbd>K</kbd> | <kbd>Ctrl</kbd> + <kbd>K</kbd> |
| **插入换行** | <kbd>Cmd</kbd> + <kbd>Enter</kbd> | <kbd>Ctrl</kbd> + <kbd>Enter</kbd> |
| **组件菜单** | <kbd>/</kbd> | <kbd>/</kbd> |

<div id="markdown-mode">
  ## Markdown 模式
</div>

Markdown 模式使用 Monaco 编辑器，并支持 VS Code 标准键盘快捷键。

### 搜索与导航

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **搜索文件** | <kbd>Cmd</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>P</kbd> |
| **查找** | <kbd>Cmd</kbd> + <kbd>F</kbd> | <kbd>Ctrl</kbd> + <kbd>F</kbd> |

<div id="editing">
  ### 编辑
</div>

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **切换注释** | <kbd>Cmd</kbd> + <kbd>/</kbd> | <kbd>Ctrl</kbd> + <kbd>/</kbd> |
| **缩进当前行** | <kbd>Tab</kbd> | <kbd>Tab</kbd> |
| **减少当前行缩进** | <kbd>Shift</kbd> + <kbd>Tab</kbd> | <kbd>Shift</kbd> + <kbd>Tab</kbd> |
| **上移当前行** | <kbd>Option</kbd> + <kbd>↑</kbd> | <kbd>Alt</kbd> + <kbd>↑</kbd> |
| **下移当前行** | <kbd>Option</kbd> + <kbd>↓</kbd> | <kbd>Alt</kbd> + <kbd>↓</kbd> |
| **复制当前行** | <kbd>Shift</kbd> + <kbd>Option</kbd> + <kbd>↓</kbd> | <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>↓</kbd> |

<div id="multiple-cursors">
  ### 多光标
</div>

| **在上方添加光标** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>↑</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>↑</kbd> |
| **在下方添加光标** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>↓</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>↓</kbd> |
