---
title: "键盘快捷键"
description: "Web 编辑器键盘快捷键参考。"
keywords: ["编辑器", "快捷键"]
---

在 Web 编辑器中使用键盘快捷键可以更快速地进行导航和编辑。编辑器支持常见的快捷键，如复制、粘贴、撤销和全选，以及下方列出的其他快捷键。

<div id="file-navigation">
  ## 文件导航
</div>

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **搜索文件** | <kbd>Cmd</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>P</kbd> |

<div id="text-formatting">
  ## 文本格式化
</div>

在可视化模式下，你可以使用以下快捷键对文本进行格式化：

| 命令 | macOS | Windows |
| :--- | :--- | :--- |
| **加粗** | <kbd>Cmd</kbd> + <kbd>B</kbd> | <kbd>Ctrl</kbd> + <kbd>B</kbd> |
| **下划线** | <kbd>Cmd</kbd> + <kbd>U</kbd> | <kbd>Ctrl</kbd> + <kbd>U</kbd> |
| **删除线** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> |
| **代码** | <kbd>Cmd</kbd> + <kbd>E</kbd> | <kbd>Ctrl</kbd> + <kbd>E</kbd> |
| **下标** | <kbd>Cmd</kbd> + <kbd>,</kbd> | <kbd>Ctrl</kbd> + <kbd>,</kbd> |
| **上标** | <kbd>Cmd</kbd> + <kbd>.</kbd> | <kbd>Ctrl</kbd> + <kbd>.</kbd> |

| **二级标题** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>2</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>2</kbd> |
| **三级标题** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>3</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>3</kbd> |
| **四级标题** | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>4</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>4</kbd> |
