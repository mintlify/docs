<div id="enabling-authentication">
  ## 启用认证
</div>

你可以在 `docs.json` 中添加认证方法，以在所有页面全局启用；也可以为单个页面分别设置。

当同时存在全局方法和页面方法时，页面的认证方法将覆盖全局方法。

<div id="bearer-token">
  ### Bearer 令牌
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "bearer"
      }
    }
}
```

```mdx Page Metadata
---
title: "Your page title"
authMethod: "bearer"
---
```

</CodeGroup>

<div id="basic-authentication">
  ### 基本认证
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "basic"
      }
    }
}
```

```mdx Page Metadata
---
title: "Your page title"
authMethod: "basic"
---
```

</CodeGroup>

<div id="api-key">
  ### API key
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "key",
        "name": "x-api-key"
      }
    }
}
```

```mdx Page Metadata
---
title: "Your page title"
authMethod: "key"
---
```

</CodeGroup>

<div id="none">
  ### 无
</div>

在 docs.json 中设置默认值后，可使用 "none" 认证方法在特定端点上禁用认证。

<CodeGroup>
```mdx Page Metadata
---
title: "Your page title"
authMethod: "none"
---
```
</CodeGroup>
