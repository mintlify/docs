---
title: "多个响应"
description: "为同一端点展示不同的响应变体"
---

如果你的 API 会根据输入参数、用户 context 或请求的其他条件返回不同的响应，你可以使用 `examples` 属性为同一端点记录多个响应示例。

该属性可添加到任何响应中，并具有以下架构。

```yaml
responses:
  "200":
    description: 成功响应
    content:
      application/json:
        schema:
          $ref: "#/components/schemas/YourResponseSchema"
        examples:
          us:
            summary: 美国的响应
            value:
              countryCode: "US"
              currencyCode: "USD"
              taxRate: 0.0825
          gb:
            summary: 英国的响应
            value:
              countryCode: "GB"
              currencyCode: "GBP"
              taxRate: 0.20
```
