使用 `docs.json` 中的 `integrations` 字段将第三方分析平台、聊天工具和其他服务连接到你的文档站点。

## 设置

在 `docs.json` 文件中将所有集成配置为 `integrations` 对象的属性：

```json docs.json
"integrations": {
  "ga4": {
    "measurementId": "G-XXXXXXXXXX"
  }
}
```

  </Expandable>
</ResponseField>

## 示例

```json docs.json
"integrations": {
  "ga4": {
    "measurementId": "G-XXXXXXXXXX"
  },
  "posthog": {
    "apiKey": "phc_XXXXXXXXXX"
  },
  "intercom": {
    "appId": "abc12345"
  },
  "telemetry": {
    "enabled": true
  }
}
```
