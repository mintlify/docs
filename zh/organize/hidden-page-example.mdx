此页面为隐藏页面！它未包含在 `docs.json` 的导航中，因此你只能通过其 URL 访问。

请参阅[隐藏页面](/zh/organize/hidden-pages)以了解更多信息。

<div id="common-use-cases-for-hidden-pages">
  ## 隐藏页面的常见用例
</div>

<Note>
  默认情况下，隐藏页面可公开访问，但不会通过导航被发现。若需限制对隐藏页面的访问，必须配置[认证](/zh/deploy/authentication-setup)。
</Note>

* **Beta 文档**：可以公开，但不应通过导航被发现的信息。
* **AI 工具的 context**：如果隐藏页面被索引，AI 工具可以引用其中内容作为 context。将对用户不重要但能帮助 AI 工具提供更准确回答的 context 放在隐藏页面中。
* **旧版页面**：通过直接链接保留对旧内容的访问，同时将其从导航菜单中移除。
* **内部工具**：为内部工具、预发布环境的 API 或开发流程编写文档。
