要在自定义 domain 上托管文档，你需要先在 Mintlify 设置中填写所需的自定义 domain，并在你的域名服务商处配置 DNS（域名系统）记录。

<Info>
  想要设置类似 `mintlify.com/docs` 的自定义子目录？请参阅[自定义子目录](/zh/deploy/cloudflare)文档。
</Info>

<div id="dashboard-settings">
  ## 控制台设置
</div>

1. 前往你的[控制台](https://dashboard.mintlify.com)
2. 选择**设置**
3. 选择**域名设置**
4. 输入你希望使用的自定义域名。例如：`docs.mintlify.com`

<div id="configuring-your-dns">
  ## 配置你的 DNS（域名系统）
</div>

1. 前往你的域名注册商网站，打开该 domain 的 DNS 设置页面。
2. 新增一条 DNS 记录，并填写以下值：

```bash
CNAME | docs | cname.vercel-dns.com.
```

<div id="provider-specific-settings">
  ## 针对不同提供商的设置
</div>

<AccordionGroup>
  <Accordion title="Vercel verification">
    如果 Vercel 是你的 domain 提供商，你需要添加一条用于验证的 `TXT` 记录。提交自定义域名后，这些信息会显示在你的控制台中，并同时通过电子邮件发送给你。
  </Accordion>

  <Accordion title="Cloudflare security settings">
    如果 Cloudflare 是你的 DNS（域名系统）提供商，你需要在 HTTPS 设置中启用“full strict”安全选项。
  </Accordion>
</AccordionGroup>
