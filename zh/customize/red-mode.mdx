---
title: "红色模式"
description: "启用红色调显示模式，在低光环境下减少眼睛疲劳。"
---

红色模式为您的文档应用红色滤镜，减少蓝光发射。这适用于：

- 在黑暗环境中阅读文档而不影响夜视能力
- 在深夜编程时减少眼睛疲劳
- 偏好暖色调的用户

## 启用红色模式

在您的 `docs.json` 配置中添加 `redMode` 选项：

```json docs.json
{
  "appearance": {
    "redMode": true
  }
}
```

## 配置选项

<ResponseField name="redMode" type="boolean | object">
  为您的文档启用红色模式。设置为 `true` 使用默认设置，或传递对象进行高级配置。

  <Expandable title="高级选项">
    <ResponseField name="intensity" type="number" default="0.5">
      红色滤镜的强度，从 `0`（微弱）到 `1`（全红）。
    </ResponseField>
    <ResponseField name="toggle" type="boolean" default="true">
      在导航栏中显示切换按钮，让用户可以开启/关闭红色模式。
    </ResponseField>
  </Expandable>
</ResponseField>

### 高级选项示例

```json docs.json
{
  "appearance": {
    "redMode": {
      "intensity": 0.7,
      "toggle": true
    }
  }
}
```

## 用户切换

当 `toggle` 启用时，用户可以使用顶部导航栏中的月亮图标来开启或关闭红色模式。他们的偏好设置会保存在本地存储中。
