## 插图

在文档中添加图像，以提供视觉上下文、示例或装饰。

### 基本图像语法

使用[Markdown 语法](https://www.markdownguide.org/basic-syntax/#images)向文档添加图像：

```mdx
![描述图像的替代文本](/zh/path/to/image.png)
```

<Tip>
  请务必包含具有描述性的替代文本（alt text），以提升无障碍性和生成引擎优化（GEO）。替代文本应清晰说明图像的内容。
</Tip>

单个图像文件必须小于 20MB。更大的文件请托管到 [Amazon S3](https://aws.amazon.com/s3) 或 [Cloudinary](https://cloudinary.com) 等 CDN 服务。

### 使用 HTML 嵌入图像

若需更细致地控制图像展示，可使用 HTML `<img>` 标签：

```html
<img 
  src="/images/dashboard.png" 
  alt="主仪表盘界面"
  height="300"
  className="rounded-lg"
/>
```

#### 禁用缩放功能

要禁用图像点击后的默认缩放，请添加 `noZoom` 属性：

```html highlight="4"
<img 
  src="/images/screenshot.png" 
  alt="描述性的替代文本"
  noZoom 
  height="200"
/>
```

#### 为图像添加链接

要将图像设为可点击链接，请将其包裹在锚点标签中，并添加 `noZoom` 属性：

```html
<a href="https://mintlify.com" target="_blank">
  <img 
    src="/images/logo.png" 
    alt="Mintlify 标志"
    noZoom 
    height="100"
  />
</a>
```

<Note>
  位于锚点标签内的图像会自动显示指针光标，提示其可点击。
</Note>

#### 适配浅色与深色模式的图像

要在浅色与深色主题下展示不同图像，请使用 Tailwind CSS 类：

```html
<!-- 浅色模式图像 -->
<img 
  className="block dark:hidden" 
  src="/images/light-mode.png" 
  alt="浅色模式界面"
/>

<!-- 深色模式图像 -->
<img 
  className="hidden dark:block" 
  src="/images/dark-mode.png" 
  alt="深色模式界面"
/>
```

## 视频

Mintlify 支持在 Markdown 中使用 [HTML 标签](https://www.markdownguide.org/basic-syntax/#html)，让你灵活创建丰富内容。

<Tip>
  请始终在 video 元素内添加回退文本内容，以便在不支持视频播放的浏览器中显示。
</Tip>

### 嵌入 YouTube

使用 iframe 元素嵌入 YouTube 视频：

```html
<iframe
  className="w-full aspect-video rounded-xl"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>
```

<Frame>
<iframe
  className="w-full aspect-video rounded-xl"

### 自托管视频

对于自托管的视频内容，请使用 HTML `<video>` 元素：

```html
<video
  controls
  className="w-full aspect-video rounded-xl"
  src="link-to-your-video.com"
></video>
```

### 自动播放视频

如需自动播放视频，请使用：

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video rounded-xl"
  src="/videos/demo.mp4"
></video>
```

<Note>
  使用 JSX 语法时，复合词属性应写为驼峰式：`autoPlay`、`playsInline`、`allowFullScreen`。
</Note>

## iframes

使用 iframe 元素嵌入外部内容：

```html
<iframe 
  src="https://example.com/embed" 
  title="Embedded content"
  className="w-full h-96 rounded-xl"
></iframe>
```
