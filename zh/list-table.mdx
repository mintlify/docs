## 列表

列表遵循官方的 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#lists-1)。

### 有序列表

要创建有序列表，在每个列表项前添加数字并跟一个句点。

1. 第一项
2. 第二项
3. 第三项
4. 第四项

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

### 无序列表

要创建无序列表，在每个列表项前添加短横线（`-`）、星号（`*`）或加号（`+`）。

- 第一项
- 第二项
- 第三项
- 第四项

```mdx
- First item
- Second item
- Third item
- Fourth item
```

### 嵌套列表

通过缩进列表项来实现嵌套。

- 第一项
- 第二项
  - 其他项
  - 其他项
- 第三项

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

## 表格

表格遵循官方的 [Markdown 语法](https://www.markdownguide.org/extended-syntax/#tables)。

要添加表格，可使用三个或以上的连字符（`---`）来定义每一列的表头，并使用竖线（`|`）分隔各列。为确保兼容性，建议在每行的开头和结尾都加上一个竖线。

| 属性     | 描述                 |
| -------- | -------------------- |
| Name     | 用户全名             |
| Age      | 填报的年龄           |
| Joined   | 是否已加入社区       |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
