<Info>
  RBAC 功能适用于 [Enterprise 方案](https://mintlify.com/pricing?ref=rbac)。
</Info>

Mintlify 提供两种仪表盘访问级别：Editor 和 Admin。

以下为仅限 Admin 角色的操作：

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| 更新用户角色            |   ❌   |  ✅   |
| 删除用户                |   ❌   |  ✅   |
| 邀请管理员用户          |   ❌   |  ✅   |
| 管理和更新账单          |   ❌   |  ✅   |
| 更新自定义域名          |   ❌   |  ✅   |
| 更新 Git 来源           |   ❌   |  ✅   |
| 删除组织                |   ❌   |  ✅   |

仪表盘上的其他操作两种角色均可使用。

你可以邀请任意数量的管理员，但我们建议仅将管理员访问权限授予确有需要的用户。
