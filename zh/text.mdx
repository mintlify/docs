## 标题

标题用于组织内容并创建导航锚点。它们会显示在目录中，帮助用户快速扫描你的文档。

### 创建标题

使用 `#` 符号来创建不同层级的标题：

```mdx
## Main section header
### Subsection header
#### Sub-subsection header
```

<Tip>
  使用描述清晰、富含关键词的标题，明确指示下文内容。这将同时提升用户的导航体验和搜索引擎优化效果。
</Tip>

## 文本格式

我们支持大多数用于强调和美化文本的 Markdown 格式。

### 基础格式

将以下格式样式应用到你的文本：

| 样式 | 语法 | 示例 | 结果 |
|-------|--------|---------|--------|
| **加粗** | `**text**` | `**important note**` | **重要提示** |
| *斜体* | `_text_` | `_emphasis_` | *强调* |
| ~~删除线~~ | `~text~` | `~deprecated feature~` | ~~已弃用功能~~ |

### 组合格式

你可以组合使用这些样式：

```mdx
**_bold and italic_**
**~~bold and strikethrough~~**
*~~italic and strikethrough~~**
```

**_加粗并斜体_**<br />
**~~加粗并删除线~~**<br />
*~~斜体并删除线~~*

### 上标和下标

对于数学表达式或脚注，请使用 HTML 标签：

| 类型 | 语法 | 示例 | 结果 |
|------|--------|---------|--------|
| 上标 | `<sup>text</sup>` | `example<sup>2</sup>` | example<sup>2</sup> |
| 下标 | `<sub>text</sub>` | `example<sub>n</sub>` | example<sub>n</sub> |

## 链接

链接帮助用户在页面之间导航并访问外部资源。使用更具描述性的链接文本可提升可访问性和用户体验。

### 内部链接

使用根相对路径链接到文档中的其他页面：

```mdx
[Quickstart](/zh/quickstart)
[Steps](/zh/components/steps)
```

[Quickstart](/zh/quickstart)<br />
[Steps](/zh/components/steps)

<Note>
  避免使用类似 `[page](../page)` 的相对路径链接，因为它们加载更慢，且无法像根相对链接那样高效优化。
</Note>

### 外部链接

对于外部资源，请使用完整的 URL：

```mdx
[Markdown Guide](https://www.markdownguide.org/)
```

[Markdown Guide](https://www.markdownguide.org/)

### 失效链接

你可以使用 [CLI](/zh/installation) 检查文档中的失效链接：

```bash
mint broken-links
```

## 引用块

引用块用于在内容中突出重要信息、引文或示例。

### 单行引用块

在文本前添加 `>` 可创建引用块：

```mdx
> This is a quote that stands out from the main content.
```

> 这是一段从主体内容中凸显出来的引文。

### 多行引用块

适用于较长的引文或多段落内容：

```mdx
> This is the first paragraph of a multi-line blockquote.
>
> This is the second paragraph, separated by an empty line with `>`.
```

> 这是多行引用块的第一段。
>
> 这是第二段，中间用带有 `>` 的空行分隔。

<Tip>
  请谨慎使用引用块，以保持其视觉效果与表达力度。对于备注、警告及其他信息，请考虑使用[标注](/zh/components/callouts)。
</Tip>

## 数学表达式

我们支持使用 LaTeX 渲染数学表达式和方程式。

### 行内数学

使用单个美元符号 `$` 编写行内数学表达式：

```mdx
The Pythagorean theorem states that $(a^2 + b^2 = c^2)$ in a right triangle.
```

勾股定理表明，在直角三角形中 $(a^2 + b^2 = c^2)$。

### 独立方程

使用双美元符号 `$$` 编写独立方程：

```mdx
$$
E = mc^2
$$
```

$$
E = mc^2
$$

<Info>
  使用 LaTeX 需要遵循正确的数学语法。完整的语法指南请参阅 [LaTeX 文档](https://www.latex-project.org/help/documentation/)。
</Info>

## 换行与间距

通过控制间距和换行来提升内容可读性。

### 段落分隔

使用空行分隔段落：

```mdx
This is the first paragraph.

This is the second paragraph, separated by a blank line.
```

这是第一段。

这是第二段，用一个空行分隔。

### 手动换行

在段落内使用 HTML `<br />` 标签实现强制换行：

```mdx
This line ends here.<br />
This line starts on a new line.
```

这一行到此结束。<br />
下一行从新的一行开始。

<Tip>
  在大多数情况下，用空行分隔段落比手动换行的可读性更好。
</Tip>

## 最佳实践

### 内容组织
- 使用标题构建清晰的内容层级
- 遵循正确的标题层级（不要从 H2 跳到 H4）
- 撰写具有描述性且包含关键词的标题文本

### 文本格式
- 使用加粗来强调，而非整段加粗
- 将斜体保留用于术语、作品名或轻度强调
- 避免过度格式化，以免分散对内容的注意力

### 链接
- 使用具有描述性的链接文本，而不是“点击这里”或“阅读更多”
- 站内链接使用根相对路径
- 定期测试链接以避免出现失效链接
