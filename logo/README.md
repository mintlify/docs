# Logo Files

Add your logo files here:

- `light.svg` - Logo for light mode
- `dark.svg` - Logo for dark mode

You can download the official Spotify logo from:
https://developer-assets.spotifycdn.com/images/logo.svg

## Logo Guidelines

Follow Spotify's brand guidelines when using their logo:
https://developer.spotify.com/documentation/design

## Placeholder

For now, you can use text-based placeholders or the Spotify logo URL directly.
