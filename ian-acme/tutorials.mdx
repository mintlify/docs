# Tutorials

Master Ian ACME with these hands-on tutorials.

## Getting started tutorials

### Create and manage projects

Learn how to set up projects effectively:

<Steps>
  <Step title="Create a project">
    Navigate to your workspace dashboard and click **New Project**. Give it a clear name and description that explains its purpose.
  </Step>
  
  <Step title="Add team members">
    Click **Team** in the project sidebar, then **Invite Members**. Assign appropriate roles for each member.
  </Step>
  
  <Step title="Set up project structure">
    Create milestones to mark major phases. Add tasks under each milestone to break down the work.
  </Step>
  
  <Step title="Configure project settings">
    Set project visibility, permissions, and notification preferences to match your team's needs.
  </Step>
</Steps>

### Organize tasks efficiently

Master task management:

1. **Create task templates** - Save time by reusing common task structures
2. **Use labels and tags** - Categorize tasks for easy filtering
3. **Set dependencies** - Link related tasks to track critical paths
4. **Add checklists** - Break complex tasks into smaller steps

### Collaborate with your team

Maximize team productivity:

- **Comments** - Discuss tasks directly where work happens
- **Mentions** - Tag teammates with @username to get their attention
- **File sharing** - Attach documents, images, and files to tasks
- **Activity feed** - Stay updated on project changes

## Advanced tutorials

### Custom workflows

Create workflows that match your processes:

1. Define your workflow stages (e.g., To Do, In Progress, Review, Done)
2. Set up automation rules to move tasks between stages
3. Configure notifications for stage transitions
4. Create custom fields to capture workflow-specific data

### API integration

Build custom integrations with the Ian ACME API:

```javascript
// Example: Create a new task via API
const response = await fetch('https://api.ian-acme.com/v1/tasks', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: 'New Task',
    project_id: 'proj_123',
    assignee_id: 'user_456',
    due_date: '2026-03-01'
  })
});
```
