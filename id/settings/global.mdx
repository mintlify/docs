Setiap situs dokumentasi memerlukan`docs.json` file yang berisi pengaturan konfigurasi inti. File ini mengontrol segala hal mulai dari gaya dan navigasi hingga integrasi dan analitik.

## Properti

### Kustomisasi

## Praktik Terbaik

Saat mengkonfigurasi file`docs.json`, pertimbangkan praktik terbaik berikut:

1. Jaga konfigurasi tetap terorganisir dengan mengelompokkan pengaturan terkait bersama-sama
2. Gunakan nama yang bermakna untuk grup dan halaman dalam struktur navigasi Anda
3. Sediakan path lengkap untuk semua aset (logo, favicon, dll.)
4. Uji konfigurasi Anda dalam mode terang dan gelap
5. Verifikasi semua tautan eksternal dan integrasi telah dikonfigurasi dengan benar
6. Gunakan kontras warna yang sesuai untuk aksesibilitas
7. Konfigurasikan pengaturan SEO untuk visibilitas mesin pencari yang lebih baik

## Validasi

File`docs.json`divalidasi terhadap skema JSON untuk memastikan konfigurasi yang tepat. Anda dapat mereferensikan skema dengan menyertakan:

```json
{
  "$schema": "https://mintlify.com/docs.json"
}
```
