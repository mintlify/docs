<div id="navigate-files">
  ## Parcourir les fichiers
</div>

Parcourez vos fichiers de documentation dans l’onglet **Navigation** du panneau latéral gauche.

* Cliquez sur les éléments de navigation pour les développer ou les réduire.
* Cliquez sur les fichiers pour les ouvrir dans l’éditeur.
* Cliquez sur l’icône de recherche ou appuyez sur <kbd>Cmd</kbd> + <kbd>P</kbd> (macOS) ou <kbd>Ctrl</kbd> + <kbd>P</kbd> (Windows) pour rechercher des fichiers par nom.

<div id="create-new-pages">
  ## Créer de nouvelles pages
</div>

1. Cliquez sur le bouton **+** dans l’élément de navigation où vous souhaitez ajouter une page.
2. Cliquez sur **Ajouter une page**.
3. Saisissez un nom de fichier. L’éditeur ajoute automatiquement l’extension `.mdx`.

<div id="edit-content">
  ## Modifier le contenu
</div>

Basculez entre le mode visuel et le Markdown mode à l’aide du bouton de bascule dans la barre d’outils. L’éditeur web enregistre vos modifications lors du changement de mode.

<div id="visual-mode">
  ### Mode visuel
</div>

Modifiez le contenu avec des aperçus en temps réel qui montrent à quoi il ressemblera une fois publié.

* **Ajouter du texte** : Saisissez du texte dans l’éditeur pour voir comment il apparaîtra une fois publié.
* **Mettre le texte en forme** : Utilisez la barre d’outils pour mettre le texte en gras, en italique ou appliquer d’autres styles de mise en forme.
* **Ajouter des composants** : Appuyez sur <kbd>/</kbd> pour ouvrir le menu des composants et sélectionner des composants.
* **Ajouter des images** : Utilisez le composant d’image depuis le menu <kbd>/</kbd>.
* **Insérer des liens** : Sélectionnez du texte et appuyez sur <kbd>Cmd</kbd> + <kbd>K</kbd>.

Consultez la documentation [Composants](/fr/components) pour la liste complète des composants disponibles.

<div id="markdown-mode">
  ### Mode Markdown
</div>

Modifiez le code source MDX.

* **Édition directe en MDX** : Écrivez en syntaxe MDX et Markdown pour un contrôle précis du contenu.
* **Coloration syntaxique** : Le code est surligné pour une meilleure lisibilité.
* **Propriétés des composants** : Définissez les propriétés et configurations des composants.
* **Frontmatter** : Modifiez les métadonnées de la page en haut du fichier.

Consultez [Mettre en forme le texte](/fr/create/text) et [Mettre en forme le code](/fr/create/code) pour plus d&#39;informations sur la syntaxe MDX.

<div id="configure-pages">
  ## Configurer les pages
</div>

Configurez les paramètres des pages pour contrôler la façon dont elles apparaissent dans la navigation, les résultats de recherche et la mise en page de votre site. Effectuez un clic droit sur un fichier et sélectionnez **Configure** pour ouvrir la feuille de configuration de la page.

<div id="customize-navigation-appearance">
  ### Personnaliser l’apparence de la navigation
</div>

Contrôlez l’apparence de la page dans la barre latérale de navigation de votre site.

* **Titre** : Définissez le titre principal. Apparaît dans la navigation, les onglets du navigateur et les résultats de recherche.
* **Titre de la barre latérale** : Affichez un texte plus court dans la navigation lorsque le titre complet est trop long pour la barre latérale.
* **Icône** : Ajoutez un repère visuel à côté de la page pour aider les utilisateurs à l’identifier rapidement.
* **URL externe** : Créez un lien vers un site externe au lieu d’une page. Utilisez cette option pour ajouter des ressources externes à votre navigation.

<div id="optimize-for-search-and-sharing">
  ### Optimiser pour la recherche et le partage
</div>

Aidez les utilisateurs à trouver votre page et améliorez son apparence lorsqu’elle est partagée.

* **Description**: Rédigez un court résumé. Apparaît dans les résultats de recherche et les balises méta SEO.
* **Keywords**: Ajoutez des termes de recherche pertinents pour aider les utilisateurs à découvrir cette page.
* **OG Image URL**: Définissez une image d’aperçu personnalisée pour les partages sur les réseaux sociaux et les aperçus de liens.

<div id="control-page-layout">
  ### Contrôler la mise en page
</div>

Choisissez comment la page s’affiche en fonction de vos besoins de contenu.

* **Mise en page standard** (`default`) : Page par défaut avec barre latérale de navigation et table des matières.
* **Mise en page pleine largeur** (`wide`) : Masque la table des matières pour permettre des mises en page plus larges pour les tableaux, diagrammes ou d’autres contenus.
* **Mise en page centrée** (`center`) : Masque la barre latérale et la table des matières pour une meilleure lisibilité des pages très textuelles, comme les journaux des modifications.
* **Largeur personnalisée** (`custom`) : Mise en page minimaliste avec uniquement la barre de navigation supérieure pour les pages de destination ou d’autres mises en page spécifiques.

<div id="manage-files">
  ## Gérer les fichiers
</div>

Pour déplacer des fichiers dans la navigation, faites-les glisser et déposez-les à l’emplacement souhaité.

Pour dupliquer ou supprimer des fichiers, cliquez avec le bouton droit sur un fichier et sélectionnez l’action souhaitée.
