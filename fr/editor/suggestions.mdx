Les suggestions vous permettent de proposer des modifications au texte sans modifier le contenu existant. Vos coéquipiers peuvent examiner, accepter ou rejeter vos suggestions.

Si votre branche a une pull request ouverte sur GitHub, Mintlify ajoute un résumé des suggestions ouvertes à la description de la pull request. Le résumé liste les fichiers avec des suggestions, un aperçu de chaque modification suggérée et un lien pour ouvrir ce fichier dans l'éditeur.

Le résumé se met à jour automatiquement lorsque vous ajoutez, acceptez ou rejetez une suggestion.

## Activer le mode suggestion

1. Surlignez n'importe quel texte sur la page.
2. Cliquez sur le bouton **Switch to Suggesting**, l'icône du stylo, dans la barre d'outils qui apparaît.

Lorsque le mode suggestion est actif, l'indicateur **Suggesting** apparaît dans la barre d'outils de l'éditeur.

Pour quitter le mode suggestion, cliquez sur l'indicateur **Suggesting**.

## Faire une suggestion

Avec le mode suggestion activé, modifiez n'importe quel texte. Toutes les suppressions ou insertions apparaissent avec une mise en surbrillance distincte afin que les réviseurs puissent voir quelles modifications vous proposez.

## Accepter une suggestion

1. Cliquez sur le texte de suggestion surligné.
2. Cliquez sur le bouton accepter, l'icône de coche, pour appliquer la modification suggérée.

Accepter une suggestion enregistre le texte mis à jour et résout le fil de suggestion.

## Rejeter une suggestion

1. Cliquez sur le texte de suggestion surligné.
2. Cliquez sur le bouton rejeter, l'icône de corbeille, pour ignorer la modification proposée.

Rejeter une suggestion préserve le texte original et résout le fil de suggestion.
