Utilisez les raccourcis clavier pour naviguer et modifier plus rapidement dans l’éditeur web. L’éditeur prend en charge les raccourcis courants comme copier, coller, annuler et tout sélectionner, ainsi que les raccourcis répertoriés ci-dessous.

<div id="visual-mode">
  ## Mode visuel
</div>

Utilisez ces raccourcis lorsque vous modifiez en mode visuel.

<div id="search-and-navigation">
  ### Recherche et navigation
</div>

| Commande | macOS | Windows |
| :--- | :--- | :--- |
| **Rechercher des fichiers et du contenu** | <kbd>Cmd</kbd> + <kbd>K</kbd> | <kbd>Ctrl</kbd> + <kbd>K</kbd> |
| **Ouvrir l'agent** | <kbd>Cmd</kbd> + <kbd>I</kbd> | <kbd>Ctrl</kbd> + <kbd>I</kbd> |

<div id="text-formatting">
  ### Mise en forme du texte
</div>

| Commande | macOS | Windows |
| :--- | :--- | :--- |
| **Gras** | <kbd>Cmd</kbd> + <kbd>B</kbd> | <kbd>Ctrl</kbd> + <kbd>B</kbd> |
| **Souligné** | <kbd>Cmd</kbd> + <kbd>U</kbd> | <kbd>Ctrl</kbd> + <kbd>U</kbd> |
| **Barré** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> |
| **Code** | <kbd>Cmd</kbd> + <kbd>E</kbd> | <kbd>Ctrl</kbd> + <kbd>E</kbd> |
| **Indice** | <kbd>Cmd</kbd> + <kbd>,</kbd> | <kbd>Ctrl</kbd> + <kbd>,</kbd> |
| **Exposant** | <kbd>Cmd</kbd> + <kbd>.</kbd> | <kbd>Ctrl</kbd> + <kbd>.</kbd> |

  ### Actions de contenu
</div>

| Commande | macOS | Windows |
| :--- | :--- | :--- |

| **Insérer un saut de ligne** | <kbd>Cmd</kbd> + <kbd>Enter</kbd> | <kbd>Ctrl</kbd> + <kbd>Enter</kbd> |

<div id="markdown-mode">
  ## Mode Markdown
</div>

Le mode Markdown utilise l’éditeur Monaco, qui prend en charge les raccourcis clavier standard de VS Code.

<div id="search-and-navigation">
  ### Recherche et navigation
</div>

| Commande | macOS | Windows |
| :--- | :--- | :--- |
| **Rechercher des fichiers** | <kbd>Cmd</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>P</kbd> |
| **Rechercher** | <kbd>Cmd</kbd> + <kbd>F</kbd> | <kbd>Ctrl</kbd> + <kbd>F</kbd> |

<div id="editing">
  ### Édition
</div>

| Commande | macOS | Windows |
| :--- | :--- | :--- |
| **Commenter/décommenter** | <kbd>Cmd</kbd> + <kbd>/</kbd> | <kbd>Ctrl</kbd> + <kbd>/</kbd> |

| **Déplacer la ligne vers le bas** | <kbd>Option</kbd> + <kbd>↓</kbd> | <kbd>Alt</kbd> + <kbd>↓</kbd> |
| **Dupliquer la ligne** | <kbd>Shift</kbd> + <kbd>Option</kbd> + <kbd>↓</kbd> | <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>↓</kbd> |
