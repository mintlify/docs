Utilisez les raccourcis clavier pour naviguer et modifier plus rapidement dans l'éditeur web. L'éditeur prend en charge les raccourcis courants comme copier, coller, annuler et tout sélectionner, ainsi que les raccourcis listés ci-dessous.

<div id="editor-wide">

## Sur l'ensemble de l'éditeur

</div>

Ces raccourcis fonctionnent quel que soit le mode utilisé.

| Commande | macOS | Windows |
| :--- | :--- | :--- |
| **Rechercher des fichiers et du contenu** | <kbd>Cmd</kbd> + <kbd>K</kbd> | <kbd>Ctrl</kbd> + <kbd>K</kbd> |
| **Activer/désactiver l'assistant IA** (en dehors de l'édition de texte) | <kbd>Cmd</kbd> + <kbd>I</kbd> | <kbd>Ctrl</kbd> + <kbd>I</kbd> |
| **Basculer entre le mode visuel et le mode source** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd> |
| **Activer/désactiver la vue diff** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd> |
| **Basculer entre l'arborescence Navigation et Fichiers** | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> |
| **Masquer/afficher la barre latérale** | <kbd>Cmd</kbd> + <kbd>\\</kbd> | <kbd>Ctrl</kbd> + <kbd>\\</kbd> |

<div id="smart-substitutions">

### Substitutions intelligentes

</div>

L'éditeur convertit automatiquement certaines séquences de caractères en symboles typographiques au fur et à mesure que vous tapez. Les substitutions sont ignorées à l'intérieur des blocs de code et du code en ligne.

| Saisie | Résultat |
| :--- | :--- |
| `->` | → |
| `<-` | ← |
| `=>` | ⇒ |
| `<=` | ≤ |
| `>=` | ≥ |
| `!=` | ≠ |
