<div id="navigating-your-files">
  ## Parcourir vos fichiers
</div>

Utilisez l’explorateur de fichiers de la barre latérale pour parcourir vos fichiers de documentation. Cliquez sur un fichier pour l’ouvrir dans l’éditeur.

Appuyez sur <kbd>Command</kbd> + <kbd>P</kbd> (<kbd>Ctrl</kbd> + <kbd>P</kbd> sous Windows) pour rechercher des fichiers par nom.

<div id="creating-new-pages">
  ## Créer de nouvelles pages
</div>

<Steps>
  <Step title="Créer un nouveau fichier">
    Sélectionnez l’icône **Créer un nouveau fichier** dans la barre latérale de l’explorateur de fichiers.

      src="/images/editor/files-menu-light.png"

    <img

  <Step title="Saisir un nom de fichier">
    Les noms de fichiers doivent se terminer par `.mdx`. Par exemple : `api-reference.mdx` ou `getting-started.mdx`.

  <Step title="Créer et ouvrir">
    Appuyez sur <kbd>Entrée</kbd> pour créer le fichier. Il s’ouvrira automatiquement dans l’éditeur, prêt à accueillir du contenu.
  </Step>
</Steps>

<div id="organizing-your-navigation">
  ## Organiser la navigation
</div>

Ajoutez de nouvelles pages à la navigation de votre site en modifiant le fichier `docs.json`.

<Steps>
  <Step title="Ouvrir docs.json">
    Recherchez et ouvrez le fichier `docs.json` dans le répertoire racine à l'aide de l'explorateur de fichiers.
  </Step>
  <Step title="Repérer la section de navigation">
    Trouvez l'emplacement dans le fichier `docs.json` où vous souhaitez ajouter votre nouvelle page.
  </Step>
  <Step title="Ajouter votre page">
    Insérez le nom de fichier de votre page (sans l’extension `.mdx`) dans le tableau `pages` du groupe approprié.
  </Step>
</Steps>

<div id="example-adding-a-themes-page">
  ### Exemple : ajout d’une page de thèmes
</div>

```json title="Ajout d’une page thèmes au groupe Profile" highlight="18"
{
    "navigation": {
        "groups": [
            {
                "group": "Démarrage",
                "pages": [
                    "index",
                    "quickstart", 
                    "installation"
                ]
            },
            {
                "group": "Profil",
                "pages": [
                    "paramètres",
                    "types de compte", 
                    "tableau de bord",
                    "thèmes"
                ]
            }
        ]
    }
}
```

<div id="editing-content">
  ## Modification du contenu
</div>

Apportez des modifications à vos pages en mode visuel ou en mode Markdown dans l’éditeur.

En mode visuel, appuyez sur <kbd>/</kbd> pour ouvrir le menu des composants. Ajoutez des blocs de contenu, des encadrés, des blocs de code et d’autres composants pour personnaliser votre documentation.

En mode Markdown, vous pouvez modifier directement le `MDX` de vos pages. Cela peut être utile lorsque vous devez :

- Définir des propriétés spécifiques de composants
- Travailler avec des composants imbriqués complexes
- Copier et coller du contenu `MDX` depuis d’autres sources
