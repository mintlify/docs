Les commentaires vous permettent, à vous et à votre équipe, de laisser des retours sur un texte spécifique dans une page sans modifier le contenu. Utilisez les commentaires pour poser des questions, signaler des problèmes ou discuter des modifications avant de les publier.

Tous les [rôles](/fr/dashboard/roles)—lecteurs, éditeurs et administrateurs—peuvent laisser et répondre aux commentaires. Seuls les éditeurs et les administrateurs peuvent résoudre les fils de commentaires.

Si votre branche a une pull request ouverte sur GitHub, Mintlify ajoute un résumé des fils de commentaires non résolus à la description de la pull request. Le résumé liste les fichiers avec des commentaires ouverts, le premier commentaire de chaque fil et un lien pour ouvrir ce fichier dans l'éditeur avec la barre latérale des commentaires ouverte.

Le résumé se met à jour automatiquement lorsque vous ajoutez, résolvez ou supprimez un commentaire.

## Ajouter un commentaire

1. Surlignez le texte que vous souhaitez commenter.
2. Cliquez sur **Comment** dans la barre d'outils qui apparaît.
3. Tapez votre commentaire et appuyez sur <kbd>Enter</kbd> pour le soumettre.

## Répondre à un commentaire

Ouvrez un fil de commentaires en cliquant sur le texte surligné ou en cliquant sur le fil dans la barre latérale des commentaires. Tapez votre réponse et appuyez sur <kbd>Enter</kbd> pour la soumettre.

## Mentionner un coéquipier

Tapez `@` dans un commentaire ou une réponse pour mentionner un coéquipier. Sélectionnez son nom dans la liste qui apparaît. Les personnes doivent être membres de votre organisation pour que vous puissiez les mentionner.

Les coéquipiers mentionnés reçoivent une notification par e-mail.

## Résoudre un commentaire

Les éditeurs et les administrateurs peuvent résoudre les fils lorsqu'une discussion est terminée.

Cliquez sur l'icône de coche pour résoudre un fil.

## Supprimer un commentaire

Cliquez sur l'icône de corbeille à côté d'un commentaire pour le supprimer. Supprimer le premier commentaire d'un fil supprime tout le fil.

Vous pouvez supprimer vos propres commentaires. Les administrateurs peuvent supprimer n'importe quel commentaire.
