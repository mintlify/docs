Le Markdown offre un texte structuré que les outils d’IA peuvent traiter plus efficacement que le HTML, ce qui se traduit par une meilleure exactitude des réponses, des temps de traitement plus rapides et une consommation de jetons réduite.

Mintlify génère automatiquement des versions Markdown des pages, optimisées pour les outils d’IA et les intégrations externes.


<div id="md-url-extension">
  ## Extension d’URL .md
</div>

Ajoutez `.md` à l’URL de n’importe quelle page pour afficher sa version Markdown.

<PreviewButton href="https://mintlify.com/docs/ai/markdown-export.md">Ouvrir cette page en Markdown</PreviewButton>



<div id="keyboard-shortcut">
  ## Raccourci clavier
</div>

Appuyez sur <kbd>Command</kbd> + <kbd>C</kbd> (<kbd>Ctrl</kbd> + <kbd>C</kbd> sous Windows) pour copier une page en Markdown dans votre presse-papiers.
