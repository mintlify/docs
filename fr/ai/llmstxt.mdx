Le [fichier llms.txt](https://llmstxt.org) est une norme du secteur qui aide les LLM à indexer le contenu plus efficacement, à l’instar d’un sitemap pour les moteurs de recherche. Les outils d’IA peuvent utiliser ce fichier pour comprendre la structure de votre documentation et trouver le contenu pertinent face aux requêtes des utilisateurs.

Mintlify héberge automatiquement un fichier `llms.txt` à la racine de votre projet qui répertorie toutes les pages disponibles de votre documentation. Ce fichier est toujours à jour et ne nécessite aucune maintenance. Vous pouvez également ajouter un fichier `llms.txt` personnalisé à la racine de votre projet.

Affichez votre `llms.txt` en ajoutant `/llms.txt` à l’URL de votre site de documentation.

## Structure de llms.txt

Un fichier `llms.txt` est un fichier Markdown classique qui contient :

- **Le titre du site** au format titre H1.
- **Des sections de contenu structurées** avec des liens et une description de chaque page de votre documentation.

```mdx Example llms.txt
# Example product docs

## Guides
- [Getting started](https://example.com/docs/start): Intro guide
- [Install](https://example.com/docs/install): Setup steps

## Reference
- [API](https://example.com/docs/api): Endpoint list and usage
```

Cette structure permet aux LLM de traiter efficacement votre documentation de manière globale et de repérer le contenu pertinent pour les requêtes des utilisateurs, améliorant ainsi la précision et la rapidité des recherches de documentation assistées par l’IA.

## llms-full.txt

Le fichier `llms-full.txt` regroupe l’ensemble de votre site de documentation en un seul fichier servant de contexte pour les outils d’IA, et il est indexé par le trafic des LLM.

Mintlify héberge automatiquement un fichier `llms-full.txt` à la racine de votre projet. Affichez votre `llms-full.txt` en ajoutant `/llms-full.txt` à l’URL de votre site de documentation.

<PreviewButton href="https://mintlify.com/docs/llms-full.txt">Ouvrir le llms-full.txt de ce site.</PreviewButton>

## Fichiers personnalisés

Pour ajouter un fichier personnalisé `llms.txt` ou `llms-full.txt`, créez un fichier `llms.txt` ou `llms-full.txt` à la racine de votre projet. L’ajout d’un fichier personnalisé remplacera le fichier généré automatiquement portant le même nom. Si vous supprimez un fichier personnalisé, le fichier par défaut sera de nouveau utilisé.

Votre fichier personnalisé `llms.txt` ou `llms-full.txt` doit inclure le titre du site en tant que titre H1. Le reste du contenu est facultatif. Consultez la section [Format](https://llmstxt.org/#format) de la spécification `llms.txt` pour plus d’informations sur les sections optionnelles et les bonnes pratiques.
