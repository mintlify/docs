Le [fichier llms.txt](https://llmstxt.org) est une norme du secteur qui aide les LLM à indexer le contenu plus efficacement, comme un sitemap pour les moteurs de recherche. Les outils d’IA peuvent utiliser ce fichier pour comprendre la structure de votre documentation et trouver le contenu pertinent pour les requêtes des utilisateurs.

Mintlify héberge automatiquement à la racine de votre projet un fichier `llms.txt` qui répertorie toutes les pages disponibles de votre documentation. Ce fichier est toujours à jour et ne nécessite aucune maintenance. Vous pouvez aussi ajouter, si vous le souhaitez, un fichier `llms.txt` personnalisé à la racine de votre projet.

Affichez votre `llms.txt` en ajoutant `/llms.txt` à l’URL de votre site de documentation.

<div id="llmstxt-structure">
  ## Structure de llms.txt
</div>

Un fichier `llms.txt` est un fichier Markdown en texte brut qui contient :

* **Titre du site** en tant que titre H1.
* **Sections de contenu structurées** avec des liens et une description de chaque page de votre documentation.

```mdx Example llms.txt
# Exemple de documentation produit

## Guides
- [Commencer](https://example.com/docs/start) : Guide d'introduction
- [Installation](https://example.com/docs/install) : Étapes de configuration

## Référence
- [API](https://example.com/docs/api) : Liste des points de terminaison et utilisation
```

Cette approche structurée permet aux LLM de traiter efficacement votre documentation à un niveau global et de repérer le contenu pertinent pour les requêtes des utilisateurs, améliorant ainsi la précision et la rapidité des recherches de documentation assistées par l’IA.


<div id="llms-fulltxt">
  ## llms-full.txt
</div>

Le fichier `llms-full.txt` regroupe l’ensemble de votre site de documentation en un seul fichier servant de contexte pour les outils d’IA et est indexé par le trafic des LLM.

Mintlify héberge automatiquement un fichier `llms-full.txt` à la racine de votre projet. Affichez votre `llms-full.txt` en ajoutant `/llms-full.txt` à l’URL de votre site de documentation.

<PreviewButton href="https://mintlify.com/docs/llms-full.txt">Ouvrir le llms-full.txt de ce site.</PreviewButton>



<div id="custom-files">
  ## Fichiers personnalisés
</div>

Pour ajouter un fichier `llms.txt` ou `llms-full.txt` personnalisé, créez un fichier `llms.txt` ou `llms-full.txt` à la racine de votre projet. L’ajout d’un fichier personnalisé remplacera le fichier généré automatiquement portant le même nom. Si vous supprimez un fichier personnalisé, le fichier par défaut sera à nouveau utilisé.

Votre fichier `llms.txt` ou `llms-full.txt` personnalisé doit comporter le titre du site en en-tête H1. Le reste du contenu est facultatif. Consultez le [Format](https://llmstxt.org/#format) dans la spécification `llms.txt` pour plus d’informations sur les sections optionnelles et les bonnes pratiques.
