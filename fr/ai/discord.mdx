<Info>
  Les intégrations Discord sont disponibles sur les [offres Pro et Custom](https://mintlify.com/pricing?ref=discord) avec accès à l'assistant.
</Info>

Le bot Discord aide votre communauté avec des réponses en temps réel basées sur votre documentation. Le bot utilise l'assistant Mintlify pour rechercher dans votre documentation et fournir des réponses précises et sourcées, afin qu'il soit toujours à jour.

Le bot ne fonctionne que dans les salons publics. Il répond aux mentions `@` ou à tous les messages dans un salon spécifique.

Chaque message envoyé par le bot Discord compte dans votre quota de messages de l'assistant.

<div id="add-the-discord-bot-to-your-server">
  ## Ajouter le bot Discord à votre serveur
</div>

<Note>
  Vous devez avoir la permission « Gérer le serveur » dans Discord pour ajouter le bot.
</Note>

1. Accédez à l’onglet [Integrations](https://dashboard.mintlify.com/products/assistant/settings/integrations) de la page **Assistant Configurations** dans votre Dashboard.
1. Dans la section Discord, cliquez sur **Connect**. Discord s’ouvre.
   <Frame>
     <img src="/images/assistant/discord-connect-light.png" alt="Le bouton Connect en mode clair." className="block dark:hidden" />
     <img src="/images/assistant/discord-connect-dark.png" alt="Le bouton Connect en mode sombre." className="hidden dark:block" />
   </Frame>
1. Dans Discord, sélectionnez le serveur sur lequel vous souhaitez ajouter le bot.
1. Autorisez le bot à accéder à votre serveur.
1. Mentionnez le bot pour l’ajouter à un canal. Le nom par défaut du bot est `@Mintlify Bot`.

<div id="create-an-ask-ai-channel">
  ## Créer un canal `#ask-ai`
</div>

Pour aider votre communauté à obtenir rapidement des réponses à ses questions, le bot peut répondre à chaque message publié dans un salon de votre choix. Par défaut, le bot répond à chaque message dans les salons nommés `#ask-ai`. Créez un salon `#ask-ai` et indiquez à votre communauté que le bot répondra aux messages publiés dans ce salon.

Si vous voulez que le bot réponde aux messages dans un autre salon, sélectionnez ce salon dans le [menu de configuration](https://dashboard.mintlify.com/products/assistant/settings/integrations) du bot Discord.

Consultez l’article [Starting Your First Discord Server](https://discord.com/blog/starting-your-first-discord-server) sur le blog Discord pour en savoir plus sur la création d’un canal.

<div id="manage-the-discord-bot">
  ## Gérer le bot Discord
</div>

Après avoir ajouté le bot Discord à votre serveur, vous pouvez gérer ou supprimer le bot depuis l’onglet [Integrations](https://dashboard.mintlify.com/products/assistant/settings/integrations) de votre Dashboard.

Dans le menu de configuration du bot Discord, personnalisez le bot en modifiant son avatar ou son nom, et choisissez le salon dans lequel il répond automatiquement à tous les messages.
