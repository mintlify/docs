<Info>
  L’application Slack est disponible avec les [offres Pro et Custom](https://mintlify.com/pricing?ref=slack-app).
</Info>

L’application Slack ajoute un bot nommé `@mintlify` à votre espace de travail Slack, capable de rechercher dans votre documentation et de répondre aux questions des utilisateurs. Le bot répond aux messages privés, aux @mentions, ainsi qu’à toute question dans un canal nommé spécifiquement `#ask-ai`.

L’utilisation de l’application Slack peut entraîner des coûts : soit via vos crédits d’Assistant IA, soit via des frais de dépassement.

<div id="setting-up-the-slack-app">
  ## Configuration de l’application Slack
</div>

<Note>
  Si le propriétaire de votre espace de travail Slack exige une approbation d’un administrateur pour installer des applications, demandez-lui d’approuver l’application Slack Mintlify avant de l’ajouter.
</Note>

1. Accédez à la page [Add-ons](https://dashboard.mintlify.com/products/addons) de votre tableau de bord.
2. Sélectionnez **Connect** dans la carte Slack Integration.
3. Suivez les instructions de Slack pour ajouter l’application à votre espace de travail.
4. Vérifiez que le bot fonctionne et répond lorsque vous :

- Envoyez un message direct à l’application Mintlify.
  - Mentionnez le bot avec `@mintlify` dans un canal.
  - Créez un canal `#ask-ai`, ajoutez le bot, puis posez une question.
