<Info>
  L'application Slack est disponible pour les [offres Pro et Custom](https://mintlify.com/pricing?ref=slack-app) avec accès à l'Assistant.
</Info>

L'application Slack ajoute un bot à votre espace de travail qui aide votre communauté en fournissant des réponses en temps réel. Le bot utilise l'Assistant Mintlify pour rechercher dans votre documentation et fournir des réponses précises, avec des sources citées, afin que celles-ci soient toujours à jour.

Le bot répond aux mentions `@` et à chaque message dans les canaux nommés `#ask-ai`.

Chaque message envoyé par le bot est comptabilisé dans votre quota de messages de l'Assistant.

<div id="set-up-the-slack-app">
  ## Configurer l’application Slack
</div>

<Note>
  Si le propriétaire de votre espace de travail Slack exige une approbation d’un administrateur pour installer des applications, demandez-lui d’approuver l’application Slack Mintlify avant de l’ajouter.
</Note>

1. Accédez à l’onglet [Integrations](https://dashboard.mintlify.com/products/assistant/settings/integrations) de la page **Assistant Configurations** dans votre Dashboard.
1. Dans la carte Slack, cliquez sur **Connect**. Slack s’ouvre.

<Frame>
    <img src="/images/assistant/slack-connect-light.png" alt="Le bouton Connect dans la carte Slack en mode clair." className="block dark:hidden" />
    <img src="/images/assistant/slack-connect-dark.png" alt="Le bouton Connect dans la carte Slack en mode sombre." className="hidden dark:block" />
  </Frame>

1. Suivez les instructions de Slack pour ajouter l’application à votre espace de travail.
1. Mentionnez le bot pour l’ajouter à un canal. Le nom par défaut du bot est `@mintlify-assistant`.

<div id="customize-the-bots-name">
  ### Personnaliser le nom du bot
</div>

1. Dans Slack, sélectionnez l'application Mintlify dans le menu **Apps**.
1. Cliquez sur **About**.
1. Cliquez sur **Configuration**. La page de configuration de l'application s'ouvre dans un navigateur Web.
1. Dans la section **Bot User**, modifiez le nom du bot.

<div id="create-an-ask-ai-channel">
  ## Créer un canal `#ask-ai`
</div>

Pour aider vos utilisateurs à obtenir rapidement des réponses à leurs questions, créez un canal nommé `#ask-ai`. Le bot répondra à chaque message dans ce canal. Consultez [Create a channel](https://slack.com/help/articles/201402297-Create-a-channel) dans le Centre d’assistance Slack pour plus d’informations.

<div id="manage-the-slack-app">
  ## Gérer l’application Slack
</div>

Après avoir ajouté l’application à votre espace de travail, vous pouvez gérer ou supprimer l’application depuis l’onglet [Integrations](https://dashboard.mintlify.com/products/assistant/settings/integrations).
