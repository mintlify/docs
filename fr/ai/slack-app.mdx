---

<Info>
  L’app Slack est disponible avec les [offres Pro et Custom](https://mintlify.com/pricing?ref=slack-app).
</Info>

L’app Slack ajoute un bot à votre espace de travail Slack capable de rechercher dans votre documentation et de répondre aux questions des utilisateurs. Le bot répond aux messages privés, aux @mentions, ainsi qu’à toute question dans un canal nommé précisément `#ask-ai`. Le nom du bot est `@yourprojectname-assistant`. Ainsi, si le nom de votre projet est Mintlify, le bot s’appelle `@mintlify-assistant`.

L’app Slack peut entraîner des coûts : soit en utilisant vos crédits d’Assistant IA, soit via des dépassements.

<div id="set-up-the-slack-app">
  ## Configurer l’application Slack
</div>

<Note>
  Si le propriétaire de votre espace de travail Slack exige une approbation d’un administrateur pour installer des applications, demandez-lui d’approuver l’application Slack Mintlify avant de l’ajouter.
</Note>

1. Accédez à la page [Modules complémentaires](https://dashboard.mintlify.com/products/addons) de votre Dashboard.
2. Sélectionnez **Connect** dans la carte Slack Integration.
3. Suivez les instructions de Slack pour ajouter l’application à votre espace de travail.
4. Vérifiez que le bot fonctionne et répond lorsque vous :

- Envoyez un message direct au bot.
  - Mentionnez le bot dans un canal.
  - Créez un canal `#ask-ai`, ajoutez le bot et posez une question.
