Pour configurer votre documentation sur un domaine personnalisé, définissez le domaine souhaité dans vos paramètres Mintlify et configurez votre DNS auprès de votre fournisseur de domaine.

<Info>
  Vous souhaitez configurer un sous-répertoire comme `mintlify.com/docs` ? Consultez la page sur le [sous-répertoire personnalisé](/fr/advanced/subpath/cloudflare).
</Info>

## Paramètres du tableau de bord

1. Accédez à votre [tableau de bord](https://dashboard.mintlify.com)
2. Sélectionnez **Settings**
3. Sélectionnez **Domain Setup**
4. Saisissez le domaine personnalisé de votre choix. Par exemple : `docs.mintlify.com`

## Configuration de votre DNS

1. Rendez-vous dans les paramètres DNS de votre domaine sur le site de votre registrar.
2. Créez une nouvelle entrée DNS et renseignez les valeurs suivantes :

```bash
CNAME | docs | cname.vercel-dns.com.
```

## Paramètres spécifiques au fournisseur

<AccordionGroup>
  <Accordion title="Vérification Vercel">
    Si Vercel est votre fournisseur de noms de domaine, vous devez ajouter un enregistrement `TXT` de vérification. Ces informations apparaîtront sur votre tableau de bord après l’envoi de votre domaine personnalisé et vous seront également envoyées par e‑mail.
  </Accordion>

  <Accordion title="Paramètres de sécurité Cloudflare">
    Si Cloudflare est votre fournisseur DNS, vous devez activer l’option de sécurité « full strict » pour la configuration HTTPS.
  </Accordion>
</AccordionGroup>
