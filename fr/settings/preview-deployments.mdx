<Info>
  Les aperçus de déploiement sont disponibles avec les [offres Pro et Enterprise](https://mintlify.com/pricing?ref=preview-deployments).
</Info>

Les aperçus de déploiement vous permettent de voir à quoi ressembleront les modifications apportées à votre documentation avant leur fusion en production. Chaque aperçu crée une URL partageable qui se met automatiquement à jour lorsque vous poussez de nouvelles modifications.

<div id="creating-preview-deployments">
  ## Créer des déploiements de prévisualisation
</div>

Les déploiements de prévisualisation sont créés automatiquement via des pull requests (demandes de fusion) ou manuellement depuis votre Dashboard.

<div id="automatic-previews">
  ### Aperçus automatiques
</div>

Lorsque vous créez une pull request (demande de fusion), le bot Mintlify ajoute automatiquement un lien pour afficher le déploiement de prévisualisation dans votre pull request. L’aperçu se met à jour chaque fois que vous poussez de nouveaux commits vers la branche.

<div id="manual-previews">
  ### Aperçus manuels
</div>

Vous pouvez créer manuellement un aperçu pour n’importe quelle branche.

1. Accédez à votre [Dashboard](https://dashboard.mintlify.com/).
2. Sélectionnez **Previews**.
3. Sélectionnez **Create preview**.
4. Saisissez le nom de votre branche.
5. Sélectionnez **Create deployment**.

<div id="redeploying-a-preview">
  ## Redéployer un aperçu
</div>

Redéployez un aperçu pour actualiser le contenu ou réessayer après un échec de déploiement.

1. Sélectionnez l’aperçu dans votre [Dashboard](https://dashboard.mintlify.com/).
2. Sélectionnez **Redéployer**.

<div id="sharing-preview-deployments">
  ## Partager les déploiements de prévisualisation
</div>

Les déploiements de prévisualisation peuvent être accessibles publiquement ou réservés aux utilisateurs authentifiés.

**Accès public** : Les URL de prévisualisation sont publiques par défaut. Partagez un lien de prévisualisation avec toute personne qui doit examiner vos modifications.

**Accès restreint** : Activez l’authentification des déploiements de prévisualisation pour limiter l’accès aux seuls utilisateurs Mintlify authentifiés. Voir [Configuration de l’Authentification](/fr/authentication-personalization/authentication-setup) pour plus d’informations.
