<Info>
  Les déploiements de prévisualisation sont disponibles avec les [offres Pro et Enterprise](https://mintlify.com/pricing?ref=preview-deployments).
</Info>

Les déploiements de prévisualisation vous permettent de voir à quoi ressembleront les modifications apportées à votre documentation avant leur fusion en production. Chaque prévisualisation crée une URL partageable qui se met automatiquement à jour lorsque vous poussez de nouvelles modifications.

<div id="creating-preview-deployments">
  ## Créer des déploiements de prévisualisation
</div>

Les déploiements de prévisualisation sont créés automatiquement via des pull requests ou manuellement depuis votre tableau de bord.

<div id="automatic-previews">
  ### Prévisualisations automatiques
</div>

Lorsque vous créez une pull request, le bot Mintlify ajoute automatiquement un lien pour afficher le déploiement de prévisualisation dans votre pull request. La prévisualisation se met à jour à chaque fois que vous poussez de nouveaux commits vers la branche.

<div id="manual-previews">
  ### Prévisualisations manuelles
</div>

Vous pouvez créer manuellement une prévisualisation pour n’importe quelle branche.

1. Accédez à votre [tableau de bord](https://dashboard.mintlify.com/).
2. Sélectionnez **Previews**.
3. Sélectionnez **Create preview**.
4. Saisissez le nom de votre branche.
5. Sélectionnez **Create deployment**.

<div id="redeploying-a-preview">
  ## Redéployer un aperçu
</div>

Redéployez un aperçu pour actualiser le contenu ou retenter après un échec de déploiement.

1. Sélectionnez l’aperçu dans votre [tableau de bord](https://dashboard.mintlify.com/).
2. Sélectionnez **Redéployer**.

<div id="sharing-preview-deployments">
  ## Partager les déploiements de prévisualisation
</div>

Les déploiements de prévisualisation peuvent être accessibles publiquement ou réservés aux utilisateurs authentifiés.

**Accès public** : Les URL de prévisualisation sont visibles publiquement par défaut. Partagez un lien de prévisualisation avec toute personne devant examiner vos modifications.

**Accès restreint** : Activez l’authentification des déploiements de prévisualisation pour limiter l’accès aux seuls utilisateurs Mintlify authentifiés. Consultez la page [Configuration de l’authentification](/fr/authentication-personalization/authentication-setup) pour en savoir plus.
