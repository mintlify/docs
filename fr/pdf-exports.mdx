<Info>
Les exports PDF sont disponibles avec les [offres personnalisées](https://mintlify.com/pricing).
</Info>

Vous pouvez exporter votre documentation dans un seul fichier PDF pour une consultation hors ligne. Le PDF inclura tout le contenu de la documentation, y compris les images et les liens, ainsi qu’une table des matières navigable.

<div id="exporting-a-pdf">
  ## Exporter un PDF
</div>

1. Accédez à la page [General](https://dashboard.mintlify.com/settings/deployment/general) dans la section Project Settings de votre Dashboard.
2. Sélectionnez le bouton **Export all content**.
3. Si nécessaire, personnalisez les options d’export :
    - **Page format** : Choisissez le format de page du PDF.
    - **Scale percentage** : Ajustez l’échelle du PDF.
    - **Include footer** : Ajoutez un pied de page avec le numéro de page et le nombre total de pages.
4. Sélectionnez **Export** pour lancer l’export.
5. Une fois l’export terminé, vous recevrez un e-mail avec un lien de téléchargement.
6. Cliquez sur ce lien pour télécharger le fichier PDF.
