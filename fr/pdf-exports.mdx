Vous pouvez exporter votre documentation en un seul fichier PDF pour une consultation hors ligne. Le PDF inclura l’ensemble du contenu de la documentation, y compris les images et les liens, ainsi qu’une table des matières navigable.


<div id="exporting-a-pdf">
  ## Exporter un PDF
</div>

1. Accédez à la page [General](https://dashboard.mintlify.com/settings/deployment/general) dans la section Paramètres du projet de votre Dashboard.
2. Cliquez sur le bouton **Exporter tout le contenu**.
3. Vous pouvez, si vous le souhaitez, personnaliser les options d’export :
    - **Format de page** : choisissez la taille des pages du PDF.
    - **Pourcentage d’échelle** : ajustez l’échelle du PDF.
    - **Inclure le pied de page** : ajoute un pied de page avec le numéro de page et le nombre total de pages.
4. Cliquez sur **Exporter** pour lancer le processus d’export.
5. Une fois l’export terminé, vous recevrez un e‑mail contenant un lien de téléchargement.
6. Cliquez sur le lien pour télécharger le fichier PDF.
