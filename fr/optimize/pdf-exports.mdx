<Info>
  Les exportations PDF sont disponibles avec les [offres Enterprise](https://mintlify.com/pricing).
</Info>

Vous pouvez exporter l’ensemble de votre documentation en un seul fichier PDF pour une consultation hors ligne. Le PDF contient tout le contenu de la doc, y compris les images et les liens, ainsi qu’une table des matières navigable.

Les exportations PDF de l’intégralité du site incluent toutes les pages, quelles que soient les restrictions de groupe, de sorte que toute personne ayant accès au PDF exporté peut lire du contenu qui pourrait autrement être soumis à des restrictions par groupe d’utilisateurs.

<Tip>
  Pour permettre aux utilisateurs de télécharger des pages individuelles au format PDF, utilisez l'option `download-pdf` du [menu contextuel](/fr/ai/contextual-menu).
</Tip>

<div id="export-all-content">
  ## Exporter tout le contenu
</div>

1. Accédez à la page [Exports](https://app.mintlify.com/settings/deployment/export-docs) de votre Dashboard.
2. Cliquez sur **Export all content**.
3. Si besoin, personnalisez les options d’export :
    - **Page format** : choisissez la taille de page du PDF.
    - **Scale percent** : ajustez l’échelle du PDF.
    - **Include footer** : inclure un pied de page avec le numéro de page et le nombre total de pages.
    - **Language** : si votre contenu est disponible en plusieurs langues, choisissez la langue à exporter.
4. Cliquez sur **Export**.
5. Une fois l’export terminé, Mintlify vous envoie un e‑mail avec un lien pour télécharger le fichier PDF.

<div id="export-select-pages">
  ## Exporter des pages spécifiques
</div>

1. Accédez à la page [Exports](https://app.mintlify.com/settings/deployment/export-docs) de votre Dashboard.
2. Cliquez sur **New export**.
3. Sélectionnez les pages que vous souhaitez exporter.
4. Si besoin, personnalisez les options d’export :
    - **Page format** : choisissez la taille de page du PDF.
    - **Scale percent** : ajustez l’échelle du PDF.
    - **Include footer** : inclure un pied de page avec le numéro de page et le nombre total de pages.
    - **Language** : si votre contenu est disponible en plusieurs langues, choisissez la langue à exporter.
5. Cliquez sur **Export**.
6. Une fois l’export terminé, Mintlify vous envoie un e‑mail avec un lien pour télécharger le fichier PDF.

<Note>
  Les exports prennent généralement quelques minutes pour les petits sites et plus de temps pour les grands sites. Si vous ne recevez pas l'e-mail, consultez la page [Exports](https://app.mintlify.com/settings/deployment/export-docs) de votre dashboard pour retélécharger les exports précédents ou relancer un export en échec.

  Les exports PDF sont déclenchés manuellement depuis le dashboard. Il n'existe pas d'option d'export via API ni d'export planifié.
</Note>
