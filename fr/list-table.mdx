## Listes

Les listes suivent la [syntaxe Markdown officielle](https://www.markdownguide.org/basic-syntax/#lists-1).

### Liste ordonnée

Pour créer une liste ordonnée, ajoutez des chiffres suivis d’un point avant chaque élément.

1. Premier élément
2. Deuxième élément
3. Troisième élément
4. Quatrième élément

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

### Liste non ordonnée

Pour créer une liste non ordonnée, ajoutez des tirets (`-`), des astérisques (`*`) ou des signes plus (`+`) avant chaque élément.

- Premier élément
- Deuxième élément
- Troisième élément
- Quatrième élément

```mdx
- First item
- Second item
- Third item
- Fourth item
```

### Liste imbriquée

Indentez les éléments pour créer une imbrication.

- Premier élément
- Deuxième élément
  - Élément supplémentaire
  - Élément supplémentaire
- Troisième élément

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

## Tables

Les tableaux suivent la [syntaxe Markdown officielle](https://www.markdownguide.org/extended-syntax/#tables).

Pour ajouter un tableau, utilisez trois tirets ou plus (`---`) pour créer l’en-tête de chaque colonne, et des barres verticales (`|`) pour séparer les colonnes. Pour des raisons de compatibilité, ajoutez également une barre verticale à chaque extrémité de la ligne.

| Propriété | Description                             |
| --------- | --------------------------------------- |
| Nom       | Nom complet de l’utilisateur            |
| Âge       | Âge déclaré                              |
| Inscrit   | Indique si l’utilisateur a rejoint la communauté |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
