<div id="lists">
  ## Listes
</div>

Les listes suivent la [syntaxe Markdown officielle](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Liste ordonnée
</div>

Pour créer une liste ordonnée, ajoutez des nombres suivis d’un point avant chaque élément.

1. Premier élément
2. Deuxième élément
3. Troisième élément
4. Quatrième élément

```mdx
1. Premier élément
2. Deuxième élément
3. Troisième élément
4. Quatrième élément
```

<div id="unordered-list">
  ### Liste à puces
</div>

Pour créer une liste à puces, ajoutez des tirets (`-`), des astérisques (`*`) ou des signes plus (`+`) avant les éléments.

* Premier élément
* Deuxième élément
* Troisième élément
* Quatrième élément

```mdx
- Premier point
- Deuxième point
- Troisième point
- Quatrième point
```

<div id="nested-list">
  ### Liste imbriquée
</div>

Indentez les éléments de liste pour créer un niveau d’imbrication.

* Premier élément
* Deuxième élément
  * Élément supplémentaire
  * Élément supplémentaire
* Troisième élément

```mdx
- Premier élément
- Deuxième élément
  - Élément additionnel
  - Élément additionnel
- Troisième élément
```


<div id="tables">
  ## Tableaux
</div>

Les tableaux suivent la [syntaxe Markdown officielle](https://www.markdownguide.org/extended-syntax/#tables).

Pour ajouter un tableau, utilisez trois tirets ou plus (`---`) pour créer l’en-tête de chaque colonne et des barres verticales (`|`) pour séparer chaque colonne. Par compatibilité, ajoutez également une barre verticale à chaque extrémité de la ligne.

| Propriété | Description                              |
| --------- | ---------------------------------------- |
| Nom       | Nom complet de l’utilisateur             |
| Âge       | Âge déclaré                              |
| Inscription | Indique si l’utilisateur a rejoint la communauté |

```mdx
| Propriété | Description                           |
| -------- | ------------------------------------- |
| Nom      | Nom complet de l’utilisateur           |
| Âge      | Âge indiqué                            |
| Inscrit  | Indique si l’utilisateur a rejoint la communauté |
```
