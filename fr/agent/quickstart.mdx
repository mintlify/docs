<div id="open-These-agent-panel">
  ## Ouvrir le panneau de l’agent
</div>

Sur ordinateur, le panneau de l’agent peut être redimensionné. Sur mobile, l’agent s’ouvre en plein écran.

Pour ouvrir le panneau de l’agent :

- Depuis n’importe quelle page de votre Dashboard, utilisez le raccourci clavier <kbd>⌘</kbd>+<kbd>I</kbd> (macOS) ou <kbd>Ctrl</kbd>+<kbd>I</kbd> (Windows/Linux).
- Sur la page [Overview](https://dashboard.mintlify.com/), cliquez sur **Ask agent**.

<div id="agent-panel-views">
  ### Vues du panneau de l'agent
</div>

<Frame>
  <img src="/images/agent/views-light.png" alt="Vues dans le panneau de l'agent : nouvelle conversation, suggestions, historique et paramètres." className="block dark:hidden" />
  <img src="/images/agent/views-dark.png" alt="Vues dans le panneau de l'agent : nouvelle conversation, suggestions, historique et paramètres." className="hidden dark:block" />
</Frame>

- **Chat** : Cliquez sur l’icône plus, <Icon icon="plus" />, pour démarrer une nouvelle conversation. Dans la vue de chat, envoyez des prompts à l’agent pour mettre à jour votre documentation ou poser des questions à son sujet. L’agent crée des pull requests (demandes de fusion) en fonction de vos instructions et affiche des liens pour consulter les pull requests ou ouvrir les modifications dans l’éditeur web.
- **Suggestions** : Si cette option est activée, le panneau des suggestions affiche des mises à jour proposées pour votre documentation, basées sur les modifications des pull requests et les conversations des utilisateurs avec l’Assistant.
- **Historique** : Cliquez sur l’icône d’historique, <Icon icon="clock" />, pour parcourir les conversations précédentes et continuer à travailler sur des demandes antérieures. Cliquez sur n’importe quelle conversation pour la charger dans la vue de chat.
- **Paramètres** : Cliquez sur l’icône de paramètres, <Icon icon="settings" />, pour configurer les intégrations de l’agent et l’accès au référentiel.

<Tip>
  Démarrez une nouvelle conversation avec l’agent pour chaque tâche. Cela permet de garder le contexte de l’agent ciblé et vous aide à associer les conversations à des projets spécifiques.
</Tip>

<div id="connect-your-github-account">
  ## Connectez votre compte GitHub
</div>

Par défaut, l'agent ouvre des pull requests (demandes de fusion) attribuées au bot Mintlify. Pour que les pull requests vous soient attribuées, connectez votre compte GitHub sur la page [Mon profil](https://dashboard.mintlify.com/settings/account) du Dashboard.

<div id="connect-repositories-as-context">
  ## Connecter des dépôts en tant que contexte
</div>

L’agent ne peut accéder qu’aux dépôts que vous connectez via la GitHub App Mintlify. Configurez les dépôts auxquels l’agent peut accéder dans le panneau **Settings** de l’agent ou dans les [paramètres de la GitHub App](https://github.com/apps/mintlify/installations/new).
