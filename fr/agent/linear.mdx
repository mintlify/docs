Connectez l'agent Mintlify à Linear pour que votre équipe puisse déclencher des mises à jour de la documentation directement depuis les tickets et commentaires Linear.

## Connecter votre espace de travail Linear

1. Ouvrez le panneau agent dans votre [tableau de bord](https://dashboard.mintlify.com/products/agent).
2. Cliquez sur le bouton **Settings**.
3. Dans la section d'intégration Linear, cliquez sur **Connect**.
4. Suivez les instructions pour autoriser Mintlify dans votre espace de travail Linear.

## Utiliser l'agent dans Linear

Une fois connecté, mentionnez l'agent dans un commentaire de ticket Linear pour déclencher une mise à jour de la documentation. L'agent lit le contexte du ticket, y compris le titre, la description et le fil de commentaires, et crée une pull request avec les modifications proposées.

Vous pouvez :

- Mentionner l'agent dans un commentaire de ticket avec des instructions sur ce qu'il faut documenter.
- Référencer des pages ou sections spécifiques à mettre à jour.
- Joindre du contexte provenant de tickets ou projets liés pour que l'agent l'utilise.

## Exemples de prompts

Voici quelques exemples de prompts que vous pouvez utiliser dans les commentaires de tickets Linear :

- `@mintlify Documentez cette nouvelle fonctionnalité dans la référence API.`
- `@mintlify Mettez à jour le guide de démarrage rapide pour refléter les changements de ce ticket.`
- `@mintlify Ajoutez une section de dépannage basée sur le bug décrit ici.`

## Bonnes pratiques

- **Soyez précis** : Dites à l'agent exactement ce que vous voulez documenter et où cela doit aller.
- **Ajoutez du contexte** : Si le ticket ne contient pas toutes les informations nécessaires, incluez des détails supplémentaires dans votre commentaire à l'agent.
- **Relisez attentivement** : Relisez toujours les pull requests créées par l'agent avant de les fusionner.
