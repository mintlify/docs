Lorsque des agents IA parcourent votre documentation via les [pages Markdown](/fr/ai-native#discovering), ils peuvent soumettre des retours sur les problèmes qu'ils rencontrent. Ces retours apparaissent dans votre tableau de bord aux côtés des retours des utilisateurs, afin que vous puissiez suivre et résoudre les problèmes de documentation signalés par les humains et les agents IA.

<div id="how-agent-feedback-works">
  ## Comment fonctionnent les retours d'agents
</div>

Vos pages Markdown incluent des instructions qui indiquent aux agents comment signaler les problèmes. Les agents soumettent leurs retours via une requête POST contenant le chemin de la page et une description du problème.

Les retours d'agents apparaissent dans le [tableau de bord des retours](https://dashboard.mintlify.com/products/analytics/v2/feedback) avec un badge <Badge color="blue">Agent</Badge> pour les distinguer des retours humains. Vous pouvez filtrer, gérer et résoudre les retours d'agents de la même manière que les autres types de retours.

Les retours d'agents sont activés par défaut pour tous les sites de documentation. Aucune configuration n'est requise.

<div id="view-agent-feedback">
  ## Consulter les retours d'agents
</div>

1. Accédez au [tableau de bord des retours](https://dashboard.mintlify.com/products/analytics/v2/feedback) dans votre tableau de bord.
2. Sélectionnez **Ratings by page** pour obtenir une vue d'ensemble, ou **Detailed feedback** pour les soumissions individuelles.
3. Recherchez le badge <Badge color="blue">Agent</Badge> pour identifier les retours soumis par des agents IA.

Utilisez la case à cocher **Show agent feedback** dans le menu de filtres pour inclure ou exclure les retours d'agents de votre vue.

<div id="manage-agent-feedback">
  ## Gérer les retours d'agents
</div>

Les retours d'agents prennent en charge les mêmes fonctionnalités de gestion que les autres types de retours :

- **Changer le statut** : Marquez les retours comme En attente, En cours, Résolu ou Rejeté.
- **Ajouter des notes internes** : Cliquez sur un retour pour ajouter des notes pour votre équipe.
- **Actions groupées** : Sélectionnez plusieurs éléments pour changer le statut, signaler ou supprimer en masse.

Pour plus d'informations sur la gestion des retours, consultez [Retours](/fr/optimize/feedback).
