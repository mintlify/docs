<Info>
  L&#39;agent est disponible sur les [plans Pro et Enterprise](https://mintlify.com/pricing?ref=agent) pour toute personne ayant accès à votre Dashboard.
</Info>

L&#39;agent crée des pull requests (demandes de fusion) avec des modifications proposées pour votre documentation, en fonction de vos instructions. Lorsque vous envoyez une requête à l&#39;agent, il s&#39;appuie sur votre documentation, les dépôts connectés et les messages Slack pour créer du contenu qui suit les bonnes pratiques de rédaction technique et respecte le schéma Mintlify.

<div id="what-you-can-do-with-the-agent">
  ## Ce que vous pouvez faire avec l&#39;agent
</div>

Utilisez l&#39;agent pour :

* Rédiger de nouveaux contenus à partir de vos prompts, de liens vers des pull requests (demandes de fusion) ou de fils de discussion Slack.
* Mettre à jour des exemples de code obsolètes et des références d&#39;API.
* Rechercher et mettre à jour du contenu existant.
* Répondre aux questions sur votre documentation et sur des sujets de rédaction technique.
* Conserver les connaissances issues des conversations Slack et des pull requests (demandes de fusion) avant qu&#39;elles ne se perdent.

<div id="where-you-can-use-the-agent">
  ## Où vous pouvez utiliser l’agent
</div>

* **Dashboard** : Depuis n’importe quelle page de votre Dashboard, utilisez le raccourci clavier <kbd>⌘</kbd>+<kbd>I</kbd> (macOS) ou <kbd>Ctrl</kbd>+<kbd>I</kbd> (Windows/Linux) pour ouvrir le panneau de l’agent. Vous pouvez aussi cliquer sur **Ask agent** sur la page [Overview](https://dashboard.mintlify.com/).
  <Frame>
    <img src="/images/obama.jpg" alt="Le panneau de l’agent ouvert dans le Dashboard." className="block dark:hidden" />

    <img src="/images/obama.jpg" alt="Le panneau de l’agent ouvert dans le Dashboard." className="hidden dark:block" />

* **Slack** : Ajoutez l’agent à votre espace de travail Slack et mentionnez `@mintlify` pour le solliciter.
* **API** : Intégrez l’agent dans des applications personnalisées à l’aide des [points de terminaison de l’agent](/fr/api-reference/agent/create-agent-job).
