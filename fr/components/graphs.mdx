Utilisez le composant `<Graph>` pour créer des graphiques interactifs et des visualisations de données dans votre documentation. Les graphiques aident les utilisateurs à comprendre les tendances, les comparaisons et les distributions de données en un coup d'œil.

## Utilisation de base

Encapsulez votre configuration de données dans un composant `<Graph>` et spécifiez le type de graphique.

<Graph type="line" title="Revenus mensuels">
  {[
    { month: "Jan", revenue: 4000 },
    { month: "Fév", revenue: 3000 },
    { month: "Mar", revenue: 5000 },
    { month: "Avr", revenue: 4500 },
    { month: "Mai", revenue: 6000 },
    { month: "Juin", revenue: 5500 }
  ]}
</Graph>

```jsx
<Graph type="line" title="Revenus mensuels">
  {[
    { month: "Jan", revenue: 4000 },
    { month: "Fév", revenue: 3000 },
    { month: "Mar", revenue: 5000 },
    { month: "Avr", revenue: 4500 },
    { month: "Mai", revenue: 6000 },
    { month: "Juin", revenue: 5500 }
  ]}
</Graph>
```

## Types de graphiques

### Graphique linéaire

Affichez les tendances dans le temps ou les données continues.

<Graph type="line" title="Croissance des utilisateurs" xKey="month" yKey="users">
  {[
    { month: "Jan", users: 1200 },
    { month: "Fév", users: 1900 },
    { month: "Mar", users: 2400 },
    { month: "Avr", users: 3100 },
    { month: "Mai", users: 4200 },
    { month: "Juin", users: 5100 }
  ]}
</Graph>

```jsx
<Graph type="line" title="Croissance des utilisateurs" xKey="month" yKey="users">
  {[
    { month: "Jan", users: 1200 },
    { month: "Fév", users: 1900 },
    { month: "Mar", users: 2400 },
    { month: "Avr", users: 3100 },
    { month: "Mai", users: 4200 },
    { month: "Juin", users: 5100 }
```

## Séries multiples

Affichez plusieurs séries de données sur le même graphique pour comparaison.

<Graph type="line" title="Revenus vs Dépenses" xKey="month">
  {{
    data: [
      { month: "Jan", revenue: 4000, expenses: 2400 },
      { month: "Fév", revenue: 3000, expenses: 1398 },
      { month: "Mar", revenue: 5000, expenses: 3800 },
      { month: "Avr", revenue: 4500, expenses: 3908 },
      { month: "Mai", revenue: 6000, expenses: 4800 },
      { month: "Juin", revenue: 5500, expenses: 3800 }
    ],
    series: [
      { key: "revenue", label: "Revenus", color: "#22c55e" },
      { key: "expenses", label: "Dépenses", color: "#ef4444" }
    ]
  }}
</Graph>

```jsx
<Graph type="line" title="Revenus vs Dépenses" xKey="month">
  {{
    data: [
      { month: "Jan", revenue: 4000, expenses: 2400 },
      { month: "Fév", revenue: 3000, expenses: 1398 },
      { month: "Mar", revenue: 5000, expenses: 3800 },
      { month: "Avr", revenue: 4500, expenses: 3908 },
      { month: "Mai", revenue: 6000, expenses: 4800 },
      { month: "Juin", revenue: 5500, expenses: 3800 }
    ],
    series: [
      { key: "revenue", label: "Revenus", color: "#22c55e" },
      { key: "expenses", label: "Dépenses", color: "#ef4444" }
    ]
  }}
</Graph>
```

## Props

<ResponseField name="type" type="string" required>
  Le type de graphique. Options : `line`, `bar`, `area`, `pie`, `donut`.
</ResponseField>

<ResponseField name="title" type="string">
  Le titre du graphique affiché au-dessus de la visualisation.
</ResponseField>

<ResponseField name="xKey" type="string">
  La clé de données à utiliser pour l'axe des x (pour les graphiques linéaires, à barres et en aires).
</ResponseField>

<ResponseField name="yKey" type="string">
  La clé de données à utiliser pour l'axe des y (pour les graphiques à série unique).
</ResponseField>

<ResponseField name="height" type="number" default="300">
  La hauteur du graphique en pixels.
</ResponseField>

<ResponseField name="color" type="string">
  La couleur principale pour les graphiques à série unique. Accepte hex, RGB ou noms de couleurs CSS.
</ResponseField>

<ResponseField name="stacked" type="boolean" default="false">
  Empiler plusieurs séries les unes sur les autres (pour les graphiques à barres et en aires).
</ResponseField>

<ResponseField name="showLegend" type="boolean" default="true">
  Afficher ou masquer la légende du graphique.
</ResponseField>

<ResponseField name="showGrid" type="boolean" default="true">
  Afficher ou masquer les lignes de grille.
</ResponseField>

<ResponseField name="children" type="array | object" required>
  Les données du graphique. Passez un tableau pour une série unique ou un objet avec `data` et `series` pour plusieurs séries.
</ResponseField>
