---
title: "Infobulles"
description: "Afficher une définition au survol d’un texte"
icon: "message-square"
---

Utilisez des infobulles pour fournir un contexte supplémentaire ou des définitions lorsqu’un utilisateur survole un segment de texte. Les infobulles peuvent inclure des liens d’appel à l’action facultatifs.

**Exemple** : <Tooltip tip="Application Programming Interface : un ensemble de protocoles permettant à des applications logicielles de communiquer entre elles." cta="Lire notre guide sur l’API" href="/fr/api-reference">API</Tooltip> La documentation aide les développeurs à comprendre comment intégrer votre service.

```mdx Tooltip example wrap
<Tooltip tip="Interface de programmation d'application : un ensemble de protocoles permettant aux applications logicielles de communiquer." cta="Lire notre guide API" href="/api-reference">API</Tooltip> La documentation API aide les développeurs à comprendre comment s'intégrer à votre service.
```

<div id="properties">
  ## Propriétés
</div>

<ResponseField name="tip" type="string" required>
  Texte affiché dans l’infobulle.
</ResponseField>

<ResponseField name="cta" type="string">
  Texte de l’appel à l’action pour un lien dans l’infobulle.
</ResponseField>

<ResponseField name="href" type="string">
  URL du lien d’appel à l’action. Requis lorsque `cta` est utilisé.
</ResponseField>
