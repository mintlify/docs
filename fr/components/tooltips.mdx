---
title: "Info-bulles"
description: "Afficher une définition au survol d’un texte"
icon: "message-square"
---

Utilisez des info-bulles pour fournir du contexte supplémentaire ou des définitions lorsque l’utilisateur survole un texte. Les info-bulles peuvent inclure des liens d’appel à l’action (CTA) optionnels.

**Exemple** : <Tooltip tip="Interface de programmation d’applications : un ensemble de protocoles permettant aux applications logicielles de communiquer." cta="Lire notre guide API" href="/fr/api-reference">API</Tooltip> La documentation aide les développeurs à comprendre comment intégrer votre service.

```mdx Enrobage d’exemple d’infobulle
<Tooltip tip="Interface de programmation d’applications : un ensemble de protocoles permettant la communication entre des applications logicielles." cta="Consultez notre guide API" href="/api-reference">API</Tooltip> La documentation aide les développeurs à comprendre comment intégrer votre service.
```

<div id="properties">
  ## Propriétés
</div>

<ResponseField name="tip" type="string" required>
  Le texte affiché dans l’infobulle.
</ResponseField>

<ResponseField name="cta" type="string">
  Le texte de l’appel à l’action pour un lien dans l’infobulle.
</ResponseField>

<ResponseField name="href" type="string">
  L’URL du lien d’appel à l’action. Obligatoire lorsque `cta` est utilisé.
</ResponseField>
