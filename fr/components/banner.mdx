title: "Bannière"
description: "Ajoutez une bannière pour afficher des annonces et notifications importantes à l’échelle du site"
icon: "layout-panel-top"
---

Utilisez des bannières pour afficher des annonces, des mises à jour ou des notifications importantes sur l’ensemble de votre site de documentation. Les bannières apparaissent en haut de chaque page, prennent en charge le formatage Markdown et peuvent être rendues dismissibles.

Pour ajouter une bannière, utilisez la propriété `banner` dans votre `docs.json` :

<CodeGroup>
  ```json Product announcements wrap
  "banner": {
    "content": "🚀 La version 2.0 est désormais disponible ! Consultez notre [journal des modifications](/changelog) pour plus de détails.",
    "dismissible": true 
  }
  ```

  ```json Maintenance notices wrap
  "banner": {
    "content": "⚠️ Maintenance programmée : l’API sera indisponible le 15 décembre, de 2 h à 4 h UTC",
    "dismissible": false
  }
  ```

  ```json Required actions wrap
  "banner": {
    "content": "**Action requise :** migrez vers notre nouvelle version d’ici le 1er janvier. [Guide de migration](/migration)",
    "dismissible": true
  }
  ```
</CodeGroup>

<div id="properties">
  ## Propriétés
</div>

<ResponseField name="content" type="string" required>
  Le message de la bannière. Prend en charge le texte brut et la mise en forme Markdown.
</ResponseField>

<ResponseField name="dismissible" type="boolean">
  Indique si les utilisateurs peuvent rejeter la bannière. Lorsque `true`, les utilisateurs peuvent fermer la bannière et elle ne réapparaît pas pendant leur session. Valeur par défaut : `false`.
</ResponseField>
