---
title: "Encadrés"
description: "Utilisez des encadrés pour mettre en forme et mettre en avant du contenu important."
---

Les encadrés peuvent être affichés sous forme de Note, Avertissement, Info, Conseil, Check, Danger, ou vous pouvez créer votre propre encadré :

<Note>Ajoute une note dans le contenu</Note>

```mdx
<Note>Cela ajoute une note dans le contenu</Note>
```

<Warning>Cela génère un avertissement à surveiller</Warning>

```mdx
<Warning>Ceci déclenche un avertissement auquel il faut prêter attention</Warning>
```

<Info>Met en évidence des informations importantes</Info>

```mdx
<Info>Attire l'attention sur des informations importantes</Info>
```

<Tip>Ceci propose un conseil utile</Tip>

```mdx
<Tip>Ceci suggère un conseil utile</Tip>
```

<Check>Cela nous donne un statut coché</Check>

```mdx
<Check>Ceci nous donne un statut vérifié</Check>
```

<Danger>Ceci est un avertissement</Danger>

```mdx
<Danger>Ceci est un avertissement de danger</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Ceci est un appel personnalisé</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Ceci est un encadré personnalisé</Callout>
```
