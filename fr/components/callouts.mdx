---
title: "Encadrés"
description: "Utilisez des encadrés pour ajouter un contexte percutant à votre contenu"
icon: "info"
---

Les encadrés peuvent être présentés sous forme de Note, Avertissement, Info, Astuce, Validation, Danger, ou vous pouvez créer votre propre encadré :

<Note>Cela ajoute une note dans le contenu</Note>

```mdx
<Note>Ajoute une note dans le contenu</Note>
```

<Warning>Ceci génère un avertissement à prendre en compte</Warning>

```mdx
<Warning>Cela génère un avertissement à surveiller</Warning>
```

<Info>Ceci met en avant une information importante</Info>

```mdx
<Info>Cela met en avant des informations importantes</Info>
```

<Tip>Astuce : conseil utile</Tip>

```mdx
<Tip>Voici une astuce utile</Tip>
```

<Check>Cela nous donne un statut validé</Check>

```mdx
<Check>Cela nous donne un statut validé</Check>
```

<Danger>Ceci est une alerte critique</Danger>

```mdx
<Danger>Ceci est un avertissement de danger</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Ceci est une alerte personnalisée</Callout>

```mdx renvoi à la ligne
<Callout icon="key" color="#FFC107" iconType="regular">Ceci est un encadré personnalisé</Callout>
```
