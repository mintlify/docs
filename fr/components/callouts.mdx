---
title: 'Encadrés'
description: 'Utilisez des encadrés pour ajouter un contexte percutant à votre contenu'
---

Les encadrés peuvent être présentés sous forme de Note, d’Avertissement, d’Info, de Conseil, de Validation, de Danger, ou vous pouvez créer votre propre encadré :

<Note>Cela ajoute une note dans le contenu</Note>

```mdx
<Note>Ceci ajoute une note dans le contenu</Note>
```

<Avertissement>Cela génère un avertissement à prendre en compte</Avertissement>

```mdx
<Warning>Ceci génère un avertissement à prendre en compte</Warning>
```

<Info>Met en évidence une information importante</Info>

```mdx
<Info>Ceci attire l'attention sur les informations importantes</Info>
```

<Tip>Voici un conseil utile</Tip>

```mdx
<Tip>Ceci suggère un conseil utile</Tip>
```

<Check>Cela nous donne un statut validé</Check>

```mdx
<Check>Ceci nous donne un statut coché</Check>
```

<Danger>Ceci est un avertissement</Danger>

```mdx
<Danger>Ceci est une alerte de danger</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Ceci est un encadré personnalisé</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Ceci est un encadré personnalisé</Callout>
```
