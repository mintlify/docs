Utilisez le composant `Visibility` pour afficher ou masquer du contenu en fonction de l'audience. Vous pouvez cibler du contenu spécifiquement pour les lecteurs humains ou les agents IA.

<Visibility for="humans">
  Ce contenu est visible uniquement par les lecteurs humains dans le navigateur.
</Visibility>

<Visibility for="agents">
  Ce contenu est visible uniquement par les agents IA qui consomment l'export Markdown.
</Visibility>

## Exemple

````mdx Contenu réservé aux humains et aux agents
<Visibility for="humans">
  Ce paragraphe est affiché uniquement aux lecteurs humains
  qui consultent la page dans un navigateur.
</Visibility>

<Visibility for="agents">
  Ce paragraphe est inclus uniquement lorsque des agents IA
  demandent la page via l'export Markdown.
</Visibility>
````

## Fonctionnement

Mintlify détecte si le visiteur actuel est un humain ou un agent IA et affiche le contenu en conséquence :

- **Dans le navigateur** — Mintlify évalue le user agent du visiteur côté serveur. Le contenu enveloppé dans `<Visibility for="humans">` est affiché aux visiteurs classiques, tandis que le contenu `<Visibility for="agents">` est affiché lorsqu'un agent IA est détecté.
- **Dans l'[export Markdown](/fr/ai/markdown-export)** — lorsqu'une page est demandée en Markdown (via l'extension `.md` ou l'en-tête `Accept: text/markdown`), seul le contenu marqué `for="agents"` est inclus. Le contenu marqué `for="humans"` est supprimé de la sortie.

Le contenu en dehors de tout composant `Visibility` est toujours affiché aux deux audiences.

## Propriétés

<ResponseField name="for" type='"agents" | "humans"' required>
  L'audience cible du contenu encadré. Accepte également les formes au singulier `"agent"` et `"human"`.
</ResponseField>
