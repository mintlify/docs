Les accordéons permettent aux utilisateurs d’ouvrir et de replier des sections de contenu. Utilisez des accordéons pour une divulgation progressive et pour organiser les informations.

## Accordéon simple

<Accordion title="Je suis un accordéon.">
  Vous pouvez ajouter n'importe quel contenu ici, y compris d'autres composants, comme du code :
   ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
</Accordion>

## Groupes d’accordéons

Regroupez des accordéons liés à l’aide de `<AccordionGroup>`. Cela crée une section cohérente d’accordéons pouvant être développés ou réduits individuellement.

<AccordionGroup>
  <Accordion title="Getting started">
    Vous pouvez insérer d’autres composants à l’intérieur des accordéons.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="Advanced features" icon="bot">
    Ajoutez des icônes pour rendre les accordéons plus distinctifs visuellement et plus faciles à parcourir.
  </Accordion>
  
  <Accordion title="Troubleshooting">
    Regroupez le contenu connexe pour le garder organisé.
  </Accordion>
</AccordionGroup>

## Propriétés

<ResponseField name="title" type="string" required>
  Titre dans l’aperçu de l’accordéon.
</ResponseField>

<ResponseField name="description" type="string">
  Détail sous le titre dans l’aperçu de l’accordéon.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Indique si l’accordéon est ouvert par défaut.
</ResponseField>
