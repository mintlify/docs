Le composant `Columns` vous permet de regrouper plusieurs composants `Card`. Il est le plus souvent utilisé pour disposer plusieurs cartes dans une grille, en précisant le nombre de colonnes.

<Columns cols={2}>
  <Card title="Première carte" icon="panel-left-close">
    Neque porro quisquam est qui dolorem ipsum quia dolor sit amet
  </Card>
  <Card title="Deuxième carte" icon="panel-right-close">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit
  </Card>
</Columns>
