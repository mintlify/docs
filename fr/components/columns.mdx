Le composant `Columns` vous permet de regrouper plusieurs composants `Card`. Il est le plus souvent utilisé pour disposer des cartes en grille, en précisant le nombre de colonnes.

<Columns cols={2}>
  <Card title="Pour commencer" icon="rocket">
    Configurez votre projet avec notre guide de démarrage rapide.
  </Card>

  <Card title="Référence de l’API" icon="code">
    Explorez les endpoints, paramètres et exemples de votre API.
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="Commencer">
    Configurez votre projet avec notre guide de démarrage rapide.
  </Card>
  <Card title="Référence API">
    Explorez les endpoints, paramètres et exemples de votre API.
  </Card>
</Columns>
```


<div id="properties">
  ## Propriétés
</div>

<ResponseField name="cols" default={2}>
  Le nombre de colonnes par ligne.
</ResponseField>
