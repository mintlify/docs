Le composant `Columns` vous permet de regrouper plusieurs composants `Card`. Il est le plus souvent utilisé pour afficher des cartes en grille en spécifiant le nombre de colonnes.

<Columns cols={2}>
  <Card title="Démarrer" icon="rocket">
    Configurez votre projet avec notre guide de démarrage rapide.
  </Card>

  <Card title="Référence API" icon="code">
    Explorez les points de terminaison, les paramètres et des exemples pour votre API.
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="Commencer">
    Configurez votre projet avec notre guide de démarrage rapide.
  </Card>
  <Card title="Référence API">
    Explorez les points de terminaison, les paramètres et les exemples pour votre API.
  </Card>
</Columns>
```


<div id="properties">
  ## Propriétés
</div>

<ResponseField name="cols" default={2}>
  Le nombre de colonnes par ligne.
</ResponseField>
