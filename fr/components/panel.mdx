---
title: "Panneau"
description: "Personnalisez le contenu du panneau droit d’une page."
---

Vous pouvez utiliser le composant `<Panel>` pour personnaliser le panneau droit d’une page avec les composants de votre choix.

Si une page contient un composant `<Panel>`, les composants [RequestExample](/fr/components/examples#request-example) et [ResponseExample](/fr/components/examples#response-example) doivent se trouver à l’intérieur de `<Panel>`.

Les composants d’un `<Panel>` remplacent la table des matières de la page.

```mdx
<Panel>
  <Info>Épinglez des informations dans le panneau latéral ou ajoutez n'importe quel autre composant.</Info>
</Panel>
```

<Panel>
  <Info>Épinglez une Info dans le panneau latéral. Ou ajoutez tout autre composant.</Info>
</Panel>
