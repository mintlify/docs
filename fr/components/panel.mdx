title: "Panneau"

---

Vous pouvez utiliser le composant `<Panel>` pour personnaliser le panneau latéral droit d'une page avec les composants de votre choix.

Si une page contient un composant `<Panel>`, tous les composants [RequestExample](/fr/components/examples#request-example) et [ResponseExample](/fr/components/examples#response-example) doivent se trouver à l’intérieur de `<Panel>`.

Les composants dans un `<Panel>` remplacent la table des matières d’une page.

```mdx
<Panel>
  <Info>Épinglez des informations dans le panneau latéral, ou ajoutez tout autre composant.</Info>
</Panel>
```
