---
title: "Panel"
description: "Définir le contenu du panneau latéral droit"
---

Vous pouvez utiliser le composant `<Panel>` pour personnaliser le panneau latéral droit d’une page avec les composants de votre choix.

Si une page comporte un composant `<Panel>`, les composants [RequestExample](/fr/components/examples#request-example) et [ResponseExample](/fr/components/examples#response-example) doivent se trouver à l’intérieur de `<Panel>`.

Les composants dans un `<Panel>` remplaceront la table des matières de la page.

```mdx
<Panel>
  <Info>Épinglez des informations dans le panneau latéral. Ou ajoutez tout autre composant.</Info>
</Panel>
```
