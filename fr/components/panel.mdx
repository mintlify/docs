Vous pouvez utiliser le composant `<Panel>` pour personnaliser le panneau latéral droit d’une page avec les composants de votre choix.

Si une page contient un composant `<Panel>`, les composants [RequestExample](/fr/components/examples#requestexample) et [ResponseExample](/fr/components/examples#responseexample) doivent se trouver à l’intérieur de `<Panel>`.

Les composants dans un `<Panel>` remplacent la table des matières de la page.

```mdx
<Panel>
  <Info>Épinglez des infos dans le panneau latéral. Ou ajoutez un autre composant.</Info>
</Panel>
```
