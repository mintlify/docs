<div id="inline-icons">
  ## Icônes en ligne
</div>

Les icônes sont placées en ligne lorsqu’elles sont utilisées dans une phrase, un paragraphe ou un titre. <Icon icon="flag" iconType="solid" /> Utilisez les icônes pour la décoration ou pour renforcer l’accent visuel.



<div id="properties">
  ## Propriétés
</div>

<IconsRequired />

<ResponseField name="color" type="string">
  La couleur de l’icon sous forme de code hexadécimal (par exemple « #FF5733 »).
</ResponseField>

<ResponseField name="size" type="number">
  La taille de l’icon en pixels.
</ResponseField>

<ResponseField name="className" type="string">
  Nom de classe CSS personnalisé à appliquer à l’icon.
</ResponseField>
