Utilisez des icônes provenant de Font Awesome, de Lucide, de fichiers SVG (Scalable Vector Graphics), d’URL externes ou de fichiers de votre projet pour enrichir votre documentation.

<Icon icon="flag" size={32} />

```mdx Icon example
<Icon icon="flag" size={32} />
```


<div id="inline-icons">
  ## Icônes en ligne
</div>

Les icônes sont placées en ligne lorsqu’elles sont utilisées dans un paragraphe.

<Icon icon="flag" iconType="solid" /> La documentation que vous voulez, en toute simplicité.

```markdown Inline icon example
<Icon icon="flag" iconType="solid" /> La documentation que vous souhaitez, sans effort.
```


<div id="properties">
  ## Propriétés
</div>

<IconsRequired />

<ResponseField name="color" type="string">
  La couleur de l’icon sous forme de code hexadécimal (par exemple « #FF5733 »).
</ResponseField>

<ResponseField name="size" type="number">
  La taille de l’icon en pixels.
</ResponseField>
