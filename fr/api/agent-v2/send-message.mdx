Send follow-up instructions or feedback to an existing agent session. This allows you to have a conversation with the agent, providing clarifications or additional tasks after the initial job has started.

## Use cases

- Provide clarification when the agent needs more context
- Add additional tasks to an ongoing session
- Request changes to work the agent has already done
- Give feedback on the agent's progress

## Example workflow

```javascript
// 1. Create initial job
const createResponse = await fetch(
  `https://api.mintlify.com/v2/agent/${projectId}/job`,
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ prompt: 'Add a quickstart guide for the CLI' })
  }
);
const job = await createResponse.json();

// 2. Send follow-up message
const messageResponse = await fetch(
  `https://api.mintlify.com/v2/agent/${projectId}/job/${job.id}/message`,
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ prompt: 'Also add code examples in Python' })
  }
);
```
