<div id="usage">
  ## Utilisation
</div>

Utilisez cet endpoint pour obtenir les compteurs de retours agrégés pour une page spécifique de votre documentation. La réponse inclut le nombre de pouces levés, de pouces baissés et de retours sur les extraits de code ainsi qu'un total.

Cela est utile pour créer des tableaux de bord ou des rapports montrant les tendances des retours par page sans récupérer les entrées de retours individuelles.

<div id="filtering">
  ## Filtrage
</div>

Filtrez les compteurs de retours par :
- **Chemin de la page** (obligatoire) : le chemin de la page de documentation pour laquelle agréger les retours
- **Plage de dates** : utilisez `dateFrom` et `dateTo` pour limiter l'agrégation à une période donnée
