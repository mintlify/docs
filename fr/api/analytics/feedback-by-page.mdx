<div id="usage">
  ## Utilisation
</div>

Utilisez cet endpoint pour récupérer les comptages agrégés des retours pour une page spécifique de votre documentation. La réponse inclut le nombre d'évaluations positives, d'évaluations négatives, de retours sur les extraits de code et le nombre total.

Contrairement à l'endpoint [Obtenir des retours](/fr/api/analytics/feedback) qui retourne les entrées individuelles de retours, cet endpoint renvoie un résumé de tous les retours pour un chemin de page donné.

<div id="filtering">
  ## Filtrage
</div>

Filtrez les retours par :
- **Chemin de la page** (requis) : Spécifiez le paramètre `path` pour cibler une page de documentation spécifique
- **Plage de dates** : Utilisez `dateFrom` et `dateTo` pour limiter les résultats à une période donnée
