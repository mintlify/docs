## Utilisation

Ce point de terminaison envoie un message de suivi à une session de tâche d'agent existante. Utilisez-le pour continuer une conversation avec l'agent après la création initiale de la tâche, par exemple pour demander des modifications supplémentaires ou fournir des précisions.

## Exemple

```javascript
const response = await fetch(
  `https://api.mintlify.com/v2/agent/${projectId}/job/${jobId}/message`,
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      prompt: 'Ajoutez également un exemple de gestion des erreurs dans la documentation de l\'API'
    })
  }
);

const job = await response.json();
```

## Limites de débit

L'API de l'agent est soumise aux limites suivantes :

- 100 utilisations par projet Mintlify et par heure
