## Utilisation

Ce point de terminaison récupère les détails et l'état d'une tâche d'agent spécifique à partir de son identifiant unique. Utilisez-le pour interroger la complétion de la tâche et récupérer le lien de la pull request.

## Statut de la tâche

Le champ `status` indique l'état actuel de la tâche :

| Statut | Description |
|--------|-------------|
| `pending` | La tâche est en file d'attente |
| `running` | La tâche est en cours de traitement |
| `completed` | La tâche s'est terminée avec succès |
| `failed` | La tâche a rencontré une erreur |

## Champs de réponse

| Champ | Description |
|-------|-------------|
| `id` | Identifiant unique de la tâche |
| `status` | Statut actuel de la tâche |
| `source.repository` | URL du dépôt GitHub |
| `source.ref` | Nom de la branche où les modifications sont effectuées |
| `model` | Modèle d'IA utilisé pour la tâche |
| `prLink` | URL de la pull request (disponible après complétion) |
| `createdAt` | Horodatage de création de la tâche |
| `archivedAt` | Horodatage de complétion de la tâche |
