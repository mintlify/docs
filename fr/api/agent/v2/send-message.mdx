Envoyez une instruction de suivi à une tâche d'agent existante. Le message est traité de manière asynchrone — interrogez l'endpoint [get agent job](/fr/api/agent/v2/get-agent-job) pour suivre la progression.

<div id="rate-limits">
  ## Limites de débit
</div>

- 100 utilisations par projet Mintlify par heure

<div id="usage">
  ## Utilisation
</div>

```bash
curl -X POST https://api.mintlify.com/v2/agent/{projectId}/job/{id}/message \
  -H "Authorization: Bearer mint_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Also add error handling examples to the quickstart guide"}'
```
