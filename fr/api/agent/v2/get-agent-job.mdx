---
title: "Obtenir une tâche d'agent"
description: "Récupère le statut et les détails d'une tâche d'agent, notamment sa progression et le lien vers la pull request."
openapi: /fr/admin-openapi.json GET /v2/agent/{projectId}/job/{id}
keywords: [ "tâche d'agent", "statut", "récupération", "détails", "interroger" ]
---

Interrogez cet endpoint pour suivre la progression d'une tâche d'agent. Le champ `status` évolue de `active` → `completed` ou `failed`.

Une fois que l'agent crée une pull request (demande de fusion), le champ `prLink` est renseigné.
