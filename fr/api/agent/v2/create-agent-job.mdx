title: "Créer une tâche d'agent"
description: "Crée une tâche d'agent qui s'exécute en arrière-plan et crée automatiquement une pull request avec les modifications de la documentation."
openapi: /fr/admin-openapi.json POST /v2/agent/{projectId}/job
keywords: [ "tâche d'agent", "créer", "automatisation", "automatiser" ]
---

Cet endpoint crée une tâche d'agent en arrière-plan. La tâche s'exécute de manière asynchrone — utilisez l'endpoint [get agent job](/fr/api/agent/v2/get-agent-job) pour surveiller sa progression.

Si l'agent modifie des fichiers avec succès, une pull request (demande de fusion) est automatiquement créée et le champ `prLink` est renseigné dans la réponse de la tâche.

<div id="rate-limits">
  ## Limites de débit
</div>

- 100 utilisations par projet Mintlify par heure

<div id="usage">
  ## Utilisation
</div>

```bash
curl -X POST https://api.mintlify.com/v2/agent/{projectId}/job \
  -H "Authorization: Bearer mint_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Add a quickstart guide for the Python SDK"}'
```
