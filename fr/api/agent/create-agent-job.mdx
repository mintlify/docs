Cet endpoint crée une tâche d'agent à partir d'un prompt fourni. La tâche s'exécute de manière asynchrone en arrière-plan et renvoie immédiatement les détails de la tâche. Interrogez l'[endpoint de récupération](/fr/api/agent/get-agent-job) pour vérifier le statut et récupérer le lien de la pull request une fois la tâche terminée.

Si des fichiers sont modifiés avec succès, une pull request est automatiquement créée à la fin de la tâche.

## Limites de débit

L'API de l'agent est soumise aux limites suivantes :

- 100 utilisations par projet Mintlify et par heure

## Interrogation pour la complétion

Après avoir créé une tâche, interrogez l'endpoint [obtenir la tâche d'agent](/fr/api/agent/get-agent-job) pour vérifier quand la tâche est terminée :

```javascript
const jobId = response.id;

// Interroger jusqu'à ce que la tâche soit terminée
let job;
do {
  await new Promise(resolve => setTimeout(resolve, 5000)); // Attendre 5 secondes
  const statusResponse = await fetch(
    `https://api.mintlify.com/v2/agent/${projectId}/job/${jobId}`,
    { headers: { 'Authorization': `Bearer ${apiKey}` } }
  );
  job = await statusResponse.json();
} while (job.status === 'pending' || job.status === 'running');

if (job.prLink) {
  console.log('Pull request créée :', job.prLink);
}
```

## Continuer une conversation

Utilisez l'endpoint [envoyer un message](/fr/api/agent/send-message) pour envoyer des instructions supplémentaires à une tâche existante.
