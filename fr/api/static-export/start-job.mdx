<Info>
  L'exportation statique nécessite un [forfait Enterprise](https://mintlify.com/pricing?ref=static-export).
</Info>

Le champ `domain` est le nom d'hôte sur lequel votre documentation est servie — le domaine personnalisé configuré pour votre projet dans le tableau de bord Mintlify (par exemple, `docs.example.com`), ou votre sous-domaine Mintlify (par exemple, `acme.mintlify.app`) si vous n'avez pas configuré de domaine personnalisé. Indiquez uniquement le nom d'hôte, sans le protocole ni aucun préfixe de chemin.
