L’onglet Aperçu affiche le nombre de visiteurs de votre documentation, les pages les plus populaires et les sources de trafic. Servez-vous de ces informations pour déterminer quelles pages apportent le plus de valeur à vos utilisateurs et suivre les tendances dans le temps.

Accédez aux indicateurs de l’aperçu depuis la page **Insights** de votre [Dashboard](https://dashboard.mintlify.com/products/insights).

<div id="metrics">
  ## Statistiques
</div>

Utilisez le sélecteur d’intervalle pour ajuster la période des données affichées. Sélectionnez Visiteurs, Vues ou Actions pour afficher un graphique en courbes montrant les tendances sur la période sélectionnée.

- **Visiteurs**: Visiteurs uniques
- **Vues**: Nombre total de vues de page
- **Actions**: Nombre cumulé d’appels API, de clics sur les liens de la barre de navigation et de clics sur les boutons d’appel à l’action (CTA)
- **Pages populaires**: Chemins des pages les plus visitées et leur nombre de vues
- **Référents**: Principales sources de trafic dirigeant les utilisateurs vers votre documentation

<div id="using-overview-data">
  ## Utiliser les données de synthèse
</div>

Passez en revue vos insights de synthèse pour :

- **Identifier les pages populaires** : Servez-vous des pages les plus consultées pour comprendre quels contenus comptent le plus pour vos utilisateurs, afin de vous assurer qu’ils sont à jour et complets.
- **Suivre les tendances de trafic** : Surveillez les évolutions du trafic pour comprendre l’impact des mises à jour ou de nouveaux contenus.
