title: 'Autorisations de l’éditeur'
description: 'Permettez à davantage de membres de votre équipe de mettre à jour votre documentation'
---

Un éditeur a accès à votre Dashboard et à l’éditeur web.

Tout le monde peut contribuer à votre documentation en travaillant en local et en poussant des modifications vers votre référentiel, mais il existe des différences clés dans la façon dont ces modifications sont déployées :

* **Modifications d’un éditeur** : Lorsqu’un éditeur publie via l’éditeur web ou fusionne une pull request (demande de fusion) dans le référentiel de votre documentation, les modifications sont automatiquement déployées sur votre site en production.
* **Modifications d’un non‑éditeur** : Lorsqu’un non‑éditeur fusionne une pull request (demande de fusion) dans votre référentiel, vous devez déclencher manuellement un déploiement depuis votre Dashboard pour que ces modifications apparaissent sur votre site en production.



<div id="add-editors">
  ## Ajouter des éditeurs
</div>

Par défaut, le membre de l’équipe qui a créé votre organisation Mintlify dispose d’un accès éditeur. Ajoutez d’autres éditeurs depuis la page [Members](https://dashboard.mintlify.com/settings/organization/members) de votre Dashboard.

Les places d’éditeur sont facturées à l’usage, et vous pouvez avoir autant d’éditeurs que nécessaire. Consultez notre [page de tarification](https://mintlify.com/pricing) pour en savoir plus.
