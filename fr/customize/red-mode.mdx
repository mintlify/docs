---
title: "Mode rouge"
description: "Activez un mode d'affichage teinté de rouge pour réduire la fatigue oculaire dans les environnements à faible luminosité."
---

Le mode rouge applique un filtre de couleur rouge à votre documentation, réduisant l'émission de lumière bleue. Ceci est utile pour :

- Lire la documentation dans des environnements sombres sans perturber la vision nocturne
- Réduire la fatigue oculaire lors de longues sessions de codage nocturnes
- Les utilisateurs qui préfèrent des températures de couleur plus chaudes

## Activer le mode rouge

Ajoutez l'option `redMode` à votre configuration `docs.json` :

```json docs.json
{
  "appearance": {
    "redMode": true
  }
}
```

## Options de configuration

<ResponseField name="redMode" type="boolean | object">
  Activez le mode rouge pour votre documentation. Définissez sur `true` pour les paramètres par défaut, ou passez un objet pour une configuration avancée.

  <Expandable title="Options avancées">
    <ResponseField name="intensity" type="number" default="0.5">
      L'intensité du filtre rouge, de `0` (subtil) à `1` (rouge complet).
    </ResponseField>
    <ResponseField name="toggle" type="boolean" default="true">
      Afficher un bouton de basculement dans la barre de navigation pour permettre aux utilisateurs d'activer/désactiver le mode rouge.
    </ResponseField>
  </Expandable>
</ResponseField>

### Exemple avec options avancées

```json docs.json
{
  "appearance": {
    "redMode": {
      "intensity": 0.7,
      "toggle": true
    }
  }
}
```

## Basculement utilisateur

Lorsque `toggle` est activé, les utilisateurs peuvent activer ou désactiver le mode rouge en utilisant l'icône de lune dans la barre de navigation supérieure. Leur préférence est enregistrée dans le stockage local.
