Définissez une police personnalisée pour l’ensemble de votre site, ou séparément pour les titres et le corps du texte. Utilisez Google Fonts, des fichiers de polices locaux ou des polices hébergées à l’externe. La police par défaut varie selon le thème.

Les polices sont gérées par la [propriété fonts](/fr/organize/settings#param-fonts) dans votre `docs.json`.

<div id="google-fonts">
  ## Google Fonts
</div>

Mintlify charge automatiquement [Google Fonts](https://fonts.google.com) lorsque vous indiquez une famille de polices dans votre `docs.json`.

```json docs.json
"fonts": {
  "family": "Inter"
}
```

<div id="local-fonts">
  ## Polices locales
</div>

Pour utiliser des polices locales, placez vos fichiers de police dans le répertoire de votre projet et faites-y référence dans votre configuration `docs.json`.

<div id="setting-up-local-fonts">
  ### Configuration des polices locales
</div>

<Steps>
  <Step title="Ajouter les fichiers de police à votre projet">
    Par exemple, créez un répertoire `fonts` et ajoutez-y vos fichiers de police :

    ```text
    your-project/
    ├── fonts/
    │   ├── InterDisplay-Regular.woff2
    │   └── InterDisplay-Bold.woff2
    ├── docs.json
    └── ...
    ```
  </Step>

  <Step title="Configurer les polices dans docs.json">
    Référencez vos polices locales en utilisant des chemins relatifs depuis la racine de votre projet :

    ```json docs.json
    {
      "fonts": {
        "family": "InterDisplay",
        "source": "/fonts/InterDisplay-Regular.woff2",
        "format": "woff2",
        "weight": 400
      }
    }
    ```
  </Step>
</Steps>

<div id="local-fonts-for-headings-and-body">
  ### Polices locales pour les titres et le corps de texte
</div>

Configurez des polices locales distinctes pour les titres et le corps du texte dans votre `docs.json` :

```json docs.json
{
  "fonts": {
    "heading": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Bold.woff2",
      "format": "woff2",
      "weight": 700
    },
    "body": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Regular.woff2",
      "format": "woff2",
      "weight": 400
    }
  }
}
```

<div id="externally-hosted-fonts">
  ## Polices hébergées en externe
</div>

Utilisez des polices hébergées en externe en indiquant l’URL de la source de la police dans votre `docs.json` :

```json docs.json
{
  "fonts": {
    "family": "Hubot Sans",
    "source": "https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2",
    "format": "woff2",
    "weight": 400
  }
}
```

      Graisse (épaisseur) de la police, par exemple 400 ou 700. Les polices variables acceptent des valeurs précises comme 550.

    <ResponseField name="source" type="string (uri)">
      URL de la source de votre police, par exemple `https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2`, ou chemin vers votre fichier de police local, par exemple `/assets/fonts/InterDisplay.woff2`. Les polices Google sont chargées automatiquement lorsque vous indiquez un nom de `family` Google, aucune URL de source n’est donc nécessaire.
    </ResponseField>
