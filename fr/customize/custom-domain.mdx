---
title: "Domaine personnalisé"
description: "Hébergez votre documentation sur le domaine personnalisé de votre site web"
---

Pour configurer votre documentation sur un domaine personnalisé, vous devez définir le domaine personnalisé souhaité dans vos paramètres Mintlify et configurer vos paramètres DNS auprès de votre fournisseur de domaine.

<Info>
  Vous souhaitez configurer un sous-répertoire personnalisé comme `mintlify.com/docs` ? Voir la documentation sur le [sous-répertoire personnalisé](/fr/deploy/cloudflare).
</Info>

<div id="dashboard-settings">
  ## Paramètres du Dashboard
</div>

1. Accédez à votre [Dashboard](https://dashboard.mintlify.com)
2. Sélectionnez **Settings**
3. Sélectionnez **Domain Setup**
4. Saisissez le domaine personnalisé souhaité. Par exemple : `docs.mintlify.com`

<div id="configuring-your-dns">
  ## Configuration de votre DNS
</div>

1. Accédez aux paramètres DNS de votre domaine sur le site web de votre fournisseur de domaine.
2. Créez une nouvelle entrée DNS en saisissant les [objet Objet] suivants :

```bash
CNAME | docs | cname.vercel-dns.com.
```

<div id="provider-specific-settings">
  ## Paramètres spécifiques au fournisseur
</div>

<AccordionGroup>
  <Accordion title="Vérification Vercel">
    Si Vercel est votre fournisseur de domaine, vous devez ajouter un enregistrement `TXT` de vérification. Ces informations apparaîtront sur votre Dashboard après l’envoi de votre domaine personnalisé et vous seront également envoyées par e‑mail.
  </Accordion>

  <Accordion title="Paramètres de sécurité Cloudflare">
    Si Cloudflare est votre fournisseur DNS, vous devez activer l’option de sécurité « full strict » pour le paramètre HTTPS.
  </Accordion>
</AccordionGroup>
