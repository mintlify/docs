## Aperçu

Ce guide vous explique comment configurer Auth0 comme fournisseur d’authentification pour votre documentation Mintlify. À la fin, vos utilisateurs pourront se connecter à votre documentation avec leurs identifiants Auth0.

<Steps>
  <Step title="Create a Regular Web Application in Auth0">
    Connectez-vous à votre tableau de bord Auth0 et allez à **Applications** > **Applications**. Cliquez sur **Create Application**, nommez votre application (par exemple « Mintlify ») et sélectionnez **Regular Web Applications** comme type d’application. Cliquez ensuite sur **Create**.
    
    <Frame>![Creating a new application in Auth0](/images/guides/auth0/auth0-create-application.png)</Frame>
  </Step>
  <Step title="Get client information">
    Après avoir créé votre application, vous serez redirigé vers la page des paramètres de l’application. Vous y trouverez les informations d’identification essentielles pour l’intégration OAuth :
    
    <Frame>![Auth0 client settings page](/images/guides/auth0/auth0clientsettings.png)</Frame>
    
    Notez les informations suivantes :
    - **Domain** : Le domaine de votre tenant Auth0 (par exemple `your-tenant.auth0.com`)
    - **Client ID** : L’identifiant public de votre application
    - **Client Secret** : La clé secrète de votre application (à conserver en lieu sûr)
    
    Vous aurez besoin de ces valeurs pour configurer Mintlify à l’étape suivante.
  </Step>
  <Step title="Setup Mintlify client">
    Accédez à votre tableau de bord Mintlify et rendez-vous dans **Settings** > **Authentication**. Sélectionnez **OAuth** comme méthode d’authentification : le formulaire de configuration OAuth s’affiche.
    
    <Frame>![Mintlify OAuth client settings](/images/guides/auth0/mintlifyoauthclientsettings.png)</Frame>
    
    Renseignez le formulaire avec les valeurs suivantes :
    
    - **Authorization URL** : `https://YOUR_AUTH0_DOMAIN/authorize` (remplacez `YOUR_AUTH0_DOMAIN` par votre domaine Auth0 de l’étape 2)
    - **Client ID** : Saisissez le Client ID de votre application Auth0
    - **Client Secret** : Saisissez le Client Secret de votre application Auth0
    - **Scopes** : Laissez vide, sauf si vous avez défini des scopes personnalisés dans Auth0
    - **Token URL** : `https://YOUR_AUTH0_DOMAIN/oauth/token` (remplacez `YOUR_AUTH0_DOMAIN` par votre domaine Auth0)
    
    Une fois ces champs renseignés, cliquez sur **Save changes** pour enregistrer votre configuration OAuth.
  </Step>
  <Step title="Configure Callback URL">
    Mintlify génère une URL de redirection unique qu’Auth0 doit reconnaître pour que le flux OAuth fonctionne correctement.
    
    Copiez l’URL de redirection depuis la section Authentication de votre tableau de bord Mintlify :
    <Frame>![Mintlify redirect URL](/images/guides/auth0/mintlifyredirecturl.png)</Frame>
    
    Retournez sur la page des paramètres de votre application Auth0, faites défiler jusqu’à la section **Application URIs**, puis collez l’URL de redirection dans le champ **Allowed Callback URLs** :
    <Frame>![Auth0 redirect URL configuration](/images/guides/auth0/auth0redirecturl.png)</Frame>
    
    Cliquez sur **Save Changes** en bas de la page Auth0 pour appliquer cette configuration.
  </Step>
</Steps>
