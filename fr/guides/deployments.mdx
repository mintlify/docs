Votre site de documentation est automatiquement déployé lorsque vous poussez des modifications vers votre dépôt connecté. Cela nécessite que l’application GitHub Mintlify soit correctement installée et connectée.

Si vos dernières modifications n’apparaissent pas sur votre site en ligne, vérifiez d’abord que l’application GitHub est installée sur le compte ou l’organisation qui possède votre dépôt de documentation. Consultez la section [dépannage GitHub](/fr/settings/github#troubleshooting) pour en savoir plus.

Si l’application GitHub est connectée mais que les modifications ne se déploient toujours pas, vous pouvez déclencher manuellement une reconstruction depuis votre tableau de bord.

## Déclencher manuellement un déploiement

<Steps>
  <Step title="Vérifiez que votre dernier commit a bien été effectué.">
    Assurez-vous que votre dernier commit apparaît dans votre dépôt de documentation et qu'il n'a rencontré aucune erreur.
  </Step>
  <Step title="Déclenchez manuellement un déploiement.">
    Rendez-vous sur votre [tableau de bord](https://dashboard.mintlify.com) et cliquez sur le bouton Déployer.
