---
title: "Configuration d’un monorepo"
description: "Déployez votre documentation depuis un dépôt qui contient plusieurs projets"
icon: "folder-git"
---

Configurez Mintlify pour déployer la documentation à partir d’un répertoire spécifique au sein d’un monorepo. Cette configuration vous permet de maintenir la documentation aux côtés de votre code dans des dépôts qui contiennent plusieurs projets ou services.

<div id="prerequisites">
  ## Prérequis
</div>

* Accès administrateur à votre projet Mintlify.
* Fichiers de documentation organisés dans un répertoire dédié au sein de votre monorepo.
* Un fichier `docs.json` valide dans votre répertoire de documentation.

<div id="configure-monorepo-deployment">
  ## Configurer le déploiement d’un monorepo
</div>

<Steps>
<Step title="Accéder aux paramètres Git">
Accédez aux [paramètres Git](https://dashboard.mintlify.com/settings/deployment/git-settings) depuis votre tableau de bord.

<Frame>
<img
  className="block dark:hidden my-0 pointer-events-none"
  src="/images/monorepo-light.png"
  alt="Le panneau des paramètres du projet dans le menu Git Settings. L’option « Set up as monorepo » est activée et un chemin vers le répertoire /docs est indiqué."
/>
<img
  className="hidden dark:block my-0 pointer-events-none"
  src="/images/monorepo-dark.png"
  alt="Le panneau des paramètres du projet dans le menu Git Settings. L’option « Set up as monorepo » est activée et un chemin vers le répertoire /docs est indiqué."
/>
</Frame>

</Step>
<Step title="Définir le chemin de votre documentation">
1. Activez l’option **Set up as monorepo**.
2. Entrez le chemin relatif vers votre répertoire de documentation. Par exemple, si votre documentation se trouve dans le répertoire `docs`, saisissez `/docs`.
<Note>
  N’incluez pas de barre oblique finale dans le chemin.
</Note>
3. Sélectionnez **Save changes**.
</Step>
</Steps>
