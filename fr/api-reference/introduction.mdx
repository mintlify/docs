L’API REST de Mintlify vous permet d’interagir avec votre documentation de manière programmatique, de déclencher des mises à jour et d’intégrer des expériences de chat propulsées par l’IA.

<div id="endpoints">
  ## Endpoints
</div>

* [Déclencher une mise à jour](/fr/api-reference/update/trigger): Déclenchez une mise à jour de votre site à la demande.
* [Obtenir le statut de mise à jour](/fr/api-reference/update/status): Obtenez le statut d’une mise à jour et d’autres informations sur votre documentation.
* [Générer un message de l’Assistant](/fr/api-reference/assistant/create-assistant-message): Intégrez l’Assistant, entraîné sur votre documentation, dans l’application de votre choix.
* [Rechercher dans la documentation](/fr/api-reference/assistant/search): Recherchez dans votre documentation.

<div id="authentication">
  ## Authentification
</div>

Vous pouvez générer une clé API depuis [le Dashboard](https://dashboard.mintlify.com/settings/organization/api-keys). Les clés API sont associées à l’ensemble d’une organisation et peuvent être utilisées sur plusieurs déploiements.

<div id="admin-api-key">
  ### Clé API administrateur
</div>

La clé API administrateur est utilisée pour les points de terminaison [Déclencher une mise à jour](/fr/api-reference/update/trigger) et [Obtenir l’état de la mise à jour](/fr/api-reference/update/status).

Les clés API administrateur commencent par le préfixe `mint_`. Gardez vos clés API administrateur secrètes.

<div id="assistant-api-key">
  ### Clé API de l’Assistant
</div>

La clé API de l’Assistant est utilisée pour les points de terminaison [Générer un message de l’Assistant](/fr/api-reference/assistant/create-assistant-message) et [Rechercher dans la documentation](/fr/api-reference/assistant/search).

Les clés API de l’Assistant commencent par le préfixe `mint_dsc_`.

La **key** de l’API de l’Assistant est un jeton côté serveur qui doit rester secret.

Le **token** de l’API de l’Assistant est un jeton public qui peut être utilisé dans votre code frontend.

<Note>
  Les appels utilisant le token de l’API de l’Assistant peuvent entraîner des coûts : soit en utilisant vos crédits d’Assistant IA, soit en générant des dépassements.
</Note>
