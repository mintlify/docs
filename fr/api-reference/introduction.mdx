L’API REST de Mintlify vous permet d’interagir programmatiquement avec votre documentation, de déclencher des mises à jour et d’intégrer des expériences de chat propulsées par l’IA.

## Endpoints

- [Déclencher une mise à jour](/fr/api-reference/update/trigger) : Déclenchez une mise à jour de votre site à tout moment.
- [Obtenir l’état de la mise à jour](/fr/api-reference/update/status) : Obtenez l’état d’une mise à jour et d’autres détails sur vos docs.
- [Générer un message de l’Assistant](/fr/api-reference/assistant/create-assistant-message) : Intégrez l’Assistant, entraîné sur vos docs, dans l’application de votre choix.
- [Rechercher dans la documentation](/fr/api-reference/assistant/search) : Recherchez dans votre documentation.

## Authentification

Vous pouvez générer une clé API depuis [le tableau de bord](https://dashboard.mintlify.com/settings/organization/api-keys). Les clés API sont associées à l’ensemble d’une organisation et peuvent être utilisées sur plusieurs déploiements.

### Clé API administrateur

La clé API administrateur est utilisée pour les points de terminaison [Déclencher une mise à jour](/fr/api-reference/update/trigger) et [Obtenir l’état de la mise à jour](/fr/api-reference/update/status).

Les clés API administrateur commencent par le préfixe `mint_`. Gardez vos clés API administrateur secrètes.

### Clé API Assistant

La clé API de l’Assistant est utilisée pour les points de terminaison [Générer un message d’assistant](/fr/api-reference/assistant/create-assistant-message) et [Rechercher dans la documentation](/fr/api-reference/assistant/search).

Les clés API Assistant commencent par le préfixe `mint_dsc_`.

La **clé** API de l’Assistant est un jeton côté serveur qui doit rester secret.

Le **jeton** API de l’Assistant est un jeton public qui peut être utilisé dans votre code frontend.

<Note>
  Les appels utilisant le jeton API de l’Assistant peuvent entraîner des coûts : soit en utilisant vos crédits d’assistant IA, soit en engendrant des dépassements.
</Note>
