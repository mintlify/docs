Ajoutez ce qui suit à votre fichier `docs.json` pour intégrer un widget [Front Chat](https://front.com).

<CodeGroup>

```json Options d’intégration dans docs.json
"integrations": {
    "frontchat": "CHAT_ID"
}
```

```json Exemple
"integrations": {
    "frontchat": "1365d046d7c023e9b030ce90d02d093a"
}
```

</CodeGroup>
