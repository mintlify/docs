title: "Intercom"
---

Ajoutez l’extrait suivant à votre fichier `docs.json` pour intégrer le widget [Intercom](https://www.intercom.com).

<CodeGroup>

```json Options d’intégration dans docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json Exemple
"integrations": {
```
