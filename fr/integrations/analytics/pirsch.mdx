---
title: "Pirsch"
---

Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des analyses à Pirsch.

Vous pouvez récupérer l’ID de votre site dans Paramètres > Développeur > Code d’identification.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
