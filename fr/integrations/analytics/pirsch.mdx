---
title: "Pirsch"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données Analytics à Pirsch.

Vous pouvez récupérer l’ID de votre site dans Settings > Developer > Identification Code.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Exemple
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
