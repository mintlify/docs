---
title: "PostHog"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des analyses à PostHog.

Vous n’avez besoin d’inclure `apiHost` que si vous auto-hébergez PostHog. Par défaut, nous envoyons les événements à `https://app.posthog.com`.

<CodeGroup>

```json Options d’analyses dans docs.json
"integrations": {
    "posthog": {
        "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
        "apiHost": "optional"
    }
}
```

```json Exemple
"integrations": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<br />

<Warning>
    Activer les analyses PostHog désactivera les analyses sur le tableau de bord Mintlify.
</Warning>

## Enregistrements de session

Vous devez ajouter l’URL de votre site de documentation aux « Domaines autorisés pour les enregistrements » de PostHog avant de pouvoir recevoir des enregistrements de session. L’option pour ajouter votre URL se trouve dans les paramètres du projet de PostHog.
