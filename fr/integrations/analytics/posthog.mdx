---
title: "PostHog"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des Analytics à PostHog.

Vous n’avez besoin d’inclure `apiHost` que si vous auto-hébergez PostHog. Par défaut, nous envoyons les événements à `https://app.posthog.com`.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "posthog": {
        "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
        "apiHost": "optional"
    }
}
```

```json Exemple
"integrations": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<br />

<Warning>
    L’activation des Analytics PostHog désactivera les Analytics sur le Tableau de bord Mintlify.
</Warning>



<div id="session-recordings">
  ## Enregistrements de session
</div>

Avant de pouvoir recevoir des enregistrements de session, vous devez ajouter l’URL de votre site de documentation à la liste « Authorized domains for recordings » de Posthog. L’option permettant d’ajouter votre URL se trouve dans les paramètres du projet de Posthog.
