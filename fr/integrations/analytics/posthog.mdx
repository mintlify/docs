---
title: "PostHog"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données d’Analytics à PostHog.

Vous n’avez besoin d’inclure `apiHost` que si vous auto-hébergez PostHog. Par défaut, nous envoyons les événements à `https://app.posthog.com`.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "posthog": {
        "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
        "apiHost": "optional",
        "sessionRecording": true
    }
}
```

```json Exemple
"integrations": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<div id="configuration-options">
  ### Options de configuration
</div>

<ParamField path="apiKey" type="string" required>
  La clé d’API de votre projet PostHog. Doit commencer par `phc_`.
</ParamField>

<ParamField path="apiHost" type="string">
  L’URL de votre instance PostHog. Requise uniquement si vous auto‑hébergez PostHog. Par défaut : `https://app.posthog.com`.
</ParamField>

<ParamField path="sessionRecording" type="boolean" default="true">
  Activer ou désactiver l’enregistrement de session. Définissez la valeur sur `false` pour désactiver l’enregistrement de session tout en laissant les Analytics activés.
</ParamField>

<br />

<Warning>
    L’activation des Analytics PostHog désactivera les Analytics sur le Tableau de bord Mintlify.
</Warning>

<div id="session-recordings">
  ## Enregistrements de session
</div>

Les enregistrements de session sont activés par défaut lorsque vous configurez PostHog. Pour désactiver les enregistrements de session tout en laissant les Analytics activées, définissez `sessionRecording` sur `false` dans votre configuration.

```json Disable session recordings
"integrations": {
    "posthog": {
        "apiKey": "phc_your-key",
        "sessionRecording": false
    }
}
```

Vous devez ajouter l’URL de votre site de documentation à l’option « Authorized domains for recordings » de PostHog avant de pouvoir recevoir des enregistrements de sessions. L’option permettant d’ajouter votre URL se trouve dans les paramètres du projet de PostHog.
