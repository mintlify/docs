Vous devrez créer une nouvelle propriété <Tooltip tip="Google Analytics 4">GA4</Tooltip> à utiliser avec Mintlify. Les données collectées seront envoyées dans le même projet que vos autres données Google Analytics.

Si vous utilisez l’ancienne version de Google Analytics, Universal Analytics, vous pourrez tout de même créer une propriété <Tooltip tip="Google Analytics 4">GA4</Tooltip>. Les données <Tooltip tip="Google Analytics 4">GA4</Tooltip> diffèrent légèrement des données UA, mais elles sont tout de même collectées dans le même projet.

## Comment connecter GA4 à Mintlify

### Créer un flux web

Vous devez créer un flux web pour obtenir l’ID de mesure à renseigner dans Mintlify.

Cliquez sur l’icône d’engrenage en bas à gauche de l’interface Google Analytics, puis sur Flux de données (Data Streams).

<Frame>![](/images/ga4-web-streams.png)</Frame>

Créez un flux web et indiquez l’URL de votre site de documentation Mintlify comme URL du flux.

Votre ID de mesure ressemble à `G-XXXXXXX` et s’affiche sous Détails du flux (Stream Details) immédiatement après la création du flux web.

### Ajouter l’ID de mesure dans docs.json

Ajoutez votre ID de mesure à votre fichier `docs.json` comme suit :

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

### Attendre

Google Analytics met deux à trois jours à afficher vos données.

Vous pouvez utiliser l’extension [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) pour vérifier que la collecte est correctement activée. L’extension journalise dans la console de votre navigateur chaque requête effectuée par GA4.

<Note>

Les liens de prévisualisation ont la collecte désactivée.

</Note>
