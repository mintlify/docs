title: "Google Analytics 4"
---

Vous devrez générer une nouvelle propriété <Tooltip tip="Google Analytics 4">GA4</Tooltip> à utiliser avec Mintlify. Les données collectées seront intégrées au même projet que vos autres données Google Analytics.

Si vous utilisez l’ancienne version de Google Analytics, Universal Analytics, vous pourrez toujours générer une propriété <Tooltip tip="Google Analytics 4">GA4</Tooltip>. Les données <Tooltip tip="Google Analytics 4">GA4</Tooltip> diffèrent légèrement des données UA, mais elles sont néanmoins collectées dans le même projet.



<div id="how-to-connect-ga4-to-mintlify">
  ## Comment connecter GA4 à Mintlify
</div>

<div id="create-a-web-stream">
  ### Créer un flux web
</div>

Vous devez créer un flux web pour obtenir l’ID de mesure à ajouter dans Mintlify.

Cliquez sur l’icône d’engrenage en bas à gauche de l’écran Google Analytics, puis sur Data Streams.

<Frame>![](/images/ga4-web-streams.png)</Frame>

Créez un flux web et renseignez l’URL de votre site de documentation Mintlify comme URL du flux.

Votre ID de mesure ressemble à `G-XXXXXXX` et s’affichera sous Stream Details immédiatement après la création du flux web.

<div id="put-measurement-id-in-docsjson">
  ### Ajouter l’ID de mesure dans docs.json
</div>

Ajoutez votre ID de mesure à votre fichier `docs.json` comme ceci :

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<div id="wait">
  ### Patientez
</div>

Google Analytics met deux à trois jours à afficher vos données.

Vous pouvez utiliser l’extension [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) pour vérifier que Google Analytics est correctement activé. L’extension consigne un message dans la console de votre navigateur chaque fois que GA4 effectue une requête.

<Note>
  Les liens d’aperçu désactivent Google Analytics.
</Note>
