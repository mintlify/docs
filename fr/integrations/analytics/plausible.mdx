---
title: "Plausible"
---

Ajoutez le domaine de votre site dans `docs.json` pour envoyer les données d’analytics à Plausible.

<Info>

    N’incluez pas `https://` pour le domaine ou le serveur.

</Info>

<CodeGroup>

```json Options d’analytics dans docs.json
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Exemple
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
