---
title: "Koala"
---

Ajoutez l’entrée suivante à votre fichier `docs.json` pour envoyer des données Analytics à Koala.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Exemple
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
