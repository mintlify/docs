---
title: "Koala"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données d’analyse à Koala.

<CodeGroup>

```json Options d’analyse dans docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Exemple
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
