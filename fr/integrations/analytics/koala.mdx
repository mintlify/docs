title: "Koala"
description: "Identifiez les entreprises et les visiteurs de votre site de documentation avec Koala."
keywords: ["Intégration Koala", "identification des visiteurs", "scoring de compte", "clé d'API publique"]
---

Ajoutez ce qui suit à votre fichier `docs.json` pour activer l'identification des visiteurs Koala.

<CodeGroup>
  ```json Options d'Analytics dans docs.json
  "integrations": {
      "koala": {
          "publicApiKey": "required"
      }
  }
  ```

  ```json Exemple
  "integrations": {
      "koala": {
          "publicApiKey": "pk_abc123def456"
      }
  }
  ```
</CodeGroup>
