Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des données d’Analytics à Microsoft Clarity.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "clarity": {
        "projectId": "required"
    }
}
```

```json Exemple
"integrations": {
    "clarity": {
        "projectId": "abc123xyz"
    }
}
```

</CodeGroup>

<div id="get-your-project-id">
  ## Récupérez l’ID de votre projet
</div>

1. Créez un [compte Clarity](https://clarity.microsoft.com/projects).
2. Cliquez sur **Obtenir le code de suivi.**
3. Copiez l’ID de votre projet à partir du code de suivi.
