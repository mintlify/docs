---
title: "Hightouch"
description: "Synchronisez les données avec Hightouch pour l’activation d’audiences."
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des Analytics à Hightouch.

<Info>

  N’incluez pas `https://` pour `apiHost`.

</Info>

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "hightouch": {
        "writeKey": "Requis"
        "apiHost": "facultatif"
    }
}
```

```json Exemple
"integrations": {
    "hightouch": {
        "writeKey": "9132c81do371p76sr11da0075469b54f77649c9a067dp0303p56q0q64n072336"
        "apiHost": "us-east-1.hightouch-events.com" # facultatif, par défaut : `us-east-1.hightouch-events.com`
    }
}
```

</CodeGroup>
