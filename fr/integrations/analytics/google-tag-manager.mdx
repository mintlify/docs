---
title: "Google Tag Manager"
---

Ajoutez votre ID de balise dans le fichier `docs.json` et nous injecterons le script Google Tag Manager sur toutes vos pages.

Vous êtes responsable de la mise en place des bannières de consentement aux cookies avec Google Tag Manager si nécessaire.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Exemple
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
