---
title: "Fathom"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des analytics à Fathom.

Vous pouvez récupérer le `siteId` depuis les paramètres de votre script.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "fathom": {
        "siteId": "Requis"
    }
}
```

```json Exemple
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
