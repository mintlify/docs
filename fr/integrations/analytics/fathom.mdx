---
title: "Fathom"
---

Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des données d’Analytics à Fathom.

Vous pouvez récupérer le `siteId` depuis les paramètres de votre script.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Exemple
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
