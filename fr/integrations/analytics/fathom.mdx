Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des Analytics à Fathom.

Vous pouvez récupérer le `siteId` dans les paramètres de votre script.

<CodeGroup>
  ```json Options d’Analytics dans docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Exemple
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
