---
title: "Segment"
description: "Envoyez des événements d’analytics à Segment et aux outils connectés."
---

Ajoutez votre write key Segment à votre fichier `docs.json` pour envoyer des analytics à Segment.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Exemple
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
