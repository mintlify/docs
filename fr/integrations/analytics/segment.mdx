---
title: "Segment"
---

Ajoutez votre clé d’écriture Segment à votre fichier `docs.json` pour envoyer des données d’analytics à Segment.

<CodeGroup>

```json Options d’analytics dans docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Exemple
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
