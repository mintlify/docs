Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des Analytics à Clearbit.

<CodeGroup>
  ```json Options d’Analytics dans docs.json
  "integrations": {
      "clearbit": {
          "publicApiKey": "required"
      }
  }
  ```

  ```json Exemple
  "integrations": {
      "clearbit": {
          "publicApiKey": "pk_1a1882"
      }
  }
  ```
</CodeGroup>
