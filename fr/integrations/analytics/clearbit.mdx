---
title: "Clearbit"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données d’analytics à Clearbit.

<CodeGroup>

```json Options d’analytics dans docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Exemple
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
