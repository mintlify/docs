Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des Analytics à Common Room.

Vous pouvez obtenir le `siteId` depuis les paramètres Signals de Common Room. Il doit être un UUID valide.

<CodeGroup>

  ```json Options d'Analytics dans docs.json
  "integrations": {
      "commonRoom": {
          "siteId": "required"
      }
  }
  ```

  ```json Exemple
  "integrations": {
      "commonRoom": {
          "siteId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
      }
  }
  ```
</CodeGroup>
