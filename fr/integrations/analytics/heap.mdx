Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des Analytics à Heap.

<CodeGroup>
  ```json Options d’Analytics dans docs.json
  "integrations": {
      "heap": {
          "appId": "required"
      }
  }
  ```

  ```json Exemple
  "integrations": {
      "heap": {
          "appId": "1234567890"
      }
  }
  ```
</CodeGroup>
