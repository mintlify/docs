Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des Analytics à Heap.

<CodeGroup>
  ```json Options d’Analytics dans docs.json
  "integrations": {
      "heap": {
          "appId": "required"
      }
  }
  ```

  ```json Exemple
  "integrations": {
      "heap": {
          "appId": "1234567890"
      }
  }
  ```
</CodeGroup>

Remplacez `1234567890` par votre identifiant d'application Heap. Vous pouvez le trouver dans les [paramètres de votre compte Heap](https://heapanalytics.com/app/settings).

## Événements suivis

Mintlify suit automatiquement les interactions utilisateur suivantes et les envoie à Heap :

- Pages vues
- Requêtes de recherche
- Soumissions de commentaires
- Interactions du menu contextuel
- Clics de navigation

Consultez la [liste complète des événements suivis](/fr/integrations/analytics/overview#tracked-events) pour plus de détails sur tous les événements envoyés à votre fournisseur d'Analytics.
