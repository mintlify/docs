---
title: "Heap"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données Analytics à Heap.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Exemple
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
