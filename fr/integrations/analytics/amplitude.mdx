Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des données d’Analytics à Amplitude.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Exemple
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
