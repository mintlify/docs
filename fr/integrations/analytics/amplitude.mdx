---
title: "Amplitude"
description: "Suivez le comportement des utilisateurs et les analytics de votre documentation avec Amplitude."
---

Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des analytics à Amplitude.

<CodeGroup>

```json Options d’Analytics dans docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Exemple
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
