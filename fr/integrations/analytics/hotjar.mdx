---
title: "Hotjar"
description: "Collectez les retours des utilisateurs et des cartes de chaleur avec Hotjar."
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données Analytics à Hotjar.

```json Analytics options in docs.json
"integrations": {
    "hotjar": {
        "hjid": "required",
        "hjsv": "required"
    }
}
```
