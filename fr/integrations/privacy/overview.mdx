## Activer les intégrations de protection des données

Vous pouvez ajouter des plateformes de protection des données à votre documentation. Ajoutez le champ `integrations` dans votre fichier `docs.json` avec vos scripts correspondants.

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

## Consentement aux cookies et désactivation de la télémétrie

Si vous devez vérifier si un utilisateur a déjà consenti aux cookies pour respecter le RGPD, vous pouvez spécifier une clé et une valeur de stockage local sous `cookies` :

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

Si ces valeurs sont définies, le stockage local sera vérifié pour déterminer si l’utilisateur a consenti aux cookies. Si ce n’est pas le cas, la télémétrie sera désactivée.

Si vous souhaitez désactiver la télémétrie pour tous les utilisateurs, ajoutez ce qui suit à votre fichier `docs.json` :

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
