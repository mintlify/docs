<div id="cookie-consent-and-disabling-telemetry">
  ## Consentement aux cookies et désactivation de la télémétrie
</div>

Si vous devez vérifier si un utilisateur a déjà donné son consentement aux cookies pour respecter le RGPD, vous pouvez spécifier une key de stockage local et un value sous `cookies` :

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

Si ces valeurs sont définies, le stockage local sera vérifié pour déterminer si l’utilisateur a consenti aux cookies. S’il n’a pas donné son consentement, la télémétrie sera désactivée.

Si vous souhaitez désactiver la télémétrie pour tous les utilisateurs, vous pouvez ajouter ce qui suit à votre fichier `docs.json` :

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```

<Note>
  Si vous désactivez la télémétrie, vous ne pourrez pas recueillir de commentaires sur vos pages de documentation, même si vous activez les commentaires dans votre Dashboard.
</Note>
