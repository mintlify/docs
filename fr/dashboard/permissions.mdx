description: 'Permettez à davantage de membres de votre équipe de mettre à jour votre documentation'
---

Un éditeur a accès à votre Dashboard et à l’éditeur web.

Tout le monde peut contribuer à votre documentation en travaillant en local et en envoyant des modifications vers votre référentiel, mais il existe des différences importantes dans la façon dont les changements sont déployés :

* **Modifications d’un éditeur** : lorsqu’un éditeur publie via l’éditeur web ou fusionne un pull request (demande de fusion) dans votre référentiel de documentation, les modifications sont automatiquement déployées sur votre site en production.
* **Modifications d’un non‑éditeur** : lorsqu’un non‑éditeur fusionne un pull request (demande de fusion) dans votre référentiel, vous devez déclencher manuellement un déploiement depuis votre Dashboard pour que ces modifications apparaissent sur votre site en production.

<div id="add-editors">
  ## Ajouter des éditeurs
</div>

Par défaut, le membre de l’équipe qui a créé votre organisation Mintlify dispose de droits d’éditeur. Ajoutez d’autres éditeurs depuis la page [Members](https://dashboard.mintlify.com/settings/organization/members) de votre Dashboard.

Les sièges éditeur sont facturés à l’usage, et vous pouvez avoir autant d’éditeurs que nécessaire. Consultez notre [page de tarification](https://mintlify.com/pricing) pour en savoir plus.
