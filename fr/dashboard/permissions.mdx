Un éditeur a accès à votre Dashboard et à l’éditeur web.

N’importe qui peut contribuer à votre documentation en travaillant en local et en poussant des modifications vers votre référentiel, mais il existe des différences clés dans la manière dont les modifications sont déployées :

* **Modifications d’éditeur** : Lorsqu’un éditeur publie via l’éditeur web ou fusionne une pull request (demande de fusion) dans votre référentiel de documentation, les modifications sont automatiquement déployées sur votre site en production.
* **Modifications d’un non‑éditeur** : Lorsqu’un non‑éditeur fusionne une pull request (demande de fusion) dans votre référentiel, vous devez déclencher manuellement un déploiement depuis votre Dashboard pour que ces modifications apparaissent sur votre site en production.

<div id="add-editors">
  ## Ajouter des éditeurs
</div>

Par défaut, le membre de l’équipe qui a créé votre organisation Mintlify dispose de droits d’éditeur. Ajoutez d’autres éditeurs depuis la page [Members](https://dashboard.mintlify.com/settings/organization/members) de votre Dashboard.

Les sièges éditeur sont facturés à l’usage, et vous pouvez avoir autant d’éditeurs que nécessaire. Consultez notre [page de tarification](https://mintlify.com/pricing) pour en savoir plus.
