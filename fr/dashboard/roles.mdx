<Info>
  Le contrôle d'accès basé sur les rôles (RBAC) est disponible avec les [offres Enterprise](https://mintlify.com/pricing?ref=rbac).
</Info>

Mintlify propose trois niveaux d'accès au Dashboard : Viewer, Éditeur et Admin.

<Note>
  Le rôle Viewer est actuellement derrière un feature flag et n'est pas disponible pour toutes les organisations.
</Note>

Les actions suivantes sont limitées par rôle :

|                         | Viewer | Éditeur | Admin |
| ----------------------- | :----: | :-----: | :---: |
| Mettre à jour les rôles des utilisateurs |   ❌   |   ❌   |  ✅   |
| Supprimer des utilisateurs |   ❌   |   ❌   |  ✅   |
| Inviter des Admins      |   ❌   |   ❌   |  ✅   |
| Inviter des Éditeurs    |   ❌   |   ✅   |  ✅   |
| Inviter des Viewers     |   ❌   |   ✅   |  ✅   |
| Gérer et mettre à jour la facturation |   ❌   |   ❌   |  ✅   |
| Mettre à jour le domain personnalisé |   ❌   |   ❌   |  ✅   |
| Mettre à jour la source Git |   ❌   |   ❌   |  ✅   |
| Supprimer l'organisation |   ❌   |   ❌   |  ✅   |

Les autres actions sur le Dashboard sont disponibles pour les deux rôles.

Vous pouvez inviter autant d'administrateurs que vous le souhaitez, mais nous recommandons de limiter l'accès administrateur aux utilisateurs qui en ont besoin.
