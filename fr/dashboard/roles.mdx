<Info>
  Le contrôle d’accès basé sur les rôles (RBAC) est disponible avec les [offres Enterprise](https://mintlify.com/pricing?ref=rbac).
</Info>

Mintlify propose trois niveaux d’accès au Dashboard : lecteur, éditeur et administrateur.

|                                          | Lecteur | Éditeur | Administrateur |
| ---------------------------------------- | :-----: | :-----: | :------------: |
| Voir le contenu dans l’éditeur           |   ✅    |   ✅    |       ✅       |
| Voir le Dashboard et les Analytics       |   ✅    |   ✅    |       ✅       |
| Modifier le contenu dans l’éditeur web   |   ❌    |   ✅    |       ✅       |
| Déclencher des déploiements              |   ❌    |   ✅    |       ✅       |
| Créer et supprimer des clés d’API        |   ❌    |   ✅    |       ✅       |
| Configurer les intégrations              |   ❌    |   ✅    |       ✅       |
| Configurer l’Assistant                   |   ❌    |   ✅    |       ✅       |
| Configurer les modules complémentaires   |   ❌    |   ✅    |       ✅       |
| Inviter des éditeurs et des lecteurs     |   ❌    |   ✅    |       ✅       |
| Mettre à jour les rôles des utilisateurs |   ❌    |   ❌    |       ✅       |
| Inviter des administrateurs              |   ❌    |   ❌    |       ✅       |
| Supprimer des utilisateurs               |   ❌    |   ❌    |       ✅       |
| Gérer la facturation                     |   ❌    |   ❌    |       ✅       |
| Gérer le SSO                             |   ❌    |   ❌    |       ✅       |
| Mettre à jour le domaine personnalisé    |   ❌    |   ❌    |       ✅       |
| Mettre à jour la source Git              |   ❌    |   ❌    |       ✅       |
| Supprimer l’organisation                 |   ❌    |   ❌    |       ✅       |

<Tip>
  Limitez l’accès administrateur uniquement aux personnes qui doivent effectuer des tâches d’administration.
</Tip>

<div id="add-members-to-your-organization">
  ## Ajouter des membres à votre organisation
</div>

Par défaut, la personne qui crée votre organisation Mintlify dispose d’un accès administrateur. Ajoutez des éditeurs supplémentaires depuis la page [Members](https://dashboard.mintlify.com/settings/organization/members) de votre Dashboard.

Vous pouvez avoir autant de membres que nécessaire dans votre organisation. Les postes peuvent entraîner des coûts supplémentaires. Consultez la [page des tarifs](https://mintlify.com/pricing) pour plus de détails.
