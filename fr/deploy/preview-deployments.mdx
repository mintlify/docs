<Info>
  Les aperçus de déploiement sont disponibles avec les [offres Pro et Custom](https://mintlify.com/pricing?ref=preview-deployments).
</Info>

Les aperçus de déploiement vous permettent de voir à quoi ressembleront les modifications de votre documentation avant la fusion en production. Chaque aperçu crée une URL partageable qui se met automatiquement à jour lorsque vous poussez de nouvelles modifications. 

Par défaut, les URL d’aperçu sont publiques. Partagez un lien d’aperçu avec toute personne qui doit examiner vos modifications.

<div id="create-preview-deployments">
  ## Créer des déploiements de prévisualisation
</div>

Les déploiements de prévisualisation sont créés automatiquement via des pull requests (demandes de fusion) ou manuellement depuis votre Dashboard.

<div id="automatic-previews">
  ### Aperçus automatiques
</div>

Lorsque vous créez un pull request (demande de fusion), le bot Mintlify ajoute automatiquement un lien pour afficher le déploiement de prévisualisation dans votre pull request. L’aperçu se met à jour chaque fois que vous poussez de nouveaux commits vers la branche.

<div id="manual-previews">
  ### Aperçus manuels
</div>

Vous pouvez créer manuellement un aperçu pour n’importe quelle branche.

1. Accédez à votre [Dashboard](https://dashboard.mintlify.com/).
2. Sélectionnez **Previews**.
3. Sélectionnez **Create custom preview**.
4. Entrez le nom de la branche à prévisualiser.
5. Sélectionnez **Create deployment**.

<div id="redeploy-a-preview">
  ## Redéployer un aperçu
</div>

Redéployez un aperçu pour actualiser le contenu ou réessayer après un déploiement ayant échoué.

1. Sélectionnez l’aperçu depuis votre [Dashboard](https://dashboard.mintlify.com/).
2. Sélectionnez **Redéployer**.

<div id="restrict-access-to-preview-deployments">
  ## Restreindre l’accès aux déploiements de prévisualisation
</div>

Par défaut, les déploiements de prévisualisation sont accessibles publiquement à toute personne disposant de l’URL. Vous pouvez en restreindre l’accès aux membres authentifiés de votre organisation Mintlify.

1. Accédez à la section **Previews** sur la page [Modules complémentaires](https://dashboard.mintlify.com/products/addons) de votre Dashboard.
2. Activez ou désactivez l’interrupteur **Preview authentication** pour gérer l’authentification des prévisualisations.
