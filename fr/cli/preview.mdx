<div id="start-the-local-preview">
  ## Démarrer la prévisualisation locale
</div>

Naviguez jusqu'au répertoire de votre documentation contenant votre fichier `docs.json` et exécutez :

```bash
mint dev
```

La prévisualisation locale démarre à l'adresse `http://localhost:3000`. Au fur et à mesure que vous modifiez, les changements apparaissent en temps réel.

Par défaut, la prévisualisation s'ouvre automatiquement dans votre navigateur. Pour empêcher l'ouverture du navigateur, utilisez le flag `--no-open` :

```bash
mint dev --no-open
```

Pour générer une prévisualisation sans installer la CLI globalement, exécutez :

```bash
npx mint dev
```

<div id="log-in-for-search-and-assistant">
  ## Se connecter pour la recherche et l'assistant
</div>

Vous devez authentifier la CLI avec votre compte Mintlify pour activer la recherche et l'[assistant](/fr/ai/assistant).

```bash
mint login
```

Après vous être authentifié avec la commande `mint login`, la CLI stocke vos identifiants dans `~/.config/mintlify/config.json` afin que vous restiez connecté entre les sessions. La CLI vous invite également à sélectionner un projet par défaut, que des commandes comme `mint analytics` utilisent.

Une fois connecté, exécutez `mint dev` pour démarrer la prévisualisation locale avec la recherche et l'assistant activés. L'assistant utilise le même contenu indexé que votre site de documentation déployé.

Pour vérifier l'état de votre authentification, exécutez :

```bash
mint status
```

Pour vous déconnecter, exécutez :

```bash
mint logout
```

<div id="custom-ports">
  ## Ports personnalisés
</div>

Par défaut, la CLI utilise le port 3000. Pour utiliser un port différent, utilisez le flag `--port` :

```bash
mint dev --port 3333
```

Si le port est déjà utilisé, la CLI essaie automatiquement le port disponible suivant.

<div id="skip-openapi-processing">
  ## Ignorer le traitement OpenAPI
</div>

Si vous avez de nombreux fichiers OpenAPI, ignorez le traitement OpenAPI pendant le développement local pour améliorer les performances avec le flag `--disable-openapi` :

```bash
mint dev --disable-openapi
```

<div id="preview-as-a-specific-group">
  ## Prévisualiser en tant que groupe spécifique
</div>

Si vous utilisez le contrôle d'accès basé sur les groupes, prévisualisez en tant que groupe d'authentification spécifique avec le flag `--groups` :

```bash
mint dev --groups admin
```

Consultez [Configuration de l'authentification](/fr/deploy/authentication-setup#control-access-with-groups) pour plus d'informations sur les groupes.
