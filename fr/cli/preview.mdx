<div id="start-the-local-preview">
## Démarrer l'aperçu local
</div>

Naviguez vers le répertoire de votre documentation contenant votre fichier `docs.json` et exécutez :

```bash
mint dev
```

L'aperçu local démarre à `http://localhost:3000`. Au fur et à mesure que vous éditez, les modifications apparaissent en temps réel.

Par défaut, l'aperçu s'ouvre automatiquement dans votre navigateur. Pour empêcher le navigateur de s'ouvrir, utilisez l'option `--no-open` :

```bash
mint dev --no-open
```

Pour générer un aperçu sans installer le CLI globalement, exécutez :

```bash
npx mint dev
```

<div id="log-in-for-search-and-assistant">
## Se connecter pour la recherche et l'assistant
</div>

Vous devez authentifier le CLI avec votre compte Mintlify pour activer la recherche et l'[assistant](/fr/ai/assistant).

```bash
mint login
```

Après vous être authentifié avec la commande `mint login`, le CLI stocke vos identifiants dans `~/.config/mintlify/config.json` afin que vous restiez connecté entre les sessions.

Une fois connecté, exécutez `mint dev` pour démarrer l'aperçu local avec la recherche et l'assistant activés. L'assistant utilise le même contenu indexé que votre site de documentation déployé.

Pour vérifier l'état de votre authentification, exécutez :

```bash
mint status
```

Pour vous déconnecter, exécutez :

```bash
mint logout
```

<div id="custom-ports">
## Ports personnalisés
</div>

Par défaut, le CLI utilise le port 3000. Pour utiliser un port différent, utilisez l'option `--port` :

```bash
mint dev --port 3333
```

Si le port est déjà utilisé, le CLI essaie automatiquement le prochain port disponible.

<div id="skip-openapi-processing">
## Ignorer le traitement OpenAPI
</div>

Si vous avez de nombreux fichiers OpenAPI, ignorez le traitement OpenAPI pendant le développement local pour améliorer les performances avec l'option `--disable-openapi` :

```bash
mint dev --disable-openapi
```

<div id="preview-as-a-specific-group">
## Aperçu en tant que groupe spécifique
</div>

Si vous utilisez le contrôle d'accès basé sur les groupes, prévisualisez en tant que groupe d'authentification spécifique avec l'option `--groups` :

```bash
mint dev --groups admin
```

Consultez [Configuration de l'authentification](/fr/deploy/authentication-setup#control-access-with-groups) pour plus d'informations sur les groupes.
