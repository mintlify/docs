## Ajouter un fichier de spécification AsyncAPI

Pour commencer à créer des pages pour vos websockets, assurez-vous de disposer d’un schéma AsyncAPI valide au format JSON ou YAML conforme à la [spécification AsyncAPI](https://www.asyncapi.com/docs/reference/specification/v3.0.0). Votre schéma doit respecter la spécification AsyncAPI 3.0+.

<Tip>
  Pour vérifier la validité de votre schéma AsyncAPI, vous pouvez le coller dans
  [AsyncAPI Studio](https://studio.asyncapi.com/)
</Tip>

## Générer automatiquement les pages WebSocket

Vous pouvez ajouter un champ `asyncapi` à n’importe quel onglet ou groupe dans la navigation de votre `docs.json`. Ce champ peut contenir soit le chemin vers un document de schéma AsyncAPI dans votre dépôt de documentation, l’URL d’un document de schéma AsyncAPI hébergé, soit un tableau de liens vers des documents de schéma AsyncAPI. Mintlify génère automatiquement une page pour chaque canal WebSocket AsyncAPI.

**Exemples avec des onglets :**

<CodeGroup>

```json Local File {5}
"navigation": {
  "tabs": [
    {
        "tab": "API Reference",
        "asyncapi": "/path/to/asyncapi.json"
    }
  ]
}

```

```json Remote URL {5}
"navigation": {
  "tabs": [
    {
        "tab": "API Reference",
        "asyncapi": "https://github.com/asyncapi/spec/blob/master/examples/simple-asyncapi.yml"
    }
  ]
}
```

</CodeGroup>

**Exemples avec des groupes :**

```json {8-11}
"navigation": {
  "tabs": [
    {
      "tab": "AsyncAPI",
      "groups": [
        {
          "group": "Websockets",
          "asyncapi": {
            "source": "/path/to/asyncapi.json",
            "directory": "api-reference"
          }
        }
      ]
    }
  ]
}
```

<Note>
  Le champ directory est facultatif. S’il n’est pas spécifié, les fichiers seront placés dans
  le dossier **api-reference** du dépôt de documentation.
</Note>

## Page de canal

Si vous souhaitez mieux contrôler l’ordre de vos canaux, ou simplement référencer un seul canal, vous pouvez créer un fichier MDX avec le champ `asyncapi` dans le frontmatter.

```mdx
---
title: "Websocket Channel"
asyncapi: "/path/to/asyncapi.json channelName"
---
```
