<Warning>
  L'éditeur web classique sera entièrement obsolète le 7 mai 2026.
  
  Le nouvel éditeur est désormais disponible avec la collaboration en temps réel, l'enregistrement automatique sur tous les appareils et une synchronisation améliorée. Consultez [Éditeur](/fr/editor) pour plus d'informations.
</Warning>

L'éditeur web vous permet de créer, modifier et publier de la documentation directement dans votre navigateur.

<div id="editor-modes">
  ## Modes d'édition
</div>

L'éditeur dispose de deux modes que vous pouvez basculer à l'aide du bouton dans la barre d'outils.

**Le mode visuel** affiche un aperçu en direct de votre contenu pendant la saisie. Appuyez sur <kbd>/</kbd> pour insérer des composants et des médias, ou pour formater le texte.

**Le mode Markdown** vous donne un accès direct à la source MDX de vos pages. Utilisez-le pour un contrôle précis des propriétés des composants et du frontmatter.

<div id="publish-changes">
  ## Publier les modifications
</div>

La branche sur laquelle vous travaillez détermine comment l'éditeur publie vos modifications. Si vous avez des règles de protection de branche sur votre dépôt, l'éditeur les applique.

- **Branche de déploiement** : La publication met à jour votre site en direct immédiatement.
- **Branches de fonctionnalité** : La publication crée une pull request pour examen avant le déploiement sur votre site en direct.
