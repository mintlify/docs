## Add your custom domain

1. Navigate to the [Custom domain setup](https://dashboard.mintlify.com/settings/deployment/custom-domain) page in your dashboard.
2. Enter your domain name (e.g., `docs.example.com`).
3. Click **Add domain**.

## Configure your DNS

On your domain provider's website, create a new DNS record:

```text
CNAME | docs | cname.mintlify-dns.com.
```

DNS changes typically take 1–24 hours to propagate. You can verify your DNS using [DNSChecker](https://dnschecker.org). Once active, Vercel automatically provisions a free TLS certificate via Let's Encrypt. Certificates are renewed automatically.

<AccordionGroup>
  <Accordion title="CAA records">
    If your domain uses CAA records, authorize Let's Encrypt by adding this DNS record:
    ```text
    0 issue "letsencrypt.org"
    ```
    The `/.well-known/acme-challenge` path is reserved for certificate validation — do not redirect or rewrite it.
  </Accordion>

  <Accordion title="Vercel verification">
    If Vercel is your domain provider, add the verification `TXT` record shown on your dashboard after submitting your custom domain.
  </Accordion>

  <Accordion title="Cloudflare settings">
    Enable **Full (strict)** SSL/TLS encryption mode and disable **Always Use HTTPS** in Edge Certificates. Cloudflare's HTTPS redirect blocks Let's Encrypt domain validation.
  </Accordion>
</AccordionGroup>

## Set a canonical URL

To ensure search engines index your preferred domain, add the `canonical` meta tag to your `docs.json`:

```json
"seo": {
    "metatags": {
        "canonical": "https://docs.example.com"
    }
}
```

Replace `https://docs.example.com` with your actual custom domain.
