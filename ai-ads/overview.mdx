AI Ads provides intelligent, context-aware advertising capabilities integrated into your Mintlify documentation. Use machine learning to deliver relevant advertisements based on documentation content and user context.

## Features

- **Smart placement**: Automatically identifies optimal ad placement locations within documentation pages.
- **Content matching**: Uses NLP to match ads with relevant documentation topics.
- **Performance analytics**: Comprehensive dashboard for tracking ad impressions, clicks, and conversions.
- **A/B testing**: Built-in experimentation framework for ad creative optimization.
- **Privacy-first**: GDPR and CCPA compliant with user consent management.

## How it works

AI Ads analyzes your documentation content and user browsing patterns to deliver contextually relevant advertisements. The system uses:

- Documentation content analysis
- User browsing patterns
- Contextual relevance scoring
- Real-time bidding integration

## Configuration

Enable AI Ads in your `docs.json` configuration:

```json docs.json
{
  "aiAds": {
    "enabled": true,
    "placement": ["sidebar", "inline", "footer"],
    "targeting": {
      "contextual": true,
      "behavioral": false
    },
    "refreshInterval": 30000
  }
}
```

### Configuration options

<ResponseField name="enabled" type="boolean" required>
  Enable or disable AI Ads for your documentation.
</ResponseField>

<ResponseField name="placement" type="string[]" required>
  Ad placement locations. Options: `sidebar`, `inline`, `footer`.
</ResponseField>

<ResponseField name="targeting" type="object">
  Targeting configuration options.
  <Expandable title="properties">
    <ResponseField name="contextual" type="boolean" default="true">
      Enable contextual targeting based on page content.
    </ResponseField>
    <ResponseField name="behavioral" type="boolean" default="false">
      Enable behavioral targeting based on user browsing patterns.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="refreshInterval" type="number" default="30000">
  Ad refresh interval in milliseconds.
</ResponseField>

## API endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/ads/serve` | GET | Fetch ads for current page context. |
| `/api/ads/impression` | POST | Record ad impression. |
| `/api/ads/click` | POST | Record ad click event. |
| `/api/ads/analytics` | GET | Retrieve ad performance metrics. |

## Architecture

The AI Ads system consists of the following components:

1. **Ad server**: Handles ad request routing and delivery.
2. **ML pipeline**: Processes content for contextual matching.
3. **Bidding engine**: Manages real-time bidding with ad networks.
4. **Analytics service**: Collects and aggregates performance data.
5. **Admin dashboard**: UI for managing campaigns and viewing reports.
